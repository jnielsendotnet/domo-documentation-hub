## Einführung

Diashows stellen eine hervorragende Möglichkeit dar, Ihre Unternehmensdaten zu repräsentieren. In den folgenden Situationen könnten sie sich auch für Sie als nützlich erweisen:

- Wandanzeigen  
  Sie können wichtige Daten bezüglich der Fortschritte Ihres Teams auf einer Wandanzeige oder einem großen Bildschirm anzeigen.
- Geschäftliche Präsentationen  
  Als Vortragender können Sie die Diashow an bestimmten Folien, auf die Sie näher eingehen möchten, anhalten und anschließend jederzeit mit der Präsentation fortfahren.
- Unternehmenskiosk  
  Sie haben die Möglichkeit, eine Diashow auf einem Bildschirm abzuspielen, um so die Tätigkeiten Ihres Unternehmens vorzustellen.
- Abteilungsbildschirme  
  Sie können eine Diashow mit Karten, die eine bestimmte Abteilung betreffen, abspielen. So könnte beispielsweise Ihr Vertriebsteam eine Diashow mit Karten zum Umsatz der letzten Woche, des letzten Quartals usw. ansehen.

Die Diashows in Domo umfassen zwei Variationen:

- Domo-Publikationen  
  Die Publikationsfunktion platziert Inhalte in eine Diashow, die entweder in andere Formate exportiert oder sicher in das Internet gestellt werden kann.
- Einmalige Seiten-Diashow  
  Dabei handelt es sich um einfache Diashows aller Karten einer Seite. Sie werden nicht gespeichert – wählen Sie einfach die Option **Seite als Diashow anzeigen** für eine Seite aus und die Diashow wird abgespielt.

Sie können die Diashow im „Kiosk“-Modus über einen Webbrowser auf einer Wandanzeige, einem Fernseher oder einem großen Bildschirm anzeigen. Melden Sie sich dazu bei Domo an und lassen Sie die Diashow abspielen.

## Erstellen und Verwalten von Publikationen

Auf der Seite „Publikationen“ können Sie Publikationen erstellen und verwalten. Dabei handelt es sich um Diashows von Karten in Domo.

Durch die Publikation können andere die Folien außerhalb von Domo sehen. Wenn Sie eine Publikation veröffentlichen, erstellt Domo eine URL, über die im Internet auf die Publikation zugegriffen werden kann. Sie können eine E-Mail mit der URL an andere versenden und sie einladen, sich die Publikation anzusehen. Sie können die Publikation aber auch so einrichten, dass sie nur privat freigeschaltet wird und für den Zugriff ein Zugriffscode erforderlich ist.

Sie können auf diese Weise auch Berichte in bestimmten Zeitabständen an Benutzer und Gruppen innerhalb Ihrer Organisation versenden.

**Hinweis**: Die Domo-Publikationen bleiben mit den Karten, die sie enthalten verknüpft, d. h., wenn sich die Daten auf einer Karte ändern, wird die Karte sofort auch in der Diashow aktualisiert, um die Änderungen widerzuspiegeln. Wenn Sie es vorziehen, eine einmalige, unveränderliche Momentaufnahme Ihrer Kartendaten in der Diashow zu haben, können Sie die Publikation als eine Microsoft PowerPoint-Präsentation exportieren. Weitere Informationen finden Sie unter [Exportieren von KPI-Karten](/s/article/360043437813).

Sie finden die Informationen über das Layout einer Publikationsseite unter [Layout der Publikationsseite](/s/article/360042933014).

### Erstellen eine Publikation

Publikationen können in Domo auf zwei Arten erstellt werden:

- Durch Auswahl der Option **Neue Diashow erstellen** auf der Seite „Publikationen“. Anschließend müssen alle Karten, die Sie in der Diashow haben möchten, einzeln hinzugefügt werden.
- Durch Auswahl der Option **Als Diashow veröffentlichen** auf einer Kartenseite. Dies erstellt eine Diashow aus allen Karten dieser Seite. Anschließend können Sie, falls gewünscht, Karten von anderen Seiten hinzufügen.

**So erstellen Sie eine Publikation durch das Hinzufügen einzelner Karten:**

1. Klicken Sie auf <img alt="new_app_icon__2_.png" src="/images/kb/de/0EM5w000005wJXE.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/>**> Publikationen**.
2. Klicken Sie auf **Neue Diashow erstellen**.
3. Fügen Sie im Dialogfeld die Karten hinzu, die Sie in der Diashow sehen möchten.  
   Sie können nach einer Karte anhand ihres Namens suchen, anschließend die Maus darüber halten und auf das „+“-Zeichen klicken, um die Karte als Folie hinzuzufügen.  
   Die Karten in Ihrer Diashow erscheinen in der gleichen Reihenfolge, wie die Karten in diesem Dialogfeld.
4. (Optional) Um Folien innerhalb einer Diashow neu anzuordnen, positionieren Sie die Karten nach Bedarf, indem Sie sie an die gewünschte neue Stelle ziehen und ablegen.
5. Klicken Sie auf **Speichern**.
6. (Optional) Führen Sie eine der folgenden Handlungen aus:

   - Geben Sie die Diashow für andere frei, indem Sie auf **Ändern** klicken und die gewünschte Publikationsoption auswählen.
   - Klicken Sie auf **Jetzt wiedergeben**, um die Diashow anzusehen.
   - Exportieren Sie die Diashow als Microsoft PowerPoint-Präsentation, indem Sie auf **Als PowerPoint-Datei herunterladen** klicken.
7. Klicken Sie auf **Fertig**.

**So erstellen Sie eine Publikation von allen Karten auf einer Seite:**

1. Navigieren Sie zur Seite mit den Karten, die Sie Ihrer Diashow hinzufügen möchten.
2. Klicken<img alt="page_gear_icon.png" src="/images/kb/de/0EM5w000005wJXj.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/> Sie auf die rechte, obere Ecke der Seite, neben den Profilbildern der Mitarbeiter.
3. Wählen Sie **Als Diashow veröffentlichen**.  
   Domo fügt der Seite die Karten als Folien in einer Diashow, die Sie speichern können, hinzu.
4. (Optional) Ändern Sie den Namen der Diashow.
5. (Optional) Fügen Sie Ihrer Diashow zusätzliche Karten hinzu, indem Sie die Kartennamen angeben.
6. Klicken Sie auf **Speichern**, um die ausgewählten Karten Ihrer Diashow als Folien hinzuzufügen.  
   Standardmäßig wird die Diashow nicht veröffentlicht.
7. Führen Sie eine der folgenden Handlungen aus:

- Veröffentlichen Sie die Diashow, indem Sie auf **Ändern** klicken, die gewünschte Publikationsoption auswählen und anschließend auf **Fertig** klicken.
- Klicken Sie auf **Jetzt wiedergeben**, um die Diashow abzuspielen.
- Exportieren Sie die Diashow als Microsoft PowerPoint-Präsentation, indem Sie auf **Als PowerPoint-Datei herunterladen** klicken.

#### Publikationsoptionen

Die folgende Tabelle beschreibt die verfügbaren Optionen für eine Publikation:

| Publikationsstatus | Beschreibung |
| --- | --- |
| Nicht veröffentlicht | Die Diashow kann in Domo angesehen werden, wurde aber *nicht* im Web veröffentlicht. |
| Öffentlich | Die Diashow ist anhand einer URL öffentlich im Web verfügbar. Sie können die URL an andere versenden. |
| Privat | Die Diashow ist anhand einer URL und einem Zugriffscode privat im Web verfügbar. Sie können die URL und den Zugriffscode an andere versenden. |

#### Einer Publikation Karten hinzufügen

Auf der Seite „Publikation“ können Sie einer gespeicherten Publikation Karten hinzufügen.

Sie können einem vorhandenen Bericht eine einzelne Karte auf einer Seite folgendermaßen hinzufügen: <img alt="page_gear_icon.png" src="/images/kb/de/0EM5w000005wJXj.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/>

Wählen Sie dazu „> Bericht planen“ und anschließend die Option „Dem Bericht hinzufügen“ aus.                 Wählen Sie jetzt den Bericht aus, dem Sie die Karte hinzufügen möchten.

**So fügen Sie einer Publikation Karten hinzu:**

1. Klicken Sie auf <img alt="new_app_icon__2_.png" src="/images/kb/de/0EM5w000005wJXE.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/>**> Publikationen**.
2. Wählen Sie für die Diashow, der Sie Karten hinzufügen möchten, <img alt="gear_menu_icon.png" src="/images/kb/de/0EM5w000005wJXq.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/>**> Bearbeiten** aus.
3. Geben Sie in das Feld **Nach zusätzlichen Karten zum Hinzufügen suchen** den Namen der Karte ein, die Sie hinzufügen möchten.  
   Die besten Ergebnisse der Suche erscheinen automatisch in einer Liste, wenn Sie Zeichen eingeben, und werden entsprechend gefiltert, wenn Sie weitere Zeichen eingeben bzw. entfernen.
4. Wählen Sie die Karte aus, die Sie der Liste hinzufügen möchten.  
   Der Name des Benutzers wird dem Feld hinzugefügt.
5. (Optional) Fügen Sie der Diashow weitere Karten hinzu, indem Sie die letzten beiden Schritte wiederholen.
6. (Optional) Entfernen Sie eine Karte aus dem Feld, indem Sie rechts neben dem Benutzernamen auf „x“ klicken.
7. Klicken Sie auf **Hinzufügen**, um der Diashow alle Karten hinzuzufügen, deren Namen im Feld enthalten sind.

In der Publikation erscheinen für alle hinzugefügten Karten Folien. Die Reihenfolge der Karten in der Diashow hängt von der Reihenfolge der Karten in diesem Dialogfeld ab. Bei Bedarf können Sie die Karten im Dialogfeld neu anordnen, indem Sie sie anklicken und an die gewünschte Position ziehen.

<Note>
**Hinweis**: Bei der Erstellung der Diashow wird die Reihenfolge der Karten festgelegt, das bedeutet, dass sich die Reihenfolge der Karten in der Diashow auch dann *nicht* ändert, wenn sich die Reihenfolge auf der Quellseite ändert.
</Note>

#### Freigeben einer Publikation

Bei jeder Publikation haben Sie die Auswahl, ob Sie die Publikation für bestimmte Personen freigeben möchten oder nicht. Wenn Sie eine Publikation veröffentlichen, kann dies öffentlich (URL) oder privat (URL und Zugriffscode) geschehen. Darüber hinaus können Sie eine E-Mail verschicken, um andere darüber zu informieren, wie sie auf die Publikation zugreifen können.

**So geben Sie eine Publikation frei:**

1. Halten Sie die Maus auf der Publikationsseite über die gewünschte Diashow und wählen Sie **Veröffentlichen** aus.
2. Wählen Sie die Publikationsoption aus, die Sie anwenden möchten.
3. (Optional) Wenn Sie die Diashow öffentlich oder privat veröffentlichen, können Sie durch Klicken auf die Option **E-Mail-Web-Link** eine E-Mail verschicken. Anschließend müssen Sie nur noch die E-Mail-Nachricht fertigstellen.
4. Klicken Sie auf **Fertig**.  
   Der Publikationsstatus erscheint auf Publikationsseite der Diashow.

### Verwalten von Publikationen

Auf der Publikationsseite können Sie die Publikationen verwalten. Auf dieser Seite können Sie sehen, welche Diashows Sie erstellt haben oder welche Diashows andere für Sie freigegeben haben. Wenn Sie die Maus über eine Diashow halten, öffnet sich das Diashow-Objekt und bietet Ihnen verschiedene Optionen an. In der folgenden Tabelle können Sie mehr über diese Optionen erfahren:

| Option | Beschreibung |
| --- | --- |
| Wiedergeben | Die Diashow wird im Viewer von Domo wiedergegeben. Weitere Informationen über den Diashow-Viewer finden Sie unter [Verwenden des Diashow-Viewers](/s/article/360043437793). |
| Bearbeiten | Öffnet ein Dialogfeld in dem Sie die Folien der Karten hinzufügen oder entfernen, neu anordnen oder umbenennen können. Alle vorgenommenen Änderungen werden gespeichert, wenn Sie auf **Speichern** klicken. |
| Veröffentlichen | Es wird ein Dialogfeld angezeigt, in dem Sie die Optionen für die Freigabe der Diashow festlegen können. |
| Herunterladen | Exportiert die Folien in der Diashow als eine Microsoft PowerPoint-Präsentation. Weitere Informationen finden Sie unter [Exportieren von Publikationen](/s/article/360043437793). |
| Löschen | Löscht die Diashow aus Domo. |

Folgende Zeilen hinzufügen:

|  |  |
| --- | --- |
| Bericht jetzt senden | Ermöglicht es Ihnen, diesen Bericht an bestimmte Personen und/oder Gruppen innerhalb Ihrer Organisation zu senden. |
| Bericht planen | Ermöglicht es Ihnen, einen Zeitplan für diesen Bericht einzurichten, demzufolge der Bericht an bestimmte Personen und/oder Gruppen innerhalb Ihrer Organisation gesendet wird. |

Darüber hinaus können Sie dem Bericht Karten von einer Kartenseite oder eine **Detailansicht** hinzufügen. Wählen Sie dazu <img alt="page_gear_icon.png" src="/images/kb/de/0EM5w000005wJXj.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/>

**Bericht planen** für die Karte aus, die Sie hinzufügen möchten und klicken Sie anschließend auf die Option **Dem Bericht hinzufügen**. Jetzt können Sie alle vorhandenen Berichte auswählen, denen Sie diese Karte hinzufügen möchten.

### Exportieren von Publikationen

Sie können Folien von KPI-Karten aus einer Publikation als Microsoft PowerPoint-Präsentation exportieren. Das Exportieren kann sich dann als nützlich erweisen, wenn Sie eine *Momentaufnahme* Ihrer Daten wünschen (im Gegensatz zu einer dynamischen Diashow, die aktualisiert wird, sobald die Kartendaten aktualisiert werden).

In der folgenden Tabelle finden Sie die wichtigsten Ähnlichkeiten und Unterschiede zwischen exportierten Publikationen und jenen, die in Domo angezeigt werden:

| In PowerPoint... | In Domo |
| --- | --- |
| - Jede Karte wird als Folie angezeigt - Karten sind *statisch*; Karten werden *nicht* automatisch aktualisiert, sobald sich die von ihnen dargestellten Daten ändern; nützlich, um eine *Momentaufnahme* Ihrer Daten wiederzugeben - Exportierte Diashows werden in Microsoft PowerPoint angezeigt | - Jede Karte wird als Folie angezeigt - Karten sind *dynamisch*; Karten werden automatisch aktualisiert, sobald sich die von ihnen dargestellten Daten ändern - Diashows werden in einem Webbrowser angezeigt |

 Zudem können Sie auch eine oder alle Karten einer Seite in PowerPoint herunterladen, ohne erst eine Diashow in Domo erstellen zu müssen. Weitere Informationen finden Sie unter [Exportieren von KPI-Karten](/s/article/360043437813).

**So exportieren Sie eine Publikation nach Microsoft PowerPoint:**

1. Suchen Sie auf der Publikationsseite die Diashow, die Sie exportieren möchten.
2. Wählen Sie <img alt="gear_menu_icon.png" src="/images/kb/de/0EM5w000005wJXq.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/>**> Herunterladen** aus.

Sie können eine Diashow auch nach deren Bearbeitung exportieren. Klicken Sie dazu auf **Speichern** und anschließend auf **Als PowerPoint-Datei herunterladen**.

<Note>
**Hinweis**: Unter den verschiedenen Kartentypen (KPI, Bild, Notizbuch, Dokument, Umfrage usw.) können Sie *nur* KPI-Karten in eine Microsoft PowerPoint-Präsentation exportieren.

### Verwalten von Publikationen

Sie können die Publikationen in den Administratoreinstellungen verwalten. Die Administratoreinstellungen umfassen u. a. die folgenden Optionen: Webzugriff auf eine Publikation zu entfernen, vereinzelte Karten aus einer Publikation zu entfernen oder eine Publikation vollständig zu löschen.

Weitere Informationen über das Layout der Administratoreinstellungen finden Sie unter [Layout der Administratoreinstellungen](/s/article/360042934234).

**So verwalten Sie Publikationen:**

1. Klicken Sie auf <img alt="new_app_icon__2_.png" src="/images/kb/de/0EM5w000005wJXE.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/>**Administrator**.  
   Die Administratoreinstellungen werden angezeigt.
2. Klicken Sie auf **Publikationen**.  
   Es wird eine List der Diashows sowie Details zur ausgewählten Diashow angezeigt.
3. Wählen Sie die Diashow aus, die Sie verwalten möchten.
4. Führen Sie eine der folgenden Handlungen aus:

   | Aktion | Beschreibung |
   | --- | --- |
   | Beenden der Freigabe der Diashow | Klicken Sie auf **Publikation beenden**, um den Webzugriff auf die Diashow zu untersagen. |
   | Entfernen einer Karte von der Diashow | Klicken Sie neben der Folie, die Sie aus der Diashow entfernen möchten, auf **Entfernen**. Klicken Sie nach Aufforderung auf **Löschen**, um zu bestätigen, dass Sie die Folie aus der Diashow löschen möchten. |
   | Löschen der Diashow | Klicken Sie auf **Löschen**, um die ausgewählte Diashow aus Domo zu löschen. Klicken Sie nach Aufforderung auf **Löschen**, um zu bestätigen, dass Sie die Diashow löschen möchten. |

   **Hinweis**: Wenn Sie an einer Diashow administrative Aktionen durchführen, wird der Besitzer der Diashow über diese Aktionen benachrichtigt.

## Anzeigen einer einmaligen Diashow aus Karten einer Seite

Sie können die Karten einer Seite als eine einmalige (d. h. nicht gespeicherte) Diashow anzeigen.

Da die Seiten-Diashows alle Karten einer Seite anzeigen, empfiehlt es sich, Seiten zu erstellen, die speziell für Diashows geeignet sind. Beispielsweise verfügt jedes Projektteam in Ihrer Entwicklungsabteilung über seine eigene Seite. Sie möchten nun die erzielten Fortschritte der einzelnen Teams bei der Fehlerbehebung der Software herausstellen. Dazu könnten Sie eine neue Seite erstellen, die die Karten von jeder Seite enthält, die diese Daten angeben und daraus eine Diashow für die neue Seite zusammenstellen.

**Hinweise:**

- Wenn Sie auf einer Seite eine Diashow aus Karten abspielen, dann ist das eine schnelle Möglichkeit, die Karten durchzusehen, allerdings wird die Diashow *nicht* gespeichert.  
  Wenn Sie auf einer Seite eine Diashow aus Karten erstellen und speichern möchten, damit andere sie auf Wunsch ansehen können, stehen Ihnen die folgenden Möglichkeiten zur Verfügung:

  - Sie können eine dynamische Diashow erstellen, indem Sie auf die <img alt="page_gear_icon.png" src="/images/kb/de/0EM5w000005wJXj.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/>obere rechte Ecke einer Seite klicken, **Als Diashow veröffentlichen** auswählen und anschließend auf **Speichern** klicken. Auf diese Weise wird die Diashow auf Ihrer Domo-Instanz gespeichert. Von hier aus können Sie die Diashow herunterladen, wiedergeben, veröffentlichen, bearbeiten oder löschen. Weitere Informationen finden Sie unter [Erstellen einer Publikation](/s/article/360043437793).
  - Sie können eine Seite mit Karten direkt nach PowerPoint exportieren. Wählen Sie dazu aus dem <img alt="page_gear_icon.png" src="/images/kb/de/0EM5w000005wJXj.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/>Seitenmenü **Exportieren nach PowerPoint** aus. In diesem Fall wird die Diashow nicht in Domo gespeichert und die Daten werden nicht automatisch aktualisiert. Allerdings ist dies eine hervorragende Möglichkeit, schnell eine Momentaufnahme der Daten zu erstellen und zu teilen. Weitere Informationen finden Sie unter [Exportieren von KPI-Karten](/s/article/360043437813).
- Von der Launcher-Seite können Sie Karten anzeigen, die von einem bestimmten Connector oder Datenanbieter als eine Diashow gehandhabt werden. Weitere Informationen finden Sie unter [Anzeigen von Karten für Datenanbieter](/s/article/360043437813).

**So geben Sie alle Karten einer Seite als Diashow wieder.**

1. Navigieren Sie zu der Seite mit den Karten, die Sie anzeigen möchten.
2. Klicken Sie auf <img alt="page_gear_icon.png" src="/images/kb/de/0EM5w000005wJXj.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/>in der rechten, oberen Ecke der Seite, neben den Profilbildern der Mitarbeiter.
3. Wählen Sie **Seite als Diashow anzeigen** aus.

Weitere Informationen über die Steuerungselemente des Diashow-Viewers finden Sie im nächsten Abschnitt.

## Verwenden des Diashow-Viewers

Domo umfasst einen integrierten Diashow-Viewer, mit dem Karten nacheinander im Vollbildmodus angezeigt werden können. Im Viewer können Sie auf grundlegende Steuerungselemente zugreifen, u. a. haben Sie die Möglichkeit, die Diashow anzuhalten, mit der nächsten Folie fortzufahren oder zur vorherigen Folie zurückzukehren. Zudem können Sie die Zeit festlegen, die jede Folie auf dem Bildschirm angezeigt wird.

Wenn Sie die Maus bewegen, erscheinen die wichtigsten Steuerungselemente der Benutzeroberfläche der Diashow vorübergehend in der oberen rechten Ecke des Bildschirms. Darüber hinaus gibt es noch andere Steuerungselemente, die in anderen Bereichen des Bildschirms erscheinen, wenn Sie die Maus darüber bewegen. Nachfolgend finden Sie deren Beschreibung.

Wenn Sie die Diashow beginnen, erscheinen die Steuerelemente in der oberen rechten Ecke des Bildschirms. Sie verschwinden nach wenigen Sekunden, können aber erneut angezeigt werden, indem Sie die Maus bewegen. Diese Steuerung umfasst die folgenden Elemente:

### Pfeile „Vorherige/Nächste“

Wenn Sie die Maus über die linke oder rechte Seite des Bildschirms bewegen, erscheinen große Pfeile, die Sie zur vorherigen oder nächsten Folie bringen, wenn Sie darauf klicken. Diese Pfeile gleichen den Pfeilen der Hauptsteuerungselemente. Sie können Ihnen beim Navigieren durch die Folien nützlich sein.

### Kartenkarussel

Wenn Sie die Maus über das untere Ende des Bildschirms bewegen, erscheint eine Reihe an Karten. Klicken Sie auf eine der Karten, um auf diese Karte in der Diashow zu springen. Wenn sich mehr als acht Karten auf der Seite befinden, können Sie die Pfeile <img alt="carousel_previous.png" src="/images/kb/de/0EM5w000005wJXc.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/> und <img alt="carousel_next.png" src="/images/kb/de/0EM5w000005wJXl.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/> verwenden, um das Kartenkarussel nach rechts oder nach links zu „verschieben“, um mehr Karten anzuzeigen. Sind die Pfeile abgeblendet, sind auf der Seite keine zusätzlichen Karten in dieser Richtung verfügbar.
