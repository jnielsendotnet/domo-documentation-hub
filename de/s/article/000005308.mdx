## MS Office-Add-ins

<Frame><img alt="domo add-ins.jpg" src="/images/kb/ka0Vq0000000Rcv-00N5w00000Ri7BU-0EM5w000006vM3r.jpg" style={{width: 94, height: 105}}/></Frame>
Die folgenden Funktionen wurden den Domo MS Office-Add-ins hinzugefügt. Weitere Informationen finden Sie im [Add-ins-Benutzerhandbuch](/s/article/000005143).

#### Dynamisches Grafikrendering

Um Verzerrungen zu vermeiden, wenn Sie in einem Dokument die Größe einer Grafik ändern, können Sie die Karte nach der Größenänderung neu laden. Dadurch wird die ursprüngliche Bildqualität mit dem korrekten Seitenverhältnis wiederhergestellt.

#### Verbesserte Importeinstellungen

Mit den folgenden Änderungen der Einstellungen für den Import von Inhalten haben Sie noch mehr Konfigurationsmöglichkeiten, wenn Sie Domo-Inhalte in Dokumente einfügen:

* Die erweiterten Grafikeinstellungen ermöglichen Ihnen die Anpassung von Breite, Höhe und Skalierung der importierten Grafiken.
* Sie können die Sammelnummer importieren und das Kartenbild einblenden (dann wird nur die Sammelnummer im Dokument angezeigt).
* Das Gesamtdesign für die Importeinstellungen wurde optimiert.
* Sie können einen transparenten Hintergrund verwenden.

## AppStore-Verbesserungen

<Frame><img alt="Screenshot 2023-07-20 at 10.14.07 AM.png" src="/images/kb/ka0Vq0000000Rcv-00N5w00000Ri7BU-0EM5w000006vM7Z.jpg" style={{width: 706, height: 365}}/></Frame>
Im AppStore-Ökosystem wurde u. a. Folgendes verbessert:

#### Automatische Erkennung eingereichter Komponenten

Wenn eine Liste eingereicht wird, erkennt das System sie jetzt automatisch und schlägt Komponenten vor, die die Liste am besten beschreiben und in den Suchergebnissen angezeigt werden.

#### Konsolidierung von App-Typ und Komponenten

Bei der Suche im AppStore wurden die App-Typ-Optionen mit den Komponenten konsolidiert, um die Suchergebnisse zu verbessern.

#### Konsolidierung von Werbe- und Servicelisten

Der Übersichtlichkeit halber haben wir die Option, eine Liste als Werbeliste zu kategorisieren, entfernt. Wir empfehlen stattdessen, den Service-Typ für diese Listen zu verwenden.

<Frame><img alt="service type.jpg" src="/images/kb/ka0Vq0000000Rcv-00N5w00000Ri7BU-0EM5w000006vKaj.jpg" style={{width: 436, height: 337}}/></Frame>
#### Sofortige Probeversion erhalten

Anstatt „Demo anfordern“ oder „Probeversion erhalten“ bieten zahlungspflichtige Apps jetzt die Option „Sofortige Probeversion erhalten“.

#### Liste enthält

Bei AppStore-Listen gibt es jetzt den Abschnitt **Liste enthält**, der angibt, welche Assets in der Liste enthalten sind und in welcher Anzahl. In diesem Beispiel enthält die Liste eine Karte, ein DataSet und eine Enterprise-App.

<Frame><img alt="listing includes.jpg" src="/images/kb/ka0Vq0000000Rcv-00N5w00000Ri7BU-0EM5w000006vGBj.jpg" style={{width: 261.996, height: 160.994}}/></Frame>
#### Rich-Text-Formatierung und Beschreibungen

Beim Einreichen von Listen gibt es jetzt mehr Optionen für die Rich-Text-Formatierung, und Herausgeber können auch das Layout von Tabellen und die Textausrichtung konfigurieren.

Weitere Informationen finden Sie in unserer [AppStore-Dokumentation](https://domo-support.domo.com/s/topic/0TO5w000000ZampGAC/appstore).

## Neue DomoStats-Berichte

<Frame><img alt="Screenshot 2023-07-18 at 5.09.58 PM.png" src="/images/kb/ka0Vq0000000Rcv-00N5w00000Ri7BU-0EM5w000006vKuy.jpg" style={{width: 148, height: 148}}/></Frame>

Der DomoStats-Bericht über Ansichtseingaben liefert Informationen zu jeder bereits vorhandenen DataSet-Ansicht in einer Domo-Instanz. Damit können Administratoren besser nachvollziehen und steuern, wie ihre DataSet-Ansichtseingaben verwendet werden.

## Flex Map v2

Flex Map v2 verbessert die Version v1: Sie bietet zusätzliche Optionen für die Anpassung der Karten, entfernt den Großteil der erforderlichen Konfigurations-DataSets, steigert die Leistung von großen DataSets und ergänzt Stil-Optionen wie neue Pinnnadel-Farben, neue Basis-Layer und eine umfassendere Bibliothek an Formen. Flex Map v2 unterstützt außerdem Domo-Filter und ermöglicht es Ihnen, beim Erstellen einer Karte Änderungen rückgängig zu machen oder erneut vorzunehmen.

Wenn Sie Flex Map v1 bereits besitzen, steht Ihnen Flex Map v2 zur Verfügung. Sie müssen die neue Version allerdings erst aus dem [AppStore](https://www.domo.com/appstore/) herunterladen.

Weitere Informationen finden Sie im [Flex Map v2-Implementierungsleitfaden](/s/article/000005073) und im [Flex Map v2-Benutzerhandbuch](/s/article/000005048).

## Regeln für die Sandbox-Neubenennung und Abhängigkeitsdiagramm

#### Regeln für die Neubenennung

Regeln für die Neubenennung ermöglichen es Ihnen, Namen und Beschreibungen von Domo-Inhalten bei der Hochstufung auf eine andere Instanz problemlos zu bearbeiten. Sie brauchen die Details für alle Daten nicht mehr manuell zu bearbeiten. Stattdessen werden die Namen und Beschreibungen von Dashboards, Karten, Beast Mode-Berechnungen, DataSet-Ansichten und DataFlows von den Regeln für die Neubenennung entfernt, ersetzt oder hinzugefügt.

#### Abhängigkeitsdiagramm

Das Abhängigkeitsdiagramm in einem Sandbox-Repository ist ein leistungsstarkes Tool für die Untersuchung und Visualisierung von Inhalten in Domo. Es zeigt Domo-Inhalte im hierarchischen Diagrammformat an. So können Sie die Beziehung zwischen den verschiedenen Elementen des Repositorys untersuchen und nachvollziehen. Wenn beim Commit oder Hochstufen eines Repositorys ein Fehler auftritt, liefert das Diagramm zusätzlichen Kontext für Fehler und ihre Beziehungen, was die Problembehebung beschleunigt.

Erfahren Sie mehr über [Domo Sandbox](/s/article/4403367344023).

## Kontroll-Toolkit-Updates

Das Kontroll-Toolkit umfasst jetzt die folgenden Funktionen.

#### Kopie der Unternehmensdaten

Mit diesem Tool können Sie Daten massenweise über mehrere Instanzen hinweg kopieren. Sie können mit ihm eine manuelle Liste von DataSets erstellen, die Sie in ein separates DataSet kopieren möchten, oder mit dynamischen Filtern eine variierende Anzahl an zu kopierenden DataSets einschließen, die spezifischen Kriterien entsprechen. Administratoren können mit diesem Tool Daten aus einer bestimmten Instanz abrufen und auf eine bestimmte Instanz pushen.

Erfahren Sie mehr über die [Kopie der Unternehmensdaten](/s/article/000005197).

#### Webhook-Benachrichtigung

Das DataSet-Watchdog-Tool bietet jetzt eine Option für Webhook-Benachrichtigungen. Mit Webhook-Benachrichtigungen können Sie automatisch Benachrichtigungen zu Änderungen an den wichtigsten DataSets und DataFlows über den bevorzugten Messaging-Dienst versenden.

Erfahren Sie mehr über [DataSet-Watchdog](/s/article/000005105).
