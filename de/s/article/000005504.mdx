# Mitteilungen senden geplante Berichte

Sie können jetzt eine Mitteilung zum Senden von geplanten Berichten für Karten und DataSets zu einer festgelegten Zeit und Häufigkeit konfigurieren.

# Horizontales Hoch-Niedrig-Diagramm

Mit dem neuen horizontalen Hoch-Niedrig-Diagramm als Ergänzung zum vertikalen Hoch-Niedrig-Diagramm können Sie längere Listen anzeigen und mehr Details zu Metriken mit erhöhtem Volumen liefern.

<Frame><img alt="Screenshot 2024-04-25 at 11.05.20 AM.png" src="/images/kb/ka0Vq0000001Im5-00N5w00000Ri7BU-0EMVq000000pNXF.jpg" style={{width: 925, height: 433}}/></Frame>
Sowohl der vertikale als auch der horizontale Hoch-Niedrig-Diagrammtyp in Analyzer beinhalten die folgenden Stiloptionen:

* Standard
* Linie
* Balken
* Gerundeter Balken
* Lollipop

Über die Balkenstil-Einstellung können Sie die prozentuale Höhe der Balken im Diagramm festlegen. Sie können auch die Spalten aus Ihrem DataSet verwenden, um die Hoch- und Niedrig-Werte in Ihrem Diagramm festzulegen.

Weitere Informationen zu Hoch-Niedrig-Diagrammen finden Sie unter [Hoch-Niedrig-Diagramm.](https://domo-support.domo.com/s/article/360043429173?language=en_US)

# API-Gateway-Verwaltung

Admins können jetzt API-Gateway-Clients in Domo erstellen und die Aktivitätenprotokolle enthalten nun mehr Details über Client-Aktionen.

Die neue Berechtigung „API-OAuth-Clients erstellen“ legt fest, ob API-Gateway-Clients erstellt werden können. Administratoren mit dieser Berechtigung können die Berechtigung für andere Benutzer in ihrer Domo-Instanz deaktivieren.

Zusammen mit diesen neuen Funktionen können Administratoren auch API-Clients programmatisch verwalten. Diese Optionen umfassen Folgendes:

* Die Nutzung und Verwaltung von Benutzerrollenberechtigungen. Weitere Informationen zur Erstellung von Clients finden Sie im
  [Entwicklerportal.](http://developer.domo.com/)

* Fügen Sie Datums- und Zeitstempel für die Client-Erstellung hinzu.

# App Studio

#### Integration veröffentlichen

Die App Studio-Integration mit Domo Everywhere ermöglicht Ihnen die Erstellung von Veröffentlichungen direkt aus Ihren App Studio-Apps und die Veröffentlichung der Inhalte für Ihre Abonnenten.

Abonnenten können Veröffentlichungen einzeln, auf Einladung oder automatisch abonnieren, um alle Veröffentlichungen von einer vertrauenswürdigen Veröffentlichungsorganisation zu erhalten.

Weitere Informationen zu App Studio finden Sie unter [App Studio | Übersicht.](https://domo-support.domo.com/s/article/000005295?language=en_US)

Wie Sie Inhalte mit Domo Everywhere veröffentlichen können, erfahren Sie unter [Veröffentlichen von Inhalten auf einer Abonnenteninstanz](https://domo-support.domo.com/s/article/360045120554?language=en_US).

#### Sandbox-Integration

Mit der Sandbox in App Studio können Sie App Studio-Anwendungen in einer separaten Umgebung erstellen und testen und Inhalte erst dann veröffentlichen, wenn sie für ein breites Publikum bereit sind. Bei der Erstellung von Repositories in der Sandbox können Sie die neue Option „App Studio-Apps“ wählen und bestimmte Apps für die Vermarktung von Inhalten bewerben.

Weitere Informationen zu App Studio finden Sie unter [App Studio | Übersicht.](https://domo-support.domo.com/s/article/000005295?language=en_US)

# Arbeitsabläufe

#### Kontodatentyp

Arbeitsabläufe, Code-Engine und Formulare unterstützen jetzt einen Kontodatentyp. Mit diesem neuen Datentyp können Sie Kontodaten in Ihren Arbeitsabläufen, Code-Engine-Funktionen und Formularen einrichten und zuordnen.

Dazu gehört eine neue Nachschlagefunktion, mit der Sie zugängliche Konten leicht finden und einem Parameter im Arbeitsablauf zuordnen können. Parameter, die mit einem Kontodatentyp konfiguriert sind, können entweder einer Variablen gleichen Typs oder einem fest codierten Wert über ein Suchfeld zugeordnet werden.

Weitere Information zu Arbeitsabläufen erhalten Sie unter [Arbeitsabläufe.](https://domo-support.domo.com/s/topic/0TO5w000000ZkAMGA0/workflows?language=en_US)

# Analyzer-Verbesserungen

#### Kontextsensitive Diagrammeigenschaften

Mit kontextsensitiven Diagrammeigenschaften können Sie mit der rechten Maustaste auf die Diagrammelemente in Alayzer klicken. Sie sehen dann die Liste mit Diagrammeigenschaften, die für dieses Element gelten. Wenn Sie in der folgenden Abbildung mit der rechten Maustaste auf die Y-Achse klicken, wird die Diagrammeigenschaft „Kategorieskala (Y)“ angezeigt.

<Frame><img alt="Screenshot 2024-05-01 at 2.45.36 PM.png" src="/images/kb/ka0Vq0000001Im5-00N5w00000Ri7BU-0EMVq000000s4GL.jpg" style={{width: 901, height: 423}}/></Frame>
[Nach oben](#top)

#### Zahleneingabetypen

Alle Diagrammtypen in Analyzer beinhalten jetzt eine Validierungsfunktion für Diagrammattribute, die eine Zahleneingabe erfordern. Diese Validierung stellt sicher, dass Sie nur Zahlenwerte in das Feld „Diagrammattribute“ eingeben.

Im untenstehenden Beispiel erfordert das Attribut „Höhe in Prozent“ eine numerische Eingabe im Bereich zwischen 0 und 100. Wenn der eingegebene Wert außerhalb des Bereichs liegt, dann wird das Attribut rot hervorgehoben und das Diagramm wird nicht gerendert.

<Frame><img alt="Screenshot 2024-05-01 at 2.55.52 PM.png" src="/images/kb/ka0Vq0000001Im5-00N5w00000Ri7BU-0EMVq000000s4oD.jpg" style={{width: 897, height: 435}}/></Frame>
[Nach oben](#top)

#### Schnelles Rendern

Die neue Funktion für schnelles Rendern in Analyzer verändert die Diagrammeigenschaften, sodass neue Daten nicht mehr bei jedem Laden abgerufen werden müssen. Das reduziert die gesamte Renderzeit. Für Diagramme mit großen DataSets können Sie mit schnellem Rendern mehrere Elemente des Diagramms, ohne Wartezeit durch mehrfache Diagrammaktualisierungen, aktualisieren.

# Spaltenmaskierung

Mit der Funktion „Spaltenmaskierung“ in DataSets können Sie sensible Informationen wie PII und PHI ausblenden. Sie können mit Spaltenmaskierung die Werte eines bestimmtes Feldes vollständig oder teilweise unkenntlich machen.

Sie können die Spaltenmaskierung im Abschnitt PDB im Bildschirm DataSet-Details konfigurieren.

Weitere Informationen über [Spaltenmaskierung](https://domo-support.domo.com/s/article/360042934614?language=en_US) und PDB erhalten Sie in der Wissensdatenbank.
