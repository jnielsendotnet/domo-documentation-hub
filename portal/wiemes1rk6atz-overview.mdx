Get data into and out of Domo. Whether data is stored in excel files, databases, cloud tools, on-prem systems, cloud warehouses, or even IoT devices, there are a wide-variety of ways to securely multiply the value of data in Domo.

* **Pre-built connectors**: [Connectors](https://www.domo.com/appstore/apps?appType=Connector) covering the most common data sources and systems.
* **Build your own connector**: Use Domo's [Connector Dev Studio IDE](/portal/5d8f965eb3469-overview) if a pre-built one doesn't exist yet.
* **[Connect via API](/portal/e0a20a74a7754-data-connection-via-api-overview)**: Leverage the DataSet or Streams APIs.
* **[Cloud Amplifier](https://domo-support.domo.com/s/article/4412849158167?language=en_US)**: Native integrations with cross-cloud systems like Snowflake and Databricks.
* **[Federated data queries](/portal/2bd2767928a2e-federated-queries)**: Use federated adapters to query data directly from where you've already hosted it.
* **[Connect to on-prem data](/portal/cb2e07663deaf-on-premises-data)**: Securely upload your on-prem data to Domo using Workbench.
* **[Jupyter Workspaces](https://domo-support.domo.com/s/article/36004740075?language=en_US)**: Domo offers Jupyter Workspaces with both Python and R kernels, which can be used to move data in and out of Domo.
* **[Domo CLI](https://domo-support.domo.com/s/article/360043437733?language=en_US)**: Often useful to ingest massive DataSets.

### Custom Connectors

---

If an existing Connector doesn't fit your needs, Domo provides the ability to build Custom Connectors through the [Connector Dev Studio IDE](/portal/5d8f965eb3469-overview). There are two kinds of connectors you can build:

1. **[Standard Custom Connector](/portal/5d8f965eb3469-overview)**: Pull data from an external cloud system into a Domo DataSet
2. **[Writeback Connector](/portal/cf8902150aae4-writeback-connectors)**: Push DataSets to third-party file storage applications instead of pulling data from those applications into Domo.

### Connect via API

---

You can programmatically interact with Domo DataSets through the DataSet API or the Streams API (for larger DataSets).

Common use-cases:

* [**Import and Export Data**](/portal/3989acf1bafff-import-and-export-data) into DataSets
* [**Manage DataSets**](/portal/6a1dd9d565f05-managing-data-sets-in-domo)
  + Change DataSet Owner
  + Update DataSet Schema
* [**Apply Personalized Data Permissions (PDP)**](/portal/7zzu6uf05maam-pdp) to DataSets

Looking to get started connecting to DataSets via API? Check out the DataSet API [**Quickstart**](/portal/278e9d611eaa6-quickstart).
