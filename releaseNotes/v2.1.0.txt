Hello everyone!

We're excited to announce that version 2.1.0 of the Domo Documentation Hub is now available! This release brings a cleaner navigation experience, several new and expanded articles, improved AI/MCP docs, and a wave of localization quality improvements.


🌟 What's New

🗂️ Navigation Restructure
The site's information architecture has been significantly simplified: the separate Welcome tab has been consolidated into the Knowledge Base tab, redundant navigation groups have been removed, and the Dev Portal tab dropdown has been cleaned up. The result is a leaner, easier-to-navigate sidebar that gets readers to the right content faster.
Thanks to Jared Peterson for the full docs.json restructure.

🔒 New Article: Set Up AWS PrivateLink with Domo
A brand-new guide walks customers through evaluating, configuring, and managing AWS PrivateLink connections with Domo — covering customer-side setup from start to finish.
Thanks to John Nielsen for authoring this guide.

🤖 MCP / AI Tools Updates
The "Connect AI Tools to Domo Using MCP" article has been substantially revised: the Claude Code server name is updated, the VS Code configuration now accepts an access token as a secure input (rather than embedding credentials), and WSL configuration scope is documented for Windows users.
Thanks to Bryce Cindrich for the improvements.

📋 SCIM Documentation Expanded
The SCIM article now includes a rich set of real-world API examples, giving administrators concrete reference material for provisioning and managing users via System for Cross-domain Identity Management.
Thanks to Jesse Kreitzman for the additions.

📐 Beast Mode Functions Reference Guide — Major Expansion
The Beast Mode Functions Reference Guide received a major overhaul: the file has been reformatted for readability, HyperLogLog aggregate functions have been added, and a new note explains engine equivalences across the different Cloud Integrations environments.
Thanks to Felipe Suarez for the expansion.

💬 LiveChat & SQL Server Connector Updates
The LiveChat Connector article now covers two new report types — Active Chats and Archive Chats — with an updated title to match. The Microsoft SQL Server Connector article also gains a Fetch Columns description in its Details section.
Thanks to Arun Raj for both updates.

🌐 Localization Improvements
  • French, German, and Spanish navigation tab and group titles are now properly translated in docs.json (previously displayed in English).
  • Japanese localization received a fresh update pass, and two Japanese articles with MDX parsing errors (causing Mintlify build warnings) have been fixed.
  • Four English nav group titles that were accidentally translated to Japanese in a prior PR have been restored.
Thanks to Masaaki Hori for the Japanese localization updates and to Jared Peterson for the nav title fixes.

🧹 Content & Housekeeping
  • Duplicate content removed from article 4403367344023 (thanks to Mamta Bolaki).
  • Minor fix in the "Intro to Domo" article — a stray ">" character removed from the intro paragraph.
  • Temporary fix scripts (fix_mdx_errors.py and related files) removed from the repo root after their work was complete.


🙏 Thank You

A heartfelt thank you to everyone who contributed to this release — John Nielsen, Bryce Cindrich, Jesse Kreitzman, Felipe Suarez, Arun Raj, Masaaki Hori, Mamta Bolaki, and Jared Peterson. Your contributions make Domo's documentation better for everyone.

If you have questions or feedback, please reach out — we're always happy to help!
