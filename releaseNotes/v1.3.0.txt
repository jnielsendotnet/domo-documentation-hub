Hello everyone!

We're excited to announce that version 1.3.0 of the Domo Documentation Hub is now available! This release delivers expanded developer guidance and important content accuracy updates to keep our docs in sync with the platform.


🌟 What's New

📚 Domo.js Versioned Documentation (v4, v5, and v6)
Developers now have dedicated, version-specific documentation for Domo.js. The main Domo.js page has been reorganized, and two new pages cover v4 and v5 in depth — giving you clear reference material no matter which version your app targets.
Special thanks to Han Kim for this major contribution (DOMO-482176).

🧪 Sandbox Unsupported Items — Refreshed
The Domo Sandbox "Unsupported Items" list has been brought up to date with the current platform, including:
  • Broader promotion support: pages, App Studio apps, and Views are now called out as supported content
  • New unsupported items documented: Annotations, non-Domo cloud warehouses, Split Join / Split Filter tiles, Bring-your-own-model AI tiles, PDP in tiles, customized UI layouts, Workspaces, and Worksheets
  • Corrected terminology (e.g., "dashboards" → "pages", "Writeback Tile DataFlows" → "Writeback tile")
  • Removed stale restrictions (Fixed Input tile, Text Generation tile, Custom Apps outside of App Studio)

Thanks to Mamta Bolaki for keeping this critical reference accurate.


🙏 Thank You

A heartfelt thank you to everyone who contributed to this release. Your work makes the Domo developer and admin experience better with every release.

If you have questions or feedback, please reach out — we're always happy to help!
