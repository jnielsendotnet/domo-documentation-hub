The Beast Mode Functions Reference Guide received a massive update, adding HyperLogLog aggregate functions and reformatting the entire article for readability. The SCIM (System for Cross-domain Identity Management) article now includes concrete examples to accompany the existing reference material, making it much easier for admins to implement SCIM provisioning correctly.

Version 2.0.3
════════════════════════════════════════

Hello everyone!

We're excited to announce that version 2.0.3 of the Domo Documentation Hub is now available! This is a focused patch release that corrects a pair of bugs introduced during recent navigation and content updates.


🌟 What's New

🌐 English Nav Titles Restored
Four navigation group titles in the English Knowledge Base and API Reference tabs were accidentally translated to Japanese during last week's Japanese localization update. The affected groups — "Connect & Integrate," "Connect Data to Domo," "Distribute Domo Content," and "AI Services" — are now restored to their original English labels. English-language users will no longer see Japanese text in those sections of the nav.
Thanks to Jared Peterson for catching and correcting this quickly.

📄 Alert Message Formatting Fixed
A special-character rendering issue in the "Create an Alert for a DataSet" article has been resolved. Dollar signs and parentheses in the example alert message were being misinterpreted by the MDX renderer, potentially causing broken or malformed output. The characters are now properly escaped so the example displays as intended.
Thanks to Jared Peterson for the fix.


🙏 Thank You

A heartfelt thank you to everyone who flagged these issues and helped maintain the quality of our documentation. Every fix — no matter how small — keeps the experience polished for our readers.

If you have questions or feedback, please reach out — we're always happy to help!

Version 2.1.0
════════════════════════════════════════

Hello everyone!

We're excited to announce that version 2.1.0 of the Domo Documentation Hub is now available! This release focuses on a cleaner, more streamlined navigation experience — with a major information architecture overhaul and expanded localization support.


🌟 What's New

🗂️ Navigation Architecture Overhaul
The Welcome tab has been merged into the Knowledge Base tab, and the Dev Portal tab dropdown has been removed, resulting in a flatter, more intuitive site structure. Redundant entries across the navigation tree were consolidated, making it easier for readers to find what they're looking for without extra clicks.
Thanks to Jared Peterson for rethinking the top-level IA and seeing the restructure through.

🌍 Localized Navigation Titles
Tab and group labels in the French, German, and Spanish sections of the documentation hub now appear in their respective languages — for example, "Knowledge Base" becomes "Base de connaissances" (FR), "Wissensdatenbank" (DE), and "Base de conocimientos" (ES). This brings the navigation experience in line with the localized content readers are already used to.
Thanks to Jared Peterson for the translations and the scripted rollout.

🇯🇵 Japanese Article Parse Fixes
Two Japanese articles had MDX formatting issues that prevented Mintlify from loading them (and generated navigation warnings). Indentation on image blocks inside numbered list items and a malformed multi-line JSX tag have both been corrected. Both articles now render cleanly.
Thanks to Jared Peterson for tracking down the root causes.

✏️ Content Polish
A handful of small but visible content corrections landed in this release:
  • Removed a stray ">" character from the introduction of the "Intro to Domo" article
  • Bolded the "Maximum String Length" label in the "Create an Alert for a Dataset" article to match UI conventions
  • Restored a missing metadata title in the French Current Release Notes page


🙏 Thank You

A heartfelt thank you to everyone who contributed to this release. Your attention to the details — from navigation structure down to a single stray character — is what keeps the documentation hub trustworthy and easy to use.

If you have questions or feedback, please reach out — we're always happy to help!
