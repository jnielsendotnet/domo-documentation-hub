### Nouvelles fonctionnalités et optimisations

# Planification avancée de la vérification de l'actualité des données

La planification avancée permet de mieux contrôler le moment où les mises à jour sont recherchées dans vos données et de n'effectuer cette vérification que lorsque c'est nécessaire. Les dépenses relatives à l'actualité globale des données sont ainsi réduites.

<Frame><img alt="Advanced scheduling - Settings (1).png" src="/images/kb/ka0Vq00000072GP-00N5w00000Ri7BU-0EMVq000003QUIT.jpg" style={{width: 899, height: 407}}/></Frame>

En savoir plus sur l'ancienneté des données dans la [vue d'ensemble de Cloud Amplifier](/s/article/4412849158167).



# Requêtes de données dans les flux de travail et Assistant d'IA

La nouvelle fonctionnalité d'IA Texte en SQL facilite encore plus la création de requêtes à l'aide de la tâche de requête de DataSet dans les flux de travail. Vous saisissez votre requête en texte standard et recevez une instruction SQL générée par l'IA que vous pouvez utiliser dans votre flux de travail.

En savoir plus sur la [tâche de requête de DataSet](/s/article/000005331#add_query_table) dans les flux de travail.

<Frame><img alt="Dataset Query.png" src="/images/kb/ka0Vq00000072GP-00N5w00000Ri7BU-0EMVq000003dvZN.jpg" style={{width: 897, height: 513}}/></Frame>

# Rapport DomoStats: Membres de groupe

<Frame><img alt="rtaImage.jpeg" src="/images/kb/ka0Vq0000002hX7-00N5w00000Ri7BU-0EMVq000001VwNJ.jpg"/></Frame>

Le nouveau rapport DomoStats: Membres de groupe vous permet de voir la liste des groupes et de leurs membres.

Grâce à ce rapport, les administrateurs Domo peuvent suivre l'appartenance à un groupe et rassembler d'autres perspectives en associant le rapport Membres de groupe à d'autres tels que les rapports Personnes, Autorisations de carte et Accès aux DataSets.

Un nouveau rapport DomoStats intitulé Groupes est publié dans la version bêta. En savoir plus [ci-dessous](#domo_stats_groups_report).

En savoir plus sur [DomoStats](/s/article/360043433813).

# Options d'exportations dans Embed dans le Studio d'applications

Les options d'exportations sont désormais disponibles dans Embed dans le Studio d'applications. Grâce à cette fonctionnalité, vous pouvez exporter vos cartes dans Embed sous la forme de fichiers CSV, PowerPoint et Excel, et également les imprimer.

En savoir plus sur [Embed dans le Studio d'applications](/s/article/000005295#export_embedded_app_content).

# Mises à jour d'Analyzer

#### Types de colonne de tableau Image et Lien

Les nouveaux types de colonne de tableau Image et Lien vous permettent d'afficher une image ou un lien dans une colonne, sans concaténer des éléments HTML

En savoir plus sur les [graphiques en tableau](/s/article/360043429573).

<Frame><img alt="New Table Column Types (Image + Link) 3.png" src="/images/kb/ka0Vq00000072GP-00N5w00000Ri7BU-0EMVq000003QG94.jpg" style={{width: 530, height: 617}}/></Frame>



#### Graphique avec image

Grâce au nouveau graphique avec image d'Analyzer, vous pouvez ajouter une colonne d'URL d'image à votre graphique. Vous pouvez ainsi afficher des images de manière dynamique, en fonction des données affichées dans la page de l'application ou dans le tableau de bord. Les graphiques avec image sont compatibles avec les filtres.

En savoir plus sur le [graphique avec image](/s/article/000005654) dans la base de connaissances.

# Éditeur de programmation professionnelle

Pro Code Editor est désormais disponible pour tous les utilisateurs de Domo.

Pro Code Editor est un éditeur de programmation professionnelle qui vous permet de développer des applications de programmation professionnelle dans un IDE Web. Écrivez et prévisualisez du code dans l'interface Domo, et accédez à d'autres outils Domo qui améliorent la fluidité du développement.

# Mises à jour du journal d'activité

Nous avons converti le journal d'activité en réaction. Cette mise à jour offre les améliorations suivantes:

* Filtre des événements plus intuitif, avec le filtre Type d'événement et Événement divisé en filtres distincts accessibles par liste déroulante.
* Les filtres Date et Personne sont désormais intégrés dans les en-têtes de colonne au lieu d'être proposés en tant que filtres distincts accessibles par liste déroulante.
* Une expérience utilisateur plus compacte, qui utilise l'espace de manière plus efficace.

Consultez l'article [Journal d'activité](/s/article/360042934574) dans la base de connaissances.

# Modèles universels de prévision par l'IA

#### Modèles universels de prévision par l'IA à partir des cartes

La fonctionnalité Prévision des cartes avec l'IA dans Studio d'applications vous permet de configurer des prévisions avancées dans une carte graphique à l'aide de [Domo.AI](http://Domo.AI). Cette fonctionnalité utilise le modèle de prévision universel de Domo et renvoie un graphique de prévisions intelligent en fonction des données de la carte.

Vous pouvez également appliquer des filtres à la page, ajuster des variables et générer une prévision en fonction des informations du graphique.

En savoir plus sur la [génération d'une prévision dans une carte Studio d'applications](/s/article/000005567#forecast_in_app).

<Frame><img alt="AI Card Forecasting (2).png" src="/images/kb/ka0Vq00000072GP-00N5w00000Ri7BU-0EMVq000003ULi1.jpg" style={{width: 649, height: 295}}/></Frame>

Cette fonctionnalité est à l'état bêta fermé. Si vous souhaitez rejoindre une prochaine session bêta et avez des questions à ce sujet, contactez l'équipe de votre compte Domo.



#### Mosaïque de prévision par l'IA dans Magic ETL

La mosaïque de prévision par l'IA dans Magic ETL vous permet de prévoir les futurs points de données en fonction de motifs et de tendances dans vos données. Le résultat de la prévision comprend deux plages d'intervalles de confiance.

En savoir plus sur le [modèle universel de prévision](/s/article/000005567).

<Frame><img src="/images/kb/ka0Vq00000072GP-00N5w00000Ri7BU-0EMVq000003lIML.jpg" style={{width: 885, height: 472}}/></Frame>

#### Modèle universel de prévision par l'IA dans le service Playground de Domo. AI

Vous pouvez utiliser le modèle universel de prévision dans le service Playground de Domo. AI pour comparer et tester des sorties de configuration. Vous pouvez également extraire le code cURL qui génère les prévisions pour l'utiliser dans d'autres configurations, par exemples pour l'interaction avec une API.

En savoir plus sur le [modèle universel de prévision par dans le service Playground de Domo.AI](/s/article/000005567#forecasting_service_domo-ai_playground)

# Rapport DomoStats: Groupes

<Frame><img alt="rtaImage.jpeg" src="/images/kb/ka0Vq0000002hX7-00N5w00000Ri7BU-0EMVq000001VwNJ.jpg"/></Frame>

Le rapport DomoStats Groupes répertorie les groupes Domo et les attributs pertinents.

Ce rapport comporte des informations importantes, notamment:

* Nom du groupe
* Type de groupe
* Date de création
* ID du créateur

Le rapport Groupes peut être associé à d'autres rapports de groupe tels que le rapport Accès aux DataSets. En savoir plus sur [DomoStats](/s/article/360043433813).

<Note>
**Remarque:** un rapport DomoStats précédent fournissait des informations sur les adhésions aux groupes. Ce rapport portait par erreur le nom de Groupes; il a été renommé Membres de groupe. En savoir plus sur le [rapport Membres de groupe](#domostats_group_members) ci-dessus.

</Note>
