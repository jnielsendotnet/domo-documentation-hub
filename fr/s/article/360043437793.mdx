## Introduction

Les diaporamas sont un excellent moyen de mettre en valeur les données de votre entreprise. Vous les trouverez très utiles dans les cas suivants :

- Panneaux muraux  
   Vous pouvez afficher les précieuses données retraçant la progression de votre équipe sur un tableau mural ou sur un grand écran.
- Présentations professionnelles  
   En tant que présentateur, vous pouvez interrompre le diaporama pour détailler certaines diapos, puis reprendre la présentation à votre gré.
- Kiosques de la société  
   Vous pouvez diffuser le diaporama sur un écran pour mettre en avant les activités de votre société.
- Écrans départementaux  
   Vous pouvez diffuser les cartes qui concernent un département donné dans un diaporama. Par exemple, votre équipe commerciale pourrait consulter le diaporama des cartes présentant les résultats commerciaux de la semaine précédente, du trimestre précédent, etc.

Dans Domo, les diaporamas prennent deux formes :

- Publications Domo  
  Les publications placent votre contenu dans un diaporama qui peut ensuite être exporté dans d'autres formats ou publié sur le Web en toute sécurité.
- Diaporamas de pages ponctuels  
   Il s'agit-là de simples diaporamas qui présentent toutes les cartes d'une page. Ces diaporamas ne sont pas enregistrés ; il vous suffit de sélectionner l'option **Afficher cette page sous forme de diaporama** pour une page pour diffuser le diaporama.

Pour afficher votre diaporama en mode « kiosque » sur un panneau mural, un téléviseur ou un grand écran dans un navigateur Web, connectez-vous à Domo et diffusez le diaporama.

## Création et gestion des publications

La page Publications vous permet de créer et de gérer les « Publications », c'est-à-dire les diaporamas de cartes dans Domo.

Grâce à ces publications, ceux qui sont extérieurs à Domo peuvent voir les diapos. Lorsque vous publiez une publication, Domo crée une URL qui permet d'y accéder via Internet. Vous pouvez alors envoyer un e-mail contenant le lien de l'URL à d'autres personnes, et les inviter à consulter la publication. Pour publier une publication de manière confidentielle, vous pouvez demander un code d'accès qui permet de la consulter.

Vous pouvez également planifier l'envoi de rapports à intervalles réguliers aux différents utilisateurs et groupes de votre organisation.

<Note>
**Remarque** : les publications Domo et les cartes qu'ils contiennent restent synchronisées. Cela signifie que lorsque les données d'une carte changent, la carte en question est automatiquement mise à jour dans le diaporama. Si vous préférez que votre diaporama présente un instantané stable et ponctuel, vous pouvez exporter la publication sous forme de présentation Microsoft PowerPoint. Vous pouvez également télécharger l'une des cartes ou toutes les cartes d'une page dans PowerPoint sans créer de publication dans Domo. Pour obtenir plus d'informations, consultez la rubrique [Exportation de cartes KPI.](/s/article/360043437813)
</Note>

Pour obtenir plus d'informations sur la page Publications, consultez la rubrique [Topologie de la page Publications.](/s/article/360042933014)

### Création d'une publication

Deux méthodes distinctes vous permettent de créer des publications dans Domo :

- Sélectionnez l'option **Créer un nouveau diaporama** dans la page Publications, puis ajoutez une à une les différentes cartes à inclure dans le diaporama.
- Sélectionnez l'option **Publier sous forme de diaporama** dans la page d'une carte. Un diaporama est alors créé à partir de toutes les cartes de cette page. Au besoin, vous pouvez ensuite ajouter des cartes depuis d'autres pages.

**Pour créer une publication via l'ajout de différentes cartes :**

1. Cliquez sur **> Publications**.
2. Cliquez sur **Créer un nouveau diaporama**.
3. Dans la boîte de dialogue, ajoutez les cartes à transformer en diapos dans le diaporama.  
   Vous pouvez rechercher une carte par son nom, la survoler avec votre souris, puis cliquer sur le + pour ajouter la carte sous forme de diapo.  
   Dans le diaporama, l'ordre des cartes est le même que dans cette boîte de dialogue.
4. (Facultatif) Pour réagencer les diapos dans le diaporama, faites glisser les cartes et déposez-les aux emplacements voulus.
5. Cliquez sur **Enregistrer**.
6. (Facultatif) Choisissez l'une des procédures suivantes :

   - Pour partager le diaporama avec d'autres personnes, cliquez sur **Modifier**, puis sélectionnez l'option Publier requise.
   - Pour voir le diaporama, cliquez sur **Jouer maintenant**.
   - Pour exporter le diaporama sous forme de présentation Microsoft PowerPoint, cliquez sur **Télécharger au format PowerPoint**.
7. Cliquez sur **Terminé**.

**Pour créer une publication à partir de toutes les cartes d'une page :**

1. Accédez à la page contenant les cartes à ajouter au diaporama.
2. En haut et à droite de la page, à côté des photos du profil du collaborateur, cliquez sur .
3. Sélectionnez **Publier sous forme de diaporama**.  
   Domo ajoute alors les cartes de la page sous forme de diapos dans le diaporama et vous pouvez enregistrer ce dernier.
4. (Facultatif) Modifiez le nom du diaporama.
5. (Facultatif) Pour ajouter d'autres cartes dans votre diaporama, spécifiez les cartes par leur nom.
6. Cliquez sur **Enregistrer** pour ajouter les cartes sélectionnées sous forme de diapos dans le diaporama.  
   Par défaut, votre diaporama n'est pas publié.
7. Choisissez l'une des procédures suivantes :

- Pour publier le diaporama, cliquez sur **Modifier**, sélectionnez l'option Publier requise, puis cliquez sur **Terminé**.
- Pour diffuser le diaporama, cliquez sur **Jouer maintenant**.
- Pour exporter le diaporama sous forme de présentation Microsoft PowerPoint, cliquez sur **Télécharger au format PowerPoint**.

#### Options de publication

Le tableau suivant décrit les options de publication disponibles :

| État de la publication | Description |
| --- | --- |
| Non publié | Le diaporama est consultable dans Domo, mais n'est *pas* publié sur le Web. |
| Public | Le diaporama est publiquement disponible sur le Web via l'URL. Vous pouvez envoyer le lien de l'URL à d'autres personnes par e-mail. |
| Privé | Le diaporama est disponible en privé sur le Web via l'URL et le code d'accès. Vous pouvez envoyer le lien de l'URL et le code d'accès à d'autres personnes par e-mail. |

#### Ajout de cartes dans une publication

Vous pouvez ajouter des cartes dans une publication enregistrée via la page Publication.

Pour ajouter l'une des cartes d'une page dans un rapport existant, sélectionnez > Planifier le rapport, puis l'option Ajouter au rapport. Choisissez ensuite le rapport auquel ajouter la carte.

**Pour ajouter des cartes dans une publication :**

1. Cliquez sur **> Publications**.
2. Pour le diaporama auquel ajouter les cartes, sélectionnez **> Modifier**.
3. Entrez le nom de la carte à ajouter dans le champ **Recherchez des cartes additionnelles à ajouter**.  
   Les résultats de votre recherche apparaissent automatiquement dans la liste à mesure que vous saisissez ou supprimez des caractères.
4. Sélectionnez la carte à ajouter dans la liste.  
   Le nom de l'utilisateur est alors ajouté dans le champ.
5. (Facultatif) Pour ajouter d'autres cartes dans ce diaporama, répétez les deux étapes précédentes.
6. (Facultatif) Pour retirer une carte du champ, cliquez sur le « x » situé à droite de son nom.
7. Pour ajouter toutes les cartes dont le nom s'affiche dans le champ dans le diaporama, cliquez sur **Ajouter**.

Les diapos correspondant aux cartes ajoutées apparaissent dans cette publication. Dans le diaporama, l'ordre des cartes est le même que dans cette boîte de dialogue. Pour en modifier l'ordre dans la boîte de dialogue, cliquez sur une carte et faites-la glisser vers la position souhaitée.

<Note>
**Remarque :** l'ordre des cartes au sein du diaporama est défini lorsque celui-ci est généré ; modifier l'ordre des cartes dans la page source n'a donc *pas* d'incidence sur le diaporama.
</Note>

#### Partage d'une publication

Quelle que soit la publication, vous pouvez choisir de la partager ou non avec des utilisateurs sélectionnés. Lorsque vous publiez une publication, vous pouvez le faire publiquement (URL) ou en privé (URL et code d'accès). De même, vous pouvez envoyer un e-mail décrivant la procédure à suivre pour accéder à la publication.

**Pour partager une publication :**

1. Dans la page Publications, survolez le diaporama souhaité avec votre souris, puis sélectionnez **Publier**.
2. Sélectionnez l'option de publication voulue.
3. (Facultatif) Lorsque vous publiez le diaporama en public ou en privé, vous pouvez envoyer un e-mail en cliquant sur **Envoyer ce lien Web par e-mail**, puis en finalisant le message.
4. Cliquez sur **Terminé**.  
   L'état de la publication dans le diaporama est indiqué dans la page Publications.

### Gestion des publications

Vous pouvez gérer les publications dans la page Publications, qui présente les diaporamas déjà créés par vos soins ou que d'autres personnes ont partagés avec vous. Lorsque vous survolez un diaporama avec votre souris, l'objet du diaporama bascule pour révéler un certain nombre d'options. Ces différentes options sont décrites dans le tableau suivant :

| Option | Description |
| --- | --- |
| Lire | Diffuse le diaporama dans la visionneuse de Domo. |
| Modifier | Ouvre une boîte de dialogue qui vous permet d'ajouter ou de retirer des diapos de carte, de les réagencer et de renommer le diaporama. Les modifications apportées sont enregistrées lorsque vous cliquez sur **Enregistrer**. |
| Publier | Affiche une boîte de dialogue qui vous permet de définir les options de partage du diaporama. |
| Télécharger | Exporte les diapo du diaporama sous forme de présentation Microsoft PowerPoint. Pour obtenir plus d'informations, consultez la rubrique [Exportation de publications.](/s/article/360043437793) |
| Supprimer | Supprime le diaporama dans Domo. |

Ajoutez les lignes suivantes :

|  |  |
| --- | --- |
| Envoyer le rapport maintenant | Permet d'envoyer ce rapport par e-mail aux utilisateurs et/ou groupes sélectionnés dans votre organisation. |
| Planifier le rapport | Permet de programmer l'envoi de ce rapport aux utilisateurs et/ou groupes sélectionnés dans votre organisation. |

Vous pouvez également utiliser une page de cartes ou l'affichage des **Détails** pour ajouter des cartes dans un rapport. Pour ce faire, sélectionnez **> Planifier le rapport** pour la carte à ajouter, puis cliquez sur l'option **Ajouter au rapport**. Vous pouvez ensuite sélectionner tous les rapports existants auxquels cette carte doit être ajoutée.

### Exportation de publications

Vous pouvez exporter les diapos des cartes KPI d'une publication sous forme de présentation Microsoft PowerPoint. Cette exportation peut se révéler utile lorsque vous voulez présenter un *instantané* de vos données (plutôt qu'un diaporama dynamique qui se met à jour à mesure que les données de la carte sont actualisées).

Le tableau suivant répertorie les grandes similitudes et différences entre les publications exportées et celles qui s'affichent dans Domo :

| Dans PowerPoint... | Dans Domo |
| --- | --- |
| - Chaque carte prend la forme d'une diapo. - Les cartes sont *statiques* ; elles ne sont *pas* automatiquement mises à jour lorsque les données qu'elles représentent sont modifiées ; ce qui se révèle utile pour présenter un *instantané* de vos données. - Les diaporamas exportés peuvent être affichés dans Microsoft PowerPoint. | - Chaque carte prend la forme d'une diapo. - Les cartes sont *dynamiques* ; elles sont automatiquement mises à jour lorsque les données qu'elles représentent sont modifiées. - Les diaporamas s'affichent dans un navigateur Web. |

Vous pouvez également télécharger l'une des cartes ou toutes les cartes d'une page dans PowerPoint sans d'abord créer un diaporama dans Domo. Pour obtenir plus d'informations, consultez la rubrique [Exportation de cartes KPI.](/s/article/360043437813)

**Pour exporter une publication vers Microsoft PowerPoint :**

1. Dans la page Publications, localisez le diaporama à exporter.
2. Sélectionnez **> Télécharger**.

<Frame>![Screenshot](/images/kb/fr/0EM5w000005wJLb.png)</Frame>

Vous pouvez aussi exporter un diaporama après l'avoir modifié en cliquant sur **Enregistrer**, puis sur **Télécharger au format PowerPoint**.

<Note>
**Remarque :** parmi les différents types de cartes (KPI, image, bloc-notes, document, sondage, etc.), *seules* les cartes KPI peuvent être exportées sous forme de présentation Microsoft PowerPoint.
</Note>

Vous pouvez afficher la présentation exportée dans Microsoft PowerPoint.

### Administration des publications

Pour administrer les Publications, servez-vous des Paramètres d'administrateur. Les options d'administration comprennent la suppression de l'accès Web à une publication, le retrait de cartes individuelles dans la publication ou la suppression d'une publication dans son intégralité.

Pour obtenir plus d'informations sur la topologie des Paramètres d'administrateur, consultez la rubrique [Topologie des Paramètres d'administrateur.](/s/article/360042934234)

**Pour administrer les publications :**

1. Cliquez sur **> Administrateur**.  
   Les Paramètres d'administrateur s'affichent.
2. Cliquez sur **Publications**.  
   La liste des diaporamas et les détails relatifs au diaporama sélectionné s'affichent.
3. Sélectionnez le diaporama à administrer.
4. Choisissez l'une des actions suivantes :

   | Action | Description |
   | --- | --- |
   | Arrêter de partager le diaporama | Cliquez sur **Arrêter de publier** pour supprimer l'accès Web au diaporama. |
   | Retirer une carte du diaporama | Cliquez sur **Retirer** à côté de la diapo à retirer du diaporama. Lorsque vous y êtes invité, cliquez sur **Supprimer** pour confirmer la suppression de la carte dans le diaporama. |
   | Supprimer le diaporama | Cliquez sur **Supprimer** pour supprimer le diaporama sélectionné dans Domo. Lorsque vous y êtes invité, cliquez sur **Supprimer** pour confirmer la suppression du diaporama. |

   <Note>
   **Remarque :** lorsque vous appliquez une action administrative à un diaporama, une notification est envoyée au propriétaire de ce dernier.
   </Note>

## Affichage d'un diaporama ponctuel des cartes d'une page

Vous pouvez afficher les cartes d'une page sous forme de diaporama ponctuel (c.-à-d. non enregistré).

Comme les diaporamas présentent toutes les cartes d'une page, vous pouvez créer des pages spécifiquement destinées aux diaporamas. Par exemple, si chaque équipe de projet de votre département Développement dispose de sa propre page, vous pouvez mettre en avant les progrès réalisés par chacune de ces équipes en matière de résolution des bogues logiciels. Vous pouvez créer une nouvelle page contenant les cartes de chaque page présentant ces données, puis afficher cette nouvelle page dans un diaporama.

<Note>
**Remarques :**

- Diffuser le diaporama des cartes d'une page est un moyen rapide de passer les différentes cartes en revue, mais ce diaporama n'est *pas* enregistré.  
  Pour créer et enregistrer un diaporama des cartes d'une page ensuite consultable à leur gré par d'autres utilisateurs, vous disposez des options suivantes :

  - Pour créer un diaporama dynamique, cliquez sur en haut et à droite de la page, sélectionnez **Publier sous forme de diaporama**, puis cliquez sur **Enregistrer**. Le diaporama est alors enregistré dans votre instance de Domo. À partir de là, vous pouvez télécharger, lire, publier, modifier ou supprimer le diaporama. Pour obtenir plus d'informations, consultez la rubrique [Création d'une publication.](/s/article/360043437793)
  - Vous pouvez directement exporter une page de cartes vers PowerPoint en sélectionnant **Exporter au format PowerPoint** dans le menu de la page. Dans ce cas, aucun diaporama n'est enregistré dans Domo et les données ne sont pas mises à jour dynamiquement. Cette option est toutefois un excellent moyen de partager rapidement un instantané des données. Pour obtenir plus d'informations, consultez la rubrique [Exportation de cartes KPI](/s/article/360043437813).
- La page Launcher vous permet d'afficher les cartes alimentées par un connecteur ou un fournisseur de données spécifique sous forme de diaporama.
</Note>

**Pour diffuser un diaporama de toutes les cartes d'une page :**

1. Accédez à la page contenant les cartes à diffuser.
2. En haut et à droite de la page, à côté des photos du profil du collaborateur, cliquez sur .
3. Sélectionnez **Afficher cette page sous forme de diaporama**.

Pour plus d'informations sur les commandes de la visionneuse de diaporama, consultez la section suivante.

## Utilisation de la visionneuse de diaporama

Domo inclut une visionneuse de diaporama intégrée qui permet de présenter les cartes une à une en mode Plein écran. Les commandes de base de la visionneuse vous permettent notamment d'interrompre le diaporama, de passer à la diapo suivante ou de revenir à une diapo précédente, ou encore de définir la durée d'affichage des diapos à l'écran.

Les principales commandes de l'interface de la visionneuse s'affichent momentanément en haut et à droite de l'écran lorsque vous déplacez la souris. D'autres commandes s'affichent également dans d'autres parties de l'écran lorsque vous les survolez avec votre souris. Ces commandes sont décrites ci-après.

### Principales commandes

Dans Domo, les différentes commandes de la fonction de diaporama vous permettent de parcourir les diapos, d'interrompre et de reprendre la lecture, de définir le délai devant s'écouler entre deux diapos et d'afficher le diaporama en Plein écran. Lorsque vous lancez le diaporama, les commandes s'affichent en haut et à droite de l'écran, puis disparaissent quelques secondes plus tard, mais il vous suffit de déplacer la souris pour les réafficher. Les commandes sont notamment les suivantes :

### Flèches Précédent/Suivant

Lorsque vous survolez le côté gauche ou droit de l'écran avec votre souris, les grandes flèches qui s'affichent vous permettent de passer à la diapo précédente ou suivante. Ces flèches sont similaires à celles du jeu principal de commandes. Elles sont parfois plus pratiques pour parcourir les diapos.

### Carrousel de cartes

Lorsque vous survolez le bas de l'écran avec votre souris, une ligne de cartes s'affiche ; cliquez sur une carte pour y accéder directement dans le diaporama. Lorsque la page contient plus de huit cartes, vous pouvez utiliser les flèches et pour « faire glisser » le carrousel vers la gauche ou la droite et afficher davantage de cartes. Lorsque la page ne contient plus de carte dans la direction indiquée, la flèche est grisée.
