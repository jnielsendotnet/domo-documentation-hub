# PDP : masquage des colonnes

PDP : la fonctionnalité Masquage des colonnes dans les DataSets vous permet de masquer les informations sensibles telles que les informations personnelles identifiables et les données de santé protégées. Vous pouvez définir les stratégies de masquage des colonnes pour expurger des valeurs intégralement ou pour masquer un certain nombre de caractères dans un champ donné.

Vous pouvez configurer le masquage des colonnes dans la section PDP de l'écran Détails du DataSet.

<Frame>![ColumnMasking (1).png](/images/kb/fr/0EMVq000000xyD3.jpg)</Frame>

Pour plus d'informations sur le [masquage des colonnes](/fr/s/article/360042934614) et PDP, consultez la base de connaissances.

# Fonctions de fenêtre dans la mosaïque SQL

Les fonctions de fenêtre dans la mosaïque SQL vous permettent d'utiliser des fonctions analytiques complexes dans vos DataFlows Magic ETL : vous n'avez pas besoin de retrier les données pour utiliser un DataFlow Redshift ni de créer plusieurs flux de données dans Magic ETL.

Vous pouvez effectuer des calculs de total cumulé, de classement, de moyennes mobiles et de statistiques cumulées directement depuis votre mosaïque SQL.

<Frame>![Screenshot](/images/kb/fr/0EMVq000007iie1.jpg)</Frame>

En savoir plus sur les [fonctions de fenêtre dans les mosaïques SQL](/fr/s/article/360044876874).

# Studio d'applications

#### Mises à jour de l'éditeur de Studio d'applications

Utilisez le Studio d'applications pour générer automatiquement des graphiques et des tableaux à l'aide de vos données en quelques étapes, sans aucune configuration initiale nécessaire.

Concevez les éléments dont vous avez besoin à l'aide des options et outils suivants :

- Sélectionnez votre colonne, ajoutez des calculs, ajustez des plages de dates et appliquez des filtres
- Vous avez le choix entre quatre types de tableaux pour vos conceptions : standard, modifiables, liés et croisés dynamiques
- Choisissez parmi des dizaines de types de graphiques : à barres, en courbes, sectoriel, carte thermique, carte du monde et nuage de points

<Frame>![Screenshot](/images/kb/fr/0EMVq000009brg5.jpg)</Frame>

#### Composants de tableau modifiables

Les composants de tableau sont désormais disponibles dans votre toile Studio d'applications.

Utilisez ces composants pour concevoir des tableaux de toutes pièces pour la planification et le suivi des flux de travail, le chargement des fichiers CSV, l'ajout de colonnes personnalisées et la connexion à des données existantes.

# Magic ETL

#### Désactivation des mosaïques

Magic ETL permet désormais de désactiver des mosaïques ou de désactiver le code dans les DataFlows. Grâce à cette fonctionnalité, vous pouvez sélectionner une ou plusieurs mosaïques à désactiver, et elles seront exclues de l'exécution du Dataflow.

Outre la désactivation de mosaïques, vous pouvez également mettre en surbrillance toutes les mosaïques en amont ou en aval d'une mosaïque sélectionnée, et les désactiver individuellement depuis leur panneau de configuration.

<Frame>![Screenshot](/images/kb/fr/0EMVq000009qslV.jpg)</Frame>

#### Mosaïque Saisie corrigée

La mosaïque Saisie corrigée vous permet d'ajouter rapidement des données, directement dans la toile Magic ETL. Vous pouvez ajouter des données manuellement en les copiant-collant d'autres sources ou en téléchargeant des fichiers Excel et CSV. Vous pouvez également utiliser la génération alimentée par l'IA pour créer des colonnes ou des DataSets complets à l'aide de données exemple réalistes. Toutes les données saisies sont stockées dans la définition du DataFlow et peuvent être utilisées comme entrée pour les transformations en aval dans ce même DataFlow.

La mosaïque Saisie corrigée présente les avantages suivants :

- Prototypage et test accélérés
- Flexibilité de la saisie des données améliorée
- Autonomisation des utilisateurs non techniciens
- Génération de données exemple réalistes

<Frame>![Screenshot](/images/kb/fr/0EMVq000009qhAA.jpg)</Frame>

#### Mosaïque Développeur JSON

La nouvelle mosaïque Développeur JSON vous permet d'étendre les données au format JSON d'une colonne à plusieurs colonnes, sans aucune analyse complexe.

Grâce à cette fonctionnalité, vous accédez aux données de colonne de manière fiable à l'aide d'un processus d'expansion hiérarchique convivial.

<Frame>![Screenshot](/images/kb/fr/0EMVq000009qwp7.jpg)</Frame>

# Feuilles de travail

Les feuilles de travail vous permettent de transformer les données brutes en perspectives rapides, sans aucune courbe d'apprentissage ni configuration de structures d'applications nécessaires. 

Elles comprennent une toile accessible, qui fonctionne par glisser-déposer et permet d'effectuer des analyses de données rapides dans Domo. EIles sont étroitement intégrées dans l'environnement de données de Domo et prennent en charge la remontée des informations lorsque vous êtes prêt à mettre une feuille de calcul dans le Studio d'applications pour la partager.
