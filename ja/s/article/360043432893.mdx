# はじめに

Open Exchange Ratesは、リアルタイムの為替レートと通貨換算のデータを提供します。DomoのOpen Exchange Ratesコネクターを使用すると、選択した基本通貨タイプの為替レートを確認することができます。Open Exchange Rates APIの詳細については、Open Exchange Rates のウェブサイト（ https://openexchangerates.org/documentation ）を参照してください。またDomoには、このコネクターの高度なバージョンもあり、世界の通貨のリストや履歴データを取得することができます。詳細については、「Open Exchange Rates Advancedコネクター」を参照してください。

| **主な使用事例**                     | このコネクターは、為替レートに関する情報を返します。 |
| ------------------------------ | -------------------------- |
| **主なメトリクス**                    | 選択した世界の通貨の為替レート。           |
| **社内における主な権限**                 | 売上<br/>財務                  |
| **平均導入時間**                     | わずか数秒で実現                   |
| **使いやすさ（1～10段階で、1が「最も使いやすい」）** | 1                          |

Open Exchange Ratesコネクターは「クラウドApp」コネクターであるため、クラウドに保管されたデータの取得が可能です。Data Centerでウィンドウ上部のツールバーで **［クラウドApp］** をクリックすると、これを含む他のクラウドAppコネクターのページにアクセスできます。

Open Exchange Ratesには、 Data Center から接続します。このトピックでは、Open Exchange Ratesコネクターユーザーインターフェース固有のフィールドとメニューについて説明します。DataSetの追加や更新スケジュールの設定、DataSet情報の編集に関する一般的な情報は、「 [データコネクターを使用してDataSetを追加する](/ja/s/article/360042926274) 」を参照してください。

## 必要条件

なし。

## Open Exchange Ratesに接続する

このセクションでは、Open Exchange Ratesコネクターページの **［詳細］** ペインのオプションについて説明します。このページの他のペインの項目である **［スケジュール設定］** や **［DataSetに名前を付けて説明を入力］** に関しては、ほとんどのタイプのコネクターで共通しています。詳細については「 [データコネクターを使用してDataSetを追加する](/ja/s/article/360042926274) 」を参照してください。

### ［詳細］ペイン

このペインには、世界の通貨を選択する単一のメニューが含まれています。

| メニュー         | 説明                  |
| ------------ | ------------------- |
| 基準値として使用する通貨 | 為替レートを表示する通貨を選択します。 |

### その他のペイン

スケジュール設定、再試行、オプションの更新など、このページにないセクションに記載のコネクターインターフェースについては、「 [データコネクターを使用してDataSetを追加する](/ja/s/article/360042926274) 」を参照してください。

---

Open Exchange Ratesコネクターを使用して基準となる通貨を入力すると、他のすべてのタイプの通貨に対するすべての為替レートが表示されます。
