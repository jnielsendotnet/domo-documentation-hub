## はじめに

Advanced Tools Launchセンターにより、Domoが提供するすべてのデータツールを1ヶ所で簡単に検索できます。データの接続と変換からDomoインスタンスの管理まで、業務に必要なソリューションを見つけてください。多くの機能が作成されると、自動的にAdvanced Tools Launchセンターに追加されるため、Domoの新機能を常に知ることができます。

## Advanced Tools Launchセンターにアクセスする

Advances Tools Launchセンターにアクセスするには、上部のナビゲーションペインの **［データ］** アイコンをクリックして、 **［高度なツール］** メニューを選択します。

## Advanced Tools Launchセンターのナビゲーション

Advanced Tools Launchセンターには、検索バーが表示されています。検索バーですばやくキーワードを入力して検索することができます。各ツールには、さらに詳細な情報を得るための **［詳細］** リンクと、ツールの取得プロセスを開始するための **［利用を開始］** リンクがあります。カテゴリー別にデータをフィルタリングするための、8つのQuick Filterタブが選択できます。

* [接続](#h_01F9F60P5FAKRAGTQH1568KZ60)
* [変換](#h_01F9F60WC36SNHV6G6QX1DRYJT)
* [データサイエンス](#h_01F9F612CD462CQA751D0ZNT54)
* [統合](#h_01F9F6180RH9DCTGGCXX4KM48W)
* [ガバナンス](#h_01F9F61NG5VR8B3M88Z3SNEJ4D)
* [IoT](#h_01F9F61VCYEQSCKPYJ7P0ME6W6)
* [サービスとその他](#h_01F9F621VACSC2X074SSN9D6AY)

## 接続

接続Quick Filterには、データをDomoに取り込むための以下のオプションがすべて表示されます。

* コネクターをUpsert
* パーティションコネクター
* データフェデレーション

<Frame><img alt="Connect_Tab.png" src="/images/kb/ka05w00000123v4-00N5w00000Ri7BU-0EM5w000005vPVW.png"/></Frame>
## 変換

変換Quick Filterには、Domoでデータを結合するための以下のオプションがすべて表示されます。

* スタッカー
* Adrenaline変換
* DataSetビュー

<Frame><img alt="Transform_Tab.png" src="/images/kb/ka05w00000123v4-00N5w00000Ri7BU-0EM5w000005vPVd.png"/></Frame>
## データサイエンス

データサイエンスQuick Filterには、一般的なデータサイエンスツールをDomoで使用するための以下のオプションがすべて表示されます。

* R SDK
* Python SDK
* AutoML

## 統合

統合Quick Filterには、データをDomoからData Warehouseに送信するための以下のオプションがすべて表示されます。

* Integration Studio
* Writebackコネクター
* ODBCドライバー

## ガバナンス

ガバナンスQuick Filterには、Domoインスタンスを管理するための以下のオプションがすべて表示されます。

* 動的PDP
* DomoStats
* Beast Mode Manager

## IoT

IoT Quick Filterには、IoTデバイスとオンプレミスデバイスに接続するための以下のオプションがすべて表示されます。

* Webhookコネクター
* Workbench向けのOPCプラグイン

<Frame><img alt="IOT_Tab.png" src="/images/kb/ka05w00000123v4-00N5w00000Ri7BU-0EM5w000005vPVc.png"/></Frame>
## サービスとその他

サービスとその他Quick Filterには、Domoのカスタムサービスを購入するための以下のオプションがすべて表示されます。

* コネクターを購入（カスタム開発）
* SDKとCLI
* カスタムのインジェストパイプラインとアーキテクチャーレビュー
