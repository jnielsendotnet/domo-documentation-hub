## はじめに

Salesforce Pardotサービスにより、品質レポートの作成とマーケティングの自動化を行うことができます。DomoでPardot QuickStartを使用すると、ウェブサイトにアクセスしたユーザーの数とそのエンゲージメント情報、見込み客の測定数、ソース、地域、進行中のオポチュニティー件数、成約率などについて、サマリービューをすばやく表示することができます。

## 使用事例

* ビジネスに関する回答可能な質問：

  + 販売目標を達成するために今日すべきことは何か
  + 見込み客の主要なソースはどこか
  + 顧客はウェブサイトをどのように利用しているのか
* このAppに付属しているメトリクス：

  + Average Sales Cycle
  + Lead Generation
  + Open Opportunities
  + Page Engagement
  + Page View Sources
  + Prospect Activity
  + Prospect Sources
  + Sales and Forecast
  + Sales Funnel
  + Sales Trend
  + Top 10 Wins
  + Top Prospecting Countries
  + Top Prospecting States
  + Visitor Behavior
  + Visitors
  + Win Rate
  + Won vs Lost Deals

## 認証

認証を行うには、ユーザー名、パスワード、ユーザーキーが必要になります。 これらの情報を確認するには、以下の手順を実行します。

1. [Pardot.com](http://Pardot.com) でSalesforce Pardotアカウントにログインします。
2. 画面右上隅のユーザー名の横に表示されている矢印をクリックします。
3. ユーザーキーをコピーし、Salesforce Pardotコネクターの **［認証情報］** ペインの **［User Key］** フィールドにペーストします。
4. **［Settings］** をクリックします。
     
   これにより、 **［My User Information］** 画面にリダイレクトされます。
