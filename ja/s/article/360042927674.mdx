## はじめに

Bazaarvoice は、ブランドと小売業者と消費者を結ぶネットワークで、商品レビューや分析、ターゲットメディアなどを通して ROI を提供しています。Domo の Bazaarvoice Connectorを使用すると、製品やレビュー、質問と回答などに関するレポートを作成することができます。Bazaarvoice API に関する詳細は、ウェブサイトを参照してください： (https://developer.bazaarvoice.com/docs/read/conversations
)

Bazaarvoice のアカウントは、Data Center で接続します。このトピックでは、Bazaarvoice Connectorユーザーインターフェースに固有のフィールドとメニューについて説明します。DataSet の追加、更新スケジュールの設定及び DataSet 情報の編集に関する一般的な情報は、 [データ Connector を使用して DataSet を追加する](/ja/s/article/360042926274 "データ Connector を使用して DataSet を追加する") で説明しています。

## 必要条件

Bazaarvoice アカウントに接続して DataSet を作成するには、以下が必要です。

* Bazaarvoice インスタンスのベース URL。一般的にこれは、製作用は

  [http://api.bazaarvoice.com](http://api.bazaarvoice.com "http://api.bazaarvoice.com")

  、そしてステージング用は

  [http://stg.api.bazaarvoice.com](http://stg.api.bazaarvoice.com "http://stg.api.bazaarvoice.com")

  になります。
* Bazaarvoice API キー。API キーを取得する方法については、
  https://developer.bazaarvoice.com/apis/conversations/key_request_process
  を参照してください。
