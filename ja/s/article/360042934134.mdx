## はじめに

SquareでシンプルなモバイルAppとカードリーダーを使用すると、場所を問わず、ビジネスでクレジットカードを使用できるようになります。Squareダッシュボードには、Squareアプリケーションによって作成された販売活動のサマリービューが表示されます。また、このダッシュボードでは、在庫に関する基本的なトレンドとメトリクスが表示されるため、業務を効率的に進めることができます。

## 使用事例

ビジネスに関する回答可能な質問：

* 顧客が求めているのはどのような商品か
* 現在の販売実績はどうなっているか
* このAppに付属しているメトリクス：
* 銀行預金
* 下位10件のSKU
* コスト明細
* 品目価格リスト
* カテゴリー別品目
* 注文トレンド
* 人気の品目
* 払い戻し回数
* カテゴリー別販売額
* セールス担当者が受け取ったチップ
* Squareの使用料金
* 上位10件のSKU
* 取引収益

## 認証

認証を行うには、Square用のメールアドレスとパスワードが必要です。

## ダッシュボードレポート

レポートの説明は、「 Squareコネクター 」を参照してください。

## レポートの設定/レポートの詳細

選択可能なレポートは、以下のように設定する必要があります。

| レポート        | スケジュール設定（更新設定の変更） | パラメーター                              | 選択                                                                                       |
| ----------- | ----------------- | ----------------------------------- | ---------------------------------------------------------------------------------------- |
| Items       |                   | なし                                  |                                                                                          |
| Merchant    |                   | なし                                  |                                                                                          |
| Payments    |                   | Date From、Date To                   | 日付の範囲として開始日と終了日を指定します（特定の日付、相対的な日付、日付のオフセットを指定することができます）。                                |
| Refunds     |                   | Date From、Date To                   | 日付の範囲として開始日と終了日を指定します（特定の日付、相対的な日付、日付のオフセットを指定することができます）。                                |
| Settlements |                   | Settlement Status、Date From、Date To | ステータスは、「送信済み」または「失敗」のいずれかになります。日付の範囲として開始日と終了日を指定します（特定の日付、相対的な日付、日付のオフセットを指定することができます）。 |
