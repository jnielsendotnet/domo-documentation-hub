## はじめに

HubSpotは、より多くの顧客を獲得するためのインバウンドマーケティングプラットフォームです。HubSpot QuickStart Appを使うと、送信、配信、サブスクライブ解除、さらにはクリックして開くことによる顧客インタラクションの測定に至るまで、すばやく見ることができます。

## 要件と注意点

* 認証にはユーザーHub IDが必要です。
* デフォルトの日付範囲：利用可能なすべてのデータを引き出します。

## 使用事例

エグゼクティブパルス：現在起きていること

* エグゼクティブパルス：現在起きていること

  + バウンス率
  + 最も多く開かれている
  + メールパフォーマンス
  + コンテンツパブリケーション
  + CTORパフォーマンス
  + 閲覧数ごとの上位キーワード
* 送信と配信：配信性を損なうことなくメーリングリストを拡大します。

  + 送信数
  + 状態別リード
  + サブスクライブ解除率
  + 配信率
  + サブスクライブ解除数
* エンゲージメント：メールメッセージに対するオーディエンスの応答を増加

  + クリック率（CTR）
  + クリック率上位20件のメール
  + 最もクリックされたメール
  + 国別訪問者数
  + 最も開かれているメール
  + 上位20件のキャンペーン
  + 状態別のページ閲覧数
  + 開封率
  + クリックして開く率
