## はじめに

Conductor Searchlight は、ウェブにおける会社の存在感を管理するために使用する、顧客が購入に進むまでの全過程のエンゲージメントを促進するソリューションです。 Domo の Conductor Searchlight Connectorを使用すると、検索エンジンの最適化に関するレポートを集積してトラフィック、コンバージョン、収益を向上させることができます。 Conductor Searchlight API についての詳細は、ウェブサイト ([http://developers.conductor.com](http://developers.conductor.com/) ) を参照してください。

Conductor Searchlight アカウントには、データセンターから接続します。 このトピックでは、Conductor Searchlight Connectorユーザーインターフェイスに固有のフィールドとメニューについて説明します。 DataSet の追加、更新スケジュールの設定及び DataSet 情報の編集に関する一般的な情報は、 [データ Connector を使用して DataSet を追加する](/ja/s/article/360042926274 "データ Connector を使用して DataSet を追加する")

で説明しています。

## 必要条件

Conductor Searchlight アカウントに接続して DataSet を作成するには、以下が必要です。

* Conductor Searchlight アカウントの API キー
* Conductor Searchlight アカウントの共有秘密キー

このセクションでは、Conductor Searchlight Connectorページの **認証情報** および **詳細** ペインのオプションについて説明します。 このページの他のペインの項目である **スケジュール設定** や **DataSet の名前と説明** に関しては、ほとんどのタイプの Connector で共通しており、 [データ Connector を使用して DataSet を追加する](/ja/s/article/360042926274 "データ Connector を使用して DataSet を追加する")
