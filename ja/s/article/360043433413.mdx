## はじめに

IBM Informixは、高速でスケーラブルなデータベースサーバーであり、従来のリレーショナルデータベース、オブジェクトリレーショナルデータベース、ディメンションデータベースを管理できます。DomoのInformix SSHコネクターを使用すると、SQLクエリにもとづいてInformixデータを取得できます。Informixについての詳細は、ウェブサイト（ [https://www.ibm.com/analytics/us/en/...logy/informix/](https://www.ibm.com/jp-ja/analytics/us/en/technology/informix/ "https://www.ibm.com/jp-ja/analytics/us/en/technology/informix/") ）を参照してください。

Informixファイルには、Data Centerから接続します。このトピックでは、Informixコネクターユーザーインターフェースに固有のフィールドとメニューについて説明します。DataSetの追加や更新スケジュールの設定、DataSet情報の編集に関する一般的な情報は、「 [コネクターを使用してDataSetを追加する](/ja/s/article/360042926274 "コネクターを使用してDataSetを追加する") 」を参照してください。

## 必要条件

Informixアカウントに接続してDataSetを作成するには、以下が必要です。

* SSHホスト名、ユーザー名、パスワード
* SSHポート
* データベースホスト名
* データベース名
* データベースポート番号
* データベースのユーザー名とパスワード
* Informixサーバー名

Informixデータベースに接続する前に、以下のIPアドレスをデータベースサーバーの接続ポートでホワイトリストに登録する必要があります。IPアドレスの詳細なリストは、「 コネクターのIPアドレスをホワイトリストに登録する 」を参照してください。

<Note>
**注記：**

Domoはssh-keygenを使用して生成されるSSHキーをサポートしていません。SSHキーは、OpenSSLが生成するDESキーまたはRSAキー（PEM形式）である必要があります。

</Note>

## Informixアカウントに接続する

このセクションでは、Informixコネクターページの **［認証情報］** および **［詳細］** ペインのオプションについて説明します。このページのほかのペインの項目である **［スケジュール設定］** や **［DataSetに名前を付け、説明を入力］** に関しては、ほとんどのタイプのコネクターで共通しています。詳細については、「 [コネクターを使用してDataSetを追加する](/ja/s/article/360042926274 "コネクターを使用してDataSetを追加する") 」を参照してください。

### ［認証情報］ペイン

このペインには、Informixアカウントに接続する認証情報の入力フィールドがあります。下表では、各フィールドに必要な内容を説明しています。

| フィールド                | 説明                                                                     |
| -------------------- | ---------------------------------------------------------------------- |
| SSH Hostname         | UNIXサーバーのホスト名またはURL（SSHトンネル経由）を入力します。                                  |
| SSH User             | UNIXアカウントのSSHユーザー名を入力します。                                              |
| SSH Password         | UNIXアカウントのSSHパスワードを入力します。                                              |
| SSH Key              | SSH秘密キーを入力します。                                                         |
| SSH Port             | UNIXアカウントのポート番号を入力します。                                                 |
| Database Hostname    | Informixデータベースのホスト名を入力します。例： `[db.company.com](http://db.company.com)` |
| Database Name        | Informixデータベース名を入力します。                                                 |
| Database Port Number | Informixのポート番号を入力します。                                                  |
| Database Username    | Informixユーザー名を入力します。                                                   |
| Database Password    | Informixパスワードを入力します。                                                   |
| Informix Server Name | Informixサーバーの名前を入力します。                                                 |

有効なInformix認証情報を一度入力しておくと、随時同じアカウントを使用して新しいInformix DataSetを作成できます。コネクターアカウントは、Data Centerの **［アカウント］** タブで管理できます。このタブの詳細については、「 [コネクターアカウントを管理する](/ja/s/article/360042926054 "コネクターアカウントを管理する") 」を参照してください。

### ［詳細］ペイン

このペインには、SQLクエリを入力するためのメニューが1つあります。

| メニュー  | 説明                                 |
| ----- | ---------------------------------- |
| Query | Domoに入力するInformixデータのSQLクエリを入力します。 |

スケジュール設定、再試行、更新オプションなど、その他のコネクターインターフェースのセクションの詳細については、「 [コネクターを使用してDataSetを追加する](/ja/s/article/360042926274 "コネクターを使用してDataSetを追加する") 」を参照してください。
