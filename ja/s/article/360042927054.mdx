## ﻿ はじめに

Outbrainは広告プラットフォームです。そのコンテンツマーケティングモジュールは、インターネットのパブリッシャーに対して関連コンテンツや面白い記事、その他の信頼できるコンテンツへのリンクを提示し、ウェブトラフックの増加を支援します。 DomoのOutbrain Connectorは、DomoとOutbrainを統合することで、Outbrain Amplify API からデータの抽出を行います。 DomoのOutbrain Connectorを利用すると、マーケティング担当者、予算、キャンペーン、有料リンクやパフォーマンスレポートなどのデータを取得することができます。 Outbrain Amplify APIに関する詳細は、同社のウェブサイト (http://developer.outbrain.com/home-page/amplify-api/ )

Outbrain アカウントには、 データセンター から接続します。 このトピックでは、Outbrain Connectorユーザーインターフェースに固有のフィールドとメニューについて説明しています。 DataSet の追加、更新スケジュールの設定及び DataSet 情報の編集に関する一般的な情報は、 [追加する](/ja/s/article/360042926274 "データ Connector を使用して DataSet を追加する") で説明しています。

## 必要条件

Outbrain アカウントに接続して DataSet を作成するには、Outbrain のユーザー名とパスワードが必要です。 Outbrain の Amplify API の登録ベータユーザーである必要もあります。 詳細は http://developer.outbrain.com/home-page/amplify-api/apply/ を参照してください。

## Outbrainアカウントに接続する

このセクションでは、Outbrain コネクターページの **認証情報** および **詳細** ペインについて説明します。 このページの他のペインの項目である **スケジュール設定** や **DataSet の名前と説明** に関しては、ほとんどのタイプの Connector で共通しており、 [追加する](/ja/s/article/360042926274 "データ Connector を使用して DataSet を追加する") でより詳細に説明しています。

### 認証情報ペイン

このペインには、 Outbrainアカウントに接続する認証情報の入力フィールドが含まれています。 下表では、各フィールドに必要な内容を説明しています。

| フィールド | 説明                    |
| ----- | --------------------- |
| ユーザー名 | Outbrain ユーザー名を入力します。 |
| パスワード | Outbrain パスワードを入力します。 |

有効なOutbrain認証情報を一度入力しておくと、随時同じアカウントを使用して、新しいOutbrain DataSetを作成することができます。 Connectorアカウントは データセンター の **アカウント** タブで管理することができます。 このタブに関する詳細については、 [Connector のユーザーアカウントを管理する](/ja/s/article/360042926054 "Connector のユーザーアカウントを管理する") を参照してください。
