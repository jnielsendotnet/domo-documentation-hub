## 新機能と機能拡張

本リリースで新しくなった機能と拡張された機能は、以下のとおりです：

### 新しいカードインタラクションのツール（CSR）

新しいカードインタラクション（クライアントサイドのレンダリング）のツールにより、チャートの詳細な分析を、今までにはなかった方法で簡単に楽しく行うことができるようになりました。 このツールには以下のものがあります：

* **新しいチャートピッカー** 。 最初にカードビルダーを開かなくても、詳細ページでほとんどのチャートのグラフタイプを変更できるようになりました。 カードビルダーのチャートタイプピッカーでは、各グラフタイプのアイコンのみが表示されるのに対し、詳細ビューチャートタイプピッカーでは、利用可能なグラフタイプのサムネールが、チャートのデータが既に適用された状態で表示されます。
    
  <Frame><img alt="" src="/images/kb/ka05w00000129vC-00N5w00000Ri7BU-0EM5w000005vOnp.png" style={{width: 810, height: 329}}/></Frame>
* **凡例アイテムの切り替え** 。 凡例のあるほとんどのグラフタイプ（マップを除く）では、凡例のアイテムをクリックすることで、チャート内で対応するデータを非表示または表示させることができるようになりました。
    
  <Frame><img src="/images/kb/ka05w00000129vC-00N5w00000Ri7BU-0EM5w000005vOnp.png" /></Frame>
* **グラフの拡大** 。 チャートの一部分をクリックそしてドラッグすると、その部分が拡大されるようになりました。 これは、ほとんどのグラフタイプで使用できます。例外は円グラフやマップ、パネル、レーダーチャートとその他いくつかのチャートです。
* **クイックソート** 。 ほとんどの棒グラフと折れ線グラフでは、カードビルダーを開かなくても、アイコンをクリックするだけでソート方法をすばやく変更できるようになりました。
* **チャートに名前を付けて保存** 。 詳細ビューでチャートに変更を加えると（上記の機能を使用するか、または Analyzer ツール使ってフィルターするかに関係なく）、編集したチャートのコピーを Domo に保存することができるようになりました。 その新しいバージョンのカードに新しい名前を付け、閲覧者がカードの変更内容を把握するのに役立つ説明を加え、この新しいカードを保存する Domo のページを選択できます。
* **データの予測** 。 棒グラフや折れ線グラフのためのこの新しいチャートのプロパティでは、チャートの現在期間の残りの部分の値を予測できるようになりました。 これは、現在の期間の値と、前の期間の値とを比較することによって行われます。

これらの全てのツールに関する詳細は、 新しいカードインタラクションのツール（2016 年 1 月） を参照してください。

### ページとカードの一括管理オプション

遂に、Domo の「管理者」ユーザーは Domo インスタンスのページやカードに一括編集を行えるようになりました! この機能により、Domo インスタンスのユーザーはカードやページの権限を手動で移動、削除、設定して余計な時間を費やす必要がなくなり、業務のニーズへの対応など、より重要な作業に携わることができるようになります。

管理者設定の **カード** タブや **ページ** タブは完全に改良され、一括変更が簡単にできるようになりました。 **カード** タブでは、カードや所有者をはじめ、カードが使われているページ数、カードタイプ（チャート、ドキュメント、アンケートなど）、最終編集日、またはそれらの組み合わせによってフィルターできます。 **ページ** タブでは、ページや親ページをはじめ、所有者、ページにあるカード数、最終更新日と時間、またはそれらの組み合わせによってフィルターできます。 インスタンス内にあるカードまたはページを全て選択し、一括で変更を加えることもできます。 どんなページまたはカードでも、行の中をクリックすると、詳細が記載されたタブが表示されます。

#### ページの一括オプション

ページの一括オプションには以下の全てが含まれます：

* ページ所有者の変更
* ユーザーとグループの追加と削除
* ページの階層構造での移動（移動されたページに権限を設定するオプションを含む）
* ページのロックとロックの解除
* ページの削除

#### カードの一括オプション

カードの一括オプションには以下の全てが含まれます：

* カード所有者の変更
* ユーザーとグループの追加と削除
* カードの複製
* ページでカードの追加と削除
* カードのロックとロックの解除
* カードの削除

これらのオプションに関する詳細は、 [管理者設定のレイアウト](/s/article/360042934234 "管理者設定のレイアウト") を参照してください。

### Domo オンラインフォーム v2

オンラインフォーム（ウェブフォーム）ツールが改良され、Domo に手動で DataSet を作成するためのオプションが増えました。 新しいオプションには以下のものがあります：

* 既存の DataSet をウェブフォームにインポートできます。
* 列のデータタイプを変更することができます。 **数値** （値）、 **テキスト** （文字列）、 **日付** のデータタイプがあります。 列の **自動検出** オプションを選択解除して、列のデータタイプ自動特定機能を無効にすることもできます。
* 特定のテキスト文字列を検索そして置換することができます。
* ボタンを 1 回クリックするだけで、オンラインフォームに列を追加できます。
* オンラインフォームエディタを拡大させて、画面全体に表示できます。
* 列をフィルターし、選択データタイプに合致しない行を確認することができます。

各列には、その列にある固有の値の数も表示されます。

詳細は、 KPI カードにデータを追加する を参照してください。

### Domo PowerPoint と Excel のプラグイン v2

PowerPoint プラグインと Excel プラグインのアップデートされたバージョンがリリースされました。  この新しいバージョンには、以下の改善点と修正が含まれています：

#### 一般

バグが修正され、プロキシを介して SSO を有効にしているユーザーも Domo インスタンスに接続できるようになりました。

#### Excel プラグイン

* **クエリ表** 。 DataSet のサブセットまたは単一の値を、集計、フィルター、制限などに基づいて取得し Excel に入力することができるようになりました。
* **新しいアップロード UI。** DataSet アップロード時の UI が新しいデザインになり、進捗バーが追加され、そしてメッセージングも改善されました。これで、何が起こっているか把握できるようになったほか、データを Domo にアップロードするのにどの位の時間がかかるかわかるようになりました。
* **新しいダウンロード UI。** DataSet ダウンロード時の UI も、進捗バーと改善されたメッセージングなどによって、デザインが新しくなりました。
* **バグ修正** 。 多数のバグが修正されました。これには、null 値の列を含む DataSet をアップロードする際のエラー、海外で最新バージョンを確認する際のエラーが含まれます。

#### PowerPoint プラグイン

* **まとめ数字** 。 チャートのまとめ数字を取得して、スライドに追加できるようになりました。
* **パブリケーショングループのサポート。** パブリケーショングループのサポートが追加されました。
* 多数のバグが修正されました。これには、海外でカードを更新する際のエラーが含まれます。

詳細は、 [Domo Excel プラグインを使用する](/s/article/360043437933 "Domo PowerPoint プラグインを使用する") を参照してください。

### ETL 拡張

Magic ETL エディターでは、以下の全てができるようになりました：

* 現在の日付と時刻(協定世界時)の選択。
* `NULL` 値と空白の文字列の置き換え。
* `MIN` 、 `MAX` 、および `NULL も含めたカウント` の集計タイプの使用。

ETL エディターの詳細については、 Magic ETL DataFlow を作成する を参照してください。
