## はじめに

CNNは、米国大手のニュースとメディアのネットワークです。このダッシュボードは、自分が関心のある記事に特化したクイックリンクや、記事の人気の変遷なども提供します。

## 利用方法

* エグゼクティブパルス：新しい記事と人気の記事

  + エンターテインメント記事
  + 新着世界の記事
  + 最も人気の記事トップ10
  + 新着テクノロジーの記事
  + 新着政治記事
  + 新着エンターテインメント記事
  + 新着ビジネス記事
  + トップニュース
* トレンド：記事の人気の変遷

  + トップニュース
  + 日別のテクノロジーニュース
  + 世界のニュース記事
  + 時間別政治記事
  + 毎日のトップ記事
  + 時間別ビジネスニュース
* リストとリンク：CNNのフルストーリーへのリンク

  + ビジネス記事
  + エンターテインメント記事
  + 世界の記事
  + 最も人気の記事
  + テクノロジー記事
  + 政治記事
