## 新機能と機能拡張

本リリースで新しくなった機能と拡張された機能は、以下のとおりです：

### Domo Everywhere：エンベッド

Domo は、他に類を見ないビジネス最適化ツールです。 データウェアハウスと、ETL ツール、コラボレーションのプラットフォーム、ハイエンドな可視化そしてクラウドの効率化を組み合わせた Domo は、どこからでも、そしてどんなデバイスからでも利用可能な、あらゆる規模のビジネスのための究極ツールと言えます。 そして今 Domo は、あなたが作成したコンテンツを、さらに顧客やパートナーなどにも配布させることができるようになりました。

弊社のお客様より、得たインサイトをさらに自分のカスタマーベースやパートナーとも共有したいとのご希望を度々いただいておりました。 Domo の Embedded Analytics 機能により、エンベッド用のコードを生成することで、ウェブサイトやカスタマー ポータル、またはアプリケーションなどに利用できるようになりました。 SSO プロバイダー経由によるセキュアな接続で、コンテンツは許可されたエンドユーザーにのみ配信されます。

Domo のエンベデッド アナリティクスは、Domo 外でも動作し続ける全機能搭載型 Domo カードです。 ユーザーは、カード内のデータを探索する機能や、ドリル機能やフィルター機能、PDP（ユーザーが自分に関係するデータのみ見れる機能）などを使えるほか、可視化やフォーカスされたインタラクションなども簡単に変更できます。

Domo 進化過程の次ステージへようこそ！お気に入りの製品を、あなたによって完全に管理された環境下で、共有したい人全員に利用してもらえるようになりました。

### 高度なメモカード

メモカードが改良され、どんなユーザーでもスタイリッシュなメモやプレゼンテーションを作成できるようになりました。 新しいメモカードのバージョンには、以下の機能があります：

* 編集中のカードをロック - カードの編集は、1 人ずつでのみ可能になり、変更を上書きされることがなくなりました。
* ハイパーリンクを追加する機能
* コンピューター、ネットワークまたはインターネットから画像を追加する機能
* 表を追加する機能
* フォントサイズを 3 種類から選べる機能
* フォントの色やスタイルを変更する機能
* ネストされたリストを追加する機能

### Java、Python そして Node 用 SDK

Domo では、Domo の公開 API とのインテグレーションをさらに簡単にするために、トップクラスの SDK の提供を開始しました。 Domo の SDK は、API ゲートウェイへの認証接続を作成そして管理するための労力を不要にします。 この SDK によりディベロッパーは、その基盤となる通信インフラを管理することなく、より簡単にそして迅速にカスタム インテグレーションを作成できるようになります。

また、このライブラリはオープンソースになっているため、Domo のパブリックなエンドポイントの活用を加速化し、そして各自のニーズに合わせて SDK を適応、カスタマイズできるようになります。

Domo の [Github レポジトリー](https://github.com/domoinc) にアクセスして、さっそく試運転してみませんか。または、ディベロッパー用の資料、 [www.developer.domo.com/public\_apis/sdks](http://www.developer.domo.com/public_apis/sdks) で詳細を参照できます。

### Buzz オンボーディング

Buzz に、さまざまなセリフのほか、素早く情報を共有するための簡単手順が装備されました。 自分の名前やプロフィール写真を編集または追加してコミュニケーションを強化しましょう。 招待状も、ワンクリックで簡単にネットワークに送信できます。ライセンスや予算上の制限などで、誰にアクセス権があるかなどを心配する必要もありません。

新しいユーザーには、控えめにゆっくりと点滅する青い点が表示され、さまざまな要素への注意を促します。

例えば、他の人を招待する、新規にチャットを作成する、重要またはお気に入りのユーザーを特定化する、あるいはデータを使ってチャットを最適化する、などです。

### 摩擦のない Buzz 招待

Buzz の堅固さは、その中にいる共同ユーザー網、つまりユーザーが他のユーザーを招待することで有機的に成長した結果のネットワークがあってこそです。 ユーザーの招待は、いつでもそして誰でもできるようになりました。直接メールアドレスを使って行うことも、または固有リンクをシェアすることでもできます。成長しすぎて制限されるような心配もありません。 必要に応じて、しかるべき人材をどんどん集め、コラボレーションしましょう。簡単なクリックで招待できます。

### モバイルページのフィルター

この度、強力で複雑なページフィルターのツールを、使いやすく直感的にモバイルのプラットフォームでリリースしました。 どんなビジネスユーザーでも、すぐに利用価値を享受できます。 あなたが役員で監督下の実店舗にまさに訪れようとしている時であれ、あるいは製造業者のマネージャーが特定製品ラインの検査をしているのであれ、ページを迅速にフィルターして、必要な情報をすぐに取り出すことができます。

是非試してみてください。 モバイルから Domo 内のページのどれかに進み、メニューをタップしてフィルターしてみましょう。
