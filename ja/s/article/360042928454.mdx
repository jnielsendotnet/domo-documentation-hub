## はじめに

Square, Inc.は、金融サービス、マーチャントサービスアグリゲーターおよびモバイル決済企業です。Square APIの詳細については、Webサイト（ [https://docs.connect.squareup.com/](https://docs.connect.squareup.com/ "https://docs.connect.squareup.com/") ）を参照してください。

このコネクターは、個別の認証を必要とするさまざまなSquare アカウントのDomo DataSetがある場合に便利です。

Data CenterでSquare DataSetへの接続を設定します。このトピックでは、Square DataSet Unionコネクターユーザーインターフェース固有のフィールドとメニューについて説明します。DataSetの追加や更新スケジュールの設定、DataSet情報の編集に関する一般的な情報は、「 [データコネクターを使用してDataSetを追加する](/ja/s/article/360042926274) 」を参照してください。

## 必要条件

Domoの中の既存のSquare DataSetに接続して新しいDataSetを作成するには、 **［管理者］ > ［セキュリティ］** で生成されたDomoアクセストークンが必要です。これには、Domoの管理者権限が必要です。詳細については、「 [アクセストークンを管理する](/ja/s/article/360042934494 "アクセストークンを管理する") 」を参照してください。

## Square DataSetを結合する

このセクションでは、Square DataSet Unionコネクターページの **［認証情報］** および **［詳細］** ペインのオプションについて説明します。このページのほかのペインの項目にある **［スケジュール設定］** と **［DataSetの名前と説明］** に関しては、コネクタータイプのほとんどで共通です。詳細については「 [データコネクターを使用してDataSetを追加する](/ja/s/article/360042926274 "データコネクターを使用してDataSetを追加する") 」を参照してください。

### 認証情報ペイン

このペインでDomoアクセストークンを入力して、既存のSquare DataSetに接続できるようにします。

| フィールド    | 説明                                                                                                                               |
| -------- | -------------------------------------------------------------------------------------------------------------------------------- |
| アクセストークン | **［管理者］ > ［セキュリティ］** で生成するDomoアクセストークンを入力します。これには、Domoの管理者権限が必要です。詳細については、「 [アクセストークンを管理する](/s/article/360042934494) 」を参照してください。 |

有効なアクセストークンを一度入力しておくと、いつでも同じアカウントを使用して新しいSquare DataSet Union DataSetを作成することができます。コネクターアカウントは、Data Centerの **［アカウント］** タブで管理できます。このタブの詳細については、「 [コネクターのユーザーアカウントを管理する](/ja/s/article/360042926054 "コネクターのユーザーアカウントを管理する") 」を参照してください。

| 設定          | レポートに必要な設定を選択します。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| 結合するDataSet | すべての一致するDataSetを結合するか、リスト（ **発見** ）から特定のDataSetを選択するかを選択します。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
