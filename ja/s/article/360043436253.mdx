## はじめに

Oracle Databaseは、Oracle Corporation社により開発・販売されたオブジェクトリレーショナルデータベース管理システムです。DomoのOracle Database SSHコネクターを使用して、OracleデータベースからSSHトンネル経由でデータをロードし、カスタマイズされたレポートを編集できます。取得するデータの指定は、SQLクエリの入力で行います。

Oracle Databaseには、Data Centerから接続します。このトピックでは、Oracle Database SSHコネクターユーザーインターフェースに固有のフィールドとメニューについて説明します。DataSetの追加や更新スケジュールの設定、DataSet情報の編集に関する一般的な情報は、「 [コネクターを使用してDataSetを追加する](/ja/s/article/360042926274) 」を参照してください。

## 必要条件

Oracle Databaseに接続してDataSetを作成するには、以下が必要です。

* Oracle Databaseのログインに使用するユーザー名とパスワード
* Oracle DatabaseのSID
* Oracle Databaseのポート番号
* Oracle Databaseのホスト名またはIPアドレス
* SSHのホスト名またはURL
* SSHのユーザー名とパスワード
* SSHポート番号
* SSH秘密鍵

また、Oracle Databaseに接続する前に、接続先となるデータベースサーバーのポートで使用するIPアドレスをいくつかホワイトリストに登録する必要があります。IPアドレスの詳細なリストは、「 [コネクターのIPアドレスをホワイトリストに登録する](/ja/s/article/360043630093 "コネクターとフェデレーテッドアダプターのIPアドレスをホワイトリストに登録する") 」を参照してください。

<Note>
**注記： Domoは、ssh-keygenを使用して生成されるSSHキーをサポートしていません。SSHキーは、OpenSSLが生成するDESまたはRSAキー（PEM形式）である必要があります。**

</Note>

## Oracle Databaseに接続する

このセクションでは、Oracle Database SSHコネクターページの **［認証情報］** および **［詳細］** ペインのオプションについて説明します。このページのほかのペインの項目である **［スケジュール設定］** や **［DataSetに名前を付け、説明を入力］** に関しては、ほとんどのタイプのコネクターで共通しています。詳細については、「 [コネクターを使用してDataSetを追加する](/ja/s/article/360042926274 "コネクターを使用してDataSetを追加する") 」を参照してください。

### ［認証情報］ペイン

このペインには、データベースとSSHサーバーに接続する認証情報の入力フィールドが含まれています。下表では、各フィールドで必要とされる内容を説明しています。

| フィールド               | 説明                          |
| ------------------- | --------------------------- |
| SSH Server Hostname | SSHサーバーのホスト名を入力します。         |
| SSH Username        | SSHユーザー名を入力します。             |
| SSH Password        | SSHパスワードを入力します。             |
| SSH Port            | SSHポート番号を入力します。             |
| Database Hostname   | Oracle Databaseのホスト名を入力します。 |
| SSH Private Key     | SSH秘密鍵を入力します。               |
| Database Name       | Oracle Database名を入力します。     |
| Database Username   | Oracleユーザー名を入力します。          |
| Database Password   | Oracleパスワードを入力します。          |
| Database Port       | Oracleのポート番号を入力します。         |

有効なOracle Databaseの認証情報を一度入力しておくと、随時同じアカウントを使用して新しいOracle Database SSH DataSetを作成することができます。コネクターアカウントは、Data Centerの **［アカウント］** タブで管理できます。このタブの詳細については、「 [コネクターのユーザーアカウントを管理する](/ja/s/article/360042926054 "コネクターのユーザーアカウントを管理する") 」を参照してください。

### ［詳細］ペイン

このペインでは、SQLクエリを作成して、データベースからデータを取得します。 **［Query］** パラメーターは必須です。ほかの3つのパラメーターは、このクエリを作成する場合に役立ちます。

| メニュー                  | 説明                                                                                                                                             |
| --------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------- |
| Query                 | 必要なデータを選択するためのSQL（構造化照会言語）クエリを入力します。例：<br/>`select * from Employee`<br/>クエリの作成にヘルプが必要な場合は、                                                     |
| Database Table（オプション） | Domoに引き込むデータを含むテーブルを選択します。選択したテーブルは、 **［Query Helper］** フィールドの自動生成されるクエリに追加されます。                                                               |
| Table Columns（オプション）  | Domoに引き込むデータを含むテーブルの列を選択します。選択した列は、 **［Query Helper］** フィールドで自動生成されるクエリに追加されます。                                                                |
| Query Helper（オプション）   | クエリの作成にサポートが必要な場合は、このクエリをコピーして **［Query］** フィールドに貼り付けます。 **［Database Table］** フィールドでテーブルを、 **［Table Columns］** フィールドで列を選択すると、このクエリが自動的に生成されます。 |
