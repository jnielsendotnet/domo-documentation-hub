## はじめに

SAP ASE（Adaptive Server Enterprise）SSHは、ビジネス向けリレーショナルデータベースサーバー製品です。以前はSybase SQLサーバーと呼ばれていました。DomoのSAP ASE SSHコネクターを使用し、SQLクエリ―を入力することにより、SSHトンネル経由でSAPデータベースからデータを取得します。SAP ASE SSHの詳細については、SAP社のウェブサイト（ [https://www.sap.com/japan/products/technology-platform/sybase-ase.html](https://www.sap.com/japan/products/technology-platform/sybase-ase.html "https://www.sap.com/japan/products/technology-platform/sybase-ase.html") ）をご覧ください。

SAP ASE SSHデータベースには、Data Centerから接続します。このトピックでは、SAP ASE SSHコネクターユーザーインターフェースに固有のフィールドとメニューについて説明します。DataSetの追加、更新スケジュールの設定およびDataSet情報の編集については、「 [コネクターを使用してDataSetを追加する](/ja/s/article/360042926274 "コネクターを使用してDataSetを追加する") 」を参照してください。

## 必要条件

SAP ASE SSHデータベースに接続してDataSetを作成するには、以下の情報が必要になります。

* SAP ASEアカウントのログインに使用するユーザー名とパスワード
* データベースのホスト名
* データベースのポート番号
* データベース名
* SSHホスト名
* SSHユーザーのユーザー名とパスワード
* SSHポート番号
* SSH秘密鍵

SAPデータベースに接続する前に、接続先となるデータベースサーバーのポートで使用するIPアドレスをいくつかホワイトリストに登録する必要があります。IPアドレスの詳細なリストは、「 [コネクターのIPアドレスをホワイトリストに登録する](/ja/s/article/360043630093 "コネクターのIPアドレスをホワイトリストに登録する") 」を参照してください。

<Note>
**注記：Domoは、ssh-keygenを使用して生成されるSSHキーをサポートしていません。SSHキーは、OpenSSLが生成するDESまたはRSAキー（PEMフォーマット）である必要があります。**

</Note>

## SAP ASE SSHデータベースに接続する

このセクションでは、SAP ASE SSHコネクターページの **［認証情報］** および **［詳細］** ペインのオプションについて説明します。このページのほかのペインの項目である **［スケジュール設定］** や **［DataSetに名前を付け、説明を入力］** に関しては、ほとんどのタイプのコネクターで共通しています。詳細については、「 [コネクターを使用してDataSetを追加する](/ja/s/article/360042926274 "コネクターを使用してDataSetを追加する") 」を参照してください。

### ［認証情報］ペイン

このペインには、SAP ASE SSHアカウントに接続するための認証情報を入力するためのフィールドが含まれます。下表では、各フィールドに必要な内容を説明しています。

| フィールド           | 説明                                                                |
| --------------- | ----------------------------------------------------------------- |
| SSH Host        | SSHホスト名を入力します。                                                    |
| SSH User        | SSHユーザーのユーザー名を入力します。                                              |
| SSH Password    | SSHユーザーのパスワードを入力します。                                              |
| SSH Port        | SSHポート番号を入力します。                                                   |
| SSH Private Key | SSH秘密鍵を入力します。                                                     |
| Username        | SAP Adaptive Server Enterpriseアカウントのユーザー名を入力します。                  |
| Password        | SAP Adaptive Server Enterpriseアカウントのパスワードを入力します。                  |
| Host            | SAPデータベースのホスト名を入力します。例： `[db.company.com](http://db.company.com)` |
| Port            | データベースのポート番号を入力します。                                               |
| Database        | データベース名を入力します。                                                    |

有効なSAP ASE SSH認証情報を一度入力しておくと、随時同じアカウントを使用して新しいSAP ASE SSH DataSetを作成できます。コネクターアカウントは、Data Centerの **［アカウント］** タブで管理できます。このタブの詳細については、「 [コネクターのユーザーアカウントを管理する](/ja/s/article/360042926054 "コネクターのユーザーアカウントを管理する") 」を参照してください。

### ［詳細］ペイン

このペインでは、SQLクエリを作成して、データベースからデータを取得します。 **［Query］** パラメーターは必須です。ほかの3つのパラメーターは、このクエリを作成する場合に役立ちます。

| メニュー                  | 説明                                                                                                                                             |
| --------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------- |
| Query                 | 必要なデータを選択するためのSQL（構造化照会言語）クエリを入力します。例：<br/>`select * from Employee`                                                                            |
| Database Table（オプション） | Domoに引き込むデータを含むテーブルを選択します。選択したテーブルは、 **［Query Helper］** フィールドの自動生成されるクエリに追加されます。                                                               |
| Table Columns（オプション）  | Domoに引き込むデータを含むテーブルの列を選択します。選択した列は、 **［Query Helper］** フィールドで自動生成されるクエリに追加されます。                                                                |
| Query Helper（オプション）   | クエリの作成にサポートが必要な場合は、このクエリをコピーして **［Query］** フィールドに貼り付けます。 **［Database Table］** フィールドでテーブルを、 **［Table Columns］** フィールドで列を選択すると、このクエリが自動的に生成されます。 |
