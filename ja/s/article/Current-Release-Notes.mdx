# AI消費レート表

AI消費レートカードには、DomoのAI機能の使用のSKUと関連するクレジットレートが一覧表示されます。

この表では、DomoのAI機能の使用が全体的なクレジット消費全体にどのように影響しているかをより深く理解できます。

AI消費レート表の詳細については、 [こちら](/ja/s/article/000005326#rate-card-tab) を参照してください。

# PDP：列マスク

PDP：DataSetの列マスクを定義して、PIIやPHIなどの機密情報を非表示にすることができます。列マスクポリシーを定義して、値全体を編集したり、特定のフィールド内の特定の文字数を非表示にしたりすることができます。

列マスクは、DataSetの詳細画面にあるPDPセクションで設定できます。

列マスクとPDPの詳細については、 [こちら](/ja/s/article/360042934614) を参照してください。

# SQLタイルのウィンドウ関数

SQLタイルのウィンドウ関数を使用すると、Redshift DataFlowの使用に頼ったり、Magic ETLで複数のデータストリームを発生させたりすることなく、Magic ETL DataFlowで複雑な分析関数を使用できます。

SQLタイル内から累積合計、ランキング、移動平均、累積統計などの計算を実行できます。

# App Studio

#### App Studioエディターの更新

App Studioを使用すると、データを使用してグラフや表をいくつかのステップで自動的に生成できます。初期設定は必要ありません。

次のツールとオプションを使用して、必要なものを作成します。

* 列を選択し、計算を追加し、日付範囲を調整し、フィルターを適用します。
* 一般的な表、編集可能な表、リンクされた表、ピボット表の4種類の表を選択して作成できます。
* 棒グラフ、折れ線グラフ、円グラフ、ヒートマップ、世界マップ、散布図などの多数のグラフオプションから選択できます。

# Magic ETL

#### タイルを無効にする

Magic ETLでDataFlowのタイルを無効にする機能または「コメントアウト」する機能が利用できるようになりました。この機能を使用すると、無効にする1つまたは複数のタイルを選択し、これらのタイルをDataFlow実行中に実行しないようにすることができます。

タイルを無効にすることに加えて、選択したタイルからすべてのアップストリームまたはダウンストリームタイルをハイライト表示し、それらを設定パネルから個別に無効にすることもできます。

<Frame><img alt="" src="/images/kb/ka0Vq000000QpN3-00N5w00000Ri7BU-0EMVq000009qslV.jpg" style={{width: 795, height: 499}}/></Frame>
[このページのトップへ](#intro)

#### Fixed Input Tile

Fixed Input Tileを使用すると、Magic ETLキャンバスにデータを直接すばやく追加できます。データを手動で追加するには、ほかのソースからコピーして貼り付けるか、ExcelファイルとCSVファイルをアップロードします。また、AIを活用した生成機能を使用して、個別の列や現実的なサンプルデータを含む完全なDataSetを作成できます。すべての入力データはDataFlow定義内に保存され、同じDataFlowのダウンストリーム変換の入力として使用できます。

Fixed Input Tileでは、次のことが実現されます。

* プロトタイピングとテストの高速化
* データ入力の柔軟性の向上
* 技術に詳しくないユーザーの能力の引き上げ
* 現実的なサンプルデータ生成

[このページのトップへ](#intro)

#### JSON Expander Tile

新しいJSON Expanderを使用すると、複雑な解析を行うことなく、JSONフォーマットデータを単一の列から複数の列に展開できます。

この機能を使用すると、簡単にナビゲートできる階層的な拡張プロセスを使用して、列データに確実にアクセスできます。

# ワークシート

ワークシートを使用すると、学習の手間もなく、アプリ構造の設定も必要とせずに、未加工データをひと目で確認できるように変換できます。

ワークシートには、アクセスしやすいドラッグ＆ドロップのキャンバスが含まれており、Domo内でデータをすばやく分析できます。これはDomoのデータ環境と密接に統合されており、ワークシートをApp Studioに移動して共有する準備ができたら、シームレスなエスカレーションをサポートします。
