## はじめに

Analyzerのデータ表では、チャートに適用したすべての列のデータと、それ以外に表示したい未使用の列のデータを表示し、操作できます。データ表のデータに加えた変更は、すべてチャートのプレビューに反映されます。同様に、フィルターやソートなど、ほかのツールを使ってチャートを変更すると、表のデータが更新されます。

## データ表を表示する/非表示にする

データ表を表示するには、Analyzerウィンドウの上部にあるツールバーで **［データ表］** をクリックします（ツールバーが表示されない場合は、 <img src="/images/kb/ja/ka05w00000124rK-00N5w00000Ri7BU-0EM5w000005wKxu.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/> をクリックします）。

<Frame><img alt="Data_Table.png" src="/images/kb/ka05w00000123kI-00N5w00000Ri7BU-0EM5w000005vOyO.png"/></Frame>
Analyzerウィンドウの下部、チャートのプレビューの下にデータ表が表示されます。表示行数を増やすには、チャートのプレビューとデータ表の境界線をクリックして上にドラッグし、減らすにはクリックして下にドラッグします。チャートのプレビューやその他のオプションを表示せずにデータ表だけを表示する場合は、 <img src="/images/kb/ja/ka05w00000124rK-00N5w00000Ri7BU-0EM5w000005wKyO.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/> をクリックしてフルスクリーン表示にすることができます。

ツールバーで **［データ表］** をもう一度クリックすると、データ表を非表示にすることができます（青で強調表示されている場合）。

## データ表のコンポーネント

次のスクリーンショットは、Analyzerのデータ表の中で特に重要な部分を示しています。

<Frame><img alt="Data_Table_Options.png" src="/images/kb/ka05w00000123kI-00N5w00000Ri7BU-0EM5w000005vOyJ.png"/></Frame>
以下の表で、これらのコンポーネントの詳細を確認してください。

| コンポーネント      | 説明                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| ------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| データタイプアイコン   | 現在の列のデータタイプを示します。アイコンには、文字列（ <img src="/images/kb/ja/ka05w00000124rK-00N5w00000Ri7BU-0EM5w000005wKxn.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/> ）、数値列（ <img src="/images/kb/ja/ka05w00000124rK-00N5w00000Ri7BU-0EM5w000005wKxw.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/> ）、日時列（ <img src="/images/kb/ja/ka05w00000124rK-00N5w00000Ri7BU-0EM5w000005wKyM.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/> ）などがあります。データタイプの詳細については、「 [チャートデータについて](/s/article/360043428693) 」を参照してください。 |
| ソートアイコン      | 特定の列にある値を昇順または降順でソートします。文字列ではアルファベット順、数値列では数値順、日付列では日付順にソートされます。データ表でソートを変更すると **［ソート］** フィールドに反映され、［ソート］フィールドでデータを変更するとデータ表に反映されます。ソートの詳細については、「 [チャートのデータをソートする](/s/article/360043428733) 」を参照してください。                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| フィルター/集計アイコン | 特定の列のフィルターおよび集計オプションにアクセスできます。フィルターは列内のすべての行に適用されますが、集計は合計行にのみ適用されます。<br/>フィルターに加えた変更は **［フィルター］** フィールドに反映され、［フィルター］フィールドに加えた変更はフィルターに反映されます。フィルターの詳細については、「 [チャートにフィルターを追加する](/s/article/360042924074) 」を参照してください。<br/>集計に加えた変更は、チャートのプレビューの上にある列のドロップダウンに反映されます。データの集計の詳細については、「 [データを集計する](/s/article/360043428713) 」を参照してください。                                                                                                                                                                                                                                                                                                          |
| 列表示オプション     | 列をデータ表に追加、またはデータ表から削除できます。ここで追加した列はグレーで表示され、データ表にのみ表示されます。チャートのプレビューの上にあるフィールドエリアには表示されません。ただし、これらの列のフィルターまたはソートによって、チャート内のデータが変更される可能性があります。例えば、一部のチャートでは、日付範囲に日付を表示することはできませんが、データ表に日付列を追加して日付をフィルターすると、指定した日付範囲を反映するようにチャートのプレビューが更新されます。<br/>チャートのプレビューの上のフィールドに列名をドラッグしてチャートに追加した列はグレー表示されているため、このビューでは削除できません（ただし、フィールド内の列名の上にカーソルを置いて［x］をクリックして削除することはできます）。<br/>**［列を検索］** ダイアログで **［列を検索］** フィールドに列名を入力することで、追加または削除する列を検索できます。                                                                                                                                                                                                     |
| ギアメニュー       | データ表のデータに対して2つの表示オプションを提供します。<br/>［合計を表示］ ：チャートの2行目の「合計」行の表示/非表示を切り替えます。詳細については、上記の「合計行」を参照してください。<br/>［チャートデータを表示］/［未加工データを表示］ ：データ表でチャートデータと未加工データの表示を切り替えることができます。「未加工データ」はチャートの作成に使用される未集計データで、「チャートデータ」は適用された集計を含むデータを表示します。                                                                                                                                                                                                                                                                                                                                                                                                            |
| 使用されている列     | チャートに適用された列（チャートのプレビューの上のフィールドに列をドラッグ＆ドロップします）。使用されている文字列と日付列は青で、使用されている数値列は緑で表示されます。これらの列は、データ表から直接削除することはできませんが、チャートのプレビューの上のフィールドで列名にカーソルを置いて［x］をクリックして削除することはできます。                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| 未使用の列        | チャートに適用されていないものの、ユーザーが **［DataSetから列を表示］** リンクをクリックし、表示したい列のボックスをチェックして表示した列。これらの列は、チャートのプレビューの上のフィールドエリアには表示されません。ただし、これらの列のフィルターまたはソートによって、チャート内のデータが変更される可能性があります。例えば、一部のチャートでは、日付範囲に日付を表示することはできませんが、データ表に日付列を追加して日付をフィルターすると、指定した日付範囲を反映するようにチャートのプレビューが更新されます。                                                                                                                                                                                                                                                                                                                                                                         |
| フルスクリーンアイコン  | フルスクリーンモードでデータ表を開きます。フルスクリーンモードで <img src="/images/kb/ja/ka05w00000124rK-00N5w00000Ri7BU-0EM5w000005wKxz.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/> をクリックすると、デフォルトモードに戻すことができます。                                                                                                                                                                                                                                                                                                                                                                                                         |
