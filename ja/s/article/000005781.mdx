## はじめに

Cvent Inc.は、ミーティング、イベント、ホスピタリティーマネジメント技術を専門とする、SaaSの上場企業です。DomoのCvent Rest Writebackコネクターを使用すると、Domo DataSetのデータをCventアカウントに書き出すことができます。

Cvent REST Writebackアカウントには、Data Centerから接続できます。このトピックでは、Cvent REST Writebackコネクターのユーザーインターフェースに固有のフィールドとメニューについて説明します。DataSetの追加、更新スケジュールの設定およびDataSet情報の編集を行うには、「 [コネクターを使用してDataSetを追加する](/ja/s/article/360042926274 "コネクターを使用してDataSetを追加する") 」を参照してください。

## 必要条件

Cvent REST Writebackアカウントに接続してDataSetを作成するには、以下が必要です。

* Cventインスタンスの地域
* クライアントIDとクライアントシークレット
* DomoクライアントID
* Domoクライアントシークレット

## Cvent REST Writebackアカウントに接続する

このセクションでは、Cvent REST Writebackコネクターページの **［認証情報］** ペインおよび **［詳細］** ペインのオプションについて説明します。このページのほかのペインの項目である **［スケジュール設定］** や **［DataSetに名前を付け、説明を入力］** に関しては、ほとんどのタイプのコネクターで共通しています。詳細については、「 [コネクターを使用してDataSetを追加する](/ja/s/article/360042926274 "コネクターを使用してDataSetを追加する") 」を参照してください。

## ［認証情報］ペイン

このペインには、Cvent REST Writebackアカウントに接続する認証情報の入力フィールドが含まれています。以下の表で、各フィールドに必要な内容を説明しています。

| フィールド              | 説明                                                                                                                                             |
| ------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------- |
| Account Name       | Cventアカウントの名前を入力します。                                                                                                                           |
| Region             | Cventインスタンスの地域を選択します。                                                                                                                          |
| Client ID          | Cvent AppからクライアントIDを入力します。                                                                                                                     |
| Client Secret      | Cvent Appからクライアントシークレットを入力します。                                                                                                                 |
| DOMO Client ID     | DOMOクライアントIDを作成するには、以下の手順を実行します。<br/>まず、 デベロッパーアカウントにログインします。<br/>新しいクライアント を作成します。<br/>データとユーザーアプリケーションのスコープを選択します。<br/>［Create］をクリックします。     |
| DOMO Client Secret | DOMOクライアントシークレットを作成するには、以下の手順を実行します。<br/>まず、 デベロッパーアカウントにログインします。<br/>新しいクライアント を作成します。<br/>データとユーザーアプリケーションのスコープを選択します。<br/>［Create］をクリックします。 |

有効なCvent REST Writeback認証情報を一度入力しておくと、いつでも同じアカウントを使用して新しいCvent REST Writebackを作成できます。コネクターのアカウントはData Centerの **［アカウント］** タブで管理できます。このタブの詳細については、「 [コネクター/アダプターアカウントを管理する](/ja/s/article/360042926054 "コネクター/アダプターアカウントを管理する") 」を参照してください。

## ［詳細］ペイン

| **フィールド**             | **サブフィールド/説明**                                                                                                                                                   | **説明**                                                                                                                                                                                                                                                                                                                                                                                                                   |
| --------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Domo DataSet          | DataSet ID                                                                                                                                                       | Cventに書き込むデータを含むDomo DataSetのIDを入力します。 このIDはDataSet URL内にあります。例： [https://customer.domo.com/datasources/aaaaaaaa-bbbb-cccc-dddd-eeeeeeeeeeee/details/settings](https://customer.domo.com/datasources/aaaaaaaa-bbbb-cccc-dddd-eeeeeeeeeeee/details/settings) 。                                                                                                                                                            |
| Configuration Mapping | DataSetのIDを入力すると、 **［Use automated mapping］** と **［Select mapping manually］** のラジオボタンが表示されます。自動マッピングを使用するか（DataSetフィールドがCventフィールドと一致する場合）、手動でマッピングを定義するかを選択します。 | **Use automated mapping：** CventフィールドとDataSetフィールドが完全に一致する場合は、自動マッピングを選択できます。<br/>**Select mapping manually：** マニュアルマッピングでは、目的のCventフィールドとDataSet列を一致させます。<br/>**Upsert Choice：** ドロップダウンリストから、データをCventにUpsertする方法を選択します。［Upsert Choice］に **［Upsert based on custom fields］** を選択すると、次のフィールドが表示されます。<br/>Upsert Filter Fields<br/>Dataset field to filter on<br/>Dataset Fields：このフィールドは、手動マッピングのオプションを選択している場合に表示されます。 |
