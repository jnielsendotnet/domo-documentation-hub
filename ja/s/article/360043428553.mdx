## はじめに

Domo における *ページ* とは、カードを閲覧する画面のことです。

カードに関する詳細は、 カードについて を参照してください。

ページのカードのレイアウトをパーソナライズするには、 [カードとページを管理する](/s/article/ "カードとページを管理する") を参照してください。

## デフォルトページ

Domoには、いくつかのデフォルトページがあります。

| ページ名   | 説明                                                                                                                                                                 |
| ------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| トップページ | 所属するグループ指定のデフォルトトップページに基づいたカードが含まれています。Domo にサインインしたユーザーがこのページでどのカードを見るのかを、管理者は管理者設定で指定することができます。詳細は、 [ユーザーのデフォルトのトップページを指定する](/s/article/360043439153) を参照してください。 |
| お気に入り  | 自分がお気に入りとして指定したカードがあります。お気に入りに指定した全てのカードのステータス報告が毎日送られてきます。詳細は、 [カードをお気に入りとして追加する](/s/article/360043428373) を参照してください。                                             |
| アラート   | Alerts Center では、有効なアラートや、フォローしているアラートのあるカードを見ることができます。Alerts Center は、フォローするのにおすすめのアラートなども知らせてくれます。詳細は、 [アラートを管理する](/s/article/360043430533) を参照して下さい。            |
| 共有済    | 他のユーザーをシェアしたカードがあります。詳細は、 [カードとページへのアクセスを制御する](/s/article/360042934254) を参照してください。                                                                                 |

## Domo にページを追加する

**ページの管理** のダイアログで、Domo にページやサブページを追加することができます。このダイアログに関する詳細は、 [ページを管理する](/s/article/360043428253 "ページを管理する") を参照してください。

## ページのシェア

Domo にページやサブページを追加すると、デフォルト設定では、自分だけがそのページにアクセスできるユーザーとなります。共有コントロールを使うと、ページやカードへのアクセスを共有することができます。詳細は、 [コンテンツへのアクセスをシェアする](/s/article/360042932994 "コンテンツへのアクセスをシェアする") を参照してください。

## 所有者ページを閲覧または変更する

ページの所有者を確認することができます。所有者のユーザー写真が、ページ上部、「共有」情報の左側に表示されます。

<Frame><img alt="" src="/images/kb/ka0Vq00000011Bl-00N5w00000Ri7BU-0EM5w000005wLk1.png" style={{width: 908, height: 1097}}/></Frame>
所有者の写真をクリックすると、アバターカードが表示されます。カードには連絡先情報、そのユーザーが最も頻繁に最近閲覧したカード、 [そのユーザーが属するグループの一覧](http://wiki.domo.com/confluence/display/DHCS/Managing+users+and+groups "ユーザーとグループを管理する") のほか、このユーザーを Buzz でフォローするための **プロフィールに進む** ボタンと **フォロー** ボタンがあります。プロフィールページに関する詳細は、 [プロフィールページのレイアウト](/s/article/360042934674 "プロフィールページのレイアウト") を参照してください。

ページの所有者を変更するには、 **変更する** リンクをクリックし、メニューから新しい所有者を選択します。このリンクは、「所有者」のセキュリティ権限を持っているか、ページの所有者である場合のみ表示されます。セキュリティ権限に関する詳細は、 [セキュリティ権限リファレンス](/s/article/360043438953 "セキュリティ権限リファレンス") を参照して下さい。

## カードを追加する

ページ右上の **カードの追加** をクリックすると（または <img src="/images/kb/ja/ka0Vq00000011Bl-00N5w00000Ri7BU-0EM5w000005wLk6.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/> **> カード** を選択すると）、カードを追加するためのオプションが表示されます。詳細については、 [Domo にカードを追加する](/s/article/ "Domo にカードを追加する") を参照してください。

## ページやサブページにナビゲートする

ページは、データの「コレクション」であり、一般的にはカードとして表示されます。各ページは、画面上部のタブ行に表示されます。ページにサブページがある時は、メインページのタブ行の下の行として表示されます。ページを開くには、ページの名前をクリックします。ページ名のマウスポインタを重ねると、トップレベルのページに進まずに、そのページのサブページを見ることができます。

<Frame><img alt="badge_tabs.png" src="/images/kb/ka0Vq00000011Bl-00N5w00000Ri7BU-0EM5w000005wLk3.png"/></Frame>
画面上部のタブ行に入りきらないものは、表示されません。マウスを **もっと** に移動すると、これらを閲覧、アクセスすることができます。

<Frame><img alt="more_menu.png" src="/images/kb/ka0Vq00000011Bl-00N5w00000Ri7BU-0EM5w000005wLjy.png"/></Frame>
**ページを管理** ダイアログでは、ページの追加、ページ名の変更、並べ替え、非表示などの設定を行うことができます。ダイアログにアクセスするには、画面の右上にある <img src="/images/kb/ja/ka0Vq00000011Bl-00N5w00000Ri7BU-0EM5w000005wLk9.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/> をクリックします。詳細は、 [ページを管理する](/s/article/360043428253 "ページを管理する") を参照してください。

## ページオプションのメニュー

右上隅の <img src="/images/kb/ja/ka0Vq00000011Bl-00N5w00000Ri7BU-0EM5w000005wLk5.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/> をクリックすると、そのページのメニューアイテムが表示されます。

| オプション                | 説明                                                                                                      |
| -------------------- | ------------------------------------------------------------------------------------------------------- |
| App の詳細を表示           | ページとカードに使用されている QuickStart App の **詳細** ビューを表示します。詳細は、 [App ストア](/s/article/) を参照してください。                |
| ページをスライドショーとして表示     | ページ内のカードを一時的なスライドショーとして表示します。詳細については、 [カードの 1 回限りのスライドショーをページで閲覧する](/s/article/360043437793) を参照してください。 |
| スライドショーを保存           | ページ内のカードをスライドとしてスライドショーに追加し、保存します。詳細については、 [スライドショーを使ってコンテンツをシェアする](/s/article/360043437793) を参照してください。 |
| PowerPoint 形式でエクスポート | ページ内の全てのカードを Microsoft PowerPoint にダウンロードします。詳細は、 [KPIカードをエクスポートする](/s/article/360043437813) を参照してください。 |
| ページフィルターをオンにする       | ページ内の全てのカードのフィルタリングを行うことができます。詳細は、 ページレベルのフィルターを適用する を参照してください。                                         |
| 新しいコレクション            | このページで新しいカードコレクションを作成できます。詳細は、 [カードコレクションを作成そして管理する](/s/article/360042923314) を参照してください。                |
| サイズのオプション            | このページ内の全てのカードでサイズ変更を行うことができます。詳細は、 [ページでカードのサイズを変更する](/s/article/360042923594) を参照してください。               |
