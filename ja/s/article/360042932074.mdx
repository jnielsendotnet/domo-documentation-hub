## はじめに

Domo Excel SFTP Connectorを使用すると、SFTP（Secure File Transfer Protocol）でサーバーにアップロードしたExcelファイルにアクセスできます。

Excelファイルには、Data Centerから接続します。このトピックでは、Excel SFTP Connetorのユーザーインターフェースに固有のフィールドとメニューについて説明します。DataSetの追加や更新スケジュールの設定、DataSet情報の編集に関する一般的な情報については、「[データコネクターを使用してDataSetを追加する](/s/article/360042926274)」を参照してください。

## 必要条件

SFTPサーバーに対する認証を獲得するにはSFTPホスト名、ユーザー名、パスワードとポート番号が必要です。次のIPアドレスのホワイトリスト化も必要です。

`54.208.95.237`

`54.208.87.122`

`54.208.95.167`

`54.208.94.194`

`50.207.240.162`

`50.207.241.62`

`52.18.90.222`

`52.62.103.83`

`34.198.214.100`

## Excelファイルに接続する

このセクションでは、Excel SFTP Connetorページの**［認証情報］**および**［詳細］ペイン**のオプションについて説明します。このページのその他のペインである［**スケジュール設定］**と**［DataSetの名前と説明］**の項目は、ほとんどのタイプのコネクターで共通しています。詳細については「[データコネクターを使用してDataSetを追加する](/s/article/360042926274)」を参照してください。

### ［認証情報］ペイン

このペインは、SFTPサーバーにあってDomoに取り込むExcelファイルに接続する認証情報の入力フィールドを含んでいます。下表では、各フィールドに必要な内容を説明しています。

| フィールド | 説明                              |
| ----- | ------------------------------- |
| ユーザー名 | SFTPサーバーへのログインに使用するユーザー名を入力します。 |
| パスワード | SFTPサーバーへのログインに使用するパスワードを入力します。 |
| ホスト名  | 接続するSFTPホストの名前またはIPアドレスを入力します。  |
| ポート番号 | SFTPサーバーのポート番号を入力します。           |

有効なExcel認証情報を一度入力しておくと、随時同じアカウントを使用して新しいExcel SFTP DataSetを作成できます。コネクターアカウントは、Data Centerの**［アカウント］**タブで管理できます。このタブの詳細については、「[コネクターのユーザーアカウントを管理する](/s/article/360042926054)」を参照してください。

### ［詳細］ペイン

このペインでは、Domoに取り込むExcelファイルのオプションを選択、設定します。

| メニュー                     | 説明                                                                                      |
| ------------------------ | --------------------------------------------------------------------------------------- |
| 検索するのに使うファイル名またはファイル名の一部 | Domoに取り込むExcelファイルの名前または名前の一部を入力します。.xlsと.xlsxの両方がサポートされています。                           |
| 実行する一致の種類                | ファイル名が**［検索するのに使うファイル名またはファイル名の一部］**フィールドに入力した文字列で始まるか、またはその文字列を含むかのどちらかを選択します。         |
| パスワード保護されていますか？          | Domoに取り込むExcelファイルがパスワード保護されているかどうかを選択します。                                              |
| ファイルのパスワード               | Domoに取り込むExcelシートのパスワードを入力します。                                                          |
| シート名（オプション）              | Domoに取り込むExcelシートの名前を入力します。ここを空白のままにすると、Domoによってファイルの最初のシートが取り込まれます。                    |
| レイアウト                    | Domoに取り込むExcelファイルのレイアウトを選択します。                                                         |
| モード                      | 希望する解析モードを選択します。**［手動］**を選択すると、列ヘッダー範囲と日付範囲の入力を求められます。**［自動］**を選択すると、これらの範囲は自動的に決定されます。 |
| 列のヘッダーの範囲                | Excelファイルのヘッダーセルの範囲を入力します。例：A2:F8。                                                      |
| データの範囲                   | Excelファイルのデータセルの範囲を入力します。例：A3:F20。                                                      |
