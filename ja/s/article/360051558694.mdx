---
title: "Connector のオプションメニューアイテムについて"
---


DataSet を編集する際、 **オプション** メニューのオプションは、 <img src="/images/kb/ja/ka05w00000124vT-00N5w00000Ri7BU-0EM5w000005wLFJ.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/> をクリックして選択します。 DataSet の所有者であるか、管理者のセキュリティ権限を持っている時だけこのメニューが表示されます。 セキュリティ権限に関する詳細は、 [セキュリティ権限に関するリファレンス](/s/article/360043438953 "セキュリティ権限に関するリファレンス") を参照して下さい。

| オプション    | 説明                                                               |
| -------- | ---------------------------------------------------------------- |
| 名前と説明を編集 | DataSet のタイトルと説明を編集することができます。                                    |
| エクスポート   | DataSet を Microsoft Excel スプレッドシートまたはCSV ファイルとしてエクスポートすることができます。 |
| 今すぐ実行    | DataSet からのデータを更新し、データ Connector 設定を保存して、 **DataSet** ビューに戻ります。  |
| インポートを停止 | 長時間にわたって実行中のデータインポートを手動でキャンセルできます。                               |
| 通知       | DataSet の通知を設定します。                                               |
| 削除       | Domo からこの DataSet を削除します。                                        |
