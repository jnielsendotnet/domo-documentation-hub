---
title: "パブリックDataSetコネクター"
---

AppStoreの **［データ］** タブから、多数のパブリックDataSetをDomoインスタンスに直接かつ簡単に展開することができます。これらのDataSetは既にDomoで利用可能であるため、コネクターアカウントを作成したり、レポートを設定したりする必要はありません。あとは、「 [データコネクターを使用してDataSetを追加する](/s/article/360042926274 "データコネクターを使用してDataSetを追加する") 」で説明されているように、スケジュールを設定し（基本的なスケジュールのみ）、DataSetに名前（およびオプションの説明）を付けるだけです。

パブリックDataSetは、Data Centerだけでなく、AppStoreの **［コネクター］** タブからも利用できます。

利用可能なパブリックDataSetは以下のとおりです。

| DataSet名                               | 説明                                                       |
| -------------------------------------- | -------------------------------------------------------- |
| BLS Unemployment by State Count        | 過去2年間の各州の失業者数を月別で返します。                                   |
| BLS Unemployment by State Rate Monthly | 過去2年間の各州の失業率を月別で返します。                                    |
| NHTSA Complaints                       | NHTSA苦情データ（概要、インシデントの日付、車両のメーカーとモデル、被害件数など）をすべて返します。     |
| NHTSA Recalls                          | NHTSAリコールデータ（概要、製造会社、メーカーとモデルなどを含むリコールのリスト）をすべて返します。     |
| NHTSA Safety Ratings                   | NHTSA安全性評価データ（総合評価、故障評価、転倒評価などの安全評価リストを含む）をすべて返します。      |
| NOAA                                   | 天候に関して場所、日付、天気の種類を返します。天気と会社のデータの相関関係を確認できます。            |
| OpenAQ                                 | 環境データを返します。                                              |
| Owler Fortune 500                      | Fortune 500企業のデータ（収益、株式評価、CEO、従業員数など）を返します。              |
| Quandl Zacks Fundamentals              | 会社の公開情報（総収益、売上原価、総利益など） を返します。                           |
| Quandl NASDAQ 100                      | NASDAQトップ100のトレンド（指標値、日単位の高指標と低指標、トップ100の総市場価値など） を返します。 |
| US Census                              | 様々な産業の国勢調査データ（年度、給与額、従業員数、会社の実績など）を返します。                 |
| ZIP to Fips                            | 米国全地域のZIPコードとFIPSコードを返します。                               |
