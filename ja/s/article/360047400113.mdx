## Workbench 4 で何ができるでしょうか？

* ウィザードやその他の便利なインターフェース要素を使い、DataSet を Domo にセキュアにアップロード
* 手動またはスケジュールに基づいてデータをアップロード
* 同じスケジュールで実行される DataSet ジョブをグループ化
* ジョブファイルをインポートそしてエクスポート
* 実行する前に、ジョブをプレビューをして検証
* アカウントを追加、作成、保存そして削除
* ジョブを追加、作成、保存、削除、開く、プレビューそしてキャンセル
* ジョブを以前の保存したバージョンに戻す
* ジョブへまたはジョブに変換を追加または削除。これには以下が含まれます：

  + 新しい列を追加
  + 日付形式を変更
  + 特定の列にある機密データを暗号化
  + DataSet の列をフィルター
  + DataSet ジョブのタイムゾーンを設定
  + DataSet の列の値を置き換える
  + DataSet を他の DataSet と結合させる

* Excel、CSV、QuickBooks、JSOE、OLAP、ODBC、XML ファイルタイプをエクスポート/インポート
* ログを見る
* ログを zip ファイルとして送信
* ログをコマンドラインから実行
* Workbench にプラグインを追加して管理する
* DataSet が正常にアップロードされたときに、またはエラーが発生したときにメールで通知するよう、DataSet ジョブの通知を設定する
* 全ての大きな DataSet ジョブでマルチパートのアップロードを有効化
* 単一のジョブで最大 5000 万行までアップロード
