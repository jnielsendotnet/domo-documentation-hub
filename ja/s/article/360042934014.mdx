## はじめに

Qualtricsは、顧客や見込み客から必要なインサイトを得るためのウェブベースのアンケートプラットフォームです。

## 使用事例

* ビジネスに関する回答可能な質問：

  + アンケート回答者のエンゲージメントレベルはどれくらいか
  + アンケートの結果はどのようなものか
  + アンケートの結果を踏まえてどのように対応すべきか
* このアプリに付属しているメトリクス：

  + 上位10件のアンケート
  + アンケートの完了率
  + 質問数と回答数の比較
  + 回答された質問数
  + 応答の進捗
  + アンケートの回答率
  + 回答率
  + アンケートのステータス
  + アンケートリスト
  + 回答された質問の合計数
  + アンケートの回答数
  + 回答数
  + アンケートの傾向
  + アンケートごとの回答数

## 認証

認証で必要な情報：

* ユーザー名
* APIトークン
* アンケートのタイプ：

  + 標準的なアンケート
  + 360度評価アンケート
