## はじめに

OnBrand360は、他に類のない専有のソフトウェアです。スペシャリストは、複雑なフォーム、条件付き質問、スコアリング、検証などをサポートするフィールドで、ソフトウェアインターフェースを利用できます。OnBrand360®は、安全でわかりやすく拡張性の高い機能を通じて、クライアントが一貫性と信頼性を保ちながらブランドの信念を貫けるよう支援します。DomoのOnBrand360コネクターを使用すると、様々な評価データを取得できます。

OnBrand360アカウントには、Data Centerから接続します。このトピックでは、OnBrand360コネクターユーザーインターフェイスに固有のフィールドとメニューについて説明します。DataSetの追加、更新スケジュールの設定、DataSet情報の編集に関する一般的な情報については、「 [コネクターを使用してDataSetを追加する](/ja/s/article/360042926274) 」を参照してください。

## 必要条件

OnBrand360アカウントに接続してDataSetを作成するには、以下の情報が必要です。

* OnBrand360 APIのクライアントIDとクライアントシークレット
* OnBrand360のユーザー名とパスワード
* OnBrand360 APIのURL

## OnBrand360アカウントに接続する

このセクションでは、 OnBrand360 コネクターページの **［認証情報］** ペインと **［詳細］** ペインのオプションについて説明します。このページのほかのペインの項目にある **［スケジュール設定］** や **［DataSetの名前を付け、説明を入力］** に関しては、ほとんどのコネクタータイプで共通です。詳細については「 [コネクターを使用してDataSetを追加する](/ja/s/article/360042926274 "コネクターを使用してDataSetを追加する") 」を参照してください。

### ［認証情報］ペイン

このペインには、 OnBrand360 アカウントへの接続に使用する認証情報の入力フィールドがあります。以下の表に、各フィールドに必要な情報を示します。

| フィールド           | 説明                                              |
| --------------- | ----------------------------------------------- |
| ［Client ID］     | OnBrand360アカウントに関連付けられているAPIクライアントIDを入力します。     |
| ［Client Secret］ | OnBrand360アカウントに関連付けられているAPIクライアントシークレットを入力します。 |
| ［Username］      | OnBrand360アカウントに関連付けられているユーザー名を入力します。           |
| ［Password］      | OnBrand360アカウントに関連付けられているパスワードを入力します。           |
| ［API URL］       | OnBrand360アカウントに関連付けられているAPI URLを入力します。         |

有効な OnBrand360 認証情報を一度入力しておくと、いつでも同じアカウントを使用して新しい OnBrand360 DataSetを作成できます。コネクターアカウントは、Data Centerの **［アカウント］** タブで管理できます。このタブの詳細は、「 [コネクターのユーザーアカウントを管理する](/ja/s/article/360042926054 "コネクターのユーザーアカウントを管理する") 」を参照してください。

### ［詳細］ペイン

このペインには、メインとなる **［Reports］** メニューのほか、選択したレポートタイプに応じて表示/非表示になる様々なメニューが含まれています。

| メニュー     | 説明                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| -------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| ［Report］ | 実行するOnBrand360レポートを選択します。以下のレポートを利用できます。<br/><table border="1" cellpadding="1" cellspacing="1"><tbody><tr><td colspan="1" rowspan="1"> ［Assessments］ </td><td colspan="1" rowspan="1"> 評価データを返します。 </td></tr><tr><td colspan="1" rowspan="1"> ［Line Item Answers］ </td><td colspan="1" rowspan="1"> 行項目の回答データを返します。 </td></tr><tr><td colspan="1" rowspan="1"> ［Observations］ </td><td colspan="1" rowspan="1"> 観測データを返します。 </td></tr></tbody></table> |

### その他のペイン

スケジュール設定、再試行、更新に関するオプションなど、コネクターインターフェースのその他のペインに表示されるオプションについては、「 [コネクターを使用してDataSetを追加する](/ja/s/article/360042926274) 」を参照してください。
