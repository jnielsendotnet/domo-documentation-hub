## はじめに

製品チームはJIRAを利用して準備を整え、すばやく効率よく仕事ができるようにします。JIRA Dashboardを使用すると、現在のプロジェクト、未解決の問題と新しい問題、高い実績を上げているチームメンバーのサマリー情報を確認できます。

## 要件と注意点

JIRAにログインするにはユーザー認証情報と利用しているURLが必要です。

デフォルトの日付範囲：直近の3ヶ月に更新された問題がアップロードられるように設定済み

## 利用状況

* エグゼクティブパルス：注目すべき5枚のカード

  + 未解決の問題
  + 問題解決担当者
  + 注目のプロジェクト
  + 新たな問題のトレンド
  + 新たな課題
* 問題：目標を達成するために今日すべきことは何か

  + 新たな問題のトレンド
  + ステータストレンド
  + 問題の鍵
  + 問題のステータス
* プロジェクト：プロジェクトのステータスと、高い業績を上げているチームメンバーを確認する

  + プロジェクトの問題のステータス
  + 進行中の作業
  + 解決した問題数が多い上位の担当者
  + 報告した問題数が多い上位の担当者
  + 高い業績を上げている上位の担当者

## 認証

| 認証情報     | 説明                                                                            |
| -------- | ----------------------------------------------------------------------------- |
| ユーザー名    | これは、JIRAにログインするためのユーザー名です                                                     |
| パスワード    | これは、JIRAにログインするためのユーザー名に関連付けられているパスワードです                                      |
| サーバーのURL | これは、JIRAインスタンスのURLです（例： [mycompany.onjira.com](http://mycompany.onjira.com) ） |
