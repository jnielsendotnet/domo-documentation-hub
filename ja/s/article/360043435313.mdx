## はじめに

AP通信（AP）は、ニューヨークに本社を置くアメリカの多国籍非営利通信社です。DomoのAPコネクターを使って、最新のAPフィードコンテンツを使用したレポートを作成することができます。AP通信APIの詳細については、 [https://developer.ap.org/api-console](https://developer.ap.org/api-console "https://developer.ap.org/api-console") を参照してください。

AP通信Connectorは「クラウドアプリ」コネクターであるため、クラウドに保管されたデータの取得が可能です。Data Centerでウィンドウ上部のツールバーで **［クラウドApp］** をクリックすると、これを含む他のクラウドAppコネクターのページにアクセスできます。

AP通信 DataSetの作成は、 **Data Center** で行います。このトピックでは、 AP 通信Connectorユーザーインターフェイスに固有のフィールドとメニューについて説明します。DataSetの追加、更新スケジュールの設定、DataSet情報の編集に関する一般的な情報は「 [データコネクターを使用してDataSetを追加する](/ja/s/article/360042926274 "データコネクターを使用してDataSetを追加する") 」で説明しています。

| **主な使用事例**                    | このコネクターは現在のニュースの見出しをDomoのカードにプルする際に役立ちます。                                                                                       |
| ----------------------------- | ------------------------------------------------------------------------------------------------------------------------------- |
| **主なメトリクス**                   | このコネクターは、カテゴリ別にニューストピックの定性的なリストを提供します。そのため、プルできるメトリクスはあまりありません。ただし、ユーザーによってはヘッドラインをカテゴリ別に数えたり、特定のトピックのメンションの結果を必要としたりする場合があります。 |
| **社内における主な権限**                | エグゼクティブ<br/>ディレクタ<br/>マネージャー<br/>アナリスト                                                                                          |
| **平均導入時間**                    | 最大数分                                                                                                                            |
| **使いやすさ（1～10段階。1が「最も使いやすい」）** | 1                                                                                                                               |

## 必要条件

なし。（AP通信のデータは公開されています）

## AP通信DataSet を作成する

このセクションでは、DomoのAP通信ConnectorページでAP通信DataSet 作成する方法を説明します。このページのほかのペインの項目である **スケジュール設定** や **DataSetの名前と説明** に関しては、ほとんどのコネクタータイプで共通しています。詳細については [データコネクターを使用してDataSetを追加する](/ja/s/article/360042926274 "データコネクターを使用してDataSetを追加する") を参照してください。

### ［詳細］ ペイン

このペインには、AP 通信レポートのタイプを選択する単一のメニューがあります。

下表に、 Domo AP 通信Connectorを使用して作成できる所定のレポート全てを記載しています。

| **メニュー**   | **説明**              |
| ---------- | ------------------- |
| ビジネス       | ビジネスニュースを返します。      |
| エンターテインメント | エンターテイメントニュースを返します。 |
| 健康         | 健康関連ニュースを返します。      |
| 政治         | 政治関連ニュースを返します。      |
| 科学         | 科学ニュースを返します。        |
| スポーツ       | スポーツニュースを返します。      |
| 変わったニュース   | 一風変わったニュースを返します。    |
| テクノロジー     | テクノロジーニュースを返します。    |
| トップ見出し     | トップの見出しを返します。       |
| 米国内        | 米国のニュースを返します。       |
| 国際         | 国際ニュースを返します。        |

### その他のペイン

スケジュール設定、再試行、オプションの更新など、コネクターインターフェースのその他のペインに表示されるオプションについては、「 [データコネクターを使用してDataSetを追加する](/ja/s/article/360042926274) 」を参照してください。
