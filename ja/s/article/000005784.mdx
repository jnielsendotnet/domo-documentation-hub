# データの鮮度チェックにおけるスケジュールの詳細設定

スケジュールの詳細設定では、いつデータ更新のクエリを実行するかをより細かく制御できるため、必要な時にのみチェックを行うことができます。これにより、データの更新全般にかかるコストを削減できます。

データの鮮度の詳細については、「 [Cloud Amplifierの概要](/s/article/4412849158167) 」を参照してください。

# ワークフローデータクエリとAIアシスト

新しい「SQLにテキストを生成」AI機能を使用すると、ワークフロー内のDataSetクエリタスクを使用してクエリをさらに簡単に設定できるようになります。プレーンテキストを使用してクエリを入力すると、AIによってSQLステートメントが生成され、それをワークフローで使用できます。

ワークフロー内でのDataSetクエリタスクの詳細については、 [こちら](/ja/s/article/000005331#add_query_table) を参照してください。

# DomoStats：グループメンバーレポート

<Frame><img alt="rtaImage.jpeg" src="/images/kb/ka0Vq0000002hX7-00N5w00000Ri7BU-0EMVq000001VwNJ.jpg"/></Frame>
新しいDomoStats：グループメンバーレポートでは、グループとそのメンバーのリストを確認できます。

このレポートを使用すると、Domo管理者はグループメンバーレポートをユーザーレポート、カードの権限レポート、DataSetアクセスレポートなどのほかのレポートと組み合わせることで、グループメンバーシップを追跡し、詳細なインサイトを収集できます。

新しいDomoStatsレポート「グループ」がベータ版でリリースされます。 [下記](#domo_stats_groups_report) を参照してください。

DomoStatsの詳細については、 [こちら](/s/article/360043433813) を参照してください。

# アクティビティログの更新

アクティビティログはReactに変更されました。この更新では、以下の点が改善されています。

* イベントタイプとイベントを別のドロップダウンフィルターに分けた、より直感的なイベントフィルター。
* 日付とユーザーのフィルターが、ドロップダウンフィルター要素として独立しているのではなく、列ヘッダーに組み込まれるようになりました。
* よりコンパクトなレイアウトになったため、スペースを有効に活用できます。

「 [アクティビティログ](/s/article/360042934574) 」を参照してください。

# AI予測ユニバーサルモデル

#### カードからのAI予測ユニバーサルモデル

App StudioのAIによるカード予測機能を使用すると、 [Domo.AI](http://Domo.AI) を使用してチャートカードに高度な予測機能を設定できます。この機能は、ユニバーサルDomo予測モデルを使用し、カードデータにもとづいてインテリジェントな予測チャートを返します。

また、ページにフィルターを適用したり、変数を調整したり、チャートの情報にもとづいて予測を生成したりすることもできます。

App Studioカードで予測を生成する方法については、 [こちら](/ja/s/article/000005567#forecast_in_app) を参照してください。

<Frame><img alt="AI Card Forecasting (2).png" src="/images/kb/ka0Vq00000072GP-00N5w00000Ri7BU-0EMVq000003ULi1.jpg" style={{width: 649, height: 295}}/></Frame>

# DomoStats：グループレポート

<Frame><img alt="rtaImage.jpeg" src="/images/kb/ka0Vq0000002hX7-00N5w00000Ri7BU-0EMVq000001VwNJ.jpg"/></Frame>
DomoStatsグループレポートには、Domoグループおよび関連する属性が表示されます。

このレポートには、以下のような重要な情報が含まれます。

* グループのタイプ
* 作成日
* 作成者ID
