## はじめに

Vimeoは、高品質な動画を共有するためのウェブサイトです。Vimeo Dashboardを使用して、再生回数の多い動画や、「いいね」やコメントの数が多い動画などをトラッキングし、ターゲットとなるオーディエンスが貴社のVimeoコンテンツにどのように反応しているかを把握することができます。

## 認証

認証を行うには、Vimeoのユーザー名（メールアドレス）とパスワードが必要です。

## ダッシュボードレポート

レポートの説明については、「 Vimeoコネクター 」を参照してください。

## レポートの設定/レポートの詳細

選択可能なレポートは、以下のように設定する必要があります。

| レポート         | スケジュール設定（更新設定の変更） | パラメーター                     | 選択                                                                                                                             |
| ------------ | ----------------- | -------------------------- | ------------------------------------------------------------------------------------------------------------------------------ |
| My Followers |                   | Show Basic Data Only (Y/N) | ［Yes］を選択すると、結果セット内の項目ごとに1件のレコードが返されます。各項目の基本データだけが返されます。<br/>［No］を選択すると、基本データ内の「リスト」を返す列が展開されます。これにより、結果セット内の項目ごとに複数の行が設定されます。 |
| My Videos    |                   | Show Basic Data Only (Y/N) | ［Yes］を選択すると、結果セット内の項目ごとに1件のレコードが返されます。各項目の基本データだけが返されます。<br/>［No］を選択すると、基本データ内の「リスト」を返す列が展開されます。これにより、結果セット内の項目ごとに複数の行が設定されます。 |
