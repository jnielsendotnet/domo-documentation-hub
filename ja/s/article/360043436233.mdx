## はじめに

Oracle Databaseは、Oracle Corporationにより生産・販売されたオブジェクトリレーショナルデータベース管理システムです。取得するデータの指定は、SQLクエリの入力で行います。

Oracle Databaseには、Data Centerから接続します。このトピックでは、Oracle Databaseコネクターユーザーインターフェースに固有のフィールドとメニューについて説明します。DataSetの追加や更新スケジュールの設定、DataSet情報の編集に関する一般的な情報については、「 [データコネクターを使用してDataSetを追加する](/ja/s/article/360042926274) 」を参照してください。

## 必要条件

Oracle Databaseに接続してDataSetを作成するには、以下が必要です。

* Oracleのユーザー名とパスワード。
* Oracleサーバーのホスト名またはIPアドレス。
* Oracleのポート番号。
* Oracle SID。

Oracle Databaseに接続する前に、接続先となるデータベースサーバーのポートで使用するIPアドレスをいくつかホワイトリストに登録する必要があります。IPアドレスの詳細なリストは、「 [コネクターのIPアドレスをホワイトリスト化する](/ja/s/article/360043630093 "コネクターのIPアドレスをホワイトリスト化する") 」を参照してください。

## Oracleデータベースに接続する

このセクションでは、［Oracle Databaseコネクター］ページの **［認証情報］** および **［詳細］** ペインのオプションについて説明します。このページのほかのペインの項目にある **［スケジュール設定］** や **［DataSetの名前と説明］** に関しては、コネクタータイプのほとんどで共通です。詳細につきましては「 [データコネクターを使用してDataSetを追加する](/ja/s/article/360042926274 "データコネクターを使用してDataSetを追加する") 」を参照してください。

### ［認証情報］ペイン

このペインには、Oracle Databaseアカウントに接続するための認証情報を入力するフィールドがあります。下表では、各フィールドで必要な内容を説明しています：

| フィールド | 説明                              |
| ----- | ------------------------------- |
| ユーザー名 | Oracleアカウントのユーザー名を入力します。        |
| パスワード | Oracleアカウントのパスワードを入力します。        |
| ホスト   | Oracleサーバーのホスト名またはIPアドレスを入力します。 |
| ポート   | Oracleのポート番号を入力します。             |
| SID   | Oracle SIDを入力します。               |

有効なOracle Databaseの認証情報を一度入力しておくと、随時同じアカウントを使用して新しいOracle Database DataSetを作成することができます。コネクターアカウントは、Data Centerの **［アカウント］** タブで管理できます。このタブの詳細については、「 [コネクターのユーザーアカウントを管理する](/ja/s/article/360042926054 "コネクターのユーザーアカウントを管理する") 」を参照してください。

### ［詳細］ペイン

このペインでは、SQLクエリを作成して、データベースからデータを取得します。 **［クエリ］** パラメーターは必須です。他の3つのパラメーターは、このクエリを作成する場合に役立ちます。

| メニュー           | 説明                                                                                                             |
| -------------- | -------------------------------------------------------------------------------------------------------------- |
| クエリ            | 必要なデータを選択するための構造化照会言語（SQL）クエリを入力します。例：<br/>`select * from Employee`<br/>クエリの作成にヘルプが必要な場合は、                     |
| データベース表（オプション） | Domoに引き込むデータを含む表を選択します。選択した表は、 **［クエリヘルパー］** フィールドの自動生成されるクエリに追加されます。                                          |
| 表の列（オプション）     | Domoに引き込むデータを含む表の列を選択します。選択した列は、 **［クエリヘルパー］** フィールドで自動生成されるクエリに追加されます。                                        |
| クエリヘルパー（オプション） | クエリの作成にサポートが必要な場合は、このクエリをコピーして **［クエリ］** フィールドに貼り付けます。 **［データベース表］** で表を、 **［表の列］** で列を選択すると、このクエリが自動的に生成されます。 |

### その他のペイン

スケジュール設定、再試行、オプションの更新など、コネクターインターフェースのその他のペインに表示されるオプションについては、「 [データコネクターを使用してDataSetを追加する](/ja/s/article/360042926274) 」を参照してください。
