## はじめに

Quandlは、数値データの検索エンジンです。 複数のソースのデータをインデックスし、様々なフォーマットでデータを検索しダウンロードすることができます。 DomoのQuandl Connectorを使用すると、一般公開用のQuandl データセットを取得することができます。 Quandl API に関する詳細は、ウェブサイト ([https://www.quandl/com/help/api](https://www.quandl/com/help/api "https://www.quandl/com/help/api") ) を参照してください。

Quandl Connector は「クラウド App」です。つまり、クラウドに保管されたデータを取得します。 データセンターでは、ウィンドウ上部のツールバーでクラウド App をクリックすることで、これを含む他のクラウド A **pp C** onnector のページにアクセスできます。

Quandl のデータセットは、 データセンター からダウンロードすることができます。 このトピックでは、Quandl Connectorユーザーインターフェースに固有のフィールドとメニューについて説明します。 DataSet の追加、更新スケジュールの設定及び DataSet 情報の編集に関する一般的な情報は、 [追加する](/ja/s/article/360042926274 "データ Connector を使用して DataSet を追加する") で説明しています。

## ベストプラクティス

この Connector は「置換」に設定する必要があります。「追加」を使用するとデータが複製されてしまいます。

## 必要条件

なし。 DomoのQuandl Connectorは一般公開用データセットの取得に使用されるため、認証情報の入力は必要ありません。
