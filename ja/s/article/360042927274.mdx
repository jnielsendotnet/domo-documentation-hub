## はじめに

Silverpopは、メールマーケティングとリードマネージメントを提供するクラウドベースのマーケティングオートメーションプロバイダーです。Sillverpop APIの詳細については、Sillverpopのページ（ [https://www.programmableweb.com/api/silverpop](https://www.programmableweb.com/api/silverpop "https://www.programmableweb.com/api/silverpop") ）を参照してください。

Silverpopアカウントには、Data Centerから接続します。このトピックでは、Silverpopコネクターユーザーインターフェイスに固有のフィールドとメニューについて説明します。DataSetの追加や更新スケジュールの設定、DataSet情報の編集に関する一般的な情報は、「 [コネクターを使用してDataSetを追加する](/ja/s/article/360042926274 "コネクターを使用してDataSetを追加する") 」を参照してください。

## 必要条件

Silverpopアカウントに接続してDataSetを作成するには、以下が必要です。

* Silverpop Engageポッド番号（アプリケーションを作成したEngageインスタンスの番号）
* SilverpopクライアントID
* Silverpopクライアントシークレット
* Silverpopリフレッシュトークン（Silverpopからメールで受信したもの）
* FTPユーザー名とパスワード（Silverpopファイルサーバーからレポートをダウンロードするためにレポートの「エクスポート」を実行する場合にのみ必要）

## Silverpopアカウントに接続する

このセクションでは、Silverpopコネクターページの **［認証情報］** および **［詳細］** ペインのオプションについて説明します。このページのほかのペインの項目にある **［スケジュール設定］** や **［DataSetに名前を付け、説明を入力］** に関しては、ほとんどのコネクタータイプで共通です。詳細については「 [コネクターを使用してDataSetを追加する](/ja/s/article/360042926274 "コネクターを使用してDataSetを追加する") 」を参照してください。

### ［認証情報］ペイン

このペインには、Silverpopアカウントに接続するための認証情報の入力フィールドがあります。以下の表に、各フィールドに必要な情報を示します。

| フィールド                          | 説明                                                                                         |
| ------------------------------ | ------------------------------------------------------------------------------------------ |
| Silverpop Engage Pod Number    | Silverpop Engageポッド番号（アプリケーションを作成したEngageインスタンスの番号）を入力します。                                 |
| Client ID                      | SilverpopクライアントIDを入力します。                                                                   |
| Client Secret                  | Silverpopクライアントシークレットを入力します。                                                               |
| Refresh Token                  | Silverpopリフレッシュトークン（Silverpopからメールで受信）を入力します。                                              |
| Unique Account Name (optional) | アカウントの一意の名前を入力します。                                                                         |
| FTP Username                   | FTPサーバーへの接続に使用するユーザー名を入力します（Silverpopファイルサーバーからレポートをダウンロードするためにレポートの「エクスポート」を実行する場合にのみ必要）。 |
| FTP Password                   | FTPサーバーへの接続に使用するパスワードを入力します（Silverpopファイルサーバーからレポートをダウンロードするためにレポートの「エクスポート」を実行する場合にのみ必要）。 |

有効なSilverpop認証情報を一度入力しておくと、随時同じアカウントを使用して新しいSilverpop DataSetを作成することができます。コネクターアカウントは、Data Centerの **［アカウント］** タブで管理できます。このタブの詳細は、「 [コネクターのユーザーアカウントを管理する](/ja/s/article/360042926054 "コネクターのユーザーアカウントを管理する") 」を参照してください。
