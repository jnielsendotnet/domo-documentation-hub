## はじめに

カンマ区切り（CSV）ファイルは、表形式データをプレーンテキスト形式で保存します。Domoが提供するコネクターによって、SFTP（Secure File Transfer Protocol）サーバーにアップロードされたCSVファイルにアクセスできます。

- Domo CSV SFTPプルコネクター（旧称は「CSV-SFTP」コネクター）を使用すると、専用のSFTPサーバーにアップロードしたCSVファイルにアクセスできます。
- Domo CSV SFTP Advanced Securityコネクターを使うと、自分のSFTPサーバーにアップロードしたディレクトリまたはZIPファイルからCSVファイルにアクセスすることができます。
- Domo CSV SFTPプッシュコネクターを使用すると、Domo自体のSFTPにアップロード済みのCSVファイルを取得し、処理できます。このコネクターを使うと、Domoにより認証情報が発行され、データをDomoに送信できるようになります。

この記事は**CSV SFTP Advanced Security**コネクターについて説明しています。CSV-SFTPプッシュコネクターの詳細については、「[CSV SFTPプッシュコネクター](/s/article/360042931914)」を参照してください。

CSV SFTP Advanced Securityコネクターは「ファイル」コネクターで、ファイルを取得してDomoに出力することを意味します。Data Centerでこのコネクターやほかのファイルコネクターのページにアクセスするには、ウィンドウ上部のツールバーで****［ファイル］****をクリックします。

CSVファイルには、Data Centerから接続します。このトピックでは、CSV SFTP Advanced Securityコネクターのユーザーインターフェースに固有のフィールドとメニューについて説明します。DataSetの追加や更新スケジュールの設定、DataSet情報の編集に関する一般的な情報は、「[コネクターを使用してDataSetを追加する](/s/article/360042926274)」を参照してください。

## 必要条件

SFTPサーバーで認証を行うには以下が必要です。

- SFTPホスト名
- SFTPポート番号
- SFTPサーバーのユーザー名
- パブリックキー

### **パブリックキーを取得するには、次の手順に従います。**

- ホスト名、ポート番号、SFTPサーバーのユーザー名を入力し、**［Generate Key］**をクリックします。
- パブリックキーをSFTPサーバーにコピーします。

また、サーバー上の「[コネクターとフェデレーテッドアダプターのIPアドレスをホワイトリストに登録する](/s/article/360043630093)」の記事にしたがって、IPアドレスをホワイトリストに登録する必要があります。

## CSVファイルに接続する

このセクションでは、CSV SFTP Advancedプルコネクターの****［認証情報］****および****［詳細］****ペインのオプションについて説明します。このページのほかのペインの項目である****［スケジュール設定］****や****［DataSetに名前を付け、説明を入力］****に関しては、ほとんどのタイプのコネクターで共通しています。詳細については「[コネクターを使用してDataSetを追加する](/s/article/360042926274)」を参照してください。

### ［認証情報］ペイン

このペインは、SFTPサーバーに接続する認証情報の入力フィールドを含んでいます。下表は、各フィールドに必要な内容を説明しています。

| フィールド    | 説明                                                              |
| -------- | --------------------------------------------------------------- |
| Host     | 接続するSFTPホスト名を入力します。これは必ずsftp://から始まる*必要があります*。例：sftp://hostname |
| Port     | SFTPサーバーのポート番号を入力します。                                           |
| Username | SFTPサーバーへのログインに使用するユーザー名を入力します。                                 |

ホスト、ポート、ユーザー名を入力したら**［Generate Key］**ボタンをクリックします。SFTPサーバーに**パブリックキー**をコピーします。

CSVファイルをアップロードするに当たり、SFTPサーバーがない場合には、Domoインスタンスからのダウンロードに使用できる[Workbenchソリューション](/s/article/360043437233)を参照してください。

### ［詳細］ペイン

このペインでは、アクセスしたSFTPサーバーから取得するCSVレポートを選択します。

| メニュー                                                         | 説明                                                     |
| ------------------------------------------------------------ | ------------------------------------------------------ |
| Directory                                                    | アクセスするファイルを含むディレクトリを選択します。                             |
| File Type                                                    | ディレクトリから選択したファイルのファイルタイプを指定します。                        |
| CSV                                                          | CSVファイルにアクセスする場合、CSVを選択します。                            |
| ZIP                                                          | ZIPファイルにアクセスする場合、ZIPを選択します。                            |
| How would you like to choose your CSV file?                  | ファイル名をどのように選択するかを選択します。                                |
| Select file                                                  | ドロップダウンから取り込むファイル名を選択します。                              |
| Enter Directory                                              | ディレクトリパスを入力します。                                        |
| Enter file name                                              | ファイル名を手入力します。                                          |
| Select CSV file name                                         | ドロップダウンからCSVファイルを選択します。                                |
| Enter CSV file name                                          | CSVファイル名を手入力します。                                       |
| How would you like to choose your zip file?                  | ZIPファイルをどのように選択するかを選択します。                              |
| How would you like to choose your directory?                 | ディレクトリをどのように選択するかを選択します。                               |
| Select zip file                                              | 解凍したフォルダから解析するCSVファイルを選択します。                           |
| Enter zip file name                                          | ZIPアーカイブ内から解析するファイル名を入力します。                            |
| How would you like to choose your subfile from the zip file? | ZIPファイルでサブファイルをどのように選択するかを選択します。                       |
| Select subfile from zip file                                 | 解凍したフォルダから解析するCSVファイルを選択します。                           |
| Enter the subfile name from zip file                         | ZIPアーカイブ内から解析するファイル名を入力します。                            |
| Delimiting character                                         | ファイルで使用する区切り文字を選択します。区切り文字がリストにない場合は、［Other］を選択します。    |
| Specify your delimiter                                       | CSVテキストを区切る文字を入力します。                                   |
| Quote Character                                              | CSVファイルを解析するのに使用する引用文字を選択します。CSVの標準のデフォルトの引用符は二重引用符です。 |
| Custom Quote Character                                       | 希望するCSV引用文字を入力します。                                     |
| Escape Character                                             | CSVファイルを解析するために使用するエスケープ文字を選択します。                      |
| Custom Escape Character                                      | 希望するCSVエスケープ文字を入力します。                                  |
