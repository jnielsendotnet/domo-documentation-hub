### はじめに

Azure SQLユーザーは、Cloud Amplifierのネイティブ統合を通じて、自分のデータをDomoに接続できます。まず接続パラメーターを指定し、Domo DataSetとして表示するテーブルを選択します。

### Azure SQL統合を作成する

この記事の手順に従って、読み取り専用のAzure SQL–Domo統合を設定します。

1. Domo Data Centerで<img alt="data warehouse.jpg" src="/images/kb/ja/0EMVq000000uR3y.jpg" style={{width: 24, height: 24, display: 'inline', verticalAlign: 'start', margin: '0'}}/>

   **［Data Warehouse］**に移動します。

2. <img alt="add.jpg" src="/images/kb/ja/0EMVq000000uSeO.jpg" style={{width: 23, height: 23, display: 'inline', verticalAlign: 'start', margin: '0'}}/>

   **［新しいクラウドアカウントを追加］**を選択します。  
     

   <Frame>![add new cloud account.jpg](/images/kb/ja/0EMVq000000uRaE.jpg)</Frame>

     
     
   **［既存のクラウドウェアハウスを拡大する］**モーダルが表示されます。
3. **［ネイティブ統合］**セクションの**［詳細を見る］**を選択して、ほかのオプションを表示します。  
     

   <Frame>![see more.jpg](/images/kb/ja/0EMVq000000uTIf.jpg)</Frame>
4. **［Azure SQL］**を選択します。  
     

   <Frame>![choose azure sql.jpg](/images/kb/ja/0EMVq000001JtzW.jpg)</Frame>
5. **［+新しい統合を追加］**を選択します。  
     

   <Frame>![Screenshot 2024-06-12 at 3.43.27 PM.png](/images/kb/ja/0EMVq000001K1Yr.jpg)</Frame>

      

   <Frame>![add new azure integration.jpg](/images/kb/ja/0EMVq000001JrL0.jpg)</Frame>
6. Azure SQLデータベースに接続するには、表示されるフィールドに以下の接続パラメーターを入力します。
   - **［統合名］ —** 将来統合を更新する必要が出てきた場合に、統合のリストで検索する名前です。
   - **［統合の説明（オプション）］ —** 統合に関する追加情報。
   - **［Azure SQL Databaseの接続URL］ —** 接続先のAzure SQLインスタンスのホストとポート。例：`<myhost.com:3306>`。
   - **［Azure SQL Databaseのデータベース名］ —** 接続先のテーブルを含むデータベースの名前。
   - **［Azure SQL Databaseユーザー名］ —** 接続先のデータにアクセスできるAzure SQLインスタンスのユーザー名。
   - **［Azure SQL Databaseパスワード］ —** Azure SQLユーザーのパスワード。  
       

     <Frame>![azure sql.jpg](/images/kb/ja/0EMVq000001JwKg.jpg)</Frame>
7. フォームの下部のキャッシュオプションを設定し、**［次へ］**を選択します。Cloud Amplifierのキャッシュのルールについては、[こちら](/ja/s/article/4412849158167#caching_rules)を参照してください。  
     

   <Frame>![Screenshot 2024-05-07 at 3.24.52 PM.png](/images/kb/ja/0EMVq000000uWur.jpg)</Frame>

     
     
   成功メッセージが表示されます。
8. **［Choose Tables to Connect］**を選択して、テーブルピッカーを開きます。  
     

   <Frame>![choose tables.jpg](/images/kb/ja/0EMVq000001K20H.jpg)</Frame>
9. テーブルピッカーを使用して、スキーマとテーブルを選択します。準備ができたら、**［DataSetを作成］**を選択します。  
     

   <Frame>![table picker.jpg](/images/kb/ja/0EMVq000001L0hB.jpg)</Frame>
10. （オプション）確認モーダルでは、DataSetをほかのDomoユーザーとシェアできます。追加するユーザーを選択し、ドロップダウンでアクセスレベルを選択した後、**［シェア］**を選択します。  
    または、**［今はやめておく］**を選択し、アクセス権をシェアせずにモーダルを閉じます。  
      

    <Frame>![share modal.jpg](/images/kb/ja/0EMVq000000uXZB.jpg)</Frame>

DomoでAzure SQL DataSetとやり取りできるようになりました。

<Frame>![interact with azure sql .jpg](/images/kb/ja/0EMVq000001K2I1.jpg)</Frame>
