## はじめに

Analyzerでは、可視化カード用にチャートや表を作成することができます。また、グラフタイプ、フィルター、ソート、まとめ数字、日付単位、高度なグラフの設定など、チャートの視覚的な特性を指定することもできます。Beast Modeを使って、

チャートの列を変換するための計算を作成することもできます。

Analyzerのユーザーインターフェースコンポーネントについては、「 [Analyzerのレイアウト](/s/article/360042924034 "Card Builderのレイアウト") 」を参照してください。

## チャートを作成する

**チャートまたは表の視覚的特性を作成および指定するには：**

1. Analyzerで目的の特性を指定します。
     
   以下の表を使って、目的のトピックを見つけます。

   | トピック           | リンク                                                 |
| -------------- | --------------------------------------------------- |
| 名前             | [可視化カードの名前や説明を編集する](/s/article/360042924174)        |
| 説明             | [可視化カードの名前や説明を編集する](/s/article/360042924174)        |
| DataSet列       | [DataSetの列をチャートに適用する](/s/article/360043428713)      |
| 表のラベル          | [チャートの表ビューの列ヘッダーラベルを編集する](/s/article/360042924194)  |
| 値のフォーマット       | [チャートの値をフォーマットする](/s/article/360042924134)          |
| 目標マーカー         | [チャートに目標の線を追加する](/s/article/360043428773)           |
| チャートタイプ        | 利用可能なチャートタイプ                                        |
| フィルター          | [チャートにフィルターを追加する](/s/article/360042924074)          |
| ソート基準          | [チャートのデータをソートする](/s/article/360043428733)           |
| まとめ数字          | [チャートのまとめ数字を設定する](/s/article/360043428753)          |

2. **［カードを保存］** をクリックします。
3. （オプション）ダイアログで、変更についてコメントを入力することができます。
     
   カードの［詳細］ビューにある **［履歴］** タブの改訂と共にコメントが表示されます。 **［履歴］** タブの詳細については、「 [カード詳細ビューのレイアウト](/s/article/360042923934) 」の「履歴ビュー」を参照してください。
4. **［保存して終了］** をクリックします。
