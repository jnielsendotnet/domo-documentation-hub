## はじめに

顧客に配信する前に、キャンペーンで日付や日時のフィールドのフォーマットを直接変更しましょう。

## フォーマットパターンの例

あらかじめフォーマットされた文字列を使用して、日付や日時形式フィールドを好きなようにフォーマットすることができます。よくあるパターンの例としては、以下のようなものがあります。

| **パターン**                        | **例**                                |
| ------------------------------- | ------------------------------------ |
| `yyyy-MM-dd`                    | 2018-07-14                           |
| `dd-MMM-yyyy`                   | 14-Jul-2018                          |
| `dd/MM/yyyy`                    | 14/07/2018                           |
| `E, MMM dd yyyy`                | Sat, Jul 14 2018                     |
| `h:mm a`                        | 12:08 PM                             |
| `EEEE, MMM dd, yyyy HH:mm:ss a` | Saturday, Jul 14, 2018 14:31:06 PM   |
| `yyyy-MM-dd'T'HH:mm:ssZ`        | 2018-07-14T14:31:30+0530             |
| `hh 'o''clock' a, zzzz`         | 12 o’clock PM, Pacific Daylight Time |
| `K:mm a, z`                     | 0:08 PM, PDT                         |

## フォーマットするパターン

こちらは、日付と日時のフィールドをフォーマットするために使用できる記号と意味の完全なリストです。2020年8月3日を例に説明します。

| **記号**              | **意味**                            | **表示**    | **例**                                          |
| ------------------- | --------------------------------- | --------- | ---------------------------------------------- |
| yy、yyyy             | 暦数。                               | 年         | 20、2020                                        |
| M、MM、MMM、MMMM       | 年の月。                              | 数値/テキスト   | 8、08、Aug、August                                |
| d、dd                | 月の日付。                             | 数値        | 3、03                                           |
| E、EEEE、EEEEE        | 曜日。                               | テキスト      | Thu、Thursday、T                                 |
| D                   | 年の日。1月1日からの日数。                    | 数値        | 216                                            |
| G、GGGG、GGGGG        | 時代または年代。                          | テキスト      | AD、Anno Domini、A                               |
|                     |                                   |           |                                                |
| h、hh                | 1から12までの12時間時計を使用した時間。            | 数値        | 一桁の数字の場合は6、06、 <br/> または <br/> 二桁の数字の場合は11、11  |
| H、HH                | 0から23までの24時間時計を使用した時間。            | 数値        | 一桁の数字の場合は6、06、 <br/> または <br/> 二桁の数字の場合は18、18  |
| K、KK                | 0から11までの12時間時計を使った時間。             | 数値        | 一桁の数字の場合は6、06、 <br/> または <br/> 二桁の数字の場合は11、11  |
| k、kk                | 1から24までの24時間時計を使用した時間。            | 数値        | 一桁の数字の場合は6、06、 <br/> または <br/> 二桁の数字の場合は18、18  |
| m、mm                | 0から59までの分。                        | 数値        | 一桁の数字の場合は3、03、 <br/> または <br/> 二桁の数字の場合は13、13  |
| s、ss                | 0から59までの1分間の秒数。                   | 数値        | 一桁の数字の場合は4、04、 <br/> または <br/> 二桁の数字の場合は11、11  |
| S、SS、SSS            | 0から999までの1秒の分数。                   | 数値        | 9、09、009                                       |
|                     |                                   |           |                                                |
| V、VVVV、VVVVV        | タイムゾーンID                          | ゾーンID     | America/Los_Angeles; Z; -08:30                 |
| z、zzzz              | タイムゾーンの名前                         | Zone名     | Pacific Standard Time; PST                     |
| O、OOOO、OOOOO        | 地域化ゾーンオフセット                       | オフセット - O | GMT+8; GMT+08:00; UTC-08:00;                   |
| X、XX、XXX、XXXX、XXXXX | ゼロのゾーンオフセット「Z」                    | オフセット - X | Z; -08; -0830; -08:30; -083015; -08:30:15;     |
| x、xx、xxx、xxxx、xxxxx | ゾーンオフセット                          | オフセット - x | +0000; -08; -0830; -08:30; -083015; -08:30:15; |
| Z、ZZZZ、ZZZZZ        | ゾーンオフセット                          | オフセット - Z | +0000; -0800; -08:00;                          |
|                     |                                   |           |                                                |
| '                   | テキストのエスケープ                        | 区切り文字     |                                                |
| ''                  | シングルクォート（2つのシングルクォート、ダブルクォートではない） | リテラル      | '                                              |
