## はじめに

NetSuite OpenAirは、プロフェッショナルサービス企業が、プロジェクト管理、リソース最適化、プロジェクト会計処理、時間と費用の管理、および売上高と収益の把握などの中核的なビジネス活動を行うことを可能にします。NetSuite OpenAir APIの詳細については、NetSuiteのページ（ [https://www.openair.com/download/OpenAirXMLAPIGuide.pdf](https://www.openair.com/download/OpenAirXMLAPIGuide.pdf "https://www.openair.com/download/NetSuiteOpenAirXMLAPIGuide.pdf") ）を参照してください。

NetSuite OpenAirアカウントには、Data Centerから接続します。このトピックでは、NetSuite OpenAirコネクターユーザーインターフェースに固有のフィールドとメニューについて説明します。DataSetの追加や更新スケジュールの設定、DataSet情報の編集に関する一般的な情報については、「 [データコネクターを使用してDataSetを追加する](/ja/s/article/360042926274 "データコネクターを使用してDataSetを追加する") 」を参照してください。

## 必要条件

NetSuite OpenAirアカウントに接続してDataSetを作成するには、以下が必要です。

* 会社のNetSuite OpenAirアカウントのユーザーIDおよびパスワード
* アカウントの会社ID
* アカウントのOpenAirインスタンスのURL
* アカウントのAPI名前空間
* アカウントのAPIキー

API名前空間とキーを取得するには、NetSuite OpenAirアカウントの代表者にお問い合わせください。

## NetSuite OpenAirアカウントに接続する

このセクションでは、[NetSuite OpenAirコネクター] ページの **[認証情報]** および **[詳細]** ペインのオプションについて説明します。 このページのほかのペインの項目にある **[スケジュール設定]** や **[DataSetの名前を付け、説明を入力]** に関しては、コネクタータイプのほとんどで共通です。詳細については「 [データコネクターを使用してDataSetを追加する](/ja/s/article/360042926274 "データコネクターを使用してDataSetを追加する") 」を参照してください。

### [認証情報] ペイン

このペインには、NetSuite OpenAirアカウントに接続するための認証情報の入力フィールドがあります。下表では、各フィールドに必要な内容を説明しています。

| フィールド      | 説明                                      |
| ---------- | --------------------------------------- |
| User ID    | 会社のNetSuite OpenAirアカウントのユーザーIDを入力します。  |
| Password   | 会社のNetSuite OpenAirアカウントのパスワードを入力します。   |
| Company ID | アカウントと関連のある会社IDを入力します。                  |
| URL        | アカウントのNetSuite OpenAirインスタンスのURLを入力します。 |
| Namespace  | NetSuiteから取得したアカウントのAPI名前空間を入力します。      |
| API Key    | NetSuiteから取得したアカウントのAPIキーを入力します。        |

有効なNetSuite OpenAir認証情報を一度入力しておくと、同じアカウントを使用して随時、新しいNetSuite OpenAir DataSetを作成することができます。コネクターアカウントは、Data Centerの **[アカウント]** タブで管理できます。このタブの詳細については、「 [コネクターのユーザーアカウントを管理する](/ja/s/article/360042926054 "コネクターのユーザーアカウントを管理する") 」を参照してください。
