## はじめに

分散チャートは、データを直接表示するほかのグラフタイプとは異なり、実際の支出と予算、実際の値と予測値など、2セットのデータの差を示します。

以下に2つのチャートを示します。1つ目は売上高と予算のデータをシンプルな2本の折れ線チャートとして示し、2つ目は分散チャートと同じデータを示しています。分散チャートではデルタ（差分）データがハイライト表示されます。これは、線の間の差を分析する場合に便利です。

## 分散チャートを作成する

分散チャートには、「日付」フィールドと「実際」の値が必要です。必須ではありませんが、チャートのパワーを強調するには「目標」値が重要です。値、カテゴリー、系列データについては、「 [チャートデータについて](/ja/s/article/360043428693) 」を参照してください。

Analyzerでは、分散チャートで使用するデータを含む列を選択します。データ列の選択の詳細については、「 [DataSet列をチャートに適用する](/ja/s/article/360043428713) 」を参照してください。

## 分散チャートをカスタマイズする

分散チャートの外観をカスタマイズする方法はいくつかありますが、その多くは **［チャートの設定］** ツールで可能です。詳細については、「 [チャートのプロパティ](/ja/s/article/360042925374) 」に関する記事を参照してください。

次の表に、分散チャート固有のプロパティの一部を示します。

| プロパティ           | 説明                                                                         | 例                                                                                           |
| --------------- | -------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------- |
| ［一般］>［値の書式］     | 通貨記号を値に追加するかどうかを決定します。                                                     | —                                                                                           |
| ［一般］>［値（小数位）］   | 値の小数位の数を決めます。                                                              | —                                                                                           |
| ［一般］>［％値（小数位）］  | パーセント値の小数点以下の桁数を決定します。                                                     | —                                                                                           |
| ［一般］>［線のスタイル］   | ポイント間の線のスタイルを［デフォルト］、［直線］、［曲線］、［ステップ］から選択できます。自分のデータに最も適した見た目になるものを確認できます。 | <img alt="image.png" src="/images/kb/ka05w000001277K-00N5w00000Ri7BU-0EM5w000006u8jU.jpg"/> |
| ［一般］>［正の値の色］    | 分散が正の場合に表示される色を選択できます。                                                     | <img alt="image.png" src="/images/kb/ka05w000001277K-00N5w00000Ri7BU-0EM5w000006u8jZ.jpg"/> |
| ［一般］>［負の値の色］    | 分散が負の場合に表示される色を選択できます。                                                     | <img alt="image.png" src="/images/kb/ka05w000001277K-00N5w00000Ri7BU-0EM5w000006u8je.jpg"/> |
| ［一般］>［差の絶対値を表示］ | ゼロを基準とした絶対的な差として値を表示します。これにより、差の大きさや範囲をより適切に可視化できます。                       | <img alt="image.png" src="/images/kb/ka05w000001277K-00N5w00000Ri7BU-0EM5w000006u8kh.jpg"/> |
| 実際の線            | 実際の線のタイプ、色、太さを設定できます。                                                      | <img alt="image.png" src="/images/kb/ka05w000001277K-00N5w00000Ri7BU-0EM5w000006uAdO.jpg"/> |
| 目標線             | 目標線の値、ラベル、タイプ、色、太さを設定できます。                                                 | <img alt="image.png" src="/images/kb/ka05w000001277K-00N5w00000Ri7BU-0EM5w000006u8jo.jpg"/> |
