## はじめに

マーケターがPPC結果を追跡および評価する際、Bing Adsはそれに関するインサイトを提供します。Bing Ads Analytics Appは、顧客の購入やフォーム記入、クリックそしてインプレッションなどから顧客とのインタラクションのサマリービューを提供します。

## 要件と注意点

APIアクセスとBing Adsプロファイルのキーが必要になります。

## 使用方法

* 広告コンバージョン：広告は売り上げをもたらしているのか、それとも費用を発生させているだけなのか？

  + コンバージョン率とコンバージョン
  + 最もコンバージョン率の高かったキーワード
  + キャンペーンのコンバージョン
  + キャンペーン別コンバージョン率
  + 広告グループのコンバージョン
* 広告エンゲージメント：どんな広告パフォーマンスを達成しているか？

  + 低クオリティのクリック
  + 予算のため表示されなかったインプレッション数
  + CTR
  + キーワード関連性のため表示されなかったインプレッション数
  + 広告ポジションのトレンド
  + クリックされたキーワード
  + キャンペーンインプレッションシェア
  + 品質スコアのトレンド
* コスト：予算は、可能な限り高いROIをもたらすよう活用されているか？

  + キャンペーン予算支出
  + CPAトレンド
  + CPCトレンド
  + キャンペーン別コンパージョン単価
  + 推定支出
* エグゼクティブパルス：毎日5メトリクスをチェックする必要があります

  + 下位キャンペーン
  + コスト
  + 最も費用のかかったキャンペーン
  + アカウントのクリック
  + 上位キーワード
