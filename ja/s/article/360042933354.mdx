## はじめに

従業員情報の追跡は大量のデータを生成します。BambooHRはそのような情報プロセスの管理を手助けします。BambooHR Dashboardは総従業員数、新規雇用者、退職者、従業員デモグラフィックを始めとするトップレベル情報のサマリーを提供します。

## 要件と注意点

**APIキー：** BambooHRインスタンスでAPIキーを取得するには、画面右上隅の **［Account］** をクリックして **［API Key］** を選択し（必要な場合は新しいAPIキーを生成します）、そのキーをDomoにコピーします。

**管理者またはパワーユーザーの認証情報：** BambooHRの機密情報を表示するための権限を持っているユーザーのみ、このAppのフルインストールを実行することができます。

**デフォルトの日付範囲：** 過去30日間

## 使用事例

* エグゼクティブパルス：すべての人事管理者が把握しておくべき5つの情報

  + 従業員名簿
  + 従業員の性別
  + 年齢階層
  + 部署別の給与
  + 部署別の婚姻区分
* デモグラフィック：適切な福利プログラムを作成

  + 従業員名簿
  + 従業員の性別
  + 年齢階層
  + 部署別の給与
  + 部署別の婚姻区分
* 多様性：従業員の年齢、性別、および人種や民族などを均衡化

  + 雇用状況
  + 部署別の年齢
  + 従業員の人種や民族
  + 部署の男女の割合
  + 人種や民族別の給与
* 従業員の回転：解雇やリテンションを分析

  + 部署別リテンション
  + 部署別勤続年数
  + 解雇された従業員
  + 解雇された従業員の勤続年数
