## はじめに

Box Inc.は、法人向けのオンラインファイル共有およびパーソナルクラウドコンテンツ管理サービスです。このコネクターを使うと、Boxアカウントからデータファイルを取得し、Domoで利用できます。データファイルは、CSV、JSON、XLS、XLSX、XML、ZIP、GZIPのいずれかのファイルタイプを使用しているものを取得できます。このコネクターは、ファイル名とタイプ別にデータを取得します。

Box Advanced Connectorアカウントには、Data Centerから接続できます。このトピックでは、Box Advanced Connectorコネクターのユーザーインターフェースに固有のフィールドとメニューについて説明します。DataSetの追加、更新スケジュールの設定およびDataSet情報の編集を行うには、「 [コネクターを使用してDataSetを追加する](/ja/s/article/360042926274 "コネクターを使用してDataSetを追加する") 」を参照してください。

## 必要条件

Box Advanced ConnectorアカウントにアクセスしてDataSetを作成するには、Boxアカウントに関連付けられたメールアドレスとパスワードが必要です。

## Box Advanced Connectorアカウントに接続する

このセクションでは、Box Advanced Connectorコネクターページの **［認証情報］** および **［詳細］** ペインのオプションについて説明します。このページのほかのペインの項目である **［スケジュール設定］** や **［DataSetに名前を付け、説明を入力］** に関しては、ほとんどのタイプのコネクターで共通しています。詳細については、「 [コネクターを使用してDataSetを追加する](/ja/s/article/360042926274 "コネクターを使用してDataSetを追加する") 」を参照してください。

## ［認証情報］ペイン

有効なBox Advanced Connector認証情報を一度入力しておくと、いつでも同じアカウントを使用して新しいBox Advanced Connector DataSetを作成できます。コネクターのアカウントはData Centerの **［アカウント］** タブで管理できます。このタブの詳細については、「 [コネクターアカウントを管理する](/ja/s/article/360042926054 "コネクターアカウントを管理する") 」を参照してください。

## ［詳細］ペイン

| **メニュー**                                               | **説明**                                                                                                      |
| ------------------------------------------------------ | ----------------------------------------------------------------------------------------------------------- |
| File Name                                              | 取得するBoxファイルの名前（または名前の一部）を入力します。コネクターは、 **［Date Field］** メニューで選択した内容にもとづき、名前が一致するものから、最後に作成または変更したファイルを返します。 |
| Match Type                                             | 取得するファイルのファイル名が **［File Name］** で入力した名前から始まるのか、それともファイル名にその名前が含まれるのかを指定します。                                 |
| File Type                                              | 取得するデータファイルのファイルタイプを選択します。                                                                                  |
| File name or portion of file name to search for in zip | このフィールドは、［File Type］で **［Zip］** を選択した場合に表示されます。取得する圧縮ファイル名を入力します。空白のままにすると、最初の圧縮ファイルを取得します。                 |
| Zipped File Type                                       | 取得したい圧縮データファイルのファイルタイプを選択します。                                                                               |
| Date Field                                             | ファイルを作成日または最終変更日のどちらで取得するかを指定します。 **これは、** ［File Name］に入力した名前が複数のファイル名に含まれている場合に便利です。                       |
| Would you like to choose file from Folder?             | 特定のフォルダからファイルを取得する必要がある場合は **［Yes］** を選択します。 **［No］** が選択されている場合は、［Date field］のパラメーターにもとづいてファイルを取得します。      |

## プレビュー

このセクションでは、選択したレポートをプレビューしたり、セルのフォーマット（列ヘッダー、行ヘッダー、クロス集計）を設定したりできます。また、高度なオプションを使用して、区切り文字やファイルエンコードを設定することもできます。 **［Edit Selection］** オプションを選択して、ファイルのレイアウトのフォーマットを設定できます。

#### フォーマット

CSVファイルのレイアウトを選択できます。使用可能なオプションは以下のとおりです。

* **Column headers：**
  CSVデータに列ベースのレイアウトを使用します（つまり、各列がヘッダーを持ちます）。このオプションはデフォルトで選択されています。［Column headers］で使用可能なオプションは以下のとおりです。

  + **Use first row as headers：**
    選択した範囲の最初の行をDataSetの列ヘッダーとして使用できます。
  + **Generate Headers：**
    データにヘッダーが含まれておらず、Domoにヘッダーを生成させたい場合は、このオプションを選択します。
  + **Choose a specific range：**
    列ヘッダーとして使用する具体的なセルの範囲を選択できます。
* **Row headers：**
  CSVデータに行ベースのレイアウトを使用します（つまり、各行がヘッダーを持ちます）。［Row headers］で使用可能なオプションは以下のとおりです。

  + **Use first column as headers：**
    選択した範囲の最初の行をDataSetの行ヘッダーとして使用します。
  + **Generate Headers：**
    データにヘッダーが含まれておらず、Domoにヘッダーを生成させたい場合は、このオプションを選択します。
  + **Choose a specific range：**
    行ヘッダーとして使用する具体的なセルの範囲を選択できます。
* **Crosstab**
  ：CSVデータにクロス集計（ピボット）レイアウトを使用します。［Crosstab］で使用可能なオプションは以下のとおりです。

  + 列ヘッダー
    - **Use first row as headers：**
      選択した範囲の最初の行をDataSetの列ヘッダーとして使用できます。
    - **Choose a specific range：**
      列ヘッダーとして使用する具体的なセルの範囲を選択できます。
  + 行ヘッダー
    - **Use first column as headers：**
      選択した範囲の最初の行をDataSetの行ヘッダーとして使用します。
    - **Choose a specific range：**
      行ヘッダーとして使用する具体的なセルの範囲を選択できます。

#### 高度なオプション

このペインでは、アップロードしたCSVファイルについて高度なオプションを設定します。

* **Include batch upload metadata**
  （チェックボックス） **：** データにバッチ列を追加するには、このボックスを選択します。
* **Keep leading zeros**
  （チェックボックス） **：** 「0123」のように数値の先頭に0（ゼロ）を含める場合は、このチェックボックスを選択します。
    
  **注記：** コネクターは、先頭に0を含む値をテキストとして解析します。
* **Delimiter：**
  CSVファイルでは、デフォルトの区切り文字としてカンマが使用されます。必要に応じて、 **［Delimiter］** メニューで別の区切り文字を選択することもできます。
* **File Encoding：**
  CSVファイルには、デフォルトでUTF-8エンコーディングが使用されます。必要に応じて **［File Encoding］** メニューで別のエンコーディングタイプを選択することもできます。
