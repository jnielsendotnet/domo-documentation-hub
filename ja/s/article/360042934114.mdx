## はじめに

CRMソフトウェアは、様々なビジネスにとって非常に重要なソフトウェアですが、SalesforceはCRM市場をリードする製品です。Salesforce Dashboardを使用すると、リードソース、地理情報、成約率、セールスパイプライン、営業担当者の業績など、様々な情報を確認することができます。

## 使用事例

ビジネスに関する回答可能な質問：

- 見込み客の主要なソースはどこか
- 販売目標を超えるためには今日何をすればよいか
- 高い業績を上げている営業担当者は誰か

## 認証

認証処理では、以下が必要になります。

- 認証の対象となるSalesforce環境（login.salesforce.comやtest.salesforce.comなど）
- Salesforceのユーザー名
- Salesforceのパスワード

SalesforceのインスタンスにAPIアクセスを含めるか、APIアクセスを個別に有効にする必要があります。

## レポートの設定/レポートの詳細

選択可能なレポートは、以下のように設定する必要があります。

| データ                       | スケジュール設定（更新設定の変更） | パラメーター                                | 選択                                                                      |
| ------------------------- | ----------------- | ------------------------------------- | ----------------------------------------------------------------------- |
| Browse Objects and Fields |                   | Salesforce Object                     | 表（オブジェクト）のリストです。                                                        |
| Browse Picklists          |                   | Salesforce Object                     | 表（オブジェクト）のリストです。                                                        |
| Deleted Objects           |                   | Salesforce Object、Start Days/End Days | 表（オブジェクト）のリスト。レポートに表示するオブジェクトの日付範囲を入力します。                               |
| Query                     |                   | Query Option、Query                    | 定義済みのクエリ（販売実績またはユーザーの画像）を選択するか、独自のカスタムクエリを作成し、Salesforce SOQLクエリを入力します。 |
| Reports                   |                   | SalesForce Reports                    | クエリを実行するSalesforceレポートを選択します。                                           |
