## はじめに

Adobe Analytics Realtimeでは、インサイトをリアルタイムのデータ駆動型の応答に変換できます。DomoのAdobe Analytics Realtimeコネクターを使用すると、選択したAdobe Analytics Report Suiteからデータを取得できます。Adobe Analytics APIの詳細については、ウェブサイト（ https://marketing.adobe.com/developer/en_US/get-started/introduction/c-introduction ）を参照してください。

Adobe Analyticsアカウントには、Data Centerから接続します。このトピックでは、Adobe Analytics Realtimeコネクターユーザーインターフェース固有のフィールドとメニューについて説明します。DataSetの追加や更新スケジュールの設定、DataSet情報の編集に関する一般的な情報は、「 [コネクターを使用してDataSetを追加する](/ja/s/article/360042926274 "コネクターを使用してDataSetを追加する") 」を参照してください。

## 必要条件

Adobe Analytics v2に接続してDataSetを作成するには、Adobe Developer ConsoleでAdobe Analyticsサービスアカウント（JWT）プロジェクトを作成し、以下の情報を入力する必要があります。

* クライアントID（APIキー）
* クライアントシークレット
* テクニカルアカウントID
* 組織ID
* Adobe統合用のPEM形式の秘密キー

### Adobe Developer Consoleで新しいOAuthプロジェクトを作成する

Adobe Developer Consoleで新しいOAuthプロジェクトを作成するには、以下の手順を実行します。

1. [https://console.adbe.io/projects](https://console.adobe.io/projects "フォローリンク") にログインします。
2. **［Create new project］** をクリックします。

<Frame><img alt="New_Project_button.png" src="/images/kb/ka0Vq00000012UP-00N5w00000Ri7BU-0EM5w000005vO15.png"/></Frame>

3. **［Add API］** をクリックしてAdobeサービスを開始します。

<Frame><img alt="Add_API.png" src="/images/kb/ka0Vq00000012UP-00N5w00000Ri7BU-0EM5w000005vO16.png"/></Frame>

4. 利用可能なAPIは **［Add an API］** ウィンドウにリストされます。 **［Adobe Analytics］** を選択します。

<Frame><img alt="Add_an_API-Adobe_Analytics.png" src="/images/kb/ka0Vq00000012UP-00N5w00000Ri7BU-0EM5w000005vO19.png"/></Frame>

5. 認証タイプを選択します。ここでは、**［Service Account (JWT)］** を選択します。

<Frame><img alt="OAuth-service_account_JWT.png" src="/images/kb/ka0Vq00000012UP-00N5w00000Ri7BU-0EM5w000005vO17.png"/></Frame>

6. **［Generate a key pair］** を選択して、 **［Generate Keypair］** をクリックします。

<Frame><img alt="JWT_Creds.png" src="/images/kb/ka0Vq00000012UP-00N5w00000Ri7BU-0EM5w000005vO1B.png"/></Frame>

7. 新しく生成されたキーペアがシステムにダウンロードされます。次に、パブリックキーがJWTプロジェクトにアップロードされ、秘密キーがシステムにダウンロードされます。

<Frame><img alt="JWT_KeyPair.png" src="/images/kb/ka0Vq00000012UP-00N5w00000Ri7BU-0EM5w000005vO18.png"/></Frame>

8. config.zipファイルを解凍してconfigディレクトリを開き、任意のテキストエディターでprivate.keyファイルを開きます。このファイルに記述されている **秘密キー** を、Domoの［認証情報］セクションに貼り付けます。

9. **［Next］** をクリックします。

10. 目的の製品プロフィールを選択して **［Save Configured API］** をクリックします。

11. **［Project APIs and Services］** ページが表示されます。画面をスクロールダウンすると、 **クライアントID、クライアントシークレット、テクニカルアカウントID** 、 **組織ID** が表示されます。これらの情報を使用して、Domo内にAdobe Analytics JWTアカウントが作成されます。

<Frame><img alt="Generate_Access_Token.png" src="/images/kb/ka0Vq00000012UP-00N5w00000Ri7BU-0EM5w000005vO1A.png"/></Frame>

12. （任意）プロジェクト名をわかりやすい名前に変更します。**［Project Overview］** ページに移動して **［Edit Project］** をクリックします。

## Adobe Analytics Realtimeアカウントに接続する

このセクションではAdobe Analytics Realtimeコネクターページの **［認証情報］** と **［データの選択］** ペインのオプションについて説明します。このページのほかのペインの項目にある **［スケジュール設定］** や **［DataSetに名前を付け、説明を入力］** に関しては、コネクタータイプのほとんどで共通です。詳細については「 [コネクターを使用してDataSetを追加する](/ja/s/article/360042926274 "コネクターを使用してDataSetを追加する") 」を参照してください。

### ［認証情報］ペイン

このペインには、Adobe Analytics Realtimeアカウントに接続するための認証情報の入力フィールドが含まれています。下表では、各フィールドに必要な内容を説明しています。

| フィールド                | 説明                           |
| -------------------- | ---------------------------- |
| Client ID (API Key)  | クライアントIDを入力します。              |
| Client Secret        | クライアントシークレットを入力します。          |
| Technical Account ID | テクニカルアカウントIDを入力します。          |
| Organization ID      | 組織IDを入力します。                  |
| Private Key          | Adobe統合環境のPEM形式の秘密キーを貼り付けます。 |

これらの認証情報の入手方法については、前述の「必要条件」を参照してください。

有効なAdobe Analytics Realtimeの認証情報を一度入力しておくと、随時同じアカウントを使用して新しいAdobe Analytics RealtimeのDataSetを作成できます。コネクターアカウントは、Data Centerの **［アカウント］** タブで管理できます。このタブの詳細は、「 [コネクターのユーザーアカウントを管理する](/ja/s/article/360042926054 "コネクターのユーザーアカウントを管理する") 」を参照してください。

### よくある質問

<AccordionGroup>
<Accordion title="Adobe Analytics v2 OAuthコネクターとAdobe Analytics Realtime JWTコネクターの違いは何ですか？">


Adobe Analytics v2 OAuthコネクターではOAuth認証機能が使用されますが、Adobe Analytics Realtime JWTコネクターではJWT認証機能が使用されます。Adobeは、JWT認証機能を使用することを推奨しています。 OAuth認証機能の場合、OAuth更新トークンの有効期限が2週間で切れてしまうため、2週間ごとにOAuthアカウントを更新する必要があります。


</Accordion>
<Accordion title="このコネクターでは、どのバージョンのAdobe Analytics APIが使用されますか？">


このコネクターでは、バージョン1.4のAdobe Analytics APIが使用されます。


</Accordion>
<Accordion title="このコネクターを作成するにはどのような認証情報が必要ですか？">


Adobe Analytics Realtime JWTプロジェクトに関連付けられているクライアントID（APIキー）、クライアントシークレット、テクニカルアカウントID、組織ID、および秘密キーが必要になります。


</Accordion>
<Accordion title="APIについて知っておくべき制約事項はありますか？">


いいえ


</Accordion>
<Accordion title="どちらのAdobe Analyticsコネクターを使用すればよいですか？">


Adobeは、Adobe Analytics Realtime JWTコネクターを使用することを推奨しています。Adobe OAuthはトークンの有効期限が2週間で切れてしまうため、アカウントを更新する必要があります。



</Accordion>
