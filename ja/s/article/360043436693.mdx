## はじめに

Microsoft Azure Blob Storageは、画像、ビデオ、オーディオ、ドキュメントなど、あらゆる種類の非構造化データを簡単かつ効率的に保存できます。DomoのMicrosoft Azure Blob Storageコネクターを使用すると、CSV、TSV、およびTXTデータファイルをMicrosoft Azure Blob StorageアカウントからDomoに引き込むことができます。Azureデータと、社内のすべてのデータソースのデータを結合することで、会社を全体的に把握することができます。主要なメトリクスが変更された場合にリアルタイムに通知を行うカスタムアラートを設定し、ビジネスの意思決定をより迅速かつ的確に行えるようにします。Azure Blob Storage APIについての詳細は、同社のページ（ [https://docs.microsoft.com/en-us/res...rvice-rest-api](https://docs.microsoft.com/en-us/rest/api/storageservices/blob-service-rest-api "https://docs.microsoft.com/en-us/rest/api/storageservices/blob-service-rest-api")
）を参照してください。

Microsoft Azure Blob Storageアカウントには、Data Centerから接続します。このトピックでは、Azure Blob Storage Connectorのユーザーインターフェースに固有のフィールドとメニューについて説明します。DataSetの追加や更新スケジュールの設定、DataSet情報の編集に関する一般的な情報については、「 [データコネクターを使用してDataSetを追加する](/ja/s/article/360042926274) 」を参照してください。

## 必要条件

Azure Blob Storageアカウントに接続してDataSetを作成するには、以下が必要です。

* Microsoft Azure Blob Storageアカウント名
* Microsoft Azure Blob Storageアカウントキー

認証情報がわからない場合は、Microsoftのアカウント担当者にお問い合わせください。

## Azure Blob Storageアカウントに接続する

このセクションでは、Azure Blob Storage Connectorページの **［認証情報］** および **［詳細］** ペインのオプションについて説明します。このページのその他のペインある **［スケジュール設定］** と **［DataSetの名前と説明］の** 項目は、ほとんどのタイプのコネクターで共通しています。詳細については「 [データコネクターを使用してDataSetを追加する](/ja/s/article/360042926274 "データコネクターを使用してDataSetを追加する") 」を参照してください。
