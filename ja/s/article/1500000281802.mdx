## Azure SQL DatawarehouseがAzure Synapse SQL Datawarehouseになりました

#### *最新情報*

2020年11月16日月曜日より、 **Azure SQL Data Warehouse** **コネクター** は **Azure Synapse SQLコネクター** と呼ばれるようになります。

#### *これにより、どのような影響がありますか？*

すべてのAzure SQL Data Warehouse DataSetは引き続き今まで通り実行されます。コネクターの機能に変更はありません。

コネクターを使用しているお客様や、使用を予定しているお客様は、次の点にお気づきになると思います。

1. AppStoreにこのコネクターの新しいアイコンがあります。既存のDataSetが新しいアイコンでData Centerに表示されるようになります。
2. お客様が「タイプ」フィルターを使用してData CenterでAzure Data Warehouse DataSetを検索する場合には、「Azure Data Warehouse」ではなく「Azure Synapse SQL」タイプのDataSetを検索する必要があります。
3. 新しいAzure SQL Data Warehouseを作成する場合は、AppStoreで「Azure SQL Data Warehouse Connector」ではなく、「Azure Synapse SQL Connector」を検索する必要があります
4. Azure SQL Data Warehouseコネクターのコネクターアカウントは、「Azure Synapse SQL」アカウントと呼ばれるようになります。

#### *Azure Data Warehouse DataSetがある場合、何か対応が必要ですか？*

お客様は何もする必要はありません。既存のAzure Data Warehouse/Azure Synapse SQL DataSetは引き続き中断されることなく実行され、必要に応じて、既存の認証情報を使用して新しいDataSetを作成できます。レポートやアカウントを更新する必要はありません。

コネクターの機能に関する質問については、「 [Azure Synapse SQLコネクター](/ja/s/article/360042931674 "https://knowledge.domo.com/Connect/Connecting_to_Data_with_Connectors/Configuring_Each_Connector/Database_Connectors/Azure_Synapse_SQL_Connector") 」の資料を参照してください。
