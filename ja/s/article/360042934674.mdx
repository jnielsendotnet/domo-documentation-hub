### はじめに

プロフィールページは、Domo内におけるユーザー個人のホームページです。すべてのユーザーのプロフィールで情報を閲覧できます（[セキュリティ権限](/s/article/360043438953)によっては、編集することもできます）。

ユーザーのプロフィールでは、以下を実行できます。

- 職場での役割、部署、連絡先情報、場所など、ユーザーの個人情報を確認する
- ユーザーをフォローする
- ユーザーが組織階層のどこに位置するのかを確認する
- フォロワーの数、Buzzでの「いいね！」の獲得数など、会社およびDomoにおけるユーザーの統計情報を確認する
- ユーザーのステータス（現在のプロジェクト）を確認する
- ユーザーのDomoスコアを確認する
- 「いいね！」を獲得したチャット、コメント、写真、タスク、ステータスの更新を確認する
- ユーザーが参加しているBuzzチャットを確認する
- ユーザーが最近または最も頻繁に表示したカードを確認する
- ユーザーが所有しているDataSetを確認する
- ユーザーがフォローしているアラートを確認する
- ユーザーがお気に入りとして追加したカードを確認する
- ユーザーのタスクとプロジェクトを確認する
- ユーザーのプロフィール写真を確認する
- ユーザーにアワードを授与する
- ユーザーが属するグループを確認する

プロフィールは、上から下まで複数のペインに分かれています。ページには、すべてのペインにすばやくアクセスできるハイパーリンクが用意されています。このトピックでは、各ペインについて上から下まで順番に説明します。

### プロフィールヘッダー

プロフィール閲覧者に最初に表示される箇所であり、プロフィールのメインのペインです。ここでは、ユーザーの個人情報と統計情報を確認することができます。

また、自分のプロフィール写真の編集、ステータスの更新の入力、上長の閲覧と編集を行うことができます。

### ［アクティビティー］ペイン

このペインには、タブが並んだ行があり、ユーザーがDomo内で何をしてきたかを垣間見ることができます。このペインのタブは以下の通りです。

| タブ            | 説明                                        |
| ------------- | ----------------------------------------- |
| ステータスの更新      | このユーザーのステータスの更新内容を、最新から最も古いものまで時系列に表示します。 |
| チャット          | このユーザーが参加したBuzzチャットが表示されます。               |
| 所有しているカード     | このユーザーが所有者として設定されているカードを表示します。            |
| 所有しているDataSet | このユーザーが所有するDataSetを表示します。                 |
| 閲覧中           | このユーザーが最近、そして最も頻繁に閲覧したカードを表示します。          |
| アラート          | このユーザーがフォローしているアラートを表示します。                |
| お気に入り         | このユーザーがお気に入りとして追加したカードを表示します。             |

各タブの詳細については、以下で説明します。

## 閲覧中

このタブでは、このユーザーが最近、または最も頻繁に表示したすべてのカードのサムネイルを閲覧できます。サムネイルをクリックすると、カードのプレビューとそのカードの詳細ビューへのリンクが表示されます。長方形のカードをクリックすると、カードの詳細ビューが直接開きます。

<Frame>![Screenshot](/images/kb/ja/0EM5w000005vOdx.png)</Frame>

詳細については、「[ユーザーが閲覧したカードを見る](/s/article/360043439673)」を参照してください。

## お気に入り

このタブでは、このユーザーがお気に入りとして設定したすべてのカードのサムネイルを閲覧できます。どのサムネイルも、クリックするとカードの詳細ビューが開きます。

<Frame>![Screenshot](/images/kb/ja/0EM5w000005vOeB.png)</Frame>

詳細については、「[ユーザーのお気に入りのカードを閲覧する](/s/article/360042934754)」を参照してください。

## アラート

このタブでは、このユーザーがフォローしているすべてのアラートを閲覧できます。アラートの上部をクリックすると情報パネルが開き、アラートの編集、通知の設定、履歴の閲覧を行うことができます。下部をクリックすると、そのカードの詳細ビューを開くことができます。また、**［通知する］**をクリックすると、通知を設定することもできます。

<Frame>![Screenshot](/images/kb/ja/0EM5w000005vOe1.png)</Frame>

詳細については、「[ユーザーがフォローしたアラートを閲覧する](/s/article/360042934874)」を参照してください。

## チャット

このタブでは、このユーザーがBuzzチャットを行ったすべてのユーザーについて、そのチャットの内容とともに確認できます。タブは2つのセクションに分かれています。左側にあるセクションには、このユーザーがチャットを行ったすべてのユーザーがバブルとして表示されます。バブルのサイズは、ユーザーが参加したチャットの数によって決定されます。右側にあるセクションには、Buzzの小型バージョンが表示され、このユーザーが参加している新しいチャットを反映して更新されます。

<Frame>![Screenshot](/images/kb/ja/0EM5w000005vOe2.png)</Frame>

デフォルトでは、このユーザーのすべてのチャットがフィードで表示されます。なお、特定のユーザーとのチャットのみ表示するようにフィードにフィルターをかけることもできます。これを行うには、そのユーザーのバブルを選択します。ユーザーのバブルは左側のセクション中央に移動し、そのユーザーのチャットを表示するよう、フィードにフィルターがかけられていることを示します。

詳細については、「[ユーザーのチャットのアクティビティを閲覧する](/s/article/360043439533)」を参照してください。

## 「いいね！」

このタブでは、このユーザーが「いいね！」したBuzzチャット（つまりBuzzでの最初の投稿）、Buzzコメント、写真、ステータスの更新、タスクをすべて閲覧できます。

<Frame>![Screenshot](/images/kb/ja/0EM5w000005vOeG.png)</Frame>

このタブには、以下のようなサブタブがあります。

| サブタブ  | 説明                                                                                                     |
| ----- | ------------------------------------------------------------------------------------------------------ |
| 全て    | ユーザーのBuzzチャットとコメント、タスク、写真やステータスで「いいね！」されたものがすべて表示されます。                                                 |
| チャット  | Buzzのディスカッションで、「いいね！」されたこのユーザーによる最初の投稿をすべて表示します。                                                       |
| コメント  | 「いいね！」されたこのユーザーによるBuzzコメントをすべて表示します。                                                                   |
| タスク   | このユーザーのタスクで「いいね！」されたものをすべて表示します。プロジェクトとタスクの詳細については、「[プロジェクトとタスク](/s/article/360042925874)」を参照してください。   |
| 写真    | このユーザーの個人写真で「いいね！」されたものをすべて表示します。個人写真の詳細については、「[個人写真を管理する](/s/article/360042934714)」を参照してください。         |
| ステータス | このユーザーのステータスで「いいね！」されたものをすべて表示します。ステータスの更新の詳細については、「[ステータス更新を入力する](/s/article/360042934734)」を参照してください。 |

「いいね！」された項目は、それぞれのカードに表示されます。いずれの項目でも、「親指」のアイコンにマウスポインターを合わせると、その項目に「いいね！」したユーザーを確認することができます。さらに自分でクリックし「いいね！」することもできます。

## 所有しているカード

このタブでは、このユーザーが所有者として設定されているすべてのカードを確認できます。どのカードもクリックすることで、そのカードの詳細ビューを表示させることができます。

詳細については、「[ユーザーが所有するカードを閲覧する](/s/article/360043439653)」を参照してください。

<Frame>![Screenshot](/images/kb/ja/0EM5w000005vOe3.png)</Frame>

詳細については、「[ユーザーが所有するDataSetを閲覧する](/s/article/360042934894)」を参照してください。

## ステータスの更新

このタブには、ユーザーのすべてのステータスの更新が、最新の日付から最も古い日付の順に表示されます。

<Frame>![Screenshot](/images/kb/ja/0EM5w000005vOe4.png)</Frame>

<Frame>![Screenshot](/images/kb/ja/0EM5w000005vOe9.png)</Frame>

### ［タスクとプロジェクト］ペイン

このペインでは、このユーザーがチームメンバーとして割り当てられているすべてのプロジェクトのタスクを確認することができます。

<Frame>![Screenshot](/images/kb/ja/0EM5w000005vOdy.png)</Frame>

プロジェクトごとに、**［プロジェクトとタスク］**ページのすべてのリストが表示されます。リストをクリックすると、そのリストにあるすべてのタスクが、割り当てられたすべてのユーザーとともに表示されます。

<Frame>![Screenshot](/images/kb/ja/0EM5w000005vOe9.png)</Frame>

アイコンをクリックして、タスクに「いいね！」することができます。また、このペインには、このユーザーのプロジェクトの更新をすべて含む、ライブの「作業ログ」が表示されます。例えば、プロジェクトでタスクが作成されると、作業ログに通知が表示されます。

タスクとプロジェクトの詳細については、「[プロジェクトとタスク](/s/article/360042925874)」を参照してください。

### ［写真］ペイン

このペインでは、組織内のほかのユーザーも見ることのできる個人写真をアップロードすることができます。個人写真ストアにある写真は、プロフィール写真のようにあなたを示すものとしてDomo全体で使われるわけではないため、プロフィール写真とは異なります。個人写真は、ほかのユーザーに自分自身を説明するのに使うものであり、プロフィールページでのみ見ることができます。

<Frame>![Screenshot](/images/kb/ja/0EM5w000005vOeC.png)</Frame>

写真のアップロードと管理の詳細については、「[個人写真を管理する](/s/article/360042934714)」を参照してください。

### ［グループ］ペイン

このペインでは、このユーザーが属するすべてのDomoユーザーグループを確認することができます。グループ名にマウスポインターを合わせると、そのグループのすべてのメンバーを確認することができます。グループの詳細については、「ユーザーとグループを管理する」を参照してください。

<Frame>![Screenshot](/images/kb/ja/0EM5w000005vOeE.png)</Frame>

詳細については、「[ユーザーのメンバーグループを閲覧する](/s/article/360042935054)」を参照してください。

### ［アワード］ペイン

このペインでは、このユーザーに授与されたアワードを見ることができます。アワードをクリックすると、そのアワードの詳細を示すウィンドウが表示されます。

また、**［アワード］**をクリックしてリストからアワードを選択し、ユーザーにアワードを授与することもできます。このボタンは、Major Domoセンターでアワードを授与する権限が付与されている場合にのみ表示されます。詳細については、「[アワードのバッジを作成する](/s/article/360043439493)」を参照してください。

<Frame>![Screenshot](/images/kb/ja/0EM5w000005vOe5.png)</Frame>
