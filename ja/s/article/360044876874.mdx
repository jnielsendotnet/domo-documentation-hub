### ［定数を追加］タイル

<Frame>![add constants.jpg](/images/kb/ja/0EMVq000003YfqE.jpg)</Frame>

［定数を追加］タイルでは、定数値の列を追加することができます。

［定数を追加］タイルをキャンバスにドラッグした後、次の手順に従って設定します。

1. タイルを選択し、キャンバスの下にタイルエディターを開きます。タイルエディター内で残りの手順を実行します。
2. （オプション）タイルの名前を変更するには、<img alt="" src="/images/kb/ja/0EMVq0000060G2y.jpg" style={{width: 23, height: 23, display: 'inline', verticalAlign: 'start', margin: '0'}}/>

   **編集**（鉛筆のアイコン）をクリックして名前を入力します。
3. 新しい列の名前を入力します。
4. 列のタイプを選択します。

   |  |  |
   | --- | --- |
   | テキスト | テキストと、（テキストとして扱われる）数値が含まれます。 |
   | 浮動小数 | 小数点を含んだ数値が含まれます。 |
   | 固定小数 | 小数点以下の桁数が固定された小数値が含まれます。 |
   | 整数 | 小数の位のない数値が含まれます。 |
   | 日付 | 日付の値が含まれます。 |
   | タイムスタンプ | 日付と時刻の値が含まれます。 |
5. 定数、日付、日時を入力するか、空白のままにします。  
     

   <Frame>![Screenshot](/images/kb/ja/0EMVq0000060Y7t.jpg)</Frame>
6. **［終了］**を選択します。  
     

   <Frame>![Screenshot](/images/kb/ja/0EMVq0000060Y7u.jpg)</Frame>

<Tip>**ヒント：**プレビューを実行してキャンバスでタイルをクリックした後、**［プレビュー］**タブをクリックすると、タイルで変換されたデータをプレビューすることができます。</Tip>

### ［列を複製］タイル

<Frame>![Screenshot 2024-11-18 at 4.34.59 PM.png](/images/kb/ja/0EMVq000003sxcT.jpg)</Frame>

［列を複製］タイルでは、列を複製して同じデータタイプの列と置き換えることができます。例えば、文字列の値を持つ列を複製して、別の文字列の値を持つ列と置き換えることができます。列は、両方とも既にDataSetに存在している必要があります。

［列を複製］タイルをキャンバスにドラッグしてから選択してタイルエディターを開き、次の手順に従って設定します。

1. タイルを選択し、キャンバスの下にタイルエディターを開きます。タイルエディター内で残りの手順を実行します。
2. （オプション）タイルの名前を変更するには、<img alt="" src="/images/kb/ja/0EMVq0000060Lqd.jpg" style={{width: 23, height: 23, display: 'inline', verticalAlign: 'start', margin: '0'}}/>

   **編集**（鉛筆のアイコン）を選択して名前を入力します。
3. 置き換えるDataSetの列を選択します。
4. 複製するDataSetの列を選択します。置き換える列と同じデータタイプを持つ列からのみ選択できます。
5. **［終了］**を選択します。  
     

   <Frame>![Screenshot](/images/kb/ja/0EMVq0000060Y7u.jpg)</Frame>

<Tip>**ヒント：**プレビューを実行してキャンバスでタイルをクリックした後、**［プレビュー］**タブをクリックすると、タイルで変換されたデータをプレビューすることができます。</Tip>

### ［上限］タイル（ベータ版）

<Frame>![limit tile.jpg](/images/kb/ja/0EMVq000003u5OH.jpg)</Frame>

［上限］タイルでは、出力に含めるデータの行数を制限できます。制限にオフセットを適用して、制限を開始するゼロから始まる行インデックスを指定することができます。［上限］タイルと[［順序］タイル](#order_tile)を一緒に使用して、出力に含める行を選択することができます。

<Warning>**重要：**制限を適用すると、DataSetの一番上の行のみが出力に含まれるようになります。[［順序］タイル](#order_tile)を使用してどの行を含めるか選択します。</Warning>

この例では、制限は25行に、オフセットは6に設定されています。つまり、出力に含まれるのは行6～30となります。

<Frame>![Screenshot](/images/kb/ja/0EMVq0000060dfR.jpg)</Frame>

［上限］タイルを構成するための手順は次の2つです。

1. 含める行数を入力します。これが制限です。
2. 制限を開始するゼロから始まる行インデックスを入力します。これがオフセットです。行1から含めていく場合は、空白のままにします。

<Tip>**ヒント：**プレビューを実行してキャンバスでタイルをクリックした後、**［プレビュー］**タブをクリックすると、タイルで変換されたデータをプレビューすることができます。</Tip>

### ［順序］タイル（ベータ版）

<Frame>![order tile.jpg](/images/kb/ja/0EMVq000003u0ja.jpg)</Frame>

［順序］タイルにより、特定の列の値を昇順または降順で表示できます。文字列照合順序とロケールには、高度なオプションもあります。

<Frame>![Screenshot](/images/kb/ja/0EMVq0000060eoP.jpg)</Frame>

複数の列を含めることができ、それらのリスト内での位置によって、並べ替えを行う順序をコントロールできます。つまり、最初の列が並べ替えられ、次にその並べ替えに合わせて、選択された2番目の列で行が並べ替えられる、というように並べ替えが続いていきます。

<Note>**注記：**［順序］タイルを使用して、DataFlowのダウンストリーム関数のパフォーマンスを向上させることもできます。値を順序付けすると、フィルターおよびJoinの操作のパフォーマンスが向上するためです。</Note>

<Tip>**ヒント：**プレビューを実行してキャンバスでタイルをクリックした後、**［プレビュー］**タブをクリックすると、タイルで変換されたデータをプレビューすることができます。</Tip>

### ［列を選択］タイル

<Frame>![Select_Columns.png](/images/kb/ja/0EM5w000005vNjv.png)</Frame>

［列を選択］タイルでは、出力に含める列を選択したり、列を並べ替えたり、列ヘッダー名を変更したりすることができます。

［列を選択］タイルをキャンバスにドラッグしてから選択してタイルエディターを開き、次の手順に従って設定します。

1. タイルを選択し、キャンバスの下にタイルエディターを開きます。タイルエディター内で残りの手順を実行します。
2. （オプション）タイルの名前を変更するには、<img alt="" src="/images/kb/ja/0EMVq0000060Lqd.jpg" style={{width: 23, height: 23, display: 'inline', verticalAlign: 'start', margin: '0'}}/>

   **編集**（鉛筆のアイコン）を選択して名前を入力します。
3. （オプション）含める（または除外する）列を選択するには、次のいずれかを行います。

   1. **［列を追加］**を選択して含めたい列を選択します。これを繰り返してほかの列を追加します。
   2. **［全ての列を追加］**を選択してから、列の**削除**（Xアイコン）を選択して列を除外します。  
        

      <Frame>![Screenshot](/images/kb/ja/0EMVq0000060ZS9.jpg)</Frame>
4. （オプション）列名の横にあるハンドルを使用して、列を構成に適した位置にドラッグして並べ替えます。  
     

   <Frame>![Screenshot](/images/kb/ja/0EMVq0000060ZaD.jpg)</Frame>
5. （オプション）列名を変更する場合は、該当するフィールドに各列の新しい名前を入力します。  
     

   <Frame>![Screenshot](/images/kb/ja/0EMVq0000060Zrx.jpg)</Frame>

<Tip>**ヒント：**プレビューを実行してキャンバスでタイルを選択した後、タイルエディターの**［プレビュー］**タブで表示させることにより、タイルで変換されたデータをプレビューすることができます。</Tip>

<Frame>![series tile.jpg](/images/kb/ja/0EMVq000003u7oI.jpg)</Frame>

### ［SQL］タイル

<Frame>![Screenshot](/images/kb/ja/0EMVq000005nFRG.jpg)</Frame>

［SQL］タイルでは、独自の自由形式のSQLを記述できます。MySQL風の構文を使用しており、ほとんどのMySQL関数を使用できます。また、Redshift構文、Common Table Expressions（CTE）、その他のほとんどの標準SQL関数の基本もサポートされています。

<Note>**注記：**［SQL］タイルを使用して、既存のSQL DataFlowをMagic ETLに変換できます。</Note>

## 既知の制限事項

- 分析/ウィンドウ関数はまだサポートされていません（今後のリリースで導入予定）。
- ストアドプロシージャと変数はサポートされていません。
- 非SELECTステートメントはサポートされていません。

<Tip>**ヒント：**Magic ETLでサポートされているSQL式については、[リファレンス資料](/s/article/000005809)を参照してください。</Tip>

## SQL DataFlowをMagic ETLに変換する

Magic ETLとしての編集オプションは、MySQLおよびRedshift DataFlowで使用できます。使用するには以下のステップに従います。

1. Data CenterでMySQLまたはRedshift DataFlowを開きます。  
     

   <Frame>![Screenshot](/images/kb/ja/0EMVq0000060p8j.jpg)</Frame>
2. <img alt="" src="/images/kb/ja/0EMVq000005nexV.jpg" style={{width: 23, height: 22, display: 'inline', verticalAlign: 'start', margin: '0'}}/>

   **オプション**（縦の三点リーダー）>**［Magic ETLとして編集］**を選択します。  
     

   <Frame>![Screenshot](/images/kb/ja/0EMVq0000060pFB.jpg)</Frame>

     
     
   DataFlowがMagic ETLキャンバスで開きます。  
     

   <Frame>![Screenshot](/images/kb/ja/0EMVq0000060l1u.jpg)</Frame>

**重要な注記：**

- 必要な表が入力としてキャンバスに取り込まれます。
- 既存の各変換は、独自の［SQL］タイルとして表示されます。
- サポートされていない操作や構文を変更するために、SQLの更新が必要になる場合があります。[SQL式のリファレンス](/s/article/000005809)を参照してください。
- この新しいDataFlowでは、**［名前を付けて保存］**オプションを使用すると、元のDataFlowが変更される代わりに、新しいMagic ETL DataFlowのコピーが保存されます。

### ［値マッパー］タイル

<Frame>![value mapper tile.jpg](/images/kb/ja/0EMVq000003u4f8.jpg)</Frame>

［値マッパー］タイルでは、入力した1組の文字列値に従って、指定した列の文字列値を検索して置換することができます（［値マッパー］操作は、略語の置換、言語コードの変換などに使用できます）。置き換える値は、既存の列を上書きするか、または新しい列に書き出すかを選ぶことができます。

<Tip>**ヒント：**テキスト値を［テキストを置換］タイルで置き換える方法については[こちら](/ja/s/article/360045402873#replace_text)、［列を複製］タイルで置き換える方法については[こちら](#duplicate_column)を参照してください。</Tip>

例えば、［値マッパー］タイルでは、以下の最初の画像のデータを2番目の画像のデータに変換します。

<Frame>![etl_valuemapper_datain.png](/images/kb/ja/0EM5w000005vNkW.png)</Frame>

 

<Frame>![etl_valuemapper_dataout.png](/images/kb/ja/0EM5w000005vNkN.png)</Frame>

<Note>**注記：**この例では、値が新しい列に書き出されています。設定された場合、元の列の値を上書きさせることもできます。</Note>

［値マッパー］タイルを設定するには、以下の手順に従います。

1. タイルを選択し、キャンバスの下にタイルエディターを開きます。タイルエディター内で残りの手順を実行します。
2. （オプション）タイルの名前を変更するには、<img alt="" src="/images/kb/ja/0EMVq0000060Lqd.jpg" style={{width: 23, height: 23, display: 'inline', verticalAlign: 'start', margin: '0'}}/>

   **編集**（鉛筆のアイコン）を選択して名前を入力します。
3. 検索対象の列を選択します。
4. 指定した列の値を上書きするか、新しい列に値を書き出すかを選択します。

   1. （条件付き）新しい列に値を書き出す場合、その列の名前を入力します。
5. 行に一致する値が見つからなかった場合に、元の値を書き込むかデフォルトの値を書き込むかを選択します。

   1. （条件付き）デフォルトの値を書き込むときは、その値を入力します。
6. 値マッパーで検索するそれぞれの値について、検索する値と置換後の値を入力します。
7. （条件付き）マッピングを追加する場合は、**［マッピングを追加］**をクリックします。

<Tip>**ヒント：**プレビューを実行してキャンバスでタイルをクリックした後、**［プレビュー］**タブをクリックすると、タイルで変換されたデータをプレビューすることができます。</Tip>
