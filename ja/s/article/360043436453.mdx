## はじめに

Anaplan Audit APIは、監査ログの配信メカニズムとして監査情報が頻繁に取得されるように設計されているため、独自のテクノロジーを利用してフィルタリング、分析、保存を行うことができます。Anaplan APIについて詳しくは、Webサイト（ [https://docs.anaplan.apiary.io/](https://anaplan.docs.apiary.io/# "https://docs.anaplan.apiary.io/") ）を参照してください。

Data CenterでAnaplanアカウントに接続します。このトピックでは、Anaplan Audit Serviceコネクターのユーザーインタフェース固有のフィールドとメニューについて説明します。DataSetの追加や更新スケジュールの設定、DataSet情報の編集に関する一般的な情報は、「 [データコネクターを使用してDataSetを追加する](/ja/s/article/360042926274) 」を参照してください。

## 必要条件

Anaplanアカウントに接続してDataSetを作成するには、Anaplanのユーザー名とパスワードが必要です。

## Anaplanアカウントに接続する

このセクションでは、Anaplan Audit Serviceコネクターページの **［認証情報］** および **［詳細］** ペインのオプションについて説明します。このページのほかのペインの項目にある **［スケジュール設定］** や **［DataSetの名前を付け説明を入力］** に関しては、ほとんどのタイプのコネクターで共通しています。詳細については、「 [データコネクターを使用してDataSetを追加する](/ja/s/article/360042926274 "データコネクターを使用してDataSetを追加する") 」を参照してください。

### 認証情報ペイン

このペインには、Anaplanアカウントに接続する認証情報を入力するフィールドが含まれています。下表では、各フィールドに必要な内容を説明しています。

| フィールド | 説明                   |
| ----- | -------------------- |
| ユーザー名 | Anaplanのユーザー名を入力します。 |
| パスワード | Anaplanのパスワードを入力します。 |

有効なAnaplan認証情報を一度入力しておくと、いつでも同じアカウントを使用して新しいAnaplan Audit Service DataSetを作成できます。コネクターアカウントは、Data Centerの **［アカウント］** タブで管理できます。このタブについて詳しくは、「 [コネクターのユーザーアカウントを管理する](/ja/s/article/360042926054 "コネクターのユーザーアカウントを管理する") 」を参照してください。

### ［詳細］ペイン

このペインには、メインとなる **［レポート］** メニューのほか、選択したレポートタイプに応じて表示/非表示になる様々なメニューが含まれています。

| メニュー           | 説明                                                                                                                                                                                                                                                                                                                                                                                       |
| -------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| レポート           | 実行するAnaplanレポートを選択します。以下のレポートが利用できます。<br/><table border="1" cellpadding="1" cellspacing="1"><tbody><tr><td colspan="1" rowspan="1"><p> 監査イベント </p></td><td colspan="1" rowspan="1"><p> ユーザーが関連付けられているテナントの監査イベントを取得します。 </p></td></tr><tr><td colspan="1" rowspan="1"><p> ユーザー </p></td><td colspan="1" rowspan="1"><p> Anaplan認証トークンに基づいてユーザーのリストを取得します。 </p></td></tr></tbody></table> |
| 監査イベント応答フォーマット | 監査イベントの出力応答フォーマットとして、CEFまたはJSONのいずれかを選択します。                                                                                                                                                                                                                                                                                                                                              |
| タイプ            | BYOK（Bring Your Own Key）またはユーザーアクティビティのいずれかで、返すイベントのタイプを選択します。                                                                                                                                                                                                                                                                                                                           |
| 開始日            | レポートの開始日を選択します。                                                                                                                                                                                                                                                                                                                                                                          |
| 終了日            | レポートの終了日を選択します。                                                                                                                                                                                                                                                                                                                                                                          |
| 時間間隔           | 返すイベントの時間間隔を入力します。例えば、 `2` を入力すると、レポートは過去2時間のイベントを返します。                                                                                                                                                                                                                                                                                                                                  |

### その他のペイン

スケジュール設定、再試行、更新に関するオプションなど、コネクターインターフェースのその他のペインに表示されるオプションについては、「 [データコネクターを使用してDataSetを追加する](/ja/s/article/360042926274) 」を参照してください。
