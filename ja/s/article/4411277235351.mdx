## はじめに

Cvent Inc.は、ミーティング、イベント、ホスピタリティーマネジメント技術を専門とする、SaaS上場企業です。DomoのCvent RESTコネクターを使用すると、出席者、連絡先、イベント、アンケートなどに関する様々なインサイトとメトリクスのデータを取得できます。

Cventデータを既存のCRMおよびマーケティングオートメーションシステムに統合することで、パフォーマンスの向上と意思決定の迅速化を実現します。CventデータとDomoの先進的なビジネスインテリジェンスプラットフォームを組み合わせることで、ビジネスを完全に最適化するための包括的な戦略を策定できるようになります。最も重要なKPIに合わせてカスタマイズされたレポートを数分で作成できます。APIの詳細については、[Cventのウェブサイト](http://www.cvent.com/en/event-management-software/cvent-integrations.shtml)を参照してください。

Cventアカウントには、Data Centerから接続します。このトピックでは、Cventコネクターのユーザーインターフェースに固有のフィールドとメニューについて説明します。DataSetの追加や更新スケジュールの設定、DataSet情報の編集に関する一般的な情報は、「[コネクターを使用してDataSetを追加する](/ja/s/article/360042926274)」を参照してください。

|                                |                                                                                                              |
| ------------------------------ | ------------------------------------------------------------------------------------------------------------ |
| **主な使用事例**                     | このコネクターを使用して、会議のメトリクスを分析およびレポートし、これらの大規模なミーティングデータから学習して最適化します。                                              |
| **主なメトリクス**                    | - イベントにサインアップしたのは何人ですか？ - 実際に参加したのは何人ですか？ - 各ミーティング、アンケート、イベントに参加したのは合計で何人ですか？ - アンケートでのフィードバックはどのような内容でしたか？ |
| **社内における主な権限**                 | 組織によって異なりますが、ほとんどの企業では、Cレベルの権限がどこかの時点でデータにアクセスできるようにする必要があります。                                               |
| **平均導入時間**                     | 1時間                                                                                                          |
| **使いやすさ（1～10の段階。1が「最も使いやすい」）** | 2                                                                                                            |

## 必要条件

Cventアカウントに接続してDataSetを作成するには、以下が必要です。

- Cventアカウント名
- Cvent Appに関連付けられているクライアントIDとクライアントシークレット
- Cventインスタンスの地域を選択する必要もあります。

DomoでCventに接続する前に、お住まいの地域の適切なIPアドレスをホワイトリストに登録する必要があります。次のウェブサイトで地域ごとのIPアドレスのリストにアクセスできます。 [「コネクターとフェデレーテッドアダプターのIPアドレスをホワイトリストに登録する」](/ja/s/article/360043630093)

## Cventアカウントに接続する

このセクションでは、Cvent RESTコネクターページの**［認証情報］**および**［詳細］**ペインのオプションについて説明します。このページのほかのペインの項目である**［スケジュール設定］**や**［DataSetに名前を付け、説明を入力］**に関しては、ほとんどのタイプのコネクターで共通しています。詳細については、「[コネクターを使用してDataSetを追加する](/ja/s/article/360042926274)」を参照してください。

## ［認証情報］ペイン

このペインには、Cventアカウントに接続する認証情報の入力フィールドが含まれています。下表では、各フィールドに必要な内容を説明しています。

| フィールド         | 説明                             |
| ------------- | ------------------------------ |
| Account Name  | Cventアカウントの名前を入力します。           |
| Region        | Cventインスタンスの地域を選択します。          |
| Client ID     | Cvent AppからクライアントIDを入力します。     |
| Client Secret | Cvent Appからクライアントシークレットを入力します。 |

これらの認証情報の入手方法については、前述の「必要条件」を参照してください。

有効なCvent認証情報を一度入力しておくと、いつでも同じアカウントを使用して新しいCvent REST DataSetを作成できます。コネクターアカウントは、Data Centerの**［アカウント］**タブで管理できます。このタブの詳細については、「[コネクター/アダプターアカウントを管理する](/ja/s/article/360042926054)」を参照してください。

詳細については、[https://domo-support.domo.com/s/article/360043630093](/ja/s/article/360043630093)を参照してください。
