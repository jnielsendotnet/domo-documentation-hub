### Nuevas funciones y mejoras

# Conectores: conector multicuentas avanzado de Facebook Ads

El nuevo conector multicuentas avanzado de Facebook Ads le permite obtener sus datos de cualquier número de cuentas en un DataSet. Esto simplifica su configuración de canalización de datos y obtiene todos sus datos de Facebook de manera automática.

# Elevación para cambios confidenciales

Con la función de elevación para cambios confidenciales, ahora es necesaria la elevación para acciones como el cambio de contraseña. Es un periodo de tiempo donde se pueden realizar acciones sensibles. Puede generar un código de acceso único generado por una aplicación de autenticación, e introducirlo para iniciar una elevación. Esta actualización mejora la seguridad para llevar a cabo acciones sensibles en su instancia.

Descubra cómo [registrar una aplicación de autenticación](https://domo-support.domo.com/s/article/000005831?language=en_US#register_domo) y utilice la función de [elevación](https://domo-support.domo.com/s/article/000005831?language=en_US#start_end_elevation).

# Plantillas de instancias

Las plantillas de instancias le permiten configurar cambios de función y comportamientos en Domo y crear múltiples plantillas para la función de instancias.

Después de configurar las plantillas de instancias, puede aplicarlas al crear nuevas instancias. También puede aplicar plantillas a instancias existentes, directamente desde la instancia principal.

Las siguientes funciones están disponibles para las plantillas de instancias:

* Personalizar los ajustes predeterminados de Domo para que sus instancias coincidan con su marca.
* Controlar a qué funciones tienen acceso los usuarios en su instancia gestionada.

Más información sobre las [plantillas de instancias.](https://domo-support.domo.com/s/article/000005786?language=en_US)



# Actualización de tablero v4 a v5

Como parte de nuestro ciclo regular de mantenimiento técnico, estamos actualizando los tableros de diseño v4 a diseño v5. Esta actualización no afecta a la funcionalidad principal de sus tableros, pero sí mejora la experiencia de editar para alinearse con App Studio.

Puede esperar las siguientes actualizaciones:

* Una interfaz de edición más intuitiva con identificadores que se pueden arrastrar.
* Diseños alineados para un mejor posicionamiento del contenido.
* Pequeños cambios en el tamaño del contenido del diseño. Esto puede requerir algunos ajustes manuales en los tableros existentes.

Recomendamos revisar los tableros tras la actualización para asegurar que todo se ve bien.

# Crear cliente de API

La creación y gestión de clientes de API ahora se puede administrar en su instancia de Domo, a través de los ajustes de administrador. Los usuarios con el permiso existente Crear cliente OAuth de API, incluido en los permisos de sistema de Domo, tienen acceso a la función de crear cliente de API.

Si es un administrador de Domo, puede utilizar esta función para establecer roles de usuario apropiados para su organización. También puede cambiar los permisos y roles de usuario según sea necesario para restringir la capacidad de los usuarios de su instancia de crear clientes de API.

<Frame><img alt="" src="/images/kb/de/ka0Vq000000HWJd-00N5w00000Ri7BU-0EMVq000006bfAP.jpg" style={{width: 318, height: 398}}/></Frame>
<Note>
  **Nota:** La capacidad de crear clientes de API está gobernada por el permiso Crear clientes OAuth de API, que se incluye en todas las funciones de sistema de Domo. Esta mejora no cambia la capacidad de sus usuarios de crear clientes de API (no cambia los permisos o funciones de usuario ya existentes). Sin embargo, como ahora se pueden crear clientes de API en Domo, es más probable que los usuarios vean esta capacidad. Revise las funciones de sus usuarios y realice los ajustes que considere necesarios.

</Note>

# Creador de informes para informes programados

El creador de informes en App Studio le permite crear reportes personalizables por correo electrónico utilizando sus aplicaciones.

El creador de informes le permite:

* Incluir múltiples páginas de aplicación
* Arrastrar, soltar, intercambiar, redimensionar y eliminar su contenido dentro del creador de informes
* Añadir más contenido para mejorar informes, incluidos encabezados, bordes, tarjetas y texto
* Configurar opciones para personalizar los encabezados y pies de los correos electrónicos con logotipos y contenido
* Añadir un botón que enlace a contenido de Domo, incluidos la propia aplicación, una tarjeta o un tablero
* Añadir componentes de temas, incluidos estilos de tarjeta, colores de fondo y densidad, así como importar temas de cualquier aplicación para sincronizar las paletas de colores
* Añadir filtros y programar el informe para usuarios y grupos

# API de SCIM: grupos y mejoras de usuario

SCIM, sistema para la gestión de identidades en distintos dominios, es un protocolo que permite a un proveedor de identidad (IdP) impulsar la creación, actualización y eliminación de grupos y usuarios entre sistemas de software. SCIM está disponible en Domo y se han implementado los extremos de grupo, lo que permite crear y eliminar grupos de directorio y reflejar los cambios en los miembros de grupo en el IdP, sin necesidad de iniciar sesión en Domo.

Más información sobre [SCIM en Domo](/s/article/000005241).
