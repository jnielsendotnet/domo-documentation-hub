## Dojo pasa a ser los Foros de la comunidad

Con el lanzamiento de marzo de 2023, presentamos los [Foros de la comunidad](https://dojo.domo.com/main), que reemplaza lo que se conocía como el Dojo.

Puede encontrar el Foro de la comunidad ubicado en Ayuda en Domo Central. Con este cambio de marca, hemos actualizado la interfaz de usuario y la experiencia de usuario para que sea más fácil encontrar el contenido que busca. Ahora, los [Grupos sociales](http://dojo.domo.com/main/groups) y las [Grabaciones de eventos](http://dojo.domo.com/main/categories/recordings) recientes se encuentran disponibles en el Foro de la comunidad.

## Nuevas características

Estas características son parte del lanzamiento de marzo de 2023.

### Complementos de Microsoft Office

Los complementos de Domo para Microsoft 365 le permiten integrar sus productos de Microsoft Office con Domo. En lugar de mover los datos manualmente, ahora puede transferirlos de manera eficiente entre Excel, Outlook, PowerPoint, Word y Domo. Por ejemplo, puede usar los complementos para importar tarjetas y DataSets a un documento de Word, o puede importar datos de una hoja de cálculo de Excel a Domo.

Para obtener más información, consulte la  [Guía de instalación de complementos de Domo para Microsoft 365](/es/s/article/000005146) y la [Guía del usuario de complementos de Domo para las aplicaciones de escritorio de Microsoft 365](/es/s/article/000005143). También puede buscar Domo en Microsoft AppSource.

Los clientes empresariales pueden implementar los nuevos complementos de Domo en su organización, lo que permite a los usuarios acceder a los complementos directamente desde sus aplicaciones de Microsoft 365 sin tener que instalarlos.

Para obtener más información, consulte la [Guía de instalación de complementos de Domo para Microsoft 365 Enterprise](/es/s/article/000005145).

### Atributos personalizados y atributos de lista para grupos

En la sección **Gobernanza** de la **Configuración del administrador**, ahora puede usar atributos para controlar la pertenencia a grupos dinámicos, como asignar a todos los usuarios en un determinado departamento o ubicación para que formen parte de un grupo que recibe acceso a ciertos tipos de contenido. Los administradores de Domo pueden importar los atributos del IdP de su organización para ayudar a asignar usuarios automáticamente a los grupos correctos.

Para obtener más información, consulte [Configuración del administrador: Gobernanza](/es/s/article/000005147).

### Cálculos de tabla

Los cálculos de tabla le permiten mostrar un valor en la tabla como un porcentaje del total de la columna, parte de un total acumulado o cómo se clasifica frente a otros valores en la columna, todo sin utilizar los cálculos de Beast Mode o DataFlows. Los cálculos de tabla están disponibles para Megatablas y Tablas dinámicas.

Para obtener más información sobre los cálculos de tablas, consulte [Diagrama de tabla](/es/s/article/360043429573).

### Nuevos tipos de diagramas

#### Diagrama Sankey circular

El diagrama Sankey circular se basa en el diagrama Sankey original al capturar conceptos recurrentes, que requieren flujos que regresan a los nodos anteriores. Para obtener más información, consulte nuestro artículo [Diagrama Sankey circular](/es/s/article/000005155).

#### Diagrama de pendientes

Un diagrama de pendientes es una visualización de línea simple de un período de tiempo que se basa en los puntos de inicio y finalización del período. Este diagrama de fácil lectura elimina el desorden de los puntos intermedios, lo que le permite concentrarse en la información que importa. Para obtener más información, consulte nuestro artículo [Diagrama de pendientes](/es/s/article/000005154).

<Frame>![release notes 3.png](/images/kb/es/0EM5w000006uPis.jpg)</Frame>

#### Diagrama de varianza

Los diagramas de varianza le permiten resaltar las diferencias entre dos conjuntos de datos. Para obtener más información, consulte nuestro artículo [Diagrama de varianza](/es/s/article/000005156).

## Nuevas características premium

### Domo Everywhere

#### Publicaciones

- Las suscripciones automáticas permiten a las organizaciones suscriptoras suscribirse a todas las publicaciones de una organización editorial específica y recibir cualquier contenido publicado automáticamente sin tener que aceptar las publicaciones individualmente.
- Junto con las suscripciones automáticas, las nuevas Reglas de uso compartido predeterminadas permiten que una organización suscriptora envíe automáticamente todas las publicaciones de una organización editorial específica a cualquier número de usuarios o grupos
- La nueva función de clasificación le permite ver de un vistazo cuándo actualizó por última vez una publicación. También puede ver si una publicación tiene un estado de error y para cuáles suscriptores.

Para obtener más información, consulte [Publicar contenido en la organización de un suscriptor](http://domo-support.domo.com/s/article/360045120554?language=en_US)**.**

El cambio programático de DataSet ya está disponible. Para obtener más información, consulte [Cambio de DataSet](http://developer.domo.com/docs/embed/dataset-switching) en el portal del desarrollador.

Ahora, Domo Everywhere admite la publicación de Domo Bricks.

## <Frame>![release notes 5.jpeg](/images/kb/es/0EM5w000006uPjC.jpg)</Frame> Características beta

Estas características están en la versión beta a partir del lanzamiento de marzo de 2023.

### Procesamiento de subconjunto

El Procesamiento de subconjunto está en una versión beta abierta. Esta característica está dentro de Magic – Extracción, transformación y carga, y le permite actualizar solo ciertas subsecciones de sus datos cuando sea necesario, en lugar de todo el DataSet, lo que ahorra tiempo y aumenta la eficiencia. Puede crear particiones en sus datos y seleccionar qué particiones actualizar durante el procesamiento.

Para obtener más información, consulte [Procesamiento de subconjuntos en Magic – Extracción, transformación y carga](/es/s/article/000005150).

Para participar en la versión beta, siga este enlace: [Versión beta del procesamiento de subconjuntos](http://domo.az1.qualtrics.com/jfe/form/SV_exOdxemYlvewU8S.).

### Flujos de trabajo

Los flujos de trabajo aprovechan poderosamente otros productos nuevos, incluidos el Motor de códigos, las Colas de trabajo y los Formularios (todos en versión beta abierta), para permitirle diseñar e implementar procesos de flujo de trabajo ejecutables con capacidades de código bajo. Puede iniciar un flujo de trabajo manualmente, a través de una Alerta, a través de un flujo de aprobación o con un temporizador. Con los flujos de trabajo, puede guiarse por sus datos para resolver la automatización de procesos, canalizaciones de datos inteligentes, acciones de aplicaciones avanzadas y administración manual de tareas sin depender de los desarrolladores.

Para obtener más información, consulte  [Descripción general de los flujos de trabajo](/es/s/article/000005108).

### Motor de códigos

El Motor de códigos es un entorno de tiempo de ejecución nativo de Domo que puede ejecutar código JavaScript o Python e interactuar con cualquier característica dentro o fuera de Domo. Domo proporciona una biblioteca general de paquetes y funciones para integraciones y servicios comunes, pero también puede crear sus propias funciones para usar en su instancia de Domo. Puede usar una función del Motor de códigos en los flujos de trabajo sin tener que escribir ningún código.

Para obtener más información, consulte [Motor de códigos](http://).

<Frame>![release notes 7.jpeg](/images/kb/es/0EM5w000006uPg4.jpg)</Frame>

### Formularios

El nuevo Editor de formularios le permite crear y publicar formularios para interacciones manuales personalizadas. Puede integrar formularios con un flujo de trabajo para que cuando un flujo de trabajo requiera una tarea manual, se genere un formulario automáticamente y se presente a un usuario final. También puede enviar datos a un flujo de trabajo mediante un formulario, lo que le permite hacer referencia a la información en otros pasos.

Para obtener más información, consulte [Formularios](http://).

<Frame>![release notes 8.jpeg](/images/kb/es/0EM5w000006uPjg.jpg)</Frame>

### Colas

Las colas de trabajo le permiten organizar tareas manuales en diferentes colas y controlar quién puede acceder a ellas. Puede configurar una cola para mostrar tareas para una persona o un grupo. Cada tarea está asociada a un formulario generado por un flujo de trabajo. Puede filtrar las colas para ver las tareas por cesionario o estado, monitorear el volumen de la cola y personalizar los diseños.

Para obtener más información, consulte [Colas](http://).
