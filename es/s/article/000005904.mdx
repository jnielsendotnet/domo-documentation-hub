### Nuevas funciones y mejoras

# Administración y gobernanza

#### Permisos de actualización de datos

Los permisos de actualización de datos permiten a los administradores configurar permisos para las opciones de frecuencia del calendario de conectores y flujos de datos que los usuarios y grupos tienen a su disposición en su instancia. 

Esto permite a los administradores controlar la cantidad de uso futuro de la instancia de Domo de cada usuario. 

Consulte información sobre cómo configurar los [permisos de actualización de datos](/s/article/000005492). 

<Frame>![Screenshot](/images/kb/es/0EMVq000007pBDV.jpg)</Frame>



#### Lista de funciones permitidas

La lista de funciones permitidas permite a determinados usuarios asignar una lista de funciones a usuarios de su instancia. Puede configurar estas funciones mediante el permiso "Administrar todas las funciones" a nivel de administrador.

Los usuarios que tengan el permiso "Asignar usuarios a una función" solo pueden asignar las funciones que se encuentren en la lista de funciones permitidas.

En cambio, los usuarios que tengan el permiso "Administrar todas las funciones" pueden asignar cualquier función y configurar la lista de funciones permitidas. 

Con la lista de funciones permitidas y el permiso relacionado "Asignar usuarios a una función", los administradores de Domo pueden delegar la administración de usuarios —incluida la asignación de funciones de usuario— a usuarios que no sean administradores. 

<Frame>![Screenshot](/images/kb/es/0EMVq000007ifoD.jpg)</Frame>

Consulte más información sobre la [lista de funciones permitidas](/es/s/article/000005839). 



#### Configuración de SSO

La nueva experiencia de usuario para la configuración del SSO de Domo es más moderna e intuitiva. Utiliza la terminología estándar del sector e incluye controles y conjuntos de herramientas centrados en la experiencia de inicio de sesión del usuario final.

Esta actualización permite a los administradores utilizar una implementación de autoservicio del SSO y reduce la dependencia de los equipos de soporte de Domo.

<Frame>![Screenshot](/images/kb/es/0EMVq000006om1C.jpg)</Frame>

Consulte más información sobre el [SSO de Domo](/es/s/article/000005827).



# Crear cliente de API

La creación y administración de clientes de APi ahora se puede gestionar en los ajustes de administrador de la instancia de Domo. Los usuarios que tengan el permiso "Crear clientes de API OAuth", incluido en las funciones del sistema de Domo, pueden acceder a la función "Crear cliente de API".

Puede restringir la capacidad de los usuarios de su instancia para crear clientes de API si modifica sus funciones y permisos de usuario según proceda.

<Frame>![Screenshot](/images/kb/es/0EMVq000006bfAP.jpg)</Frame>

Consulte información sobre cómo [crear clientes de API](/es/s/article/000005240).

<Note>
**Nota:** La capacidad de crear clientes de API está gobernada por el permiso Crear clientes OAuth de API, que se incluye en todas las funciones de sistema de Domo. Esta mejora no cambia la capacidad de sus usuarios de crear clientes de API (no cambia los permisos o funciones de usuario ya existentes). Sin embargo, como ahora se pueden crear clientes de API en Domo, es más probable que los usuarios vean esta capacidad. Revise las funciones de sus usuarios y realice los ajustes que considere necesarios.
</Note>

# Funcionalidad de clonación de DataSet

La nueva funcionalidad de clonación de DataSet orientada al cliente le permite crear un nuevo DataSet que sea una copia exacta de un DataSet existente. El nuevo DataSet copiado mantiene la misma configuración que el original, salvo los ajustes de calendario, por lo que le ahorra tener que volver a crear los DataSets manualmente.

# Diseño de tableros v5

Como parte de nuestro ciclo regular de mantenimiento técnico, estamos actualizando los tableros de diseño v4 a diseño v5. Esta actualización no afecta a la funcionalidad principal de sus tableros, pero sí mejora la experiencia de editar para alinearse con App Studio.

Puede esperar las siguientes actualizaciones:

- Una interfaz de edición más intuitiva con identificadores que se pueden arrastrar.
- Diseños alineados para un mejor posicionamiento del contenido.
- Pequeños cambios en el tamaño del contenido del diseño. Esto puede requerir algunos ajustes manuales en los tableros existentes.

Recomendamos revisar los tableros tras la actualización para asegurar que todo se ve bien.



# Mejoras en filtros y variables

<Note>
**Nota:** Estas funciones no son compatibles actualmente para móviles, pero pretendemos habilitarlas para móviles en el futuro.
</Note>

#### Control avanzado de filtros de fecha

Las opciones de periodos de tiempo anteriores ahora están disponibles en Analyzer, lo que le permite filtrar datos excluyendo periodos específicos, como el trimestre o el día actuales. Los periodos con el filtro "Anterior" siempre excluyen el periodo actual (hoy, semana actual, mes actual, trimestre actual y año actual).

La opción de filtro "Último" se ha movido al principio de la lista de filtros y abarca el periodo actual de forma predeterminada.

Esta actualización está disponible en el menú desplegable de rangos de fecha en Analyzer, en los tableros y en App Studio.

<Frame>![Screenshot](/images/kb/es/0EMVq000007gNSP.jpg)</Frame>



#### Mejoras de variables de fecha

Las nuevas mejoras de las variables de fecha le permiten utilizar la sintaxis para definir fechas relativas mediante las variables de fecha. Esto le aporta mayor flexibilidad y la posibilidad de utilizar una sintaxis de fechas relativas para definir periodos de tiempo dinámicos en los datos mediante el editor de variables.

<Frame>![Screenshot](/images/kb/es/0EMVq000007gN7S.jpg)</Frame>



#### Contenido de los filtros de página

La función de contenido de los filtros de página el permite seleccionar una columna específica y aplicar un filtro "contiene" para buscar coincidencias con valores específicos del filtro de su DataSet, lo que le permite filtrar rápidamente valores únicos.

# Nuevas opciones de iconos de Magic - Extracción, transformación y carga

Las siguientes actualizaciones para la ampliación de los iconos están disponibles en Magic - Extracción, transformación y carga:

- **Dividir unión**: administre varios flujos de salida desde una única unión e identifique las filas que están unidas y las que no lo están.
- **Dividir filtro**: capture las filas que cumplen los criterios de filtro y visualice las que no utilizan un flujo independiente.
- **Clasificación y ventanas**: añada nuevas columnas con funciones de clasificación, delimitación o desplazamiento directamente desde su DataSet.
- **Dinamización**: transforme los pares clave-valor en varias columnas.
- **Anular dinamización**transforme columnas en filas.
- **Escritura diferida**transforme la salida de sus datos en un nuevo DataSet y envíelo a otro sistema de integración de datos.
- **Anexión de filas mejorada**: gestione uniones de DataSets diferentes con el nuevo icono de anexión de filas.
