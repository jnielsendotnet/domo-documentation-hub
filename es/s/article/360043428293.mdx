## Introducción

Domo incluye dos métodos distintos para combinar datos mediante DataFusions: "Agregar columnas" y "Agregar filas". "Agregar columnas" combina dos o más DataSets mediante la asociación de columnas similares. Con este método, solo las columnas coincidentes deben tener el mismo encabezado o tipo de datos. "Agregar filas" combina DataSets mediante la unión de las filas de dichos DataSets. Con este método, todas las columnas deben tener el mismo encabezado.

<Note>
**Nota:** Si alguno de los DataSets de un DataFusion está desactualizado, el propio DataFusion también aparecerá como desactualizado.
</Note>

## Combinación de DataSets con el método Combinar columnas

En este método, seleccione dos o más DataSets, cada uno de los cuales contiene al menos una columna que coincide con una columna o un tipo de datos de alguno de los otros. A continuación, combine todos los DataSets en un único DataSet mediante la asociación de columnas similares. Este método es equivalente a la utilización de una instrucción JOIN en SQL.

En el ejemplo siguiente, un usuario sigue este método para cargar y combinar dos DataSets. El usuario asocia las columnas "N.º de pedido" y "Número de pedido", y el segundo DataSet aparece a la derecha del primero.

**Para combinar DataSets con el método Agregar columnas:**

1. Haga clic en <img alt="new_app_icon.png" src="/images/kb/es/0EM5w000005wKLC.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/> **> Data Center**.
2. Haga clic en **Combinar** en la barra de herramientas **Transformación Magic** en la parte superior de la ventana.

   <Tip>
   **Consejo:** También puede abrir el editor de DataFusion desde cualquier ubicación de Domo al seleccionar en la barra de herramientas de la aplicación y, a continuación, **Datos > Combinar**.
   </Tip>
3. (Opcional) Cambie el título predeterminado del DataSet.  
   El título se usa para buscar o seleccionar el DataSet al alimentar una tarjeta.
4. Agregue el DataSet "principal" o el primero que desee agregar; para ello, siga estos pasos:

   1. Haga clic en **Seleccionar DataSet**.
   2. Seleccione el DataSet en la lista.  
      Puede filtrar los DataSets que aparecen en la lista; para ello, seleccione el DataSet en el menú **Todos los tipos** o escriba su nombre en el campo **Buscar**.  
      Aparece una lista de los nombres de columna del DataSet.
5. Siga estos pasos para agregar un DataSet secundario:

   1. Haga clic en **Seleccionar DataSet**.
   2. Seleccione el DataSet en la lista.  
      Puede filtrar los DataSets que aparecen en la lista; para ello, seleccione el DataSet en el menú **Todos los tipos** o el propietario en el menú **Todos los usuarios** o escriba el nombre del DataSet en el campo **Buscar**.  
      Aparece una lista de los nombres de columna del DataSet.
6. (Opcional) Agregue DataSets adicionales para combinarlos.
7. (Opcional) Haga clic en el icono **X** para quitar los DataSets seleccionados.
8. Especifique la configuración para combinar DataSets.

   1. Determine qué columnas de los DataSets contienen los mismos valores o tipos de datos.  
      Puede consultar las listas de nombres de columnas de cada DataSet para obtener ayuda.
   2. Para cada combinación de DataSet que desee, seleccione las columnas que desea asociar.

      1. Haga clic en el primer menú **Seleccionar un campo**, seleccione el DataSet principal y, a continuación, una columna.
      2. Haga clic en el segundo menú **Seleccionar un campo**, seleccione el DataSet secundario y, por último, la columna con los valores o el tipo de datos que coinciden con los de la primera columna seleccionada.
      3. Seleccione la opción que desea incluir en el menú <img alt="join_options_icon.png" src="/images/kb/es/0EM5w000005wKL7.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/>.

         | Opción | Descripción |
         | --- | --- |
         | Combinación externa izquierda | Incluye todas las filas en el primer DataSet seleccionado y agrega las filas con el campo del valor coincidente especificado del segundo DataSet seleccionado. |
         | Combinación interna | Incluye las filas de ambos DataSets con el campo del valor coincidente especificado. |
   3. (Condicional) Si ha agregado más de un DataSet secundario, repita los pasos para las columnas coincidentes de cada par adicional de menús **Seleccionar un campo** que aparecen.
9. Seleccione **Haga clic para cargar la vista previa** a fin de previsualizar el DataSet.  
   La opción está disponible cuando las columnas coincidentes se seleccionan correctamente para todos los DataSets seleccionados.
10. (Opcional) Haga clic en la flecha situada a la derecha de cualquier columna en la vista previa para configurar sus opciones.

    | Opción | Descripción |
    | --- | --- |
    | Cambiar el nombre de la columna | Permite cambiar el nombre de esta columna. Solo puede cambiar el nombre de las columnas antes de guardar el DataFusion. Esta opción dejará de estar disponible después de guardar. |
    | Omitir columna | Excluye esta columna del DataSet combinado. |
11. Haga clic en **Guardar** para guardar el DataSet combinado.

El nuevo DataSet se agrega a la pestaña **DataSets** en Data Center. Para obtener más información sobre esta pestaña, consulte [Diseño de Data Center](https://domohelp.domo.com/hc/en-us/search/click?data=BAh7CzoHaWRsKwjlqoHUUwA6D2FjY291bnRfaWRpA%2FtFkjoJdHlwZUkiDGFydGljbGUGOgZFVDoIdXJsSSJQaHR0cHM6Ly9kb21vaGVscC5kb21vLmNvbS9oYy9lbi11cy9hcnRpY2xlcy8zNjAwNDc1NTMyNTMtRGF0YS1DZW50ZXItTGF5b3V0BjsIVDoOc2VhcmNoX2lkSSIpOTIyYjAxYmQtYjY5My00ODY3LWJhZmEtNDA0MTc3OWMyOWE5BjsIRjoJcmFua2kG--762db1d168d36319c5a6f74c8e38a56908fe0161).

## Combinación de DataSets con el método Agregar filas

En este método, seleccione dos o más DataSets en los que todos los encabezados de columna sean idénticos. Domo combina posteriormente las filas de cada DataSet en un único DataSet. Este método es equivalente a la utilización de una instrucción UNION en SQL.

En el ejemplo siguiente, un usuario sigue este método para cargar y combinar dos DataSets. El segundo DataSet se adjunta directamente debajo del primero.

<Frame>![combine_rows_result.png](/images/kb/es/0EM5w000005wKLB.png)</Frame>

<Warning>
**Importante:** Las columnas de todos los DataSet combinados mediante este método *deben* tener el mismo encabezado y orden. Además, las columnas coincidentes también deben contener los mismos tipos de datos. Por ejemplo, si una columna de un DataSet tiene el formato de FECHA, esa misma columna de todos los demás DataSets también debe tener ese mismo formato. Los tipos de datos posibles son los siguientes:

- LARGO almacena números como valores numéricos, incluidos decimales, fraccionales y enteros.
- DOBLE almacena valores numéricos de punto flotante con doble precisión.
- CADENA almacena caracteres alfanuméricos como texto.
- FECHA Y HORA almacena valores de año, mes, día, hora, minuto y segundo.
- FECHA almacena valores de año, mes y día.
- HORA almacena valores de hora, minuto y segundo.
</Warning>

**Para combinar DataSets con el método Combinar filas:**

1. Haga clic en <img alt="new_app_icon.png" src="/images/kb/es/0EM5w000005wKLC.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/> **> Data Center**.
2. En el menú **Nuevo DataSet**, haga clic en <img alt="datasourcemenuicon.png" src="/images/kb/es/0EM5w000005wKL4.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/> y, a continuación, seleccione **Combinar DataSets**.
3. (Opcional) Cambie el título predeterminado del DataSet.  
   El título se usa para buscar o seleccionar el DataSet al alimentar una tarjeta.
4. Agregue el primer DataSet; para ello, siga estos pasos:

   1. Haga clic en **Seleccionar DataSet**.
   2. Seleccione el DataSet en la lista.  
      Puede filtrar los DataSets que aparecen en la lista; para ello, seleccione el DataSet en el menú **Todos los tipos** o el propietario en el menú **Todos los usuarios** o escriba el nombre del DataSet en el campo **Buscar**.  
      Aparece una lista de los nombres de columna del DataSet.
5. Haga clic en **Seleccionar DataSet** y seleccione el DataSet de la lista, como lo ha hecho anteriormente, para agregar un DataSet secundario.

   <Note>
   **Nota:** Las columnas de todos los DataSets agregados con este método *deben* tener los mismos encabezados o, de lo contrario, no podrá previsualizar ni guardar la fusión.
   </Note>
6. (Opcional) Agregue DataSets adicionales para combinarlos.
7. Haga clic en **Combinar filas**.  
   Si ha seleccionado correctamente dos DataSets con los mismos encabezados de columna, aparece un vínculo a continuación en el que se lee **Haga clic para cargar la vista previa**.
8. Seleccione **Haga clic para cargar la vista previa** a fin de previsualizar el DataSet.
9. (Opcional) Haga clic en la flecha situada a la derecha de cualquier columna en la vista previa para configurar sus opciones.

   | Opción | Descripción |
   | --- | --- |
   | Omitir columna | Excluye esta columna del DataSet combinado. |
10. Haga clic en **Guardar** para guardar el DataSet combinado.

El nuevo DataSet se agrega a la pestaña **DataSets** en Data Center. Para obtener más información sobre esta pestaña, consulte [Diseño de Data Center](https://domohelp.domo.com/hc/en-us/search/click?data=BAh7CzoHaWRsKwjlqoHUUwA6D2FjY291bnRfaWRpA%2FtFkjoJdHlwZUkiDGFydGljbGUGOgZFVDoIdXJsSSJQaHR0cHM6Ly9kb21vaGVscC5kb21vLmNvbS9oYy9lbi11cy9hcnRpY2xlcy8zNjAwNDc1NTMyNTMtRGF0YS1DZW50ZXItTGF5b3V0BjsIVDoOc2VhcmNoX2lkSSIpOTIyYjAxYmQtYjY5My00ODY3LWJhZmEtNDA0MTc3OWMyOWE5BjsIRjoJcmFua2kG--762db1d168d36319c5a6f74c8e38a56908fe0161).
