## Nuevas características y mejoras

Las características y mejoras de esta versión incluyen lo siguiente:

### Motor de automatización empresarial: acciones de alerta

Las acciones de alerta impulsan acciones poderosas para su negocio al activar eventos como la integración de Webhook con sistemas de terceros e integraciones de proyectos y tareas en Domo. Las posibilidades de acciones de alerta son literalmente infinitas. Por ejemplo...

* Como gerente de una tienda minorista, es posible que ya tenga una alerta que le notifique cuando un determinado producto haya alcanzado un umbral de unidades vendidas; con las acciones de alerta puede ampliar esta alerta básica haciendo que también active una acción de  "reordenar" en su sistema de inventario cuando se active.
* Como líder de ventas, puede establecer una acción de alerta que haga que suenen sirenas o música en su oficina cuando haya cerrado un negocio importante para que todos lo puedan celebrar juntos.
* Como jefe de soporte, puede establecer acciones de alerta para crear tareas de forma automática en Domo y asignarlas a personas específicas siempre que el número de errores abiertos exceda una cantidad determinada.

Actualmente hay dos tipos de acciones de alerta disponibles en Domo: Webhooks y Tareas. Con la acción Webhook, puede conectarse a cualquiera de varios sistemas de terceros. Con las acciones Tarea, puede crear una tarea en un proyecto específico y asignarla a una persona determinada. Configure estas acciones en el asistente de alertas cuando cree una alerta o edite una alerta existente.

### Navegación actualizada

La nueva navegación de Domo facilita la organización y administración de sus páginas para que sea más fácil encontrarlas, independientemente de si tiene unas pocas páginas o cientos de ellas. También permite navegar a las características más usadas de Domo con unos pocos clics, lo que reduce el tiempo que toma realizar acciones en Domo y obtener respuestas. Para obtener más información, vea este [vídeo de demostración](https://youtu.be/0Ekqq3CvvV4).

La navegación actualizada estará disponible para todos los clientes en nuestra versión de octubre. Debido a la magnitud de este cambio, la nueva estructura de navegación reemplazará oficialmente a la antigua estructura en marzo de 2020. Hasta entonces, todos los clientes tienen la opción de activar la nueva navegación en Configuración del administrador seleccionando **Configuración de la empresa > Control de características**, y luego activando la opción "Habilitar la navegación actualizada para todos los usuarios". (Debe tener privilegios de seguridad de administrador para hacer esto). De esta manera, su organización tendrá la libertad de hacer la transición cuando esté lista. Tenga en cuenta que las páginas favoritas y de tercer nivel solo se admiten en la navegación actualizada, de modo que si vuelve a la navegación anterior desde la navegación nueva, estas páginas no estarán visibles en la interfaz de usuario.

### Transmisión de vídeos P4 en Doc Cards

Cree fácilmente más claridad para sus lectores con transmisión de vídeos MP4 en Doc Cards. Puede ver estos vídeos en el nivel de página (se admiten páginas estándar e historias de Domo) o en la vista Detalles de Doc Card. También puede elegir una imagen personalizada para la miniatura de su vídeo o dejar que Domo elija una miniatura predeterminada para usted.

Los vídeos se pueden ver en las versiones web y móvil de Domo. Tenga en cuenta que *solo* se admiten vídeos MP4 en este momento (no AVI, WMV, QuickTime, etc.). Si su vídeo no es un archivo MP4, no se transmitirá; solo verá el ícono del archivo. También tenga en cuenta que el límite de tamaño de archivo para vídeos es de 1 GB y el límite para imágenes en miniatura personalizadas es de 10 MB.

### Conector JSON sin código

Conecte datos personalizados a Domo sin escribir ningún código mediante el conector JSON sin código.

Hasta ahora, si deseaba incorporar sus datos personalizados a Domo, tenía que pedirle a alguien de Domo que creara un conector personalizado o crear la conexión utilizando nuestra API o IDE de conector, ambas opciones con uso intensivo de códigos. Ahora puede extraer sus datos personalizados siguiendo los pasos en una sencilla interfaz de asistencia. El asistente le ayudará a conectarse a sus datos como un archivo JSON e incorporarlo a Domo.

Para usar esta herramienta, necesitará lo siguiente:

* Un identificador del cliente y una clave secreta del cliente desde
  [developer.domo.com](http://developer.domo.com)
* Una URL de autorización y una URL de identificación de acceso de su proveedor de datos
* La URL para acceder al archivo JSON

Acceda a la interfaz de este conector buscando "Conector JSON OAuth sin código" en la AppStore.

<Frame><img alt="no_code_json_tile_rn.png" src="/images/kb/ka05w00000129uT-00N5w00000Ri7BU-0EM5w000005vOjQ.png"/></Frame>
Para obtener más información, consulte [Conector JSON sin código](/s/article/360052105454 "Conector JSON sin código").

### DomoStats: DataSets y DataFlows

La aplicación DataSets y DataFlows, la última adición a nuestro repertorio en constante expansión de DomoStatsApps, viene con un tablero prediseñado que permite a los usuarios administradores…

* Limpiar Domo rápidamente al ver qué DataSets y DataFlows necesitan atención, si están huérfanos o duplicados, etc., y luego seleccionar un vínculo directo para tomar medidas inmediatas.
* Comprender hacia dónde dirigir los esfuerzos para arreglar DataSets y DataFlows con tarjetas que muestran lo que necesita atención en función de los recuentos de tarjetas, propietarios, etc.
* Mantenga la seguridad de Domo observando quién está creando datos y qué datos están llegando a Domo.

¿Está listo para probar esta aplicación? Simplemente busque "DomoStats" en la AppStore, seleccione la aplicación "DomoStats: DataSets y DataFlows", y haga clic en **Probar**. Tenga en cuenta que debe tener un rol de seguridad de nivel "Administrador" para implementar aplicaciones DomoStats en Domo.

<Frame><img alt="domostats_datasets_rn.png" src="/images/kb/ka05w00000129uT-00N5w00000Ri7BU-0EM5w000005vOjO.png"/></Frame>
Para obtener más información, consulte DomoStats: DataSets y DataFlows (BETA).
