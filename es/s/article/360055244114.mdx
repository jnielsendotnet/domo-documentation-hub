# Nuevas características y mejoras

Las características y mejoras de esta versión incluyen lo siguiente:

## Mejoras de las tarjetas

Mejore sus diagramas para contar historias muy atractivas.

### Tarjeta de tabla

#### Espaciado interno

Incorpore el espaciado interno en sus tablas para que luzcan un nuevo aspecto organizado. Ahora puede controlar más que nunca el aspecto de las tablas. Para obtener más información, consulte Diagrama de tabla.
  
  <Frame><img alt="Padding_Sizes.png" src="/images/kb/ka0Vq0000000cJp-00N5w00000Ri7BU-0EM5w000005vOiY.png"/></Frame>
  <Frame><img alt="Padding_Examples.png" src="/images/kb/ka0Vq0000000cJp-00N5w00000Ri7BU-0EM5w000005vOiT.png"/></Frame>

#### Paginación

Interactúe con tablas más grandes que nunca con la paginación dinámica de filas de datos en las tarjetas de tabla mega.

#### Copiar

La opción de copia se ha añadido a las tablas mega para facilitar el movimiento de los datos con la acción de copiar y pegar.

### Indicador de valor múltiple

Ahora dispone de más opciones personalizables para mejorar el aspecto del texto en el diagrama. Elija entre las nuevas opciones Negrita y Cursiva. Para obtener más información, consulte Indicador de valor múltiple.

### Tarjeta de mapa

Nuevas opciones para animar los mapas, como: opciones de borde, escalas de valores mínimos y máximos, y permitir tamaños tres veces más grandes en las burbujas de los mapas de latitud y longitud para crear una historia de datos convincente. Destaque los números más grandes de una forma nueva y atrevida. Para obtener más información, consulte Mapa de latitud y longitud.

### Aumento del límite de los puntos de datos de las tarjetas

Ahora tiene la capacidad de representar volúmenes de datos muy grandes en los diagramas. Ahora puede usar el comando **Limitar filas** dentro de Analyzer para definir valores más grandes de lo que se permitirían de cualquier otra forma. Para mantener una capacidad de respuesta rápida en los tableros, el motor del diagrama limita el número de filas que descarga en el navegador. Ahora puede anular estos límites con la configuración explícita de estos valores. Para obtener más información, consulte Preguntas frecuentes sobre la creación de tarjetas.
  
<Frame><img alt="Limit_Rows.png" src="/images/kb/ka0Vq0000000cJp-00N5w00000Ri7BU-0EM5w000005vOiZ.png"/></Frame>

## Indicadores radiales actualizados

Nuevo indicador radial con opciones para configurar el aspecto deseado, como:

* Ancho del indicador
* Intervalos (largo y corto)
* Grados de visualización
* Aspecto de la aguja
* Colores
    
  Para obtener más información, consulte Indicador radial.
    
  <Frame><img alt="Radial_Gauges.png" src="/images/kb/ka0Vq0000000cJp-00N5w00000Ri7BU-0EM5w000005vOiX.png"/></Frame>
## Opciones de Guardar como en los tableros

Nueva opción **Duplicar todas las tarjetas** al seleccionar **Guardar como** en un tablero. Ahorra tiempo al crear nuevas versiones de un tablero, ya que permite hacer referencia con facilidad a las tarjetas originales sin necesidad de crear copias. Desmarque la casilla para usar las tarjetas originales. Para obtener más información, consulte Duplicar tableros ("Guardar como").
  
<Frame><img alt="Save_As_Modal.png" src="/images/kb/ka05w00000123T9-00N5w00000Ri7BU-0EM5w000005vOFv.png"/></Frame>
## Administración de grupos

Administración de grupos optimizada para mejorar la supervisión, simplificar la administración e integrarse con los sistemas empresariales (como Active Directory). Para obtener más información, consulte Administración de grupos.
  
<Frame><img alt="Group_Management.png" src="/images/kb/ka0Vq0000000cJp-00N5w00000Ri7BU-0EM5w000005vOiP.png"/></Frame>
## Administrador de Beast Mode

Con el Administrador de Beast Mode, puede ver todos los Beast Modes en un solo lugar. También puede realizar acciones en masa, ver qué Beast Modes no están en uso y bloquear los Beast Modes para que no se editen. Para obtener más información, consulte Administrador de Beast Mode.
  
<Frame><img alt="Beast_Mode_Manager.png" src="/images/kb/ka0Vq0000000cJp-00N5w00000Ri7BU-0EM5w000005vOiU.png"/></Frame>
