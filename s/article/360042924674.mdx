## Intro

A Running Total Stacked Bar chart is similar to a vertical [Stacked Bar chart](/s/article/360042923334 "Stacked Bar Graph"), except that each successive bar in a series displays the cumulative total (or "running total") of all bars in that series up to that point. Although you can use any kind of categorical data in a Running Total chart, typically, you use date or date-time data. Other chart types in Domo that utilize this same "running total" concept include [Running Total Bar charts](http://wiki.domo.com/confluence/display/DHCS/Building+a+Running+Total+Bar+graph "Running Total Bar Graph"), which include only one series bar per category; [Running Total Grouped Bar charts](http://wiki.domo.com/confluence/display/DHCS/Building+a+Running+Total+Grouped+Bar+graph "Running Total Grouped Bar Graph"), in which series bars in a category are shown side by side rather than stacked; and [Running Total Line charts](http://wiki.domo.com/confluence/display/DHCS/Building+a+Running+Total+Line+graph "Running Total Line Graph"), which use lines instead of bars.

## Powering Running Total Stacked Bar charts

A Running Total Stacked Bar chart requires three data columns or rows from your DataSet—one for categories, one for the series within each category; and one for values. For information about value, category, and series data, see [Understanding Chart Data](/s/article/360043428693 "Understanding Chart Data").

In the Analyzer, you choose the columns containing the data for your Running Total Stacked Bar chart. For more information about choosing data columns, see [Applying DataSet Columns to Your Chart](/s/article/360043428713 "Applying DataSet Columns to Your Chart").

For more information about formatting charts in the Analyzer, see [Visualization Card Building Part 2: The Analyzer](/s/article/360043428673 "KPI Card Building Part 2: The Card Builder").

### Changing the size of bars in a Running Total Stacked Bar chart

For any Bar chart, you can set the percentage of space taken up by bars in that chart or specify a fixed bar width. For more information, see [Specifying the Size of Bars in Bar Charts](/s/article/360042925294 "Specifying the Size of Bars in Bar Graphs").
