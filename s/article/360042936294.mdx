## New features and enhancements

Features and enhancements in this release include the following:

### Japanese Knowledge Base

Our Help Center/Knowledge Base site has been live in Domo for a few months, but we’ve finally gotten the entire portal translated and ready for our Japanese customers. The link to the new site will go live with this release.

<Frame><img alt="" src="/images/kb/ka05w00000129uI-00N5w00000Ri7BU-0EM5w000005vOod.png" style={{width: 800, height: 610}}/></Frame>
### DataFlow kill ability

You can now cancel any DataFlow mid-run. This option is available in the DataFlow listings page or in the <img src="/images/kb/ka05w00000123Qn-00N5w00000Ri7BU-0EM5w000005vOAb.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/> menu on the DataFlow details view.

<Frame><img alt="" src="/images/kb/ka05w00000129uI-00N5w00000Ri7BU-0EM5w000005vOof.png" style={{width: 523, height: 384}}/></Frame>
For more information, see [Running a DataFlow](/s/article/360042923034 "Running a DataFlow").

### Automatic retry for failed DataFlows

Failed DataFlows will be retried once automatically without a notification being sent. If the DataFlow fails a second time, a notification will be sent as usual.

For more information, see [Running a DataFlow](/s/article/360042923034 "Running a DataFlow").

For more information, see [Setting Up Publication Groups](/s/article/360043438993 "Setting Up Publication Groups").

### "Started By" column

In the Details view for a DataFlow you can now see whether a given run was executed manually or as a result of input DataSets being updated.

<Frame><img alt="" src="/images/kb/ka05w00000129uI-00N5w00000Ri7BU-0EM5w000005vOoc.png" style={{width: 700, height: 259}}/></Frame>
### End of Support for Workbench 2

Beginning in September, Domo will be ending support for Workbench 2.
