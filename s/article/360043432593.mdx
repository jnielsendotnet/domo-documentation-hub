## Intro

SAP Fieldglass is a Vendor Management System that helps you manage services procurement and external workforce programs. Use Domo's SAP Fieldglass connector to retrieve data from SAP Fieldglass based on  connector name and arguments. To learn more about the SAP Fieldglass API, visit their page ([http://www.fieldglass.com/resources/topic/integration](http://www.fieldglass.com/resources/topic/integration "http://www.fieldglass.com/resources/topic/integration") ).

You can connect to your SAP Fieldglass account in the Data Center. This topic discusses the fields and menus that are specific to the SAP Fieldglass connector user interface. To add DataSets, set update schedules, and edit DataSet information, see [Adding a DataSet Using a Data Connector](/s/article/360042926274 "Adding a DataSet Using a Data Connector").

## Prerequisites

To connect to your SAP Fieldglass account and create a DataSet, you must have a SAP Fieldglass username and password.

## Connecting to Your SAP Fieldglass Account

This section enumerates the options in the **Credentials** and **Details** panes in the SAP Fieldglass Connector page. The components of the other panes in this page, **Scheduling** and **Name & Describe Your DataSet**, are universal across most connector types and are discussed in great detail in [Adding a DataSet Using a Data Connector](/s/article/360042926274 "Adding a DataSet Using a Data Connector").
