---
title: "Migrate from Federated to Cloud Amplifier"
---

### Intro

Only Federated v2 DataSets can be converted to Cloud Amplifier. Federated v1 DataSets must be converted to v2 before the migration can be performed.

---

### Migrate a Federated DataSet to Cloud Amplifier

1. In the Domo Data Center, go to **DataSets** and and select the DataSet you want to migrate to view its Details page.
2. From the Details page, select **More** (three vertical dots icon) > **Convert to Cloud Amplifier** to display the conversion modal.
     
   <Frame><img alt="convert to cloud amplifier.jpg" src="/images/kb/ka0Vq0000006Y8f-00N5w00000Ri7BU-0EMVq000003fPqM.jpg" style={{width: 899, height: 379}}/></Frame>
3. Use the **Select Cloud Account** dropdown in the modal to select the Snowflake-Cloud Amplifier integration you want to use. Learn how to configure a [Snowflake-Cloud Amplifier integration](/s/article/4402322966807).
     
   <Frame><img alt="choose snow ca integration.jpg" src="/images/kb/ka0Vq0000006Y8f-00N5w00000Ri7BU-0EMVq000003fPyP.jpg" style={{width: 415, height: 224}}/></Frame>
4. Select **Next** and check the box to acknowledge that the DataSet will be unavailable during the conversion process. Then select **Convert**.
     
   <Frame><img alt="confirm conversion.jpg" src="/images/kb/ka0Vq0000006Y8f-00N5w00000Ri7BU-0EMVq000003fETX.jpg" style={{width: 415, height: 221}}/></Frame>
   A successful conversion displays a "Conversion Complete" message.

### Revert a DataSet to a Federated Connection

Follow these steps to revert a DataSet from a Cloud Amplifer connection to a federated connection:

1. In the Domo Data Center, go to **DataSets** and and select the DataSet you want to migrate to view its Details page.
2. From the Details page, select **More** (three vertical dots icon) > **Revert to federated data** to display the reversion modal.
     
   <Frame><img alt="revert to federated.jpg" src="/images/kb/ka0Vq0000006Y8f-00N5w00000Ri7BU-0EMVq000003fTh3.jpg" style={{width: 898, height: 380}}/></Frame>
3. Use the checkbox to acknowledge that the DataSet will not be available during the reversion process, then select **Revert**.
     
   A successful reversion displays a "Reversion Complete" message.
