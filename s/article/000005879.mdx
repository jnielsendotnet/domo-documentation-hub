### Intro

 In this tutorial, we’ll show you how to use Domo to build a dashboard from start to finish. We’ll walk you through connecting data, cleaning it through a simple ETL (Extract, Transform, Load) process, building a dashboard with four to five dynamic cards, setting a Card Alert, and pushing that alert to your email or phone in real time and sharing it with colleagues. This guide serves as a foundation for your journey with Domo, setting the stage for more advanced functionalities and applications.
 
 As we move through the tutorial, you'll gain hands-on experience with Domo's powerful features. We'll provide step-by-step instructions, practical examples, and useful tips to make your learning process smooth and enjoyable.

 <Note>
   **Why use connectors?** 
  
   Connectors provide an automated "set it and forget it" method for data integration, contrasting the manual effort needed with other data connection methods.
 </Note>

### Build a Sample ETL

 This guide provides step-by-step instructions for building an ETL (Extract, Transform, Load) process that connects sales data to zip codes and FIPS codes. This enriches the sales data with geographical details like city, state, and county, which are critical for location-based analytics.
 
 <Note>
   **Note:** 
   It's not always necessary to join data sources and clean them in Domo, but this tutorial is good practice for how to join different systems.

 </Note>

### Get Started with Alerts

 Domo Alerts is a powerful tool that can help you monitor specific conditions within your data and notify you when these conditions are met. This guide describes how to set up effective alerts for the five cards you built using the Marketing DataSet.
 
1. From the dashboard or appendix, hover over a card and select
 **Card options**  (wrench icon) > **Alert Me...**.
   
<Frame><img alt="" src="/images/kb/ka0Vq000000HcYj-00N5w00000Ri7BU-0EMVq00000999Bl.jpg" style={{width: 651, height: 367}}/></Frame>
2. Select
 **+ New Alert**  to open the alert modal.
   
<Frame><img alt="" src="/images/kb/ka0Vq000000HcYj-00N5w00000Ri7BU-0EMVq0000098sKd.jpg" style={{width: 650, height: 258}}/></Frame>
3. Use the
 **Metric**  dropdown to choose the metric you want to monitor. We'll choose **Summary number**.
4. Use the conditions dropdown to choose "Changes by —" and enter the value for that condition.
5. Select
 **Next**  and compose the message you want to receive when the alert triggers.
   
<Frame><img alt="" src="/images/kb/ka0Vq000000HcYj-00N5w00000Ri7BU-0EMVq0000098iJf.jpg" style={{width: 650, height: 406}}/></Frame>
6. Select
 **Next**  and choose anyone with whom to share the alert, then select **Save**. Close the alert modal.

 <Tip>
   **Tip:** 
   After setting an alert, you can choose how you want to receive the alert from the
   **See Alerts** 
   section on the card. Options include via text, email, and mobile app notification.

 </Tip>
