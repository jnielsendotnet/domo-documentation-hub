---
title: "Adding a Doc Card to Domo"
---


You can add a document, image, or video to Domo to share with others.

In Domo, the document, image, or video appears as a Doc Card. For many file types, the document, image, or video will be embedded right into the Doc Card, allowing users to view it without having to first download it. If you go to the Details view for that Doc Card, you can view a full-sized version of the document, image, or video. These embeddable file types include all of the following:

* DOC/DOCX
* PDF
* JPEG
* PNG
* PPT/PPTX
* MP4
* XLS/XLSX
* TXT
* BMP

For other permitted file types not in this list, a download button appears instead of an embedded view.

<Note>
  **Note:** In the interest of preventing users from uploading malicious files into Domo accounts, Domo allows a specific set of file types to be uploaded. If you attempt to upload a file not within this list, an error message appears. For more information, see [Permitted File Types for Attachments](/s/article/360042925634 "Permitted File Types for Attachments").

</Note>

**To add a Doc Card to Domo,**

1. In the Page where you want the Doc Card to appear, click  the
   <img src="/images/kb/ka05w00000123Ti-00N5w00000Ri7BU-0EM5w000005vOHg.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/> button in the upper right corner.
2. Click **Create new Card**.

   A dialog appears with a number of Card upload options.

   <Tip>
     **Tip:** You can also open this dialog from anywhere in Domo by selecting <img src="/images/kb/ka05w0000012340-00N5w00000Ri7BU-0EM5w000005vNwt.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/> in the app toolbar and selecting **Card**.

   </Tip>

3. Click **Doc Card**.
4. Do one of the following:

   * Drag and drop a file from your machine onto the dashboard.
   * Click **Browse**, browse to the desired file on your hard drive, then click **Open**.

     A preview of the document, image, or video appears. A thumbnail preview will NOT display if the file is password protected.

     For documents and images, you can navigate through the preview (if more than one dashboard is available) using the <img src="/images/kb/ka05w00000124Aa-00N5w00000Ri7BU-0EM5w000005vOGx.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/> and <img src="/images/kb/ka05w00000124Aa-00N5w00000Ri7BU-0EM5w000005vOH6.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/> buttons, download the document or image to your machine using <img src="/images/kb/ka05w00000124Aa-00N5w00000Ri7BU-0EM5w000005vOGh.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/>, or open a full-screen view of the document or image using <img src="/images/kb/ka05w00000124Aa-00N5w00000Ri7BU-0EM5w000005vOGz.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/>. For more information about the full-screen view for a Doc Card, see [Previewing a Document](/s/article/360043430213 "Previewing a Document").

     For videos, you can choose to use a default black thumbnail or drag and drop or browse to a JPEG/PNG image to use as a thumbnail. The image for the thumbnail must be 10 MB or less.
5. Click **Save** to save the file as a Doc Card.

A card is added for the document, image, or video.

If you want, you can resize the card or move it to another position in the dashboard or to another dashboard. To learn about resizing cards, see [Moving or Copying Cards to Other Pages](/s/article/360042923254).

You can also change the owner of the new Card. For more information, see [Assigning or Changing a Card Owner](/s/article/360042923794 "Assigning or Changing a Card Owner").

<Tip>
  **Tip:** You can download a document, image, or video from a Doc Card in Domo by mousing over the preview in the Details view then clicking <img src="/images/kb/ka05w00000123TM-00N5w00000Ri7BU-0EM5w000005vOGs.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/>. For more information, see [Downloading a Document from Domo](/s/article/360043430193 "Downloading a Document from Domo").

</Tip>
