## Intro

In the Workbench 5 plugins manager you can view registered plugins and add new plugins. Plugins display with their publisher, version number, description (if available), and assembly name.

---

## Manage Plugins in Workbench 5.2

Follow these steps to add plugins to Workbench:

1. Open Workbench 5 as an administrator and select
   <img src="/images/kb/ka05w00000123t2-00N5w00000Ri7BU-0EM5w000005vPP4.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/> **More**
   >
   **Plugins** in the left sidebar.
2. In the
   **Plugins** section under **Register new plugins**, select **Add New**.

   <Frame><img alt="select add new.jpg" src="/images/kb/ka05w000001292M-00N5w00000Ri7BU-0EM5w000006v1EN.jpg" style={{width: 651.989, height: 453.999}}/></Frame>

3. Select the Domo Workbench plugin file and select
   **Open**.
     
   You can now validate the plugin.
4. Select
   **Install** after you have validated the plugin information.

   <Frame><img alt="install.jpg" src="/images/kb/ka05w000001292M-00N5w00000Ri7BU-0EM5w000006v1Eh.jpg" style={{width: 651.982, height: 453.999}}/></Frame>

5. When the plugin is imported, you receive a
   Success message and the plugin displays in the list of Registered Plugins.

   <Frame><img alt="success.jpg" src="/images/kb/ka05w000001292M-00N5w00000Ri7BU-0EM5w000006v1F1.jpg" style={{width: 657.99, height: 421.996}}/></Frame>

## Manage Plugins in Workbench 5 and 5.1

<Frame><img alt="wb5_plugin_manager.png" src="/images/kb/ka05w000001292M-00N5w00000Ri7BU-0EM5w000005vPNm.png"/></Frame>

All Workbench plugins are contained within Dynamic Link Libraries (DLLs). You can add plugins to Workbench by selecting the desired DLL on your machine or network.

Follow these steps to add plugins to Workbench:

<Note>
**Note:** Your desired plugin must be located in a subdirectory of the chosen directory.
</Note>

1. Select
   <img src="/images/kb/ka05w00000123t2-00N5w00000Ri7BU-0EM5w000005vPP4.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/> **More** in the left sidebar.
2. Select
   **Plugins.** (If you do not see this option, select <img src="/images/kb/ka05w00000123tU-00N5w00000Ri7BU-0EM5w000005vPQj.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/> to expand it.)
     
   The plugin manager opens.
3. Under
   **Register new plugins**, select the ellipsis (...) next to the **Search path** field.
     
   <Frame><img alt="wb5_register_plugin.png" src="/images/kb/ka05w000001292M-00N5w00000Ri7BU-0EM5w000005vPOW.png"/></Frame>

4. On your machine or network, locate and open the DLL containing the plugins you want to add.
5. Select
   **Add** after you have chosen your directory.
