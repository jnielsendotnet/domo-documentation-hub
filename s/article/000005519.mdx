## Intro

ABLSoft is a SaaS lending software solution that helps lenders grow profitably with intelligent automation. Use Domo's ABLSoft connector to retrieve detailed data about your Borrowers, loans, ledgers, and balances. If you want to learn more about the ABLSoft API, you can see ([https://apidocs.ablsoft.com/](https://apidocs.ablsoft.com/ "https://apidocs.ablsoft.com/") ).

You can connect to your ABLSoft account in the Data Center. This topic discusses the fields and menus that are specific to the ABLSoft connector user interface. To add DataSets, set update schedules, and edit DataSet information, see [Adding a DataSet Using a Connector](/s/article/360042926274 "Adding a DataSet Using a Connector").

## Prerequisites

To connect to your ABLSoft account and create a DataSet, you must have the following:

* Username
* Password
* Client ID
* Client Secret
* Domain

**Note:** To obtain the Client ID and Client Secret, contact the technical support or the system administrator of your company.

## Connecting to Your ABLSoft Account

This section enumerates the options in the **Credentials** and **Details** panes on the ABLSoft Connector page. The components of the other panes on this page, **Scheduling**, and **Name & Describe Your DataSet**, are universal across most connector types and are discussed in great detail in [Adding a DataSet Using a Connector](/s/article/360042926274 "Adding a DataSet Using a Connector").
