## Intro

This article explains how to compose tables and visualizations with immediate feedback using Worksheets. 

When your analysis is ready for a wider audience, or requires formal governance, you can save your Worksheet to an App Studio App. [Learn how to create an App Studio App](/s/article/000005295#create-an-app).

---


## Required Grants

To access Worksheets, the following grant must be enabled for your role:
* **Edit Worksheet**

Learn more about [grants](/s/article/360043438973).

## Edit Your Worksheet in Edit Mode 

When you're in edit mode, you can access the following features: 
* Rename your Worksheet
* Add a table to the canvas
* Add a chart/graph to the canvas
* Edit Worksheet Settings
* Add a new Sheet
* Undo/Redo changes
* Save and Export


## Rename Your Worksheet

Keep your Worksheets organized by giving them clear, descriptive names. 

To rename a Worksheet: 
1. Select the edit icon in the title section of your Worksheet.
   A text box is enabled.
   <Frame>
   ![](/images/kb/worksheets-screenshot-16.png)
   </Frame>
2. Select directly on the text of your Worksheet name to enter edit mode.
3. Create your new name. 
4. Press **Enter** to save.


## Add a Table to the Canvas

Follow these steps to add a table to the canvas:
1. Go to the left navigation menu.
2. Select the **Tables** icon.
   <Frame>
   ![](/images/kb/worksheets-screenshot-17.png)
   </Frame>
   A pop-up menu opens.
3. Select one of the available table types.
   <Frame>
   ![](/images/kb/worksheets-screenshot-18.png)
   </Frame>
4. Drag and drop your new table onto the canvas.
5. (Optional) You can add multiple tables on the canvas and arrange them as desired.
   <Frame>
   ![](/images/kb/worksheets-screenshot-19.png)
   </Frame>
   <Note>
   In the above example, the user has placed three distinct tables onto the canvas. Tables and charts may be organised as needed by selecting each item and moving it to the desired location, or by adjusting its dimensions using the square handles to modify both height and width.
   </Note>


## Configure and Format Your Table Data 

Use the right panel, the toolbar, and filters to configure your table data. 

You can format your table data using column formatting, as well as charts and graphs. 


### Configure Your Table Data Using the Right Panel

The right panel is your command center for any selected object on the canvas.
1. Select a table or chart to activate the panel.
2. Use the **Settings** tab to change column order, hide/display columns, change column names, set column types, configure group by aggregation, copy columns, set vertical alignment, set date ranges, filters, sort tables, and limit rows. 
   <Frame>
   ![](/images/kb/worksheets-screenshot-20.png)
   </Frame>
3. Use the **Styles** tab to change/set colors, font types, font family, conditional formatting, general table formatting, sorting settings, row heights, add header, total and sub-total rows, and Excel export settings. 
   <Frame>
   ![](/images/kb/worksheets-screenshot-21.png)
   </Frame>

### Format Your Columns

Ensure your data is readable by applying specific formatting to table columns.

1. Start by selecting a table on the canvas. Then use the [right panel](#format-your-columns-in-the-right-panel) or [table header toolbar](#format-your-columns-in-the-table-header-toolbar) to edit and format your columns.

#### Format Your Columns in the Right Panel

2. In the right panel, select the specific field under the **Select Columns** list.
3. Select the more options menu.
4. Select desired formatting. 
5. Set the Data Label and the Data Type in the **Display As** option (Number, Currency, Percentage) and decimal places.

#### Format Your Columns in the Table Header Toolbar

Use the table header toolbar to quickly edit and format columns. 

<Frame>
![](/images/kb/worksheets-screenshot-22.png)
</Frame>

* Use the <img src="/images/kb/worksheets-icon-1.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/> option to format values as currency.
* Use the <img src="/images/kb/worksheets-icon-2.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/> option to format column values as percentages.
* Use the <img src="/images/kb/worksheets-icon-3.png" style={{width: 40, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/> options to adjust the number of decimal points to increase or decrease precision.
* Use <img src="/images/kb/worksheets-icon-4.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/> to align values to the left, center, or right.
* Use the <img src="/images/kb/worksheets-icon-5.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/> to align values to the top, center, or bottom.


### Filters

Narrow your focus by applying filters to your data.

<Frame>
![](/images/kb/worksheets-screenshot-23.png)
</Frame>

1. Select the **Filter** <img src="/images/kb/worksheets-icon-6.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/> icon. 
2. Select **+ Add Filter** to add a new filter. 
3. Search and select the column by which you want to filter.
4. Select the values you want to include in your filter.
5. Select **Apply** to save the filter. 
   Or select **Cancel** to discard your changes.
   Select the **Delete** icon to remove the filter.
   <Frame>
   ![](/images/kb/worksheets-screenshot-24.png)
   </Frame>

### Use Functions and Beast Mode

Apply logic to your data without leaving the canvas.

<Frame>
![](/images/kb/worksheets-screenshot-25.png)
</Frame>

1. In the toolbar, select the **Function** <img src="/images/kb/worksheets-icon-7.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/> icon.
   A text box is enabled.
2. Write your logic using standard **Beast Mode** (SQL-based) syntax.
   Use familiar functions for quick aggregations, such as `SUM`, `AVG`, or `COUNT`.
3. Press **Enter** to validate your function.
   Use the <img src="/images/kb/worksheets-icon-8.png" style={{width: 40, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/> buttons to save or cancel your function.

   A new column is added to the table containing the function's results.
   <Frame>
   ![](/images/kb/worksheets-screenshot-26.png)
   </Frame>

## Edit Worksheet Settings

Configure general worksheet settings using the Settings option in the left toolbar.
1. Select the **Settings** icon in the left toolbar.
   <Frame>
   ![](/images/kb/worksheets-screenshot-32.png)
   </Frame>
   The Worksheet Settings panel opens in the right pane.
   <Frame>
   ![](/images/kb/worksheets-screenshot-33.png)
   </Frame>
2. Select the setting you would like to configure. 

Here are the settings you can configure: 
* **Name —** Change or edit the name of your Worksheet
* **Description —** Add or edit your Worksheet's description
* **Icon —** Change your Worksheet's icon
* **Permissions**
    * **Owner —** Add or remove owners
    * **Lock this content —** Toggle to allow admins and owners to make changes to your Worksheet
    * **AI Chat Sources —** Select the specific data sources the AI should use to inform its responses within the app
* **Delete —** Allows you to delete the Worksheet
  <Note>
  Deleting a Worksheet also removes all associated pages and elements, such as headers, buttons, borders, and controls. However, it does not delete Cards, Domo Bricks, or Forms used within the Worksheet. **This action cannot be undong**.
  </Note>

   ![](/images/kb/worksheets-screenshot-36.png)

## Undo and Redo Changes

You can use the **Left Arrow (Undo)** and **Right Arrow (Redo)** icons in the top header to undo or redo any changes made to the Worksheet.

<Frame>
![](/images/kb/worksheets-screenshot-42.png)
</Frame>

<Note>
The unde and redo features only work on the page you are currently on. Any changes you make to other pages take effect after you save your progress.
</Note>
