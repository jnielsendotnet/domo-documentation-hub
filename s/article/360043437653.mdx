title: "Connecting to SAP BW in Domo"
---

## Overview

To connect to SAP BW data in Domo, you must establish a DOMO-SAP data link. To do this, you will go through all of the following steps:

1. [Establish an RFC connection between DOMO and BW](#h_a2635c0d-ff35-4b30-bf47-ee85a8733104 "SAP BW Connector")
2. [Create a Logical File](#h_dcb3c085-e171-4d5d-b6e2-deed0188d285 "SAP BW Connector")
3. [Find the Data Structure of your BW Object](#h_69801cf8-6231-4453-93f1-4259332b6b5f "SAP BW Connector")
4. [Build your APD](#h_2a5ec744-50f8-4a9b-8bed-0e274093b27f "SAP BW Connector")

## Step 1: Establishing an RFC connection between Domo and BW

1. In mySAP ERP, run Transaction
   SM59.
     
   <Frame><img alt="sapbw1.png" src="/images/kb/ka05w000001233r-00N5w00000Ri7BU-0EM5w000005vNwD.png"/></Frame>

2. Select
   **HTTP Connections to External Server** and click **Create**.
     
   <Frame><img alt="sapbw2.png" src="/images/kb/ka05w000001233r-00N5w00000Ri7BU-0EM5w000005vNwF.png"/></Frame>

3. Fill in the following fields:

   * **RFC Destination**
   * **Description 1**
   * **Target Host**
   * **Service No.**
   * **Path Prefix**
       
     <Frame><img alt="sapbw3.png" src="/images/kb/ka05w000001233r-00N5w00000Ri7BU-0EM5w000005vNwG.png"/></Frame>

4. Click the
   **Logon & Security** tab.
5. Select the
   **Basic Authentication** radio button.
     
   <Frame><img alt="sapbw4.png" src="/images/kb/ka05w000001233r-00N5w00000Ri7BU-0EM5w000005vNw7.png"/></Frame>

6. Scroll down and enter a username and password.
     
   <Frame><img alt="sapbw5.png" src="/images/kb/ka05w000001233r-00N5w00000Ri7BU-0EM5w000005vNw6.png"/></Frame>

7. Click
   **Save**.

Your RFC connection is now established.

## Step 3: Finding the Data Structure of your BW Object

1. In Data Warehousing Workbench, run transaction
   RSA1.
2. Select
   **BI Content**.
     
   <Frame><img alt="sapbw13.png" src="/images/kb/ka05w000001233r-00N5w00000Ri7BU-0EM5w000005vNwA.png"/></Frame>

3. Navigate to the desired BW object and double-click it.
4. Click
   **Key Figures**.
     
   <Frame><img alt="sapbw14.png" src="/images/kb/ka05w000001233r-00N5w00000Ri7BU-0EM5w000005vNvx.png"/></Frame>

The data structure of the object opens for you to view.

## Step 4: Building Your APD

1. In Data Warehousing Workbench, run transaction
   RSA1.
2. Open the Analysis Process Designer (APD).
     
   <Frame><img alt="sap145.png" src="/images/kb/ka05w000001233r-00N5w00000Ri7BU-0EM5w000005vNwB.png"/></Frame>

3. Select
   **General** then click **Create**.
     
   <Frame><img alt="sapbw15.png" src="/images/kb/ka05w000001233r-00N5w00000Ri7BU-0EM5w000005vNwE.png"/></Frame>

4. Select
   **General Application**.
     
   <Frame><img alt="sapbw16.png" src="/images/kb/ka05w000001233r-00N5w00000Ri7BU-0EM5w000005vNw8.png"/></Frame>

5. Enter a description in the
   **Description** field.
6. Select your Data Source (in this case an InfoProvider).
     
   <Frame><img alt="sapbw17.png" src="/images/kb/ka05w000001233r-00N5w00000Ri7BU-0EM5w000005vNw4.png"/></Frame>

7. Name the Data Source.
8. Select an InfoProvider.
     
   You can also limit the fields sent.
     
   <Frame><img alt="sapbw18.png" src="/images/kb/ka05w000001233r-00N5w00000Ri7BU-0EM5w000005vNw9.png"/></Frame>

   <Frame><img alt="sapbw19.png" src="/images/kb/ka05w000001233r-00N5w00000Ri7BU-0EM5w000005vNwC.png"/></Frame>

   <Frame><img alt="sapbw20.png" src="/images/kb/ka05w000001233r-00N5w00000Ri7BU-0EM5w000005vNw3.png"/></Frame>

9. Select
   **Write Data to File**.
     
   <Frame><img alt="sapbw21.png" src="/images/kb/ka05w000001233r-00N5w00000Ri7BU-0EM5w000005vNvy.png"/></Frame>

   <Frame><img alt="sapbw22.png" src="/images/kb/ka05w000001233r-00N5w00000Ri7BU-0EM5w000005vNvz.png"/></Frame>

10. In the
    **Data Target: File** dialog, do the following:

    1. Enter a description for the file in the
       **Description** field.
    2. Select the
       **Application Server** radio button.
    3. Check the box for
       **Logical File Name**.
    4. Enter your new logical file name in the
       **File** field.
    5. Click the green check box.
         
       <Frame><img alt="sapbw23.png" src="/images/kb/ka05w000001233r-00N5w00000Ri7BU-0EM5w000005vNw1.png"/></Frame>

11. Click and drag to connect the nodes.
      
    <Frame><img alt="sapbw24.png" src="/images/kb/ka05w000001233r-00N5w00000Ri7BU-0EM5w000005vNw5.png"/></Frame>

12. Save your changes.
13. Click the
    **Activate** icon.
