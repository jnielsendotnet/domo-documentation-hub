## About

Sizmek provides tools to build, serve, analyze and target ads to global audiences across a variety of channels, including mobile and video. Combine your Sizmek data with data from sales, finance, and other departments across your organization so you can maximize your ROI. Get easy-to-use, real-time insights into impressions, clicks, interactions, conversions, viewability, and more. Domo's Sizmek connector brings in data from the Sizmek MDX platform and includes reports for Advertisers, Agencies, Campaigns, Markets, Placements, Sites and more.

You can connect to your Sizmek account in the Data Center. This article explains the fields and menus that are specific to the Sizmek Connector user interface. To add DataSets, set update schedules, and edit DataSet information, see [Adding a DataSet Using a Data Connector](https://domo-support.domo.com/s/article/360042926274?language=en_US "Adding a DataSet Using a Data Connector").

## Prerequisites

To connect to your Sizmek account and create a DataSet, you must have the following:

* Username
* Password
* API Key

## Connecting to Your Sizmek Account

This section lists the options in the **Credentials** and **Details** panes in the Sizmek Connector page. The components of the other panes in this page, **Scheduling** and **Name & Describe Your DataSet**, are universal across most connector types and are discussed in great detail in [Adding a DataSet Using a Data Connector](https://domo-support.domo.com/s/article/360042926274?language=en_US "Adding a DataSet Using a Data Connector").

## Credentials Pane

This pane contains fields for entering credentials to connect to your Sizmek  account. The following table describes what is needed for each field:

| Field    | Description           |
| -------- | --------------------- |
| Username | Your Sizmek username. |
| Password | Your Sizmek password. |
| API Key  | Your Sizemek API key. |

Once you have entered valid Sizmek credentials, you can use the same account to create a new Sizmek DataSet. You can manage connector accounts in the **Accounts** tab in the Data Center. For more information about this tab, see [Managing User Accounts for Connectors](https://domo-support.domo.com/s/article/360042926054?language=en_US "Managing User Accounts for Connectors").
