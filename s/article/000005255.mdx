## Intro

TikTok is a social media platform for creating and sharing short videos with music and filters. TikTok connector allows you to use TikTok API to import statistics provided by TikTok into your Domo instance. This data includes your user profile and your video statistics.

You can connect to your TikTok account in the Data Center. This topic discusses the fields and menus that are specific to the TikTok Connector user interface. General information for adding DataSets, setting update schedules, and editing DataSet information is discussed in [Adding a DataSet Using a Data Connector](/s/article/360042926274 "Adding a DataSet Using a Data Connector").

## Prerequisites

To connect to your TikTok account and create a DataSet, you must have the authentication information to login your TikTok account.

## Connecting to Your TikTok Account

This section enumerates the options in the **Credentials** and **Details** panes in the TikTok Connector page. The components of the other panes in this page, **Scheduling** and **Name & Describe Your DataSet**, are universal across most connector types and are discussed in greater length in [Adding a DataSet Using a Data Connector](/s/article/360042926274 "Adding a DataSet Using a Data Connector").

### Credentials Pane

If you already have an authenticated TikTok account and want to use it, select it from the Account drop-down in this pane. If you want to add another account, click **Add Account**. If you do not have any TikTok account, click the **Connect** button in this pane.

The TikTok authentication screen will pop up, use your TikTok credentials to log in.

Once you have entered valid TikTok credentials, you can use the same account any time you go to create a new TikTok DataSet. You can manage connector accounts in the **Accounts** tab in the Data Center. For more information about this tab, see [Managing User Accounts for Connectors](/s/article/360042926054 "Managing User Accounts for Connectors").
