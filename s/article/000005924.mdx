### New Features and Enhancements

# AI Consumption Rate Table

The AI Consumption Rate card lists SKUs for your Domo's AI feature usage and the associated credit rates. 

This table enables you to better understand how your usage of Domo's AI features contributes to your overall credit consumption.

Learn more about the [AI Consumption Rate Table](https://domo-support.domo.com/s/article/000005326?language=en_US#rate-card-tab).



# PDP: Column Masking

PDP: Column Masking in DataSets allows you to hide sensitive information, such as PII and PHI. You can define column masking policies to redact entire values or hide a certain number of characters in a given field. 

You can configure column masking in the PDP section of DataSet Details screen.

![ColumnMasking (1).png](https://domo.file.force.com/servlet/rtaImage?eid=ka0Vq000000Vqjt&feoid=00N5w00000Ri7BU&refid=0EMVq000000xyD3)

Learn more about [column masking](https://domo-support.domo.com/s/article/360042934614?language=en_US) and PDP in the Knowledge Base.


# Window Functions in SQL Tile

Window functions in the SQL tile allow you to use complex analytical functions in your Magic ETL DataFlows without resorting to a Redshift DataFlow or creating multiple datastreams in Magic ETL. 

Perform calculations like running totals, rankings, moving averages, and cumulative statistics from within your SQL tile.

![](https://domo.file.force.com/servlet/rtaImage?eid=ka0Vq000000Vqjt&feoid=00N5w00000Ri7BU&refid=0EMVq00000AQ4yJ)

Learn more about [window functions in SQL tiles](https://domo-support.domo.com/s/article/360044876874?language=en_US#sql_tile).

# App Studio

#### App Studio Editor Updates

Use App Studio to automatically generate charts and tables using your data in just a few steps, with no initial setup required.  

Build what you need using the following tools and options:

# Magic ETL

#### Disable Tiles

The ability to disable or "comment out" tiles in DataFlows is now available in Magic ETL. This feature allows you to select one or multiple tiles to disable, and have these tiles not be considered for execution during the Dataflow run. 

Along with disabling tiles, you can also highlight all upstream or downstream tiles from a selected tile and disable them individually from their configuration panels. 

![](https://domo.file.force.com/servlet/rtaImage?eid=ka0Vq000000Vqjt&feoid=00N5w00000Ri7BU&refid=0EMVq000009qslV)



#### Fixed Input Tile

The Fixed Input tile lets you quickly add data directly on the Magic ETL canvas. You can add data manually, by copying and pasting from other sources, or by uploading Excel and CSV files. You can also use AI-powered generation to create individual columns or full DataSets with realistic sample data. All input data is stored within the DataFlow definition and can be used as input for downstream transformations in the same DataFlow.

The Fixed Input tile allows for:

*   Faster prototyping and testing
*   Improved flexibility in data input
*   Empowerment of non-technical users
*   Realistic sample data generation

![](https://domo.file.force.com/servlet/rtaImage?eid=ka0Vq000000Vqjt&feoid=00N5w00000Ri7BU&refid=0EMVq000009qhAA)



#### JSON Expander Tile

The new JSON Expander allows you to expand JSON-formatted data from a single column to multiple columns without complex parsing.

This feature allows you to access column data reliably using an easy-to-navigate, hierarchical expansion process.

![](https://domo.file.force.com/servlet/rtaImage?eid=ka0Vq000000Vqjt&feoid=00N5w00000Ri7BU&refid=0EMVq000009qwp7)



# Worksheets 

Worksheets allow you to transform raw data into quick insights without the learning curve overhead or the need to set up app structures. 

Worksheets include an accessible, drag-and-drop canvas for quick data analysis within Domo. It is closely integrated with Domo's data environment and supports seamless escalation when you're ready to move your worksheet to App Studio for sharing.
