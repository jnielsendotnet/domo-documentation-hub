## Intro

If a card already exists in your Domo instance, you can add it to a dashboard without having to recreate it. You can add any kind of card, such as a KPI, Sumo, and Doc Card. You must have access to the card you want to add as well as editing capability in the dashboard.

## Adding an Existing Card to a Dashboard

1. Go to
**Dashboards** in the navigation header.

<Frame><img alt="2022-07-06_11-51-48.png" src="/images/kb/ka05w00000123Sp-00N5w00000Ri7BU-0EM5w000005vOFQ.png"/></Frame>
2. Hover over the
<InlineImage src="/images/kb/ka05w00000123Sp-00N5w00000Ri7BU-0EM5w000005vOFR.png" /> **Menu** to open the **Dashboards** drawer.

3. Open a dashboard. You can search for one in the
**Filter by name** bar or select one from the list.

<Frame><img alt="Screen_Shot_2022-07-06_at_4.03.26_PM.png" src="/images/kb/ka05w00000123Sp-00N5w00000Ri7BU-0EM5w000005vOFW.png" style={{width: 228, height: 170}}/></Frame>
4. Select
<InlineImage src="/images/kb/ka05w00000123Sp-00N5w00000Ri7BU-0EM5w000005vOFP.png" /> **Dashboard options** and choose **Edit Dashboard**.

The editing view displays.

5. Click and drag the
**Card**

option to the dashboard template.

The **Add Content** menu displays.

<Frame><img alt="2022-08-02_16-02-21__1_.gif" src="/images/kb/ka05w00000123Sp-00N5w00000Ri7BU-0EM5w000005vOFV.gif"/></Frame>
6. From the
**Add Content** menu, select **Add existing Card**.

<Frame><img alt="2022-08-02_15-50-05.png" src="/images/kb/ka05w00000123Sp-00N5w00000Ri7BU-0EM5w000005vOFS.png" style={{width: 257, height: 385}}/></Frame>
7. Select a card from the list, or locate the card using the search bar.

8. Select the card.

9. Select
**Save.**

The existing card has been added to the dashboard.

<Frame><img alt="Screen_Shot_2022-08-02_at_4.27.21_PM.png" src="/images/kb/ka05w00000123Sp-00N5w00000Ri7BU-0EM5w000005vOFU.png"/></Frame>
