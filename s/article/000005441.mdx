## Intro

Gong is a real-time platform that provides insights for sales conversations. With its Revenue Intelligence, Gong enables you to gain critical insights into your remote sales team, your deals, and your market. Use Domo's Gong Connector to get real-time insights into your sales conversations and know your revenue strategy for success.

You can connect to your Gong Domo account in the Data Center. This topic discusses the fields and menus that are specific to the Gong Domo connector user interface. To add DataSets, set update schedules, and edit DataSet information, see  [Adding a DataSet Using a Connector](/s/article/360042926274).

## Prerequisites

To connect to your Gong Domo account and create a DataSet, you must have the email address and password associated with your Gong Oauth account.

## Connecting to Your Gong Domo Account

This section enumerates the options in the **Credentials** and **Details** panes on the Gong Domo Connector page. The components of the other panes on this page, **Scheduling**, and **Name & Describe Your DataSet**, are universal across most connector types and are discussed in great detail in [Adding a DataSet Using a Connector](/s/article/360042926274).

## Credentials Pane

Once you have entered valid Gong Domo credentials, you can use the same account at any time to create a new Gong Domo DataSet. You can manage connector accounts in the **Accounts** tab in the Data Center. For more information about this tab, see [Manage Connector/Adapter Accounts](/s/article/360042926054).

## Details Pane

This pane contains a primary **Reports** menu, along with various other menus which may or may not appear depending on the report type you select.

| Menu | Description |
| --- | --- |
| Report | You can just select the Gong Domo report you want to run. The following reports are available:   |  |  | | --- | --- | | **Report Name** | **Description** | | Calls | Returns call data for calls that occur in the selected date range. | | Call Details | Returns call detail information broken down by the selected sub-report for the selected date range. | | Calls Details Extensive | Lists detailed call data for calls that took place during a specified date range and have specified call ids. | | Call Transcripts | Returns call transcripts broken down by sentence for the selected date range. | | Company Productivity (Calls & Interaction Stats) | Returns the metrics based on calls & interaction stats data for the selected date range. | | Daily Activity for applicable users | Retrieves the daily activity of multiple users within the Gong system for a range of dates. | | Flows | This report returns all company flows, together with personal flows and flows shared with any user. | | Interaction Stats | Returns interaction statistics for calls that occurred in the selected date range. | | Scorecards | Returns scorecard data for scorecards with a review date in the selected range | | User Activity | Returns user activity by selected aggregation and date range. | | Users | Returns a list of users. | | Workspaces | Returns a list of workspaces. | |
| Fetch Data Related To | Select the parameter to fetch the data. The connector returns only the metadata, if 'None' is selected. |
| Details About | Select the sub-parameters. |
| Aggregation By Day | Check if records need to be aggregated by day. |
| Aggregation Period | Select the period you would like for data aggregation. |
| Subreport | Select the sub-report data of your choice. |
