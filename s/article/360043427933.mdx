---
title: "Join DataSets by Adding a Unique Key Column"
---

### Intro

Sometimes you want to join DataSets together but there is no unique key column to join on. In this case, you can create a unique key column.

### Create a Unique Key Column

Follow these steps to create a unique key column:

1. Open an existing DataFlow in Magic ETL or begin creating a new DataFlow.
2. Add the
   [Rank and Window](/s/article/360042922814 "ETL Actions: Rank and Window") tile to the canvas and connect it to your Input DataSet.
     
   <Frame><img alt="" src="/images/kb/ka0Vq0000000l8b-00N5w00000Ri7BU-0EM5w000005vOxE.png" style={{width: 350, height: 234}}/></Frame>
3. Select the **Rank & Window** tile.
4. In the configuration pane below the canvas, select **Add Function**.
     
   <Frame><img alt="add function.jpg" src="/images/kb/ka0Vq0000000l8b-00N5w00000Ri7BU-0EMVq000000nTda.jpg" style={{width: 755, height: 176}}/></Frame>
5. In step 1, name the new column.
     
   <Frame><img alt="Screenshot 2024-04-22 at 11.58.45 AM.png" src="/images/kb/ka0Vq0000000l8b-00N5w00000Ri7BU-0EMVq000000nUHt.jpg" style={{width: 303, height: 233}}/></Frame>
6. In step 2, expand the **Select the function to apply** dropdown and select **Row Number**.
     
   <Frame><img alt="Screenshot 2024-04-22 at 11.58.57 AM.png" src="/images/kb/ka0Vq0000000l8b-00N5w00000Ri7BU-0EMVq000000nUJW.jpg" style={{width: 301, height: 350}}/></Frame>
7. Select **Apply**.
     
   <Frame><img alt="apply function.jpg" src="/images/kb/ka0Vq0000000l8b-00N5w00000Ri7BU-0EMVq000000nUUn.jpg" style={{width: 750, height: 294}}/></Frame>
8. In step 2, choose a column to order the functions on to determine how to apply the row numbers.
9. In step 3, choose to order your column by either ascending or descending values.
     
   <Frame><img alt="Screenshot 2024-04-22 at 12.03.20 PM.png" src="/images/kb/ka0Vq0000000l8b-00N5w00000Ri7BU-0EMVq000000nUjJ.jpg" style={{width: 751, height: 299}}/></Frame>
10. To have a unique key value for every row of your data, leave the optional **What columns make up the partition** option blank.
11. Complete your DataFlow and run it.

After running your DataFlow, the output DataSet has a new column with a unique value for each row.
