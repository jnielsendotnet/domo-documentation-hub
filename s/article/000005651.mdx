## Intro

Amazon Kendra is a highly accurate and easy-to-use enterprise search service that's powered by machine learning. Use Domo's Amazon Kendra connector to help sync jobs.

You can connect to your Amazon Kendra account in the Data Center. This topic discusses the fields and menus that are specific to the Amazon Kendra connector user interface. To add DataSets, set update schedules, and edit DataSet information, see [Adding a DataSet Using a Connector](https://domo-support.domo.com/s/article/360042926274?language=en_US "Adding a DataSet Using a Connector").

## Prerequisites

To connect to your Amazon Kendra account and create a DataSet, you need the following:

* Your AWS Access Key
* Your AWS Secret Key
* You need to select the Amazon Kendra Region

## Connecting to Your Amazon Kendra Account

This section enumerates the options in the **Credentials** and **Details** panes on the Amazon Kendra Connector page. The components of the other panes on this page, **Scheduling**, and **Name & Describe Your DataSet**, are universal across most connector types and are discussed in great detail in [Adding a DataSet Using a Connector](https://domo-support.domo.com/s/article/360042926274?language=en_US "Adding a DataSet Using a Connector").
