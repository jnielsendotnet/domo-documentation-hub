## Intro

Google Spreadsheets is a Web-based application that enables you to create, update, modify spreadsheets and share the data live online. To learn more about the Google Sheets API, see ([https://developers.google.com/sheets/api/guides/concepts](https://developers.google.com/sheets/api/guides/concepts "https://developers.google.com/sheets/api/guides/concepts") ).

You can connect to your Google Sheets account in the Data Center. This topic discusses the fields and menus that are specific to the Google Sheets connector user interface. To add DataSets, set update schedules, and edit DataSet information, see [Adding a DataSet Using a Connector](https://domo-support.domo.com/s/article/360042926274?language=en_US "Adding a DataSet Using a Connector").

## Prerequisites

To connect to your Google Sheets account and create a DataSet, you must have the Service Token JSON key.

To generate a Google Sheets service account JSON key, do the following:

1. Log into Google Cloud Console using [https://console.cloud.google.com/](https://console.cloud.google.com/ "https://console.cloud.google.com/").
2. Select the project.

   <Frame>
     <img src="/images/kb/ka0Vq0000007POH-00N5w00000Ri7BU-0EMVq000001P4hN.jpg" style={{ width: 366, height: 90 }} />
   </Frame>

3. On the left pane select **IAM & Admin** > **Service Accounts**.

   <Frame>
     <img src="/images/kb/ka0Vq0000007POH-00N5w00000Ri7BU-0EMVq000001OxzH.jpg" style={{ width: 367, height: 338 }} />
   </Frame>

4. Click **+ CREATE SERVICE ACCOUNT**.

   <Frame>
     <img src="/images/kb/ka0Vq0000007POH-00N5w00000Ri7BU-0EMVq000001P4np.jpg" style={{ width: 667, height: 161 }} />
   </Frame>

5. Under **Service account details**, enter the service account name and click **CREATE AND CONTINUE**.
6. From the **Select a role** filter, under Quick Access select **Basic** > **Owner** and click **CONTINUE**.

   <Frame>
     <img src="/images/kb/ka0Vq0000007POH-00N5w00000Ri7BU-0EMVq000001P5Yb.jpg" style={{ width: 670, height: 505 }} />
   </Frame>

7. Click **Done**.
8. On the left pane select **IAM & Admin** > **Service Accounts** and click the service account created.
9. Click **Keys** > **Add Key** > **Create new key**. A pop-up window opens.

   <Frame>
     <img src="/images/kb/ka0Vq0000007POH-00N5w00000Ri7BU-0EMVq000001P5f3.jpg" style={{ width: 348, height: 228 }} />
   </Frame>

10. Select the Key type as JSON and click **Create**.
11. The JSON file containing the JSON key downloads automatically.
12. Save the downloaded file.

## Enabling Google Sheets API

To enable the Google Sheets API, perform the following steps:

1. Navigate to the GCP Console.
2. On the left pane, under **APIs and services**, select **Library**.

   <Frame>
     <img src="/images/kb/ka0Vq0000007POH-00N5w00000Ri7BU-0EMVq000001U8UD.jpg" style={{ width: 178, height: 115 }} />
   </Frame>

3. Search and click the Google Sheets API.
4. On the top of the page, click **Enable**.
   <Frame>
     <img src="/images/kb/ka0Vq0000007POH-00N5w00000Ri7BU-0EMVq000001U8Vp.jpg" style={{ width: 227, height: 90 }} />
   </Frame>

**Note**: The Google service account's email address must be added as a Google Sheet Viewer.

## Connecting to Your Google Sheets Account

This section enumerates the options in the **Credentials** and **Details** panes in the Google Sheets Connector page. The components of the other panes in this page, **Scheduling** and **Name & Describe Your DataSet**, are universal across most connector types and are discussed in great detail in [Adding a DataSet Using a Connector](https://domo-support.domo.com/s/article/360042926274?language=en_US "Adding a DataSet Using a Connector").
