## Intro

Ad-Juster is a premier, pioneering data aggregation and reporting organization. Our platform provides actionable insights that allow media and technology companies to generate more revenue. This data aggregation platform for digital publishers, ad agencies, ad networks and ad tech platforms is an automated reconciliation and aggregate reporting system in the digital ad industry.

You can connect to your Ad-Juster account in the Data Center. This topic discusses the fields and menus that are specific to the Ad-Juster connector user interface. To add DataSets, set update schedules, and edit DataSet information, see [Adding a DataSet Using a Connector](/s/article/360042926274 "Adding a DataSet Using a Connector").

## Prerequisites

To connect to your Ad-Juster account and create a DataSet, you must have the API Key and API Secret.

## Connecting to Your Ad-Juster Account

This section enumerates the options in the **Credentials** and **Details** panes in the Ad-Juster Connector page. The components of the other panes in this page, **Scheduling** and **Name & Describe Your DataSet**, are universal across most connector types and are discussed in great detail in [Adding a DataSet Using a Connector](/s/article/360042926274 "Adding a DataSet Using a Connector").

## Credentials Pane

This pane contains fields for entering credentials to connect to your Ad-Juster account. The following table describes what is needed for each field:

| Field      | Description                            |
| ---------- | -------------------------------------- |
| API Key    | Enter the Adjuster account API Key     |
| API Secret | Enter the Adjuster account API Secret. |

Once you have entered valid Ad-Juster credentials, you can use the same account any time to create a new Ad-Juster DataSet. You can manage connector accounts in the **Accounts** tab in the Data Center. For more information about this tab, see [Manage Connector Accounts](/s/article/360042926054 "Manage Connector Accounts").
