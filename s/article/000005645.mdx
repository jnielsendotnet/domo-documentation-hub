## Intro

Modernizing Medicine, Inc. (ModMed) is a software company that provides cloud-based practice management and electronic health records (EHR) systems for medical settings. ModMed's software is designed to help medical practices improve patient care by providing clinical, financial, and operational solutions. Use Domo's ModMed connector to pull your data about patients, appointments, slots, charge items, and so on from your ModMed system. To learn more about the ModMed API, see [https://www.modmed.com/modmed-synapsys-api/.](https://www.modmed.com/modmed-synapsys-api/ "https://www.modmed.com/modmed-synapsys-api/")

You can connect to your ModMed account in the Data Center. This topic discusses the fields and menus specific to the ModMed connector user interface. To add Data Sets, set update schedules, and edit Data Set information, see [Adding a DataSet Using a Connector](https://domo-support.domo.com/s/article/360042926274?language=en_US "Adding a DataSet Using a Connector").

**Note:** **ModMed recommends running large dataloads during the off-peak hours. Running large dataloads during normal business hours may result in performance issues in ModMed, which could result in the ModMed team disabling the integration.**

## Prerequisites

To connect to your ModMed account and create a DataSet, you need the following:

* Environment
* Username
* Password
* Firm URL Prefix
* API Key

To obtain your username and password, submit a request to **[practicesolutions@modmed.com](mailto:practicesolutions@modmed.com)**. The ModMed Practice Solutions team will work through the process of enabling the integration and will provide you with instructions to generate a username and password. 

To obtain the API key, send an email request to **[synapsys@modmed.com](mailto:synapsys@modmed.com)**.

## Connecting to Your ModMed Account

ModMed uses OAuth 2.0 authentication.

This section enumerates the options in the **Credentials** and **Details** panes on the ModMed Connector page. The components of the other panes on this page, **Scheduling**, and **Name & Describe Your DataSet**, are universal across most connector types and are discussed in great detail in [Adding a DataSet Using a Connector](https://domo-support.domo.com/s/article/360042926274?language=en_US "Adding a DataSet Using a Connector").
