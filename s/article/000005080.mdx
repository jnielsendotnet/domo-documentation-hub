## Intro

The **Asset Library** stores all of your assets in Domo, but each card is unique and can be powered by a different design version of an asset. If you update an asset, you may need to update each card individually. You can find instructions for doing this below in [Update an Asset](#updating_asset).

In some cases, you may need to connect a card to a different asset. A possible use case is moving from a demo or custom version of an app to another version. This requires you to switch any cards to the correct asset. See instructions for how to [Switch an Asset](#switching_asset) below.

## Update an Asset

1. From the navigation header, select **More** > **Asset Library**.
2. In the
   **Asset Library**, locate the app you want to update.

   <Frame><img alt="my apps.png" src="/images/kb/ka05w0000012BmE-00N5w00000Ri7BU-0EM5w000005wHPw.jpg" style={{width: 921, height: 346}}/></Frame>

3. Select the app to display the
   **Overview** tab. Under the app name, there is a list of the app's versions.

   <Frame><img alt="version list.png" src="/images/kb/ka05w0000012BmE-00N5w00000Ri7BU-0EM5w000005wcL5.jpg" style={{width: 773.956, height: 265}}/></Frame>

4. Select the version you want or select
   **Latest version**, which automatically updates the app to the latest version in the future.
5. Go to the
   **Cards** tab. A list of cards deployed by the app displays.
6. Select the cards you want to update by checking the boxes.

   <Frame><img alt="cards to update.png" src="/images/kb/ka05w0000012BmE-00N5w00000Ri7BU-0EM5w000005wcLF.jpg" style={{width: 565.987, height: 260.994}}/></Frame>

7. Select
   **Update Version** and choose the version you would like to update the selected cards to.

   <Frame><img alt="card update version.png" src="/images/kb/ka05w0000012BmE-00N5w00000Ri7BU-0EM5w000005wcLK.jpg" style={{width: 483.999, height: 309.972}}/></Frame>

The cards you selected are now updated.

## Switch an Asset

1. Locate the card you want to switch the asset for.
2. Hover over the card until
   <InlineImage src="/images/kb/ka05w00000127Q7-00N5w00000Ri7BU-0EM5w000005vsLM.jpg" /> **Card options** displays in the top right corner. Select the arrow to display the card options, then **Edit Card**.

   The card editor displays.
3. In the card editor, locate the icon for the app in the top left corner and select it.

   <Frame><img alt="select icon.png" src="/images/kb/ka05w0000012BmE-00N5w00000Ri7BU-0EM5w000005wHPc.jpg" style={{width: 543, height: 301}}/></Frame>

   The **Overview** tab displays.
4. Select
   <InlineImage src="/images/kb/ka05w0000012BmE-00N5w00000Ri7BU-0EM5w000005wHPh.jpg" /> **Back** in the top left corner.

   <Frame><img alt="back button.png" src="/images/kb/ka05w0000012BmE-00N5w00000Ri7BU-0EM5w000005wcKl.jpg" style={{width: 536.989, height: 427.997}}/></Frame>

   The **Asset Library** displays.
5. Search for and find the asset you want to power the card.
6. Click
   **Select** **Design**.

   <Frame><img alt="select design.png" src="/images/kb/ka05w0000012BmE-00N5w00000Ri7BU-0EM5w000005wHPm.jpg" style={{width: 317, height: 524}}/></Frame>

   You can also do this from the design **Details** screen.

   <Frame><img alt="design details.png" src="/images/kb/ka05w0000012BmE-00N5w00000Ri7BU-0EM5w000005wHPr.jpg" style={{width: 674, height: 256}}/></Frame>

7. Connect the card according to the instructions for that app in the Knowledge Base.
8. Select
   **Save & Finish**.

The card is now connected to the new asset. You can confirm this by searching for the asset in the **Asset Library**, going to the **Cards** tab, and verifying that the card you connected now displays in the list.
