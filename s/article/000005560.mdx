## How To Use A Connector Built With The Web Data Connector 3.0 SDK

After you build a connector with the Web Data Connector 3.0 SDK, follow the below steps to be used with Tableau:

1. Download the connector file, domo-tableau-connector-1.0.0.taco using the link ([https://app.domo.com/DomoConnectors/DomoTableauWDC/domo-tableau-connector-1.1.taco](https://app.domo.com/DomoConnectors/DomoTableauWDC/domo-tableau-connector-1.1.taco "https://app.domo.com/DomoConnectors/DomoTableauWDC/domo-tableau-connector-1.1.taco")).
2. Place the.taco file in the following directory (default location):

   <Frame><img alt="Dialog box 1.png" src="/images/kb/ka0Vq0000003kMT-00N5w00000Ri7BU-0EMVq000002v1pR.jpg" style={{width: 735, height: 283}}/></Frame>

After you have installed the connector, it will appear in the list of connectors in Tableau.
