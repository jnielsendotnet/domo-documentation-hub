### Intro

Connector Permissioning allows you to decide which connectors people can use to bring data into Domo, making it easier to securely control which data comes into Domo and which data you leave outside.

---

### Access Connector Permissioning

Connector Permissioning is available to Domo admins. To access Connector Permissioning, select the slider icon from the left panel, and then **Connectors**.

### Connector Permissioning Options

There are three different options available for Connector Permissioning.

<Frame><img alt="Screen_Shot_2021-03-24_at_5.04.14_PM.png" src="/images/kb/ka0Vq0000006eCT-00N5w00000Ri7BU-0EM5w000005vPVQ.png" style={{width: 900, height: 457}}/></Frame>

#### All Connectors available

This means all the Connectors in an instance are available to anyone granted permission to connect to data in Domo. This is the most lenient policy.

#### Restrict Connector access

Restrict Connector access means Connectors will be available to anyone granted permission to connect to data in Domo if they are not in the policy. Anyone included in a policy will be restricted from using the Connectors in the policy.

#### Allow Connector access

This means any Connectors or people not in a policy will be restricted from being used or using Connectors. Anyone included in a policy will be granted permission to the Connectors in the policy.

To change options in your instance, select the option you would like and save it by choosing **Update**.

### Add Connectors to your Policy

To add Connectors to your policy, select **+** **Add Connector**.

<Frame><img alt="Picture1.png" src="/images/kb/ka0Vq0000006eCT-00N5w00000Ri7BU-0EM5w000005vPVS.png" style={{width: 220, height: 79}}/></Frame>

This will open a modal where you can choose the connectors you want to create policies for.

<Note>
**Note:** It takes a few minutes for the connectors list to fully populate.
</Note>

<Frame><img alt="Screen_Shot_2021-03-25_at_9.55.58_AM.png" src="/images/kb/ka0Vq0000006eCT-00N5w00000Ri7BU-0EM5w000005vPVT.png" style={{width: 899, height: 516}}/></Frame>

From this screen, you can select one or multiple Connectors to add to your policy. To start over, select **Deselect All**. Once you have finished selecting Connectors to add to your policy, select the **Add X Connectors** button.

### Add People to Your Policy

To add people to your policy, select the Connectors you would like to add people to, and then select the pencil icon to update user attributes:

<Frame><img alt="Screen_Shot_2021-01-20_at_6.33.54_PM.png" src="/images/kb/ka0Vq0000006eCT-00N5w00000Ri7BU-0EM5w000005vPVR.png" style={{width: 901, height: 442}}/></Frame>

From here, search for people or Groups to add to the policy and select the **+** icon next to their name to include them in the policy. Those included in the policy will appear at the top of the list. To remove someone from a policy, select the **X** next to their name.

<Frame><img alt="Screen_Shot_2021-01-20_at_6.34.26_PM.png" src="/images/kb/ka0Vq0000006eCT-00N5w00000Ri7BU-0EM5w000005vPVV.png" style={{width: 500, height: 567}}/></Frame>

### FAQ

<AccordionGroup>
<Accordion title="Which grant controls who can use Connector Permissioning?">

The **Manage Connectors** grant controls access to use Connector Permissioning.

</Accordion>

<Accordion title="How long does it take for the connectors to reflect after setting up the Connector access permissions?">

The connector access permissions are cached and refreshed every **5 minutes**.

</Accordion>
</AccordionGroup>
