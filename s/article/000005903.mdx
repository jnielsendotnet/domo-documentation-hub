## Intro

Consensus is a demo automation platform that helps sales and revenue teams scale by delivering on-demand, personalized video demos, interactive product tours, and automated product simulations. Consensus connector provides bulk data on key platform elements, including users, demo boards, and customer sales. For information, refer to the API documentation [https://app.goconsensus.com/api-documentation/](https://app.goconsensus.com/api-documentation/ "https://app.goconsensus.com/api-documentation/").

You can connect to your Consensus account in the Data Center. This topic discusses the fields and menus that are specific to the Consensus connector user interface. To add DataSets, set update schedules, and edit DataSet information, see [Adding a DataSet Using a Connector](https://domo-support.domo.com/s/article/360042926274?language=en_US "Adding a DataSet Using a Connector").

## Prerequisites

To connect to your Consensus account and create a DataSet, you need the following:

* The API Key and API Secret associated with your Consensus account
* Enter the email ID used to log in to your Consensus account

## Connecting to Your Consensus Account

This section enumerates the options in the **Credentials** and **Details** panes on the Consensus Connector page. The components of the other panes on this page, **Scheduling**, and **Name & Describe Your DataSet**, are universal across most connector types and are discussed in great detail in [Adding a DataSet Using a Connector](https://domo-support.domo.com/s/article/360042926274?language=en_US "Adding a DataSet Using a Connector").

## Credentials Pane

This pane contains fields for entering credentials to connect to your Consensus account. The following table describes what is needed for each field:

| Field      | Description                                                                                                                                                                                 |
| ---------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| API Key    | Enter the API Key associated with your Consensus account. The API Key is provided in the **Integration** > **API Credentials** section on [goconsensus.com.](http://goconsensus.com/)       |
| API Secret | Enter the API Secret associated with your Consensus account. The API Secret is provided in the **Integration** > **API Credentials** section on [goconsensus.com.](http://goconsensus.com/) |
| User Email | Enter the user email ID used to log in to your Consensus account.                                                                                                                           |

Once you have entered valid Consensus credentials, you can use the same account at any time to create a new Consensus DataSet. You can manage connector accounts in the **Accounts** tab in the Data Center. For more information about this tab, see [Manage Connector Accounts](https://domo-support.domo.com/s/article/360042926054?language=en_US "Manage Connector Accounts").
