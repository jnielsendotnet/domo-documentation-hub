---
title: "Deleting a Document Version"
---


You can delete unwanted versions of a Doc Card file. You do this in the Details view for the Doc Card. In the Details view, all versions of the file are stored as thumbnails. You can delete any version by locating the thumbnail for that version and clicking the <img src="/images/kb/ka05w00000123TL-00N5w00000Ri7BU-0EM5w000005vOGr.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/> icon.

**To delete a version of a file,**

1. Click the Doc Card to open its Details view.
2. Locate the thumbnail for the version of the document you want to delete and mouse over it.
3. Click
   <img src="/images/kb/ka05w00000123TL-00N5w00000Ri7BU-0EM5w000005vOGr.png" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/>.

The version is deleted.
