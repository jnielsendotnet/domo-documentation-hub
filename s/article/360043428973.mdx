### Intro

The following table lists the types of Filter charts available in Domo.

| Chart Type                                                                                                                                                                                      | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       | Example                                                                                                         |

When you add them to a Card Page in Domo, users visiting that Page can click on one or more values to filter the other Cards in the Page. For example, you could build a Slicer Card with names of U.S. states and add it to a Card Page. If you then clicked "Texas" and "Oklahoma," all of the Cards in the Page would be filtered to show only data for those states.<br/>For more information about creating Slicer Cards, see [Page Filters and Filter Views on Dashboards](/s/article/360042923914).                                                                                                                            | <img alt="slicer_edit.gif" src="/images/kb/ka0Vq00000017yj-00N5w00000Ri7BU-0EM5w000005vP7U.gif"/>               |

As with Slicer Cards, you can filter on multiple values when you use Checkbox Cards.<br/>For more information about creating Slicer Cards, see [Page Filters and Filter Views on Dashboards](/s/article/360042923914).                                                                                                                                                                                                                                                                                                                                                | <img alt="checkbox.gif" src="/images/kb/ka0Vq000000Cr3l-00N5w00000Ri7BU-0EM5w000005vPBC.gif"/>                  |

When you add a range Card to a Card Page in Domo, users visiting that Page can use the slider buttons to narrow the range between the minimum and maximum values. All other Cards in the Page then filter to show the data for the newly selected range.<br/>For more information about creating Range Cards, see [Page Filters and Filter Views on Dashboards](/s/article/360042923914)/s/article/360042923914.                                                                                                                                                                                                                   | <img alt="range.gif" src="/images/kb/ka0Vq00000018Uz-00N5w00000Ri7BU-0EM5w000005vP4L.gif"/>                     |

When you add them to a Card Page in Domo, users visiting that Page can select a range of dates from the calendar as a filter. For example, if you selected all dates from April 1st to July 31st on the calendar, all other Cards in the Page would be filtered to show just the data for these dates.<br/>For more information about creating Date Selector Cards, see [Page Filters and Filter Views on Dashboards](/s/article/360042923914). | <img alt="date_selector.gif" src="/images/kb/ka0Vq00000018px-00N5w00000Ri7BU-0EM5w000005vPAJ.gif"/>             |

When you add them to a Dashboard in Domo, users visiting that Dashboard can click on values to filter the other Cards on the Dashboard.
