### Intro

Domo provides various in-product notifications about maintenance and new features, including pop-ups like this feature release announcement:

<Frame><img alt="Screenshot 2024-07-30 at 2.27.55 PM.png" src="/images/kb/ka0Vq0000007TYL-00N5w00000Ri7BU-0EMVq000002045i.jpg" style={{width: 350, height: 242}}/></Frame>
These notifications display when you first log into Domo if you have the required grant enabled for your role. This article describes the required grant and the different types of notifications you might see if they're enabled for your role.

---

### Required Grant

To receive in-product notifications when logging into Domo, the Receive Domo System Messages grant must be enabled for your role. Users with an Admin system role have this grant by default. If any other admin-level roles are created, this grant must be added to them as needed so that everyone who needs to receive system messages has access. Learn how to [add grants to custom roles](/s/article/360043438973).

<Warning>
  **Important:** If your Domo admin is not actively using the platform, we recommend assigning the Receive Domo System Messages grant to at least one active user so that your organization can stay updated on what's new in Domo and receive important security announcements. Learn how to [manage grants](/s/article/360043438973).
</Warning>

### Notification Types

# Feature Release

Several times a year, including every March at Domopalooza, Domo releases a slate of new features and enhancements, including products going to beta. A pop-up notifies you of the release and provides a link to explore detailed announcements about each item. This pop-up displays for about four weeks if not dismissed.

<Frame><img alt="Screenshot 2024-07-30 at 2.27.55 PM.png" src="/images/kb/ka0Vq0000007TYL-00N5w00000Ri7BU-0EMVq000002045i.jpg" style={{width: 349, height: 241}}/></Frame>
Domo also notifies admins about new feature releases via emails, [release notes](https://domo-support.domo.com/s/article/Current-Release-Notes?language=en_US) in the Knowledge Base, and the [New Product Features](https://www.domo.com/product/new-features#/) <InlineImage src="/images/kb/ka0Vq00000001FR-00N5w00000Ri7BU-0EMVq0000000XDh.jpg" /> page.

# Upcoming Product Release

A week before our product release goes live, we publish an announcement with brief descriptions/titles of every feature that will be included. This announcement is three pages long and lists our new, enhanced, and beta features.

# Planned System Maintenance

System maintenance messages are displayed for admin users. **Note:** Sometimes, these messages only display for specific products, so if you do not own or use that product in Domo, you may not receive the pop-up.

<Frame><img alt="Screenshot 2024-08-02 at 11.00.50 AM.png" src="/images/kb/ka0Vq0000007TYL-00N5w00000Ri7BU-0EMVq0000023cpB.jpg" style={{width: 350, height: 550}}/></Frame>
# Product Walkthroughs

For some newly updated or released products, a walkthrough tour pops up to guide you through using the tool. These tours are product-specific, often have multiple steps, and don't display in other parts of the Domo platform.
