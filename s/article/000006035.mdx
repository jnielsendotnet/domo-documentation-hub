# New Features and Enhancements



## App Components

The new drag-and-drop app component builder includes various features for creating visually appealing app components. You can use the six new app components and twenty templates to assemble and edit your apps in one place. Learn how to [create and customize app components in App Studio](https://domo-support.domo.com/s/article/000005864?language=en_US).

The following components are available:

* List
* Gallery
* Grid
* Filter List
* Details
* Banner

Available widgets include:

* Images
* Hyperlinks
* Pills
* Buttons
* Text
* Sort
* Search

## DomoStats: Balance Statements

<Frame><img alt="" src="/images/kb/ka0Vq000000QpLR-00N5w00000Ri7BU-0EMVq00000C9tJJ.jpg" style={{width: 114, height: 114}}/></Frame>
The new Balance Statement DomoStats reports include information on your account's balance summary and balance details. These reports enable you to easily navigate between the data in the report and the data presented in the Credit Balance tab of the Credit Utilization screen in the Admin Settings.

Balance Statement reports are available in the primary Domo instance for an account and are only available to those with Consumption. Learn more about your [DomoStats reports](https://domo-support.domo.com/s/article/000005326?language=en_US#domostats_reports).

## Workflows



### Auto Save

The new Auto Save functionality in Workflows ensures that any changes made to your canvas are continuously preserved in real time, reducing the risk of lost work due to unexpected interruptions. Learn more about [Auto Save in Workflows](https://domo-support.domo.com/s/article/000005331?language=en_US#auto_save).

<Frame><img alt="" src="/images/kb/ka0Vq000000QpLR-00N5w00000Ri7BU-0EMVq00000CAMUz.jpg" style={{width: 890, height: 506}}/></Frame>
### Sub Flow

Call one workflow from within another and receive output values to be used in subsequent steps in the parent workflow with the new Sub Flow option in Workflows.

This feature enables you to build modular and scalable workflows by reusing existing logic and processes, reducing redundancy and simplifying workflow maintenance.

Learn how to [use Sub Flow to streamline your automations](https://domo-support.domo.com/s/article/000005930?language=en_US) in Workflows.

## Magic ETL

### "Add from Canvas" for Magic ETL Tiles

Magic ETL now includes the ability to add a tile directly to the canvas using a plus button that appears next to all non-terminal tiles on the canvas. This button opens a menu that enables you to search for the tile you want to add, or scroll through the list of tiles to choose from, with the most commonly used tiles listed at the top of the list.

The feature improves the speed of DataFlow creation and overall workflow efficiency.

<Frame><img alt="" src="/images/kb/ka0Vq000000QpLR-00N5w00000Ri7BU-0EMVq00000CFA6b.jpg" style={{width: 892, height: 474}}/></Frame>
### Power User Updates

We are releasing a suite of Magic ETL feature enhancements over the next few months that will improve the lives of technical Magic ETL users.

The three new enhancements in this release include:

* The ability to configure a tile regardless of whether there are valid upstream tiles.
* The "Run to Here" functionality only runs the preview to the point where "Run to Here" is selected, allowing you to test streams of logic without needing to preview an entire DataFlow.
* An
  **Inputs** tab on each tile that lets you select the input to the tile without having to drag the line from the previous tile to it.

<Frame><img alt="" src="/images/kb/ka0Vq000000QpLR-00N5w00000Ri7BU-0EMVq00000CFNob.jpg" style={{width: 495, height: 413}}/></Frame>
### Split Join and Split Filter Tiles

The new Split Join and Split Filter tiles provide more visibility into your data, easier troubleshooting, and pipeline development.

* **Split Join —**
  Manage multiple output streams from a single Join and easily identify rows that joined and those that did not.
* **Split Filter —**
  Capture rows that meet your filter criteria while viewing those that do not in a separate stream.

Learn how to [use the split join and split filter tiles](https://domo-support.domo.com/s/article/000005919?language=en_US) in Magic ETL.

## Workspaces

Workspaces, now available on the web and mobile, provide a centralized location for organizing Domo objects, such as Cards, Dashboards, Apps, DataSets, DataFlows, Workflows, and Worksheets.

Workspaces enable you to find relevant content more efficiently, support team and department-specific organization using folders and subfolders, and enhance the navigation experience on both web and mobile platforms. Additionally, Workspaces make it easy to control sharing the right content with the right people and also revoking access when needed.
