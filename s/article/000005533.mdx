## Intro

LiveRail is an industry-leading online video monetization platform for agencies, advertisers, publishers, and networks. LiveRail provides video ad serving, real-time bidding, delivery, optimization, and analytics along with VAST and VPAID compliance. LiveRail’s API system is a powerful interface that enables clients to interact programmatically with the LiveRail Platform.

You can connect to your LiveRail account in the Data Center. This topic discusses the fields and menus that are specific to the LiveRail connector user interface. To add DataSets, set update schedules, and edit DataSet information, see [Adding a DataSet Using a Connector](https://domo-support.domo.com/s/article/360042926274?language=en_US "Adding a DataSet Using a Connector").

## Prerequisites

To connect to your LiveRail account and create a DataSet, you must have the username and password associated with your LiveRail account.

## Connecting to Your LiveRail Account

This section enumerates the options in the **Credentials** and **Details** panes on the LiveRail Connector page. The components of the other panes on this page, **Scheduling**, and **Name & Describe Your DataSet**, are universal across most connector types and are discussed in great detail in [Adding a DataSet Using a Connector](https://domo-support.domo.com/s/article/360042926274?language=en_US "Adding a DataSet Using a Connector").

## Credentials Pane

This pane contains fields for entering credentials to connect to your LiveRail account. The following table describes what is needed for each field:

| Field    | Description                                               |
| -------- | --------------------------------------------------------- |
| Username | Enter the username associated with your LiveRail account. |
| Password | Enter the password associated with your LiveRail account. |

Once you have entered valid LiveRail credentials, you can use the same account at any time to create a new LiveRail DataSet. You can manage connector accounts in the **Accounts** tab in the Data Center. For more information about this tab, see [Manage Connector Accounts](https://domo-support.domo.com/s/article/360042926054?language=en_US "Manage Connector Accounts").
