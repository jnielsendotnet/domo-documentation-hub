title: "Duplicating Cards (\"Save As\")"
---


### Intro

Domo users with an Editor, Privileged, or Admin system role; or with a custom role with the Page Edit grant enabled, can duplicate a card using the **Save As** option. Duplicated cards can be edited independently from the original card. This differs from the [copy](/s/article/360042923254) functionality because copied cards are still linked to the original—if you edit a copied card, the changes are reflected on the original and on any other copies.

When you duplicate a card, you are assigned as the owner of the new card.

You can also duplicate multiple cards at once in the Admin Settings. You must have an Admin security role to use this feature. For more information about security roles, see [Security Role Reference](/s/article/360043438953 "Security Role Reference").

<Note>
  **Note:** If a dashboard has been locked, you can only move or copy cards if you are the dashboard owner or have an Admin system role or a custom role with the Manage All Cards, Pages, and Apps (App Studio) grant enabled. For more information about locking and unlocking content on dashboards, see [Managing Custom Roles](/s/article/360043438973 "Locking or Unlocking Page Content").
</Note>

### Duplicate a Single Card

You can duplicate a card from a dashboard or the card's Details view. Both processes are explained below.

1. Hover over the card you want to duplicate.
2. Select

   <Frame><img alt="" src="/images/kb/ka0Vq0000004MJh-00N5w00000Ri7BU-0EMVq000005Yn8H.jpg" style={{width: 430, height: 332}}/></Frame>
3. Make changes to elements like the card title and description.
     
   <Frame><img alt="" src="/images/kb/ka0Vq0000004MJh-00N5w00000Ri7BU-0EMVq000005YleM.jpg" style={{width: 514, height: 489}}/></Frame>
4. Use the location dropdown to choose on which dashboard to store the card or use the option to **Create New Dashboard**.

6. Select **Save**.

The duplicate card displays on the dashboard you selected.

This is especially useful after you have performed in-depth analysis on the card by [filtering the data](/s/article/360042924014 "Filtering Data in a Card Details View"). You can save your updated view of the card as a new version that can then be favorited or shared with other users.

### Duplicate Cards in Bulk

Admins can duplicate multiple cards with a single action.

<Note>
  **Note:** Admins are those with an Admin system role or a custom role with the Manage All Cards, Pages and Apps (App Studio) grant enabled. Learn more about [roles and grants](/s/article/360043438973).
</Note>

**To begin:** Go to the Admin Settings (**More** > **Admin**).

1. Under **Content**, select **Cards**.
     
   A list of all the cards in your Domo instance appears.
2. Check the boxes for all cards you want to duplicate.
     
   You can use the column filters to filter the cards in the list.
3. Select **Edit** > **Duplicate Cards** to display the confirmation modal.
     
   <Frame><img alt="" src="/images/kb/ka0Vq0000004MJh-00N5w00000Ri7BU-0EMVq000005Ypd7.jpg" style={{width: 849, height: 188}}/></Frame>
4. Use the dropdown to choose one or more dashboards on which to store the new duplicate cards.
     
   <Frame><img alt="" src="/images/kb/ka0Vq0000004MJh-00N5w00000Ri7BU-0EMVq000005YptF.jpg" style={{width: 550, height: 276}}/></Frame>
5. Select **Duplicate Cards**.

The duplicate cards display on the dashboard(s) you specified.
