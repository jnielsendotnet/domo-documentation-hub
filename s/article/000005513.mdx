## Intro

Box is an online file-sharing and personal content cloud management service for businesses. Use Domo's Box Analytics connector to pull data on files and folders, users, enterprise events, and single-user events from the Box. The Box Analytics Advanced connector enables you to filter options for 'File and Folder' and 'Enterprise Events' reports. To learn more about the Box Analytics Advanced API, see ([https://developer.box.com/reference/](https://developer.box.com/reference/ "https://developer.box.com/reference/") ).

You can connect to your Box Analytics Advanced account in the Data Center. This topic discusses the fields and menus that are specific to the Box Analytics Advanced connector user interface. To add DataSets, set update schedules, and edit DataSet information, see [Adding a DataSet Using a Data Connector](/s/article/360042926274 "Adding a DataSet Using a Data Connector").

## Prerequisites

To connect to your Box Analytics Advanced account and create a DataSet, you need the email address and password associated with your Box account.

## Connecting to Your Box Analytics Advanced Account

This section enumerates the options in the **Credentials** and **Details** panes in the Box Analytics Advanced Connector page. The components of the other panes in this page, including **Scheduling** and **Name & Describe Your DataSet**, are universal across most connector types and are discussed in great detail in [Adding a DataSet Using a Data Connector](https://domo-support.domo.com/s/article/360042926274?language=en_US "Adding a DataSet Using a Data Connector").

## Credentials Pane

Once you have entered valid Box Analytics Advanced credentials, you can use the same account any time to create a new Box Analytics Advanced DataSet. You can manage connector accounts in the **Accounts** tab in the Data Center. For more information about this tab, see [Managing User Accounts for Connectors](https://domo-support.domo.com/s/article/360042926054?language=en_US "Managing User Accounts for Connectors").
