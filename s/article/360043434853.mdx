## Intro

Basecamp V3 is a web-based project management tool. The Basecamp V3 connector integrates Domo with Basecamp V3 to extract data from the Basecamp V3 API. You can use this connector to retrieve data about documents, attachments, calendar events, projects, people, and the like. To learn more about the Basecamp V3 API, visit their Github page ([https://github.com/Basecamp V3/bc3-api](https://github.com/basecamp/bc3-api "https://github.com/basecamp/bc3-api")).

The Basecamp V3 connector is a "Cloud App" connector, meaning it retrieves data stored in the cloud. In the Data Center, you can access the connector page for this and other Cloud App connectors by clicking **Cloud App** in the toolbar at the top of the window.

You connect to your Basecamp V3 account in the Data Center. This topic discusses the fields and menus that are specific to the Basecamp V3 connector user interface. General information for adding DataSets, setting update schedules, and editing DataSet information is discussed in [Adding a DataSet Using a Data Connector](/s/article/360042926274 "Adding a DataSet Using a Data Connector").

## Prerequisites

To connect to your Basecamp V3 account and create a DataSet, you must have the following:

* The username or email address you use to log into Basecamp V3
* The password you use to log into Basecamp V3

## Connecting to Your Basecamp V3 Account

This section enumerates the options in the **Credentials** and **Details** panes in the Basecamp V3 Connector page. The components of the other panes in this page, **Scheduling** and **Name & Describe Your DataSet**, are universal across most connector types and are discussed in greater length in [Adding a DataSet Using a Data Connector](/s/article/360042926274 "Adding a DataSet Using a Data Connector").

## Credentials Pane

The Domo Basecamp V3 connector uses OAuth to connect, so there is no need to enter credentials within Domo. Click **Connect** (or select **Add Account** if you have existing Basecamp V3 accounts in Domo) to open the Basecamp V3 OAuth screen where you can enter your Basecamp V3 username/email address and password. Once you have entered valid Basecamp V3 credentials and authorized Domo to connect to Basecamp V3, you can use the same account any time you go to create a new Basecamp V3 DataSet. You can manage connector accounts in the **Accounts** tab in the Data Center. For more information about this tab, see [Managing User Accounts for Connectors](/s/article/360042926054 "Managing User Accounts for Connectors").

<Note>
**Note:** If you are already logged into Basecamp V3 when you connect in Domo, you are authenticated automatically when you click **Add account**. If you want to connect to an account that is different from the one you are logged into, you must first log out of Basecamp V3.

</Note>

## FAQ

<AccordionGroup>
<Accordion title="How do I know my Basecamp V3 3 account credentials are secure?">

The login process uses the OAuth process so your Basecamp V3 credentials are never seen or stored by Domo. This keeps your login secure.

</Accordion>

<Accordion title="How frequently will my data update?">

As often as needed.

</Accordion>

<Accordion title="Can this connector pull custom fields?">

No. Reports and columns are predefined by Basecamp V3.

</Accordion>

<Accordion title="What is the difference between the Basecamp V3 and the Basecamp V3 3 connector?">

The Basecamp V3 3 connector provides more reports and also has more options available to retrieve filtered, customized, and meaningful data.

</Accordion>
</AccordionGroup>
