## About

HipChat is a chat client produced by Atlassian which features cloud storage and flexible communications. Domo's HipChat connector inputs all your HipChat communications into information you can see and understand.

You can connect to your HipChat account in the Data Center. This article explains the fields and menus that are specific to the HipChat connector user interface. To add DataSets, set update schedules, and edit DataSet information, see [Adding a DataSet Using a Data Connector](https://domo-support.domo.com/s/article/360042926274?language=en_US "Adding a DataSet Using a Data Connector").

## Prerequisites

To connect to your HipChat account and create a DataSet, you should add an HipChat account name and you must have the **API Auth Token**.

## Connecting to Your HipChat Account

This section lists the options in the **Credentials** and **Details** panes in the HipChat connector page. The components of the other panes in this page, **Scheduling** and **Name & Describe Your DataSet**, are universal across most connector types and are discussed in great detail in [Adding a DataSet Using a Data Connector](https://domo-support.domo.com/s/article/360042926274?language=en_US "Adding a DataSet Using a Data Connector").
