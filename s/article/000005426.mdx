## Intro

Adaptive Insights, formerly Adaptive Planning, provides online software that enables users to replace multiple spreadsheets with an integrated system that can consolidate data. Use Domo's Adaptive Insights connector to generate reports about data from standard, modeled, and cube sheets, dimensions, accounts, instances, and so on.

You can connect to your Adaptive Insights account in the Data Center. This topic discusses the fields and menus that are specific to the Adaptive Insights connector user interface. To add DataSets, set update schedules, and edit DataSet information, see [Adding a DataSet Using a Data Connector](/s/article/360042926274).

## Prerequisites

To connect to your Adaptive Insights account and create a DataSet, you must have the following:

- If you select **Authentication** as **Authenticate through Adaptive Planning**, then you need the Username, Password, and Instance.
- If you select  **Authentication** as **Authenticate through Workday**, then you need the Token Endpoint, REST Endpoint, Workday Client ID, Integration System User ID, and Private Key.

## Connecting to Your Adaptive Insights Account

This section enumerates the options in the **Credentials** and **Details** panes on the Adaptive Insights Connector page. The components of the other panes on this page, **Scheduling**, and **Name & Describe Your DataSet**, are universal across most connector types and are discussed in great detail in [Adding a DataSet Using a Data Connector](/s/article/360042926274).

## Credentials Pane

This pane contains fields for entering credentials to connect to your Adaptive Insights account. The following table describes what is needed for each field:  

| Field | Description |
| --- | --- |
| Token Endpoint | Enter the Token Endpoint. |
| Rest Endpoint | Enter the Rest Endpoint. |
| Authentication | Select, if you want to use the credentials for Adaptive Planning or Workday. |
| Username | Enter your Adaptive Insights username. |
| Password | Enter your Adaptive Insights password. |
| Instance | Find your instance code in Admin -> System -> General Setup under Setup Information -> Code. |
| Workday Client ID | Enter your Workday Client ID. |
| Integration System User ID | This ID should start with PublicAPIISU. |
| Private Key | Enter the private key that matches the public key you uploaded in Workday. |

Once you have entered valid Adaptive Insights credentials, you can use the same account any time you go to create a new Adaptive Insights DataSet. You can manage connector accounts in the **Accounts** tab in the Data Center. For more information about this tab, see [Managing User Accounts for Connectors](/s/article/360042926054).

## Details Pane

This pane contains a primary **Reports** menu, along with various other menus which may or may not appear depending on the report type you select.

| Menu | Description |
| --- | --- |
| Report Type | You can select **Data** or **Assets**.   |  |  | | --- | --- | | Data | Retrieves the data. | | Assets | Retrieves the assets such as accounts, dimensions, levels | |
| Report | Select the Adaptive Insights report you want to run. The following reports are available:   |  |  | | --- | --- | | **Report Name** | **Description** | | Accounts | Retrieves metadata for all accounts in the system. | | Active Currencies | Retrieves metadata for all currencies configured in the system. | | Attributes | Returns the list of attributes. | | Export Data | Retrieves a set of data from the requested version in the requested instance. | | Export Data from Sheets (Standard, Modeled or Cube) | Retrieves data for the accounts, dimensions and levels that correspond to the selected sheet. | | Customize data export | Retrieves the data for the accounts, dimensions and levels specified. | | Dimensions Structure | Retrieves metadata about custom dimensions in the system preserving their structure. | | Dimensions with Attributes | Represents a single member value of a custom dimension being returned in the response to an exportDimensions Call with Attributes passed in the request. | | Instances | Retrieve metadata about the instances a user has access to if the user has access to multiple instances. | | Levels | Retrieves metadata about organization levels in the system. | | Roles | Retrieves metadata about roles in the system. | | Versions | Retrieves metadata about versions in the system. | |
| Rollup Values | Select if you want to include the rollup values in the retrieved data. |
| Data Filtering | Choose whether or not you want to filter by dates. |
| Duration | Select the version name. |
| Data Version | Select the version name. |
| Currency | Select the currency. |
| Date Sheet | Select the sheet name. |
| Dimension Name | Select the dimension name. |
| Select Accounts to Import | Select the accounts belonging to the selected sheet that you want to import. |
| Select Dimensions To Import | Select the dimensions belonging to the selected sheet that you want to import. |
| Dimension Values | Select the dimensions belonging to the select sheet that you want to import. |
| Levels Values | Select the dimensions belonging to the select sheet that you want to import. |
