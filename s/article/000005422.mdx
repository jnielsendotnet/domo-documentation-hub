## Intro

The Magazine Manager is a global leader in magazine publishing solutions, serving over 15,000 publications worldwide. The Magazine Manager connector enables you to pull data from your Magazine Manager system into Domo.

You can connect to your Magazine Manager account in the Data Center. This topic discusses the fields and menus that are specific to the Magazine Manager connector user interface. To add DataSets, set update schedules, and edit DataSet information, see [Adding a DataSet Using a Data Connector](https://domo-support.domo.com/s/article/360042926274?language=en_US "Adding a DataSet Using a Data Connector").

## Prerequisites

To connect to your Magazine Manager account and create a DataSet, you must have the following:

* Password
* Username
* Domain

## Connecting to Your Magazine Manager Account

This section enumerates the options in the **Credentials** and **Details** panes on the Magazine Manager Connector page. The components of the other panes on this page, **Scheduling**, and **Name & Describe Your DataSet**, are universal across most connector types and are discussed in great detail in [Adding a DataSet Using a Data Connector](https://domo-support.domo.com/s/article/360042926274?language=en_US "Adding a DataSet Using a Data Connector").
