### Intro

<Warning>
  **Important:** These instructions only apply to Standard Pages, an older version of dashboards. If you are using a Domo dashboard and need to resize cards, choose **Settings menu** (wrench icon) > **Edit Dashboard** and interact with the drag and drop interface.
</Warning>

You can resize cards on a Standard Page. By default, cards are medium-sized, except for Domo App Cards, which are full-sized. The reference table below shows the size options. Enlarging a card makes it easier to view all of the information in the card. Shrinking a card makes it easier to view more cards at time, but less information is visible.

<Note>
  **Note:** If a Standard Page has been locked, you can only resize cards if you are the owner, have an Admin default security role, or a custom role with the Manage All Cards, Pages and Apps (App Studio) grant enabled.
</Note>


---

### Configure Dimensions for a Full-Size Card

The <img src="/images/kb/ka0Vq000000BQGv-00N5w00000Ri7BU-0EMVq000006qa2L.jpg" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/> **Full** size o ption for a single card is 4 cards medium width x 3 medium cards height. A full-size card may leave large amounts of white space in the card or cut off data you want your viewers to see. If this happens, consider changing the default height and width for a Full card.

Follow the steps below to change the default height and width for a table when **Full** is selected:

1. Open the card in **Analyzer** by selecting <img src="/images/kb/ka0Vq000000BQGv-00N5w00000Ri7BU-0EMVq000006qZZK.jpg" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/> **Card options** > **Edit in Analyzer**.
2. Under **Chart properties**, select **Full size settings**.
     
   <Frame><img alt="" src="/images/kb/ka0Vq000000BQGv-00N5w00000Ri7BU-0EMVq000006qa2M.jpg" style={{width: 801, height: 358}}/></Frame>
3. In the **Full Size Settings** modal, set the width and height for your card by dragging the desired number of squares across the grid. Or, you can select your desired width and height from the menus under the grid.
     
   <Frame><img alt="" src="/images/kb/ka05w00000129v8-00N5w00000Ri7BU-0EM5w000005vOm4.png" style={{width: 265, height: 477}}/></Frame>
4. After choosing your desired dimensions, select **Apply** to save changes.

The default Full size for this specific card is now updated.
