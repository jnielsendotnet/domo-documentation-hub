### New Features and Enhancements

# Advanced Scheduling for Data Freshness

Advanced Scheduling gives you more control over when your data is queried for updates, allowing you to check only when necessary. This can reduce expenses related to overall data freshness.

<Frame><img alt="Advanced scheduling - Settings (1).png" src="/images/kb/ka0Vq00000072GP-00N5w00000Ri7BU-0EMVq000003QUIT.jpg" style={{width: 899, height: 407}}/></Frame>
Learn more about data freshness in the [Cloud Amplifier Overview](/s/article/4412849158167).

# Workflows Data Query and AI Assist

The new "Text to SQL" AI feature makes it even easier to create queries using the DataSet Query Task in Workflows. You can enter your query using plain text and receive an AI-generated SQL statement to use in your workflow.

Learn more about the [DataSet Query Task](https://domo-support.domo.com/s/article/000005331?language=en_US#add_query_table) in Workflows.

The new ON property in the Join Data tile allows you to write your own freeform ON clause for joins. This allows you to write complex joins, including non-equijoins and ranged joins.

We have converted the Activity Log to React. The following improvements come with this update:

* More intuitive event filtering with Event Type and Event broken out into separate dropdown filters.
* Date and Person filtering are now built into the column header rather than being separated into dropdown filter elements.
* A more compact user experience, making better use of space.

See [Activity Log](/s/article/360042934574) in the Knowledge Base.

# AI Forecasting Universal Models

#### AI Forecasting Universal Models from Cards

AI Card Forecasting in App Studio allows you to configure advanced forecasting on a chart card using [Domo.AI](http://Domo.AI). This feature uses the Universal Domo Forecasting Model and returns an intelligent forecast chart based on the card data.

You can also apply filters to the page, adjust variables, and generate a forecast based on the chart's information.

Learn how to [generate a forecast on an App Studio card](https://domo-support.domo.com/s/article/000005567?language=en_US#forecast_in_app).

<Frame><img alt="AI Card Forecasting (2).png" src="/images/kb/ka0Vq00000072GP-00N5w00000Ri7BU-0EMVq000003ULi1.jpg" style={{width: 649, height: 295}}/></Frame>
This feature is in limited beta. If you have questions about joining a future beta session for this feature, contact your Domo Account team.

#### AI Forecasting Tile in Magic ETL

The AI Forecasting Tile in Magic ETL allows you to predict future data points based on patterns and trends in your data. The forecast output includes two confidence interval ranges.

Learn more about the [Forecasting Universal Model.](https://domo-support.domo.com/s/article/000005567)

<Frame><img src="/images/kb/ka0Vq00000072GP-00N5w00000Ri7BU-0EMVq000003lIML.jpg" style={{width: 885, height: 472}}/></Frame>
This feature is in limited beta. If you have questions about joining a future beta session for this feature, contact your Domo Account team.

#### AI Forecasting Universal Model in the Domo. AI Playground

You can use the Forecasting Universal Model in the Domo. AI Playground to compare and test various configuration outputs. You can also extract the cURL code that generates the forecast and use the code in other settings, such as interacting with an API.

Learn how to [use the Forecasting Universal Model in the Domo.AI Playground](https://domo-support.domo.com/s/article/000005567?language=en_US#forecasting_service_domo-ai_playground).

# DomoStats: Groups Report

<Frame><img alt="rtaImage.jpeg" src="/images/kb/ka0Vq0000002hX7-00N5w00000Ri7BU-0EMVq000001VwNJ.jpg"/></Frame>
The DomoStats Groups report lists Domo groups and relevant attributes.

This report includes valuable information like:

* Group Name
* Group Type
* Created Date
* Creator Id

The Groups report can be joined with other group-related reports, such as the DataSet Access report. Learn more about [DomoStats](/s/article/360043433813).

<Note>
  **Note:** A previous DomoStats report provided information on group memberships. This report was incorrectly named "Groups" and has been renamed "Group Members." Learn about the [Group Members report](#domostats_group_members) above.
</Note>
