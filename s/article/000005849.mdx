### Intro

<Note>
  **Note:** This feature is in beta. Contact your account team if you want to join.
</Note>

Domo FileSets allow you to gain insights from images, documents, transcripts, and other types of unstructured data. You can extract structured information from images or customer reviews to power visualizations and data products, automate document-driven business processes, or leverage AI Search to ground AI Agent responses using non-public documentation.

---

### Required Grants

The following grants allow you to use the FileSets feature. FileSets have the same permission requirements as DataSets.

* **Manage DataSet —** This grant allows you to view, edit, and delete any DataSets in an instance.
* **Edit DataSet —** This grant allows you to create and edit DataSets you have access to.

Learn about [grants and custom roles](https://domo-support.domo.com/s/article/360043438973?language=en_US).

# Upload Files

Follow these steps to upload files directly from your device to FileSets:

1. Select **Upload Files** > **Upload** from the FileSets screen.

   <Frame><img alt="" src="/images/kb/ka0Vq000000CZYH-00N5w00000Ri7BU-0EMVq0000078OXS.jpg" style={{width: 361, height: 143}}/></Frame>
2. You can browse files or drag and drop them. FileSets supports all common image and document file types.

   <Frame><img alt="" src="/images/kb/ka0Vq000000CZYH-00N5w00000Ri7BU-0EMVq000007FL6Z.jpg" style={{width: 900, height: 464}}/></Frame>
3. Name your FileSet (required) and give it an optional description.

   <Frame><img alt="" src="/images/kb/ka0Vq000000CZYH-00N5w00000Ri7BU-0EMVq000007FQRB.jpg" style={{width: 850, height: 447}}/></Frame>
Your uploaded file is now available in the FileSets list.

After uploading your files, you can enable [AI Search](#ai_search) and organize your data using folder management in FileSets. You can also add new folders and files, view, download, and delete files or folders.

<Frame><img alt="" src="/images/kb/ka0Vq000000CZYH-00N5w00000Ri7BU-0EMVq0000078yNx.jpg" style={{width: 897, height: 404}}/></Frame>

### Enable AI Search

Enabling AI Search in your FileSets allows you to search the contents of your files from the file listing tab. This also makes your files accessible to Domo AI and allows AI Chat and other agents to surface more accurate insights based on your uploaded data and files.

To enable AI Search for your FileSet, go to the **Overview** tab and select **Enable AI Search**.

<Frame><img alt="" src="/images/kb/ka0Vq000000CZYH-00N5w00000Ri7BU-0EMVq000007FSMX.jpg" style={{width: 898, height: 170}}/></Frame>
Select **Enable** to confirm.

### Access FileSets From an Agent

AI Agents can use FileSets within an **AI Agent Task** inside Workflows.

Follow these steps to enable FileSets in Workflows:

1. Navigate to the workflow you'd like to add your FileSet to.

   <Frame><img alt="" src="/images/kb/ka0Vq000000CZYH-00N5w00000Ri7BU-0EMVq000007DMza.jpg" style={{width: 897, height: 229}}/></Frame>
2. Add a new shape or edit an existing shape and choose **AI Agent Task**.
     
   <Frame><img alt="" src="/images/kb/ka0Vq000000CZYH-00N5w00000Ri7BU-0EMVq000007FWef.jpg" style={{width: 450, height: 421}}/></Frame>
3. In the configuration panel, select **Configure AI Agent**.

   <Frame><img alt="" src="/images/kb/ka0Vq000000CZYH-00N5w00000Ri7BU-0EMVq000007DPas.jpg" style={{width: 894, height: 351}}/></Frame>
   The **AI Agent Task** configuration modal displays.
4. In the modal, go to the **Tools** tab, select **+** **Add Tools** and search for FileSets.


     <Frame><img alt="" src="/images/kb/ka0Vq000000CZYH-00N5w00000Ri7BU-0EMVq000007FWpx.jpg" style={{width: 899, height: 400}}/></Frame>
     <Frame><img alt="" src="/images/kb/ka0Vq000000CZYH-00N5w00000Ri7BU-0EMVq000007FWtB.jpg" style={{width: 899, height: 409}}/></Frame>
   
5. Check the box for the **Ai Search File Set** method, then select **Add Functions**.
     

     <Frame><img alt="" src="/images/kb/ka0Vq000000CZYH-00N5w00000Ri7BU-0EMVq000007FX65.jpg" style={{width: 599, height: 240}}/></Frame>
     <Frame><img alt="" src="/images/kb/ka0Vq000000CZYH-00N5w00000Ri7BU-0EMVq000007FXAv.jpg" style={{width: 252, height: 467}}/></Frame>
   
   The FileSet is added to the list of tools in the **Tools** tab.
6. Select the FileSet to view the FileSet details. Enter the required information, including the FileSet ID.

  **Find the File Set ID:** You can find the FileSet ID by viewing the FileSet in Domo. The FileSet ID appears in the URL, as shown in the highlighted area below.<img src="/images/kb/ka0Vq000000CZYH-00N5w00000Ri7BU-0EMVq000007FVAl.jpg" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/>

### FAQ

This section answers some frequently asked questions. Select a question to expand and view its answer.

<Accordion title="What file types are supported?">
  FileSets can store all common image and document file types. AI Search indexing is limited to the following file types: txt, md, CSV, CSS, JSON, HTML, doc, docx, xls, xlsx, pdf, jpg, jpeg, and png.

</Accordion>

<Accordion title="What is the size limit per file?">
  Images: 3.75MB

  Documents: 50MB (manual upload limit to 5MB)

</Accordion>

<Accordion title="How many files can I have?">
  Unlimited, but consumption rates do apply. There are consumption charges for Storage, Retrieval,  and AI Search Indexing.

</Accordion>

<Accordion title="Can I connect files from a cloud service like Google Drive?">
  Yes, you can connect to Amazon S3, SFTP, Google Drive, or Confluence, and more will become available.

</Accordion>

<Accordion title="Can I visualize/drop files in a dashboard or App Studio app?">
  Not at this time. This is on our roadmap.

</Accordion>

<Accordion title="How do I manage access to FileSets, both for uploading and usage?">
  Currently, permissions are based on the entire FileSet.

</Accordion>

<Accordion title="Will I be able to use FileSets with the text generation tile?">
  This is on our roadmap.

</Accordion>
