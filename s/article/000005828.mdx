## Intro

The Cvent Supplier Network is a comprehensive online platform designed to streamline the process of sourcing venues and suppliers for meetings and events, connecting event planners with a global database of hotels, unique venues, and other event-related service providers.

Using Cvent Supplier Network Connector, you can fetch information on digital advertising campaigns.

You can connect to your Cvent Supplier Network - Planners account in the Data Center. This topic discusses the fields and menus that are specific to the Cvent Supplier Network - Planners connector user interface. To add DataSets, set update schedules, and edit DataSet information, see [Adding a DataSet Using a Connector](https://domo-support.domo.com/s/article/360042926274?language=en_US "Adding a DataSet Using a Connector").

## Prerequisites

To connect to your Cvent Supplier Network - Planners account and create a DataSet, you need the following:

* Select the region for your Cvent instance.
* Client ID and Client ID from your Cvent app

## Connecting to Your Cvent Supplier Network - Planners Account

This section enumerates the options in the **Credentials** and **Details** panes on the Cvent Supplier Network - Planners Connector page. The components of the other panes on this page, **Scheduling**, and **Name & Describe Your DataSet**, are universal across most connector types and are discussed in great detail in [Adding a DataSet Using a Connector](https://domo-support.domo.com/s/article/360042926274?language=en_US "Adding a DataSet Using a Connector").

## Credentials Pane

This pane contains fields for entering credentials to connect to your Cvent Supplier Network - Planners account. The following table describes what is needed for each field:

| Field         | Description                                  |
| ------------- | -------------------------------------------- |
| Account Name  | Choose a name for your Cvent Account.        |
| Region        | Select the region for your Cvent instance.   |
| Client ID     | Enter the Client ID from your Cvent app.     |
| Client Secret | Enter the Client Secret from your Cvent app. |

Once you have entered valid Cvent Supplier Network - Planners credentials, you can use the same account at any time to create a new Cvent Supplier Network - Planners DataSet. You can manage connector accounts in the **Accounts** tab in the Data Center. For more information about this tab, see [Manage Connector Accounts](https://domo-support.domo.com/s/article/360042926054?language=en_US "Manage Connector Accounts").
