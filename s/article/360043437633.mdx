---
title: "Connecting to Sage 300 Data in Domo"
---

Sage 300 (formally Timberline) uses a database that you can access via a Sage ODBC driver. To connect to Sage data, you must use SQL to query and process the data via Workbench, which then sends it to Domo.

You can access Sage 300 in one of the following ways:

* Connect directly to the database using the ODBC driver (which has been reported as slow). You can learn more about the ODBC connector in Workbench here:
  [Connecting to ODBC data](/s/article/360042932734 "Creating a Job for Uploading Data Using Workbench 4").
* Use a product called Anterra Data Center, which basically copies the Sage data to a SQL-based database. You can then connect to that database using Workbench. Most users prefer Anterra, as it is generally a faster way to access Sage data.

To access the Sage 300 ODBC driver, contact your Sage rep.

To use the Anterra Data Center method instead, you can reach out to Anterra here:
 https://anterratech.com/sql-warehouse-for-sage/
