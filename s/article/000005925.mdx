## Intro

Google Sheets enables you to create and edit spreadsheets online while collaborating with other users in real-time. With Domo's Google Sheet OAuth Writeback WIF connector, you can transfer your data from a Domo dataset to your Google Sheet.

You can connect to your Google Sheets OAuth Writeback WIF account in the Data Center. This topic discusses the fields and menus that are specific to the Google Sheets OAuth Writeback WIF connector user interface. To add DataSets, set update schedules, and edit DataSet information, see [Adding a DataSet Using a Connector](https://domo-support.domo.com/s/article/360042926274?language=en_US "Adding a DataSet Using a Connector").

## Prerequisites

To connect to your Google Sheets OAuth Writeback WIF account and create a DataSet, you need the following:

**Workload Identity Federation** must be configured in the Google Cloud Console **before** an account can be created in Domo.

* If you have an existing Workload Identity Federation, you need the following:
  + If you select to enter the configuration values manually, then you need:
    - Project Number
    - Project ID
    - Pool ID
    - Provider ID
    - Service Account Email
    - Choose whether to use the user email address or a unique GUID ID representing the connector and your Domo instance
  + If you prefer to enter the JSON Configuration from the Google Cloud Console, then you need the following:
    - Enter the JSON configuration for the Workload Identity Federation
    - Select how you want to limit access to your External Identities
* If you select the **Go through prerequisites steps**, then enter the following:
  + - Project Number
    - Project ID
    - Pool ID
    - Provider ID
    - Service Account Email
    - Choose whether to use the user email address or a unique GUID ID representing the connector and your Domo instance

## Connecting to Your Google Sheets OAuth Writeback WIF Account

This section enumerates the options in the **Credentials** and **Details** panes on the Google Sheets OAuth Writeback WIF Connector page. The components of the other panes on this page, **Scheduling**, and **Name & Describe Your DataSet**, are universal across most connector types and are discussed in great detail in [Adding a DataSet Using a Connector](https://domo-support.domo.com/s/article/360042926274?language=en_US "Adding a DataSet Using a Connector").
