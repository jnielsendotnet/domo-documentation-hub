## Intro

DTCC provides industry-leading solutions to secure and shape the global financial marketplace. The Depository Trust and Clearing Corporation (DTCC) is a financial services company that provides clearing and settlement services for the financial markets. Settlement is integral to securities transactions. It boosts investor confidence and reduces market risk. Connect with DTCC to access applications and self-service tools to create custom software solutions.

You can connect to your DTCC account in the Data Center. This topic discusses the fields and menus that are specific to the DTCC connector user interface. To add DataSets, set update schedules, and edit DataSet information, see [Adding a DataSet Using a Connector](https://domo-support.domo.com/s/article/360042926274?language=en_US "Adding a DataSet Using a Connector").

## Prerequisites

To connect to your DTCC account and create a DataSet, you need the following:

* Client ID
* Username
* Password

## Connecting to Your DTCC Account

This section enumerates the options in the **Credentials** and **Details** panes on the DTCC Connector page. The components of the other panes on this page, **Scheduling**, and **Name & Describe Your DataSet**, are universal across most connector types and are discussed in great detail in [Adding a DataSet Using a Connector](https://domo-support.domo.com/s/article/360042926274?language=en_US "Adding a DataSet Using a Connector").
