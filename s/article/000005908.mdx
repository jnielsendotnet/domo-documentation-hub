### 

### October 1, 2025 Release

# App Components

**Updates:**

- Add new rows in Linked Tables.
- Rename column headers by typing directly in them without having to double-click into them first.

**Issues Resolved:**

- Hidden calculations were not being properly deleted from cards.
- Calculated columns in Editable Tables can now only be edited in the formula editor bar, preventing unintended changes from actions like double-clicking, copy-pasting, or dragging fill.
- Hiding a key column on a Linked Table caused problems when editing an app for a second time.

# App Studio

**Issues Resolved:**

- The "More" button in the left navigation menu was not responding as expected.
- Turning off the "Allow global date" setting in App Studio did not work as expected. This update ensures the smart text adheres to the global date setting, even when it is not disabled on a specific card.
- All content above the no-scroll line was being pushed out when saving an app in App Studio.
- Content layouts in App Studio are centered when using a fixed width.

# Cards

**Issues Resolved:**

- Smart text in App Studio did not reference persisted page filters until the user clicked on the card.
- Notebook Cards were not showing the correct variable values.

# Cloud Integrations

**Updates:**

- Added a back button on the integration creation pages for all cloud integration engines. This button takes the user to the "Manage Integrations" page.

**Issues Resolved:**

- The Integration Update endpoint was returning an invalid new integration state after updating an integration.

# Connectors

**Issues Resolved:**

- Data was being written incorrectly to Google Sheets when a filter was applied. The fix ensures that the filter is cleared before writing the data to the sheet.
- The Cvent REST connector experienced an 'out of memory' error when processing the "meeting\_request\_forms" report.

# Dashboards

**Issues Resolved:**

- Variables in smart text on dashboard cards were not displaying values correctly.
- Page filters were not working on dashboards when global date filters were disabled.

# Notifications

**Issues Resolved:**

- The subscribe button was only visible on your own profile, and not on other users' profiles.

# Scheduled Reports

**Updates:**

- The selector for the Reports area is now expanded to provide more space for the options.
