## Intro

EDGE^ By Ascential provides ecommerce data measurement, sales analytics and search optimization for brand manufacturers. Use Domo's EDGE^ By Ascential connector to retrieve your data from EDGE^ By Ascential. To learn more about EDGE^ By Ascential, see ([https://www.flywheeldigital.com/](https://www.flywheeldigital.com/ "https://www.flywheeldigital.com/") ).

You can connect to your EDGE^ By Ascential account in the Data Center. This topic discusses the fields and menus that are specific to the EDGE^ By Ascential  connector user interface. To add DataSets, set update schedules, and edit DataSet information, see [Adding a DataSet Using a Data Connector](https://domo-support.domo.com/s/article/360042926274?language=en_US "Adding a DataSet Using a Data Connector").

## Prerequisites

To connect to your EDGE^ By Ascential account and create a DataSet, you must have the following:

* API key.
* Client UUID.

For more information, reach out to your EDGE^ By Ascential account representative.

## Connecting to Your EDGE^ By Ascential Account

This section enumerates the options in the **Credentials** and **Details** panes in the EDGE^ By Ascential Connector page. The components of the other panes in this page, **Scheduling** and **Name & Describe Your DataSet**, are universal across most connector types and are discussed in great detail in [Adding a DataSet Using a Data Connector](https://domo-support.domo.com/s/article/360042926274?language=en_US "Adding a DataSet Using a Data Connector").
