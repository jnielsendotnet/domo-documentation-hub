## Intro

Salesforce Marketing Cloud's ExactTarget simplifies your email campaigns and helps drive better results. With the Domo ExactTarget connector, you can combine your ExactTarget data with any other marketing data to get a holistic picture of your email effectiveness. Get a quick, interactive view of your campaigns, with an all-in-one real-time dashboard. With real-time data at your fingertips, you can reach more customers for better results.

You can connect to your Salesforce ExactTarget (Marketing Cloud) OAuth account in the Data Center. This topic discusses the fields and menus that are specific to the Salesforce ExactTarget (Marketing Cloud) OAuth connector user interface. To add DataSets, set update schedules, and edit DataSet information, see [Adding a DataSet Using a Data Connector](/s/article/360042926274 "Adding a DataSet Using a Data Connector").

## Prerequisites

To connect to your Salesforce ExactTarget (Marketing Cloud) OAuth account and create a DataSet, you must have the following:

* Salesforce Marketing Cloud Subdomain
* Client ID
* Client Secret

## Connecting to Your Salesforce ExactTarget (Marketing Cloud) OAuth Account

This section enumerates the options in the **Credentials** and **Details** panes in the Salesforce ExactTarget (Marketing Cloud) OAuth Connector page. The components of the other panes on this page, **Scheduling**, and **Name & Describe Your DataSet**, are universal across most connector types and are discussed in great detail in [Adding a DataSet Using a Data Connector](/s/article/360042926274 "Adding a DataSet Using a Data Connector").

## Credentials Pane

This pane contains fields for entering credentials to connect to your Salesforce ExactTarget (Marketing Cloud) OAuth account. The following table describes what is needed for each field:

| Field                                | Description                                                                                           |
| ------------------------------------ | ----------------------------------------------------------------------------------------------------- |
| Salesforce Marketing Cloud Subdomain | Your Salesforce Marketing Cloud subdomain. It is a 28-character string starting with the letter "mc". |
| Client ID                            | The client ID from your OAuth 2.0 API integration.                                                    |
| Client Secret                        | The client secret from your OAuth 2.0 API integration.                                                |

Once you have entered valid Salesforce ExactTarget (Marketing Cloud) OAuth credentials, you can use the same account at any time to create a new Salesforce ExactTarget (Marketing Cloud) OAuth DataSet. You can manage connector accounts in the **Accounts** tab in the Data Center. For more information about this tab, see [Managing User Accounts for Connectors](/s/article/360042926054 "Managing User Accounts for Connectors").
