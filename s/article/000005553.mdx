# DomoStats: New Columns Used in Beast Modes Report & Beast Modes Updates

<Frame><img alt="Screenshot 2024-06-26 at 8.22.31 AM.png" src="/images/kb/ka0Vq0000002hX7-00N5w00000Ri7BU-0EMVq000001VwNJ.jpg" style={{width: 106, height: 108}}/></Frame>
With the new Columns Used in Beast Modes report, admins have more visibility into what DataSet columns are being used in calculated fields within their instance.

The Columns Used in Beast Modes DataSet includes two columns: BeastModeID and ColumnName.

Additionally, two new fields are available in the Beast Modes DomoStats report—ArchivedStatus and Validity. These fields can help you determine whether each calculation has been archived and if the calculation syntax is valid.

Learn more about [DomoStats](/s/article/360043433813).

# European Number Formatting

We have added the 'English (Netherlands)' option to the language localization settings. When the 'English (Netherlands)' option is enabled, number formatting throughout the instance defaults to European conventions, including using a dot (.) as a thousands separator and a comma (,) as a decimal separator. This feature includes default, company-wide, personal, and user-specific settings that override the default setting.

For more information, see [Use European Number Formatting](https://domo-support.domo.com/s/article/360043439133?language=en_US#use_european_number_formatting).

<Frame><img src="/images/kb/ka0Vq0000002hX7-00N5w00000Ri7BU-0EMVq000001TC0Q.jpg" style={{width: 303, height: 404}}/></Frame>
# DataSet Views Replace Blend Transform

With the [Data Center updates](#discoverability_updates), we have removed the Blend transform. Existing DataFusions are not being deprecated and continue to function as expected.

When editing a DataFusion in DataSet Views, your DataFusion will be saved as a DataSet View. This action cannot be undone.

DataSet Views allow you to filter, group, aggregate, and combine data in a spreadsheet interface without having to build a transform.

Learn more about [DataSet Views](https://domo-support.domo.com/s/article/360046074774?language=en_US) in the Knowledge Base.

# Analyzer Updates

#### Chart Property Number Input Types

All chart types in Analyzer now include a validation feature for input fields that accept numbers. Validation ensures clean numeric values throughout your chart.

In the example below, the height percentage attribute requires a numeric input between zero and one hundred. If the entered value is outside that range, then the attribute is highlighted in red and the chart does not render.

Context Sensitive Chart Properties in Analyzer allow you to right-click on chart elements and see the list of applicable chart properties. In the image below, right-clicking on the y-axis displays the category scale (Y) chart property.

The sort icons on Mega Tables have been updated for greater visibility. The icons now display as up and down arrows alongside numbers identifying the sort order of each column.

#### Quick Render in Analyzer

The new Quick Render feature in Analyzer no longer queries the data on every load, decreasing the overall render time. For charts using large DataSets, Quick Render allows you to update multiple chart elements without waiting for repeated chart refreshes.

# Workflows Updates

#### Canvas Facelift

The Workflows canvas receives a facelift with the following upgrades to aesthetics and usability:

* **Centrally positioned modals** **—** Modals for custom values and conditional rules are now centered in the canvas.
* **Collapsible sections in the configuration panel —** Each section in a shape’s configuration panel can be collapsed or expanded for greater viewing flexibility.
* **Fonts and colors —** The entire canvas has been updated with fonts and colors consistent with the Domo style guide.
* **Left gallery** **—** The left gallery now shows navigation controls for the Workflows canvas only, consistent with other Domo products.
* **New icons —** Shapes and data types are represented by new icons.
* **Shape dimensions** **—** Overall shape width has been reduced to maximize canvas area.
* **Shape titles** **—** Shape titles are now the only labels that appear in a workflow shape and can be wrapped to show multiple lines of characters.
* **Stacked parameters and mapping fields** **—** Parameters and mapping fields in the configuration panel are now stacked on top of each other.
* **Variable list** **—** The variable list has better spacing between entries.
