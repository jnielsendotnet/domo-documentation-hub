## MS Office Add-ins

<Frame><img alt="domo add-ins.jpg" src="/images/kb/ka0Vq0000000Rcv-00N5w00000Ri7BU-0EM5w000006vM3r.jpg" style={{width: 94, height: 105}}/></Frame>

The following functionality has been added to the Domo MS Office Add-ins. Learn more in the [Add-ins User Guide](/s/article/000005143).

#### Dynamic Image Rendering

To avoid distortion when resizing an image in a document, you can refresh the card after resizing it to restore the original image quality with the correct aspect ratio.

* Advanced Image Settings allow you to adjust the width, height, and scale of imported images
* Import Summary Number and hide card image (only the Summary Number displays in the document)
* Overall design improvement for import Settings
* Use transparent background

The Outlook add-in has enhanced support for both Android and iOS.

## Appstore Improvements

<Frame><img alt="Screenshot 2023-07-20 at 10.14.07 AM.png" src="/images/kb/ka0Vq0000000Rcv-00N5w00000Ri7BU-0EM5w000006vM7Z.jpg" style={{width: 706, height: 365}}/></Frame>

Improvements to the Appstore ecosystem include the following:

#### Auto-Detect Submission Components

When a listing is submitted, the system now auto-detects and suggests Components that best describe the listing and show up in search results.

#### Consolidating App Type and Components

When searching the Appstore, App Type options have been consolidated with Components to improve search results.

#### Consolidating Promotional and Service Listings

For greater clarity, we have removed the option to categorize a listing as Promotional and recommend that you use the Service type for these listings.

<Frame><img alt="service type.jpg" src="/images/kb/ka0Vq0000000Rcv-00N5w00000Ri7BU-0EM5w000006vKaj.jpg" style={{width: 436, height: 337}}/></Frame>

#### Get Instant Trial

Instead of "Request Demo" or "Get Trial," paid apps now offer an option to "Get Instant Trial."

#### Listing Includes

Appstore listings now have a **Listing Includes** section that displays the assets included in the listing and how many of each. In this example, the listing includes one card, one DataSet, and one Enterprise App.

<Frame><img alt="listing includes.jpg" src="/images/kb/ka0Vq0000000Rcv-00N5w00000Ri7BU-0EM5w000006vGBj.jpg" style={{width: 261.996, height: 160.994}}/></Frame>

#### Rich Formatting and Descriptions

When submitting listings, there are more options for rich formatting; publishers can also lay out tables and align text.

Learn more in our [Appstore documentation](https://domo-support.domo.com/s/topic/0TO5w000000ZampGAC/appstore).

## New DomoStats Reports

<Frame><img alt="Screenshot 2023-07-18 at 5.09.58 PM.png" src="/images/kb/ka0Vq0000000Rcv-00N5w00000Ri7BU-0EM5w000006vKuy.jpg" style={{width: 148, height: 148}}/></Frame>

The following new reports are available. Users with an Admin default security role can access the DomoStats connectors and connect to the DomoStats apps.

### Beast Modes

The Beast Modes DomoStats report provides information about all Beast Mode calculations in a Domo instance. This allows admins to see how Beast Mode calculations are being used and to which assets they are saved.

### User Landing Resource

The User Landing Resource DomoStats report retrieves user data from Domo and allows admins to view the landing page/dashboard or landing card for all Domo users in their instance.

### View Inputs

The View Inputs DomoStats report provides information regarding every pre-existing DataSet View in a Domo instance. This allows admins to better understand and manage how their DataSet View Inputs are being used.

## Flex Map v2

Flex Map v2 builds on v1 by providing more options to customize your maps; removing most of the required configuration DataSets; improving performance for large DataSets; and adding styling options like new pin colors, new base layers, and a more extensive library of shapes. Flex Map v2 also supports Domo Filters and allows you to undo and redo changes as you build your map.

If you already have Flex Map v1, you have access to Flex Map v2, but you need to download it from the [Appstore](https://www.domo.com/appstore/).

To learn more, see the [Flex Map v2 Implementation Guide](/s/article/000005073) and [Flex Map v2 User Guide](/s/article/000005048).

## Jupyter Workspaces File Sharing

File Sharing allows Jupyter Workspaces users to create, edit, and delete files shared across Domo Jupyter Workspaces. Instead of copying and pasting script resources and data models between projects, these can be uploaded to one location and accessed from multiple workspaces.

The Create Filesharing Directories and Manage Filesharing Directories grants allow access to this feature.

Learn more about File Sharing in our [Jupyter Workspaces User Guide](https://domo-support.domo.com/s/article/36004740075?language=en_US#file_sharing).

## Sandbox Renaming Rules & Dependency Graph

#### Renaming Rules

Rename Rules allows you to seamlessly edit names and descriptions for Domo content upon promotion to another instance. Instead of editing details for all data manually, Rename Rules save time by removing, replacing, or adding names and descriptions for dashboards, cards, Beast Mode calculations, DataSet Views, and DataFlows.

#### Dependency Graph

The Dependency Graph inside a Sandbox Repository is a powerful tool for content exploration and visualization in Domo. It depicts Domo content in a hierarchical graph format, enabling you to explore and understand the relationship between the different elements of the repository. If you have an error while committing or promoting a repository, the graph provides additional context for errors and their relationships for more efficient resolution.

Learn more about [Domo Sandbox](/s/article/4403367344023).

## Governance Toolkit Updates

The Governance Toolkit now includes the following features.

#### Enterprise Data Copy

This tool allows you to copy data in bulk across multiple instances. With it, you can make a manual list of DataSets you want to copy into a separate DataSet, or you can use dynamic Filters to include an ever-changing number of DataSets to be copied that meet your specific criteria. Admins can use this tool to push or pull data to and from a given instance.

Learn more about [Enterprise Data Copy](/s/article/000005197).

#### Webhook Notification

The DataSet Watchdog tool now includes a Webhook Notification option. Webhook Notifications allows you to automatically send notifications about changes in your most crucial DataSets and DataFlows through your favorite messaging service.

Learn more about [DataSet Watchdog](/s/article/000005105).
