## About

InfusinonSoft is a complete sales and marketing automation software for small businesses combining CRM, email marketing, lead capture, and e-commerce. To learn more about the inContact API, see [InfusinonSoft](https://developer.infusionsoft.com/docs/xml-rpc/ "InfusionSoft").

You can connect to your InfusinonSoft account in the Data Center. This article explains the fields and menus that are specific to the InfusinonSoft Connector user interface. To add DataSets, set update schedules, and edit DataSet information, see [Adding a DataSet Using a Data Connector](https://domo-support.domo.com/s/article/360042926274?language=en_US "Adding a DataSet Using a Data Connector").

## Prerequisites

To connect to your InfusinonSoft account and create a DataSet, you need the Email and Password associated with your InfusionSoft account.

## Connecting to Your InfusinonSoft Account

This section lists the options in the **Credentials** and **Details** panes on the InfusinonSoft Connector page. The components of the other panes on this page, **Scheduling**, and **Name & Describe Your DataSet**, are universal across most connector types and are discussed in great detail in [Adding a DataSet Using a Data Connector](https://domo-support.domo.com/s/article/360042926274?language=en_US "Adding a DataSet Using a Data Connector").

## Credentials Pane

You can use the Email and Password associated with your InfusionSoft account. Once you have entered valid InfusinonSoft credentials, you can use the same account to create a new InfusinonSoft DataSet. You can manage connector accounts in the **Accounts** tab in the Data Center. For more information about this tab, see [Managing User Accounts for Connectors](https://domo-support.domo.com/s/article/360042926054?language=en_US "Managing User Accounts for Connectors").
