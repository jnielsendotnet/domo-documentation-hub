### Intro

You can add Smart Text to a card title or description while working in Analyzer or from a dashboard. Smart Text references certain data elements and updates in place whenever the underlying data changes. You can use more than one piece of Smart Text at a time, if needed.

Anyone with permission to edit a card or dashboard can add/modify associated Smart Text.

Personalized Data Permissions (PDP) do apply when adding Smart Text to a card title or description. For more information about PDP, see [Personalized Data Permissions (PDP)](/s/article/360042934614).

The Smart Text options are described below:

* **Date range —**
  Shows dates in text. Ex: this quarter, graphed by week
* **Filters —**
  Displays the filters applied. Ex: order status.
* **Variables —**
  When a variable is present, the selected value of the variable is listed.
* **Segments used —**
  Names the visible segments.
* **DataSet info —**
  Displays information about the DataSet such as the date last updated or owner.

  <Note>
    **Note:**
    You can also add Smart Text to Notebook Cards. See
    [Adding a Notebook Card](/s/article/360043430233)
    to learn more.

  </Note>

### Configure Smart Text

1. In the modal, select
   **+ Smart Text** for either the **Card title** or **Description** fields to open the Smart Text menu.
     
   <Frame><img alt="Screenshot 2024-02-07 at 3.18.29 PM.png" src="/images/kb/ka0Vq0000000Qyb-00N5w00000Ri7BU-0EMVq000000K33N.jpg" style={{width: 600, height: 464}}/></Frame>
   <Note>
     **Note:**
     If the card does not have any variables, the
     **Variables**
     option is not available. Add a variable to the card to have the option for Variables Smart Text.

   </Note>

2. Select the type of Smart Text you want to add and follow the prompts to apply logic. The logic depends on the type of Smart Text. See the table below.

  Your Smart Text configuration can make your card title or description quite lengthy. We recommend being thoughtful about how you apply Smart Text for usability.

   **Apply** to keep the logic, **Cancel** to discard the logic, or **Remove** to delete that piece of Smart Text from the card title or description.

   **Save** to keep your configured Smart Text and close the modal or **Cancel** to discard your changes.
