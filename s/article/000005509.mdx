## Intro

Jamf is an Apple management device company and home to the largest community of Apple IT admins in the world. With the power of JAMF and Domo, you can view all your JAMF reports in real-time using one simple dashboard. Customized reports in your team’s hands can make them available on any device. You can get an overview of your devices by building, department, group, OS version, model, and much more. To learn more about the Jamf V2 API, visit their page ([https://developer.jamf.com/apis/classic-api/index](https://developer.jamf.com/apis/classic-api/index "https://developer.jamf.com/apis/classic-api/index") ).

You can connect to your Jamf V2 account in the Data Center. This topic discusses the fields and menus that are specific to the Jamf V2 connector user interface. To add DataSets, set update schedules, and edit DataSet information, see [Adding a DataSet Using a Data Connector](https://domo-support.domo.com/s/article/360042926274?language=en_US "Adding a DataSet Using a Data Connector").

## Prerequisites

To connect to your Jamf V2 account and create a DataSet, you must have the following:

* Account Name
* URL
* Client ID
* Client Secret

## Connecting to Your Jamf V2 Account

This section enumerates the options in the **Credentials** and **Details** panes in the Jamf V2 Connector page. The components of the other panes in this page, **Scheduling** and **Name & Describe Your DataSet**, are universal across most connector types and are discussed in great detail in [Adding a DataSet Using a Data Connector](https://domo-support.domo.com/s/article/360042926274?language=en_US "Adding a DataSet Using a Data Connector").

## Credentials Pane

This pane contains fields for entering credentials to connect to your Jamf V2 account. The following table describes what is needed for each field:

| Field         | Description                                                                                                                                                   |
| ------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Account Name  | Enter the account name associated with your Jamf account.                                                                                                     |
| URL           | Enter the URL associated with your Jamf instance. For example: [https://domo.jamfcloud.com.](https://domo.jamfcloud.com)                                      |
| Client ID     | Enter the Client Id.<br/>Log into Jamf.<br/>Click Settings > API roles and Clients > API Clients .<br/>Create a new client by selecting the existing one.     |
| Client Secret | Enter the Client Secret.<br/>Log into Jamf.<br/>Click Settings > API roles and Clients > API Clients .<br/>Create a new client by selecting the existing one. |

Once you have entered the valid Jamf V2 credentials, you can use the same account any time to create a new Jamf V2 DataSet. You can manage connector accounts in the **Accounts** tab in the Data Center. For more information about this tab, see [Managing User Accounts for Connectors](https://domo-support.domo.com/s/article/360042926054?language=en_US "Managing User Accounts for Connectors").
