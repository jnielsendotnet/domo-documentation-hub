title: "Installing Salesforce's Domo Connector Package"

The "Domo Connector Package" found in Salesforce AppExchange allows Salesforce users to import their data into Domo. Follow the instructions in this article to download and install the Connector Package from the AppExchange.

**Note:**

If the Managed Package is installed successfully, both old and new Salesforce DataSets will work without changes.

## Production Environment

**To install Domo's Salesforce Managed Package in the Production Environment:**

1. Log in to Salesforce.
2. Navigate to the Salesforce AppExchange at [https://appexchange.salesforce.com](https://appexchange.salesforce.com "https://appexchange.salesforce.com"), search for "Domo Connector Package," then click the package.
     
   <Frame><img alt="sfmp_1.png" src="/images/kb/ka05w000001235y-00N5w00000Ri7BU-0EM5w000005vNyZ.png"/></Frame>

   Alternatively, you can go directly to [https://appexchange.salesforce.com/a...A00000EcrsyUAB](https://appexchange.salesforce.com/appxListingDetail?listingId=a0N3A00000EcrsyUAB "https://appexchange.salesforce.com/appxListingDetail?listingId=a0N3A00000EcrsyUAB").
3. Click **Get It Now**.
     
 <Frame><img src="/images/kb/ka0Vq000000ACUX-00N5w00000Ri7BU-0EMVq000006KFCN.jpg" style={{width: 803, height: 345}}/></Frame>

4. Click **Install in Production.** **<InlineImage src="/images/kb/ka0Vq000000ACUX-00N5w00000Ri7BU-0EMVq000006KGoL.jpg" />**
5. Select the **I have read and agree to the terms and conditions** checkbox, and click **Confirm and Install**.
   
   <Frame><img src="/images/kb/ka0Vq000000ACUX-00N5w00000Ri7BU-0EMVq000006KGrZ.jpg" style={{width: 671, height: 439}}/></Frame>

   For more details, see Salesforce's documentation at [https://help.salesforce.com/articleV...ges.htm&type=5](https://help.salesforce.com/articleView?id=distribution_installing_packages.htm&type=5 "https://help.salesforce.com/articleView?id=distribution_installing_packages.htm&type=5").

## Sandbox Environment

**To install Domo's Salesforce Managed Package in the Sandbox Environment:**

1. Log in to Salesforce.
2. Navigate to the Salesforce AppExchange at [https://appexchange.salesforce.com](https://appexchange.salesforce.com "https://appexchange.salesforce.com"), search for "Domo Connector Package," then click the package.
   
   <Frame><img alt="sfmp_1.png" src="/images/kb/ka05w000001235y-00N5w00000Ri7BU-0EM5w000005vNyZ.png"/></Frame>

3. Click **Try It Free**.

   <Frame><img src="/images/kb/ka0Vq000000ACUX-00N5w00000Ri7BU-0EMVq000006KH65.jpg" style={{width: 797, height: 339}}/></Frame>

4. Click **Continue to Installation**.
     
   <Frame><img src="/images/kb/ka0Vq000000ACUX-00N5w00000Ri7BU-0EMVq000006KFtu.jpg" style={{width: 758, height: 302}}/></Frame>

5. Click **Log In & Install**.
     
   <Frame><img src="/images/kb/ka0Vq000000ACUX-00N5w00000Ri7BU-0EMVq000006KHIz.jpg" style={{width: 720, height: 249}}/></Frame>

For more details, see Salesforce's documentation at https://help.salesforce.com/s/articleView?language=en_US&id=distribution_installing_packages.htm&type=5.

### Verifying your Installation

Use Domo's Salesforce Managed Package Validator to verify that your Salesforce Managed Package has been installed correctly.

**To verify your installation,**

1. In the Connectors page in either the Data Center or Appstore in Domo, search for "Salesforce Managed Package" and double-click on on the icon.
     
   <Frame><img alt="sfmp_v1.png" src="/images/kb/ka05w000001235y-00N5w00000Ri7BU-0EM5w000005vNyJ.png"/></Frame>

2. Click
   **Get the Data**.
     
   <Frame><img alt="sfmp_v2.png" src="/images/kb/ka05w000001235y-00N5w00000Ri7BU-0EM5w000005vNyQ.png"/></Frame>

3. Under
   **Credentials**, select your Salesforce account name, then click **Next**.
     
   <Frame><img alt="sfmp_v3.png" src="/images/kb/ka05w000001235y-00N5w00000Ri7BU-0EM5w000005vNyT.png"/></Frame>

4. Under
   **Details**, click **Next**.
     
   <Frame><img alt="sfmp_v4.png" src="/images/kb/ka05w000001235y-00N5w00000Ri7BU-0EM5w000005vNyL.png"/></Frame>

5. In the
   **Update** menu, select **Manually**, then click **Next**.
     
   <Frame><img alt="sfmp_v5.png" src="/images/kb/ka05w000001235y-00N5w00000Ri7BU-0EM5w000005vNyN.png"/></Frame>

6. Give your DataSet a name, then click
   **Save**.
     
   <Frame><img alt="sfmp_v6.png" src="/images/kb/ka05w000001235y-00N5w00000Ri7BU-0EM5w000005vNyb.png"/></Frame>

The DataSet now runs. If everything has been set up correctly, the run will be successful, and a message will appear reading, "Validation Success: You have set up your Salesforce Managed Package correctly."

<Frame><img alt="sfmp_v7.png" src="/images/kb/ka05w000001235y-00N5w00000Ri7BU-0EM5w000005vNyM.png"/></Frame>

If everything has *not* been set up correctly, the DataSet fails, and a message appears reading, "You have not installed Salesforce's Domo Connector Package," with a help link to this article.

<Frame><img alt="sfmp_v8.png" src="/images/kb/ka05w000001235y-00N5w00000Ri7BU-0EM5w000005vNyO.png"/></Frame>
