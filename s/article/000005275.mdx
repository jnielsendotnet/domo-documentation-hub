## Intro

Zalando is an online fashion and lifestyle retailer that offers a wide range of clothing, accessories, and footwear. It provides a convenient and user-friendly platform for customers to browse and purchase fashion items from various brands. For more information, refer to the API documentation [https://developers.merchants.zalando.com/docs/api-ref-intro.html](https://developers.merchants.zalando.com/docs/api-ref-intro.html "https://developers.merchants.zalando.com/docs/api-ref-intro.html").

This topic discusses the fields and menus that are specific to the Zalando Connector user interface. General information for adding DataSets, setting update schedules, and editing DataSet information is discussed in [Adding a DataSet Using a Connector](https://domo-support.domo.com/s/article/360042926274?language=en_US "Adding a DataSet Using a Connector").

## Prerequisites

To connect to your Zalando account and create a DataSet, you must have the following:

* Account Name
* Client ID
* Client Secret

## Connecting to Your Zalando Account

Zalando uses OAuth 2.0 for authentication.

This section enumerates the options in the Credentials and Details panes in the Zalando Connector page. The components of the other panes in this page, **Scheduling** and **Name & Describe Your DataSet**, are universal across most connector types and are discussed in greater length in [Adding a DataSet Using a Connector](https://domo-support.domo.com/s/article/360042926274?language=en_US "Adding a DataSet Using a Connector").

### Credentials Pane

This pane contains fields for entering credentials to connect to your Domo developer account as well as your LYNX Fleet account. The following table describes what is needed for each field:

| Menu          | Description                                                   |
| ------------- | ------------------------------------------------------------- |
| Account Name  | Enter a custom name you want to use for this account in Domo. |
| Client ID     | Enter the client ID associated with the Zalando account.      |
| Client Secret | Enter your Client Secret associated with the Zalando account. |

Once you have entered valid Zalando credentials, you can use the same account any time you go to create a new Zalando DataSet. You can manage Connector accounts in the **Accounts** tab in the Data Center. For more information about this tab, see [Manage Connector/Adapter Accounts](https://domo-support.domo.com/s/article/360042926054?language=en_US "Manage Connector/Adapter Accounts").
