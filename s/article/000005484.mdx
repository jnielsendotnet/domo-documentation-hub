## Intro

BLS Unemployment by State Count pulls unemployment data for each state. Use this connector to gather all your needed unemployment data into Domo's data framework.

You can connect to your BLS Unemployment by State Count account in the Data Center. This topic discusses the fields and menus that are specific to the BLS Unemployment by State Count connector user interface. To add DataSets, set update schedules, and edit DataSet information, see [Adding a DataSet Using a Data Connector](/s/article/360042926274 "Adding a DataSet Using a Data Connector").

## Prerequisites

No Prerequisites are required.
