## Intro

Atom Really Simple Syndication (RSS) is a means by which publishers of content offer feeds to their consumers. Using the Domo Atom RSS Connector, you can retrieve data from any content provider using the standard Atom RSS format and integrate it into Domo.

The Atom RSS connector is a "Cloud App" connector, meaning it retrieves data stored in the cloud. In the Data Center, you can access the connector page for this and other Cloud App connectors by clicking **Cloud App** in the toolbar at the top of the window.

You create an Atom RSS DataSet in the Data Center. This topic discusses the fields and menus that are specific to the Atom RSS connector user interface. To add DataSets, set update schedules, and edit DataSet information, see [Adding a DataSet Using a Data Connector](/s/article/360042926274 "Adding a DataSet Using a Data Connector").

## Prerequisites

To pull Atom RSS feed data from a content provider, you must have the URL of the desired Atom RSS feed.

## Creating an Atom RSS DataSet

This section describes how to create an Atom RSS DataSet in the Atom RSS Connector page. The components of the other panes in this page, **Scheduling** and **Name & Describe Your DataSet**, are universal across most connector types and are discussed in great detail in [Adding a DataSet Using a Data Connector](/s/article/360042926274 "Adding a DataSet Using a Data Connector").

### Details Pane

| Menu | Description                                                                                                                |
| ---- | -------------------------------------------------------------------------------------------------------------------------- |
| URL  | Enter the URL to the RSS or Atom feed you want to retrieve (ensure to provide with an https protocol, http is prohibited). |
