### Intro

When you [create an App Studio app](https://domo-support.domo.com/s/article/000005295?language=en_US#create_app), the first task is to choose a theme.

<Frame><img alt="Screenshot 2024-09-10 at 2.14.53 PM.png" src="/images/kb/ka0Vq0000002n4f-00N5w00000Ri7BU-0EMVq000002e3xK.jpg" style={{width: 900, height: 455}}/></Frame>
Themes determine how your app displays to users, including elements like the following:

* Fonts
* Colors
* Card and table styling
* Tabs
* Rounded corners

Themes are divided into two types: app themes and custom themes.

This article covers [choosing](#app_themes) and [editing](#edit_your_theme) themes, as well as how to copy and [reuse themes](#copy_custom_theme) between App Studio apps.

### App Themes

App Studio comes with several pre-built packages that include fonts, colors, and styles—these packages are called app themes. They display in the first tab of the themes modal and have names like Essential and Saturated Brand.

Either select a **theme tile** > **Apply** to apply the theme to your app and open the [app editor](https://domo-support.domo.com/s/article/000005295?language=en_US#edit_app) or select **Explore theme** to learn more about a certain theme.

<Frame><img alt="Screenshot 2024-09-06 at 12.56.32 PM.png" src="/images/kb/ka0Vq0000002n4f-00N5w00000Ri7BU-0EMVq000002ahlF.jpg" style={{width: 451, height: 376}}/></Frame>
When you explore a theme, you can see more samples from the package.

Select **Use This Theme** to apply it to your app and open the [app editor](https://domo-support.domo.com/s/article/000005295?language=en_US#edit_app). You can edit a theme or [switch to another theme](https://domo-support.domo.com/s/article/000005543?language=en_US#change_theme) at any time.

<Frame><img alt="use this theme.jpg" src="/images/kb/ka0Vq0000002n4f-00N5w00000Ri7BU-0EMVq000002aixR.jpg" style={{width: 901, height: 282}}/></Frame>
### Custom Themes

The other tab in the themes modal is **Custom Themes**.

###

<Frame><img alt="custom themes.jpeg" src="/images/kb/ka0Vq0000002n4f-00N5w00000Ri7BU-0EMVq000002c0Dx.jpg" style={{width: 899, height: 369}}/></Frame>
When you first use App Studio, there are no custom themes available. You can create custom themes by choosing a pre-built [app theme](#app_themes), making changes to it, and saving it as a custom theme. Your custom themes display in the **Custom Themes** tab of the themes modal for that app. You can [reuse a custom theme](#copy_custom_theme) in another app. After you create at least one custom theme, you can duplicate it, edit it, and save it as another custom theme.

# Save a Custom Theme

These steps describe how to save a theme to your **Custom Themes** after you have made changes to an app theme or another custom theme.

1. After making changes to a theme in the
   [app editor](https://domo-support.domo.com/s/article/000005295?language=en_US#edit_app), go to <InlineImage src="/images/kb/ka0Vq0000002n4f-00N5w00000Ri7BU-0EMVq000002dsfJ.jpg" /> **App theme**.
2. Under **Theme**, select **Save as custom**.
     
   **Note:** This option doesn't display until you make changes to the theme.
     
   <Frame><img alt="save as customa.jpg" src="/images/kb/ka0Vq0000002n4f-00N5w00000Ri7BU-0EMVq000002bsOV.jpg" style={{width: 300, height: 333}}/></Frame>
3. Name your new Custom Theme and save. The max character length is 30.

   <Frame><img alt="Screenshot 2023-10-23 at 12.08.32 PM.png" src="/images/kb/ka0Vq0000002n4f-00N5w00000Ri7BU-0EM5w000006wCpi.jpg" style={{width: 325, height: 221.989}}/></Frame>
Your new theme now displays in the **Custom Themes** tab of the themes modal.

### Copy a Custom Theme

Unless an app is [locked](https://domo-support.domo.com/s/article/000005295?language=en_US#app_configuration), app owners and editors can choose to reuse a [custom theme](#custom_themes) created in another App Studio app to preserve your branding. They can also download a custom theme as a JSON file for future use. If an app is locked, only owners can copy and reuse themes.

To copy or reuse a custom theme, you must complete two processes: downloading and uploading. These processes are described below, depending on whether you download from the [Apps Home](#download_custom_theme_apps_home) or [inside the app](#download_custom_theme_inside_app) and whether you upload to a [new app](#upload_custom_theme_json_file) or to an [existing app](#import_from_existing_app).

<Note>
  **Notes:**

  * The JSON theme file only includes information
  about the theme styles, colors, and app branding. It does not include any other app or data-related information.
  * When you import a theme from another app, it imports and applies the theme currently live in the other
  app. If you want to use a previous theme stored in that app's custom theme library, go to the app and [download the JSON theme file](#download_custom_theme_inside_app). Then you can upload it to any app.

</Note>

<Note>
  **Note:** Theme background images are *not* copied over and must be applied individually if required.
</Note>

# Download a Custom Theme From the Apps Home

1. Navigate to the Apps Home and find the app with the custom theme you want to download.
2. Hover over the app and select **More** (vertical three-dot menu) > **Download** **theme file**.
     
   This downloads a JSON file of the custom theme being used in the app with all the current customizations. If you want to use an older theme with different customizations, see [downloading a custom theme from inside an app](#download_custom_theme_inside_app).

# Upload a Custom Theme to a New App — JSON File

To reuse a custom theme in a different app, you must first create the new app so that it displays in the Apps Home.

1. Navigate to the Apps Home and find the app from which you want to reuse the custom theme.
2. Hover over the app and select **More** (vertical three-dot menu) > **Edit**.
     
   The app editor opens.
     
   <Frame><img alt="edit app theme.jpg" src="/images/kb/ka0Vq0000002n4f-00N5w00000Ri7BU-0EMVq000002c3uz.jpg" style={{width: 900, height: 399}}/></Frame>
3. In the left navigation under
   <InlineImage src="/images/kb/ka0Vq0000002n4f-00N5w00000Ri7BU-0EMVq000002dsfJ.jpg" /> **App theme**, hover over the preview thumbnail and select **Change Theme** to open the themes modal.
     
   <Frame><img alt="change theme upload.jpg" src="/images/kb/ka0Vq0000002n4f-00N5w00000Ri7BU-0EMVq000002c4EL.jpg" style={{width: 300, height: 259}}/></Frame>
4. Go to the **Custom Themes** tab of the modal and select **Upload Custom Theme**.
     
   <Frame><img alt="upload custom theme.jpg" src="/images/kb/ka0Vq0000002n4f-00N5w00000Ri7BU-0EMVq000002c05v.jpg" style={{width: 900, height: 477}}/></Frame>
5. Choose your downloaded JSON file. After your theme uploads successfully, it displays in the **Custom Themes** tab of the modal.
     
   <Warning>
     **Important:** The theme file must be less than 150 KB. If the file is too large, you'll receive an error.

   </Warning>

6. Select the theme thumbnail and then **Apply** to bring the uploaded theme into your app.
7. Select **Save** in the app editor to ensure your theme changes are complete.
