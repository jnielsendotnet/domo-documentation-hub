# Intro

The packed bubble chart allows you to visualize and compare data by relative size. This chart combines the functions of word clouds and bubble charts without the need for x and y axes.

# Powering a Packed Bubble Chart

A packed bubble chart requires the following:

* **Names —** A name for the value that displays in each bubble.
* **Values —** The values that correspond to the names inside each bubble.
* **Series —** How the names are grouped.

In Analyzer, you can choose the columns containing the data for your packed bubble chart. For more information about choosing data columns, see our article about [Applying DataSet Columns to Your Chart](https://domo-support.domo.com/s/article/360043428713?language=en_US).
