### October 2024 Updates

# AI Chat

Domo. AI Chat is now available to everyone!

Ask a question about your data and AI Chat answers, providing a detailed breakdown of how it reached the response. You'll also get visualizations, data context, and prompts for additional questions. Go beyond natural language queries and let AI Chat help you accomplish tasks like setting an alert, calling an AI model, and more.

Learn more about [AI Chat](/s/article/000005539).

# AI Readiness

The new AI Readiness feature for DataSets helps prepare your data to realize the full potential of Domo. AI. Get an excellent readiness score by using the data dictionary and adding context to columns for maximum AI impact.

Learn more about AI Readiness.

# App Studio

#### Clear Filters

The **Clear Filters** action is now available in App Studio. This action allows you to clear all page filters with a single click, eliminating the need to go to the control panel.

<Frame><img src="/images/kb/ka0Vq000000GovJ-00N5w00000Ri7BU-0EMVq0000037xXt.jpg" style={{width: 419, height: 408}}/></Frame>
#### Card-on-Card Pop-ups

When viewing a page in an App Studio app, you can now open a second card via a pop-up on top of an existing card. Card-on-card pop-ups allow you to gather secondary information quickly with a single click.

<Frame><img src="/images/kb/ka0Vq000000GovJ-00N5w00000Ri7BU-0EMVq0000037xw5.jpg" style={{width: 408, height: 375}}/></Frame>
#### Exports in App Studio Embed

Card exports are available with App Studio Embed. This feature allows you to export your cards in Embed as a CSV, PDF, PowerPoint, or Excel sheet.

<Frame><img src="/images/kb/ka0Vq000000GovJ-00N5w00000Ri7BU-0EMVq0000037wDe.jpg" style={{width: 895, height: 482}}/></Frame>
Learn more about [App Studio.](https://domo-support.domo.com/s/article/000005295?language=en_US)

# Bulk Filter Paste

A global filter update allows you to paste multiple comma-separated values directly into the filter search bar in Analyzer and card Details and when using dashboards and Quick Filters.

# Variables Support in PowerPoint Add-in

The Domo add-in for Microsoft PowerPoint now supports Variables at the document level. Variable controls can be selected and saved in your filters panel. Any updates made to the Variable controls apply to all of your cards using that variable at the document level.

Learn more about [Variables support for PowerPoint](https://domo-support.domo.com/s/article/000005143?language=en_US#enable_variable_display_powerpoint) and the [MS Office add-ins.](/s/article/000005143)

# Workflows Updates

#### Bulk Cancel

The new Bulk Cancel feature in Workflows allows you to select multiple in-progress or failed executions and submit a single cancellation request. Save time by submitting bulk cancellation requests instead of canceling executions one by one.

Learn about [canceling workflows in bulk](https://domo-support.domo.com/s/article/000005179?language=en_US#cancel_workflows_in_bulk).

<Frame><img alt="BulkCancel1 (1).png" src="/images/kb/ka0Vq000000GovJ-00N5w00000Ri7BU-0EMVq000002TJqv.jpg" style={{width: 900, height: 516}}/></Frame>
#### DataSet Query Task

The DataSet Query Task shape has been added to Workflows. This shape allows you to implement logic that queries and retrieves data from a Domo DataSet in your workflows.

The DataSet Query Task includes v ariable injections into queries, preview options, and auto-generated outputs.

<Frame><img alt="DataSetQueryTask1.png" src="/images/kb/ka0Vq000000GovJ-00N5w00000Ri7BU-0EMVq000002LimT.jpg" style={{width: 897, height: 515}}/></Frame>
Learn how to [add a DataSet Query Task](https://domo-support.domo.com/s/article/000005331?language=en_US#add_query_table) to your workflow.

#### Queues Task Center Homepage

The Queues home page has been rebranded to Task Center (consistent with the mobile app) and redesigned with a more sleek and modern interface for accessing queues and tasks.

With the release, Task Center also receives the following updates:

* **Improved Usability —** Enhanced layout, styling, and usability.

* **Task Counts —** More visibility into volumes in queues.

* **Accessibility —** Queues are searchable and centrally positioned on the page.

* **Task List** **—** Rearranged column headers display task rows more logically.

<Frame><img alt="TaskCenterHomepageUI (1).png" src="/images/kb/ka0Vq000000GovJ-00N5w00000Ri7BU-0EMVq000002TDAQ.jpg" style={{width: 898, height: 516}}/></Frame>
Learn more about [Task Center](/s/article/000005172).

# Magic ETL

#### Increased Canvas Size

The Magic ETL canvas size has been increased to give you more space to build DataFlows. The canvas is no longer limited to a set number of tiles, alleviating the pain of having to scroll back and forth on your screen. The new [Mini map](#mini_map) helps you visualize and manage your larger canvas.

<Frame><img alt="Screenshot 2024-08-22 at 10.41.25 AM.png" src="/images/kb/ka0Vq000000GovJ-00N5w00000Ri7BU-0EMVq000002LZ6T.jpg" style={{width: 896, height: 458}}/></Frame>
#### Update to the "Remove Duplicates" Tile

The Remove Duplicates tile now includes "Add All Columns" and "Remove All Columns" actions so you can easily add and remove columns in bulk.

<Frame><img alt="RemoveDuplicatesTileCrop (1).png" src="/images/kb/ka0Vq000000GovJ-00N5w00000Ri7BU-0EMVq000002LMZ4.jpg" style={{width: 572, height: 478}}/></Frame>
#### Upsert

Upsert in Magic ETL allows you to update rows in your output DataSet without reprocessing the full DataSet based on a unique row-level key. Upsert replaces the need to create recursive DataFlows and makes it easier to recover if any issues occur in the pipeline.

<Frame><img alt="Upsert 2.png" src="/images/kb/ka0Vq000000GovJ-00N5w00000Ri7BU-0EMVq000002LVZ8.jpg" style={{width: 579, height: 418}}/></Frame>
Learn more about [upsert](https://domo-support.domo.com/s/article/000005541?language=en_US#upsert) and [partitioning](https://domo-support.domo.com/s/article/000005541?language=en_US#partition) in Magic ETL.

# Workbench Enterprise 2.0

Workbench Enterprise has been enhanced with the following actions for admins:

* Import Workbench jobs to
  [Workbench Enterprise.](https://domo-support.domo.com/s/article/000005303?language=en_US)

* Upload
  [data partitions.](https://domo-support.domo.com/s/article/360062446514?language=en_US)

* Remotely execute Workbench jobs.

* Use File Watch. This action looks for the file to update and runs the job automatically.

* Use upserts as a top-level upload type.

Learn more about [implementing Workbench Enterprise.](https://domo-support.domo.com/s/article/000005303?language=en_US)

# Workflow Action Menu

The new action menu replaces the existing notation pallet, simplifying the process of selecting actions for specific steps in your workflow.

The following changes are included with the new action menu:

* Workflow actions are displayed in logical categories: Flow Controls, DataSet Query Tasks, User Tasks, and Code Functions.

* Each category has sub-menus where you can drill down to a specific action.

* New search capabilities based on matching character sets make it easy to find an action.

* The action menu can be dismissed and recalled anytime during the editing process.
