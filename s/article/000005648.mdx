### Intro

You can create different navigation configurations for your Domo mobile apps, allowing app users to more quickly access the content most relevant to them. This is done through creating and customizing mobile app configurations. The Mobile Configuration feature is available for all organizations. If your organization doesn't use Mobile Configuration, individuals can still [personalize their individual app navigation](#personalize_your_navigation).

Need support? Check out our [FAQ](#faq) or contact your Domo account team.

### Required Grants

There are two available grants for mobile configuration:

* **Edit Mobile App Configurations —** This allows users to create, edit, and delete mobile app configurations to which they have access. **Note:** This grant does not allow the holder to grant [permissions](#manage_configuration_permissions) in a configuration. This grant is for non-admin users who need to make changes to mobile app configurations.
* **(Admin-only) Manage All Mobile App Configurations —** This allows admin-level users to create, edit, and delete all mobile app configurations.

### Access Mobile App Configuration Settings

Access the mobile app configuration settings from Admin Settings.

In Admin Settings under **Content**, select **Mobile**.

In the **App registry** tab, you can view the mobile apps in your instance.
  
**Note:** Most instance registries only contain the Domo and Domo Task Center apps (if you're using Domo Task Center).

# Manage App Properties

There are four possible properties to manage (Bottom Tab Bar, Navigation top drawer section, Only show company icon, and Allow personalized navigation bar?). Selecting **Edit** for any of the properties opens that property in the **Properties** tab.

The interactive preview allows you to test your property configuration, including available languages and dark mode.

Select the **All properties** breadcrumb at the top of the pane to return to the properties list.

<Frame><img alt="select all properties.jpg" src="/images/kb/ka0Vq0000004fHV-00N5w00000Ri7BU-0EMVq000003mJ1p.jpg" style={{width: 900, height: 711}}/></Frame>
#### Edit the Bottom Tab Bar (Navigation Toolbar)

The Bottom Tab Bar (Navigation Toolbar) includes four icons and a **More** option.

Selecting **Edit** for this property displays the list of current items in the Bottom Tab Bar. When you select an item, the **Properties** tab allows you to add information like an ID, title, description, and icon for the selected item.
  
You can remove or reorder items in the list.

To add an item, select **Add** and choose Domo Content (cards, App Studio apps, or dashboards) or Domo Services like Approvals and Search.

Select **Save** after making your changes.

<Frame><img alt="bottom tab bar crop.jpg" src="/images/kb/ka0Vq0000004fHV-00N5w00000Ri7BU-0EMVq000003mJGL.jpg" style={{width: 601, height: 451}}/></Frame>
#### Edit the Navigation Top Drawer

The Navigation Top Drawer includes a list of as many items as you would like.

Selecting **Edit** for this property displays the list of current items in the Navigation top drawer. When you select an item, the **Properties** tab allows you to add information like an ID, title, description, and icon for the selected item.
  
You can remove or reorder items in the list.

To add an item, select **Add** and choose Domo Content (cards, App Studio apps, or dashboards) or Domo Services like Approvals and Search.

Select **Save** after making your changes.

<Frame><img alt="Screenshot 2024-11-12 at 8.45.37 PM.png" src="/images/kb/ka0Vq0000004fHV-00N5w00000Ri7BU-0EMVq000003mJJZ.jpg" style={{width: 600, height: 477}}/></Frame>
#### Only Show Company Icon

Selecting **Edit** for this property displays a switch in the **Properties** tab. Enabling the switch removes the Domo icon from the navigation top drawer and preserves your branding, as displayed in these images.

#### Allow Personalized Navigation Bar?

<Warning>
  **Important:** This property is enabled by default.
</Warning>

Selecting **Edit** for this property displays a switch in the **Properties** tab. Enabling the switch allows end users to [personalize their Bottom Tab Bar/navigation bar](#personalize_your_navigation).

Select **Save** after making your changes.

### Manage Configuration Permissions

The **Permissions** pane allows you to assign individuals and groups to a particular configuration.

1. Access the
   [mobile app **Configurations** pane](#access_mobile_app_configuration).
2. Select a configuration to view its configuration settings.
3. Go to the **Permissions** pane.
4. Select **Add** to open the permissions modal.
     
   <Frame><img alt="Screenshot 2024-11-12 at 4.45.15 PM.png" src="/images/kb/ka0Vq0000004fHV-00N5w00000Ri7BU-0EMVq000003m6z4.jpg" style={{width: 800, height: 522}}/></Frame>
5. Find a person or group and select them to add them to the permissions list and close the modal.
     
   <Frame><img alt="Screenshot 2024-11-12 at 4.47.51 PM.png" src="/images/kb/ka0Vq0000004fHV-00N5w00000Ri7BU-0EMVq000003mAsz.jpg" style={{width: 550, height: 475}}/></Frame>
6. Back in the **Permissions** pane, choose whether the person or group can View or Modify the configuration.
   <Note>
     **Note:** When a person or group has been given permission to a configuration, that is the configuration they will see when using the Domo mobile app. They do not need to sign out and log back in to the app to view the new configuration.

   </Note>

<Frame><img alt="Screenshot 2024-11-12 at 4.52.08 PM.png" src="/images/kb/ka0Vq0000004fHV-00N5w00000Ri7BU-0EMVq000003mAzR.jpg" style={{width: 698, height: 273}}/></Frame>
7. Select **Save** to keep your changes.

You can delete or reorder the people/groups in the permissions list. Save your changes before navigating away from the **Permissions** pane.

2. Select **More** (three horizontal lines icon).

### Personalize Your Navigation

Whether or not your organization uses mobile configuration, all users can personalize the navigation in their own mobile app. Follow the steps below.

1. In the Domo mobile app, select **More** (three horizontal lines icon).
     
   <Frame><img alt="more mobile app.jpg" src="/images/kb/ka0Vq0000004fHV-00N5w00000Ri7BU-0EMVq000003lt5l.jpg" style={{width: 300, height: 599}}/></Frame>
2. Select **Personalize nav bar**.
     
   <Frame><img alt="personalize nav bar.jpg" src="/images/kb/ka0Vq0000004fHV-00N5w00000Ri7BU-0EMVq000003lvva.jpg" style={{width: 300, height: 681}}/></Frame>
3. From the personalization screen, you can select **More** (three horizontal dots icon) for a current navigation item to edit its name and icon or remove it from the navigation.
     
   <Frame><img alt="edit name and icon.jpg" src="/images/kb/ka0Vq0000004fHV-00N5w00000Ri7BU-0EMVq000003lxni.jpg" style={{width: 300, height: 322}}/></Frame>
4. Use the handles (three horizontal lines icons) to drag and reorder items in the navigation.
     
   <Frame><img alt="drag items.jpg" src="/images/kb/ka0Vq0000004fHV-00N5w00000Ri7BU-0EMVq000003lyQP.jpg" style={{width: 299, height: 229}}/></Frame>
5. Select **Add** (plus icon) to add items to the navigation using search or your saved favorites.
     
   <Frame><img alt="add item.jpg" src="/images/kb/ka0Vq0000004fHV-00N5w00000Ri7BU-0EMVq000003lydJ.jpg" style={{width: 300, height: 225}}/></Frame>
6. (Optional) Select **Close** (x icon) or **Reset to default** to discard your changes.
     
   <Frame><img alt="close or reset.jpg" src="/images/kb/ka0Vq0000004fHV-00N5w00000Ri7BU-0EMVq000003lwek.jpg" style={{width: 300, height: 599}}/></Frame>
7. When finished, select **Save** (checkmark icon) to apply your changes to your personal app.
     
   <Frame><img alt="save personal nav.jpg" src="/images/kb/ka0Vq0000004fHV-00N5w00000Ri7BU-0EMVq000003ly3q.jpg" style={{width: 299, height: 229}}/></Frame>
### FAQ

<AccordionGroup>
<Accordion title="Can I create a custom theme for my mobile configuration?">


This is not currently supported. To create a branded Domo mobile experience, contact your Domo account team or [Domo Support](https://domo-support.domo.com/s/) <InlineImage src="/images/kb/ka0Vq00000001FR-00N5w00000Ri7BU-0EMVq0000000XDh.jpg" />.

<Accordion title="Are there any navigation elements, such as Home, that I can't remove from my configuration?">


No, all navigation elements are personalizable, including the Home screen.


</Accordion>
<Accordion title="What do I do if I'm unable to view a configuration I've been added to?">


Close and restart the Domo mobile app to view the new configuration.


</Accordion>
<Accordion title="Can the default Domo app be edited or deleted?">


No, you cannot edit or delete the default Domo app configuration.

</Accordion>
