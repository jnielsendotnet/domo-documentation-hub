## Intro

Wireless Glue is part of the SAP IoT Startup Accelerator program, which focuses on building an ecosystem of IoT partners for SAP, that complements SAP's strategy for Digital Supply Chain, Manufacturing, and Assets. Use Domo's Wireless Glue IoT Connector to receive information about the IoT data that matters to you, combine it with other datasets for more insights, and get alerts when critical data changes.

You can connect to your Wireless Glue IoT account in the Data Center. This topic discusses the fields and menus that are specific to the Wireless Glue IoT connector user interface. To add DataSets, set update schedules, and edit DataSet information, see [Adding a DataSet Using a Connector](https://domo-support.domo.com/s/article/360042926274?language=en_US "Adding a DataSet Using a Connector").

## Prerequisites

To connect to your Wireless Glue IoT account and create a DataSet, you need the following:

* Your Wireless Glue IoT Password
* Your Wireless Glue IoT Username

## Connecting to Your Wireless Glue IoT Account

This section enumerates the options in the **Credentials** and **Details** panes on the Wireless Glue IoT Connector page. The components of the other panes on this page, **Scheduling**, and **Name & Describe Your DataSet**, are universal across most connector types and are discussed in great detail in [Adding a DataSet Using a Connector](https://domo-support.domo.com/s/article/360042926274?language=en_US "Adding a DataSet Using a Connector").

## Credentials Pane

This pane contains fields for entering credentials to connect to your Wireless Glue IoT account. The following table describes what is needed for each field:

| Field    | Description                            |
| -------- | -------------------------------------- |
| Password | Enter your Wireless Glue IoT password. |
| Username | Enter your Wireless Glue IoT username. |

Once you have entered valid Wireless Glue IoT credentials, you can use the same account at any time to create a new Wireless Glue IoT DataSet. You can manage connector accounts in the **Accounts** tab in the Data Center. For more information about this tab, see [Manage Connector Accounts](https://domo-support.domo.com/s/article/360042926054?language=en_US "Manage Connector Accounts").
