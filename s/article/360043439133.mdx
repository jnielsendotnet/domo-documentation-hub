### Intro

Domo has two types of language settings: user language settings and company language settings.

* The user language settings determine the interface language for your personal view of Domo.
* The company language settings determine the default interface language for everyone in the instance.

<Note>
  **For either setting, you can choose from the following languages:**

  * Dutch
  * French
  * German
  * Japanese
  * Portuguese
  * Simplified Chinese
  * Spanish

</Note>


---

### Set Your Personal Interface Language

If your primary language differs from the default interface language set by your company, Domo allows you to choose a different language for your personal interface. You can change this in one of two places: your Domo login screen or your user localization settings.

# Set Your Personal Interface Language From the Login Screen

To set the language for your personal interface from the Domo login screen, expand the language dropdown under the login panel before logging in.

<Frame><img alt="language dropdown.png" src="/images/kb/ka0Vq00000010Qz-00N5w00000Ri7BU-0EMVq000000tyYD.jpg" style={{width: 175, height: 708}}/></Frame>
Find and select the language you want to use.
  
Your personal view of Domo now displays in that language.

# Set Your Personal Interface Language in the User Localization Settings

1. In the header, select your profile picture.
     
<Frame><img src="/images/kb/ka0Vq00000010Qz-00N5w00000Ri7BU-0EMVq000000z9Xp.jpg"/></Frame>

2. Select **User Locale Settings/Settings** to open the user settings.
     
   <Frame><img alt="user settings.jpg" src="/images/kb/ka0Vq00000010Qz-00N5w00000Ri7BU-0EMVq000000z9b3.jpg" style={{width: 300, height: 115}}/></Frame>
3. Go to the **Localization** tab.
4. In the **Language** section dropdown, choose the language you want to use.
     
   <Frame><img alt="language settings.jpg" src="/images/kb/ka0Vq00000010Qz-00N5w00000Ri7BU-0EMVq000000z8n3.jpg" style={{width: 400, height: 469}}/></Frame>
5. In the confirmation dialog that displays, select **OK**.
     
   <Frame><img alt="confirm personal language.jpg" src="/images/kb/ka0Vq00000010Qz-00N5w00000Ri7BU-0EMVq000000z96P.jpg" style={{width: 375, height: 151}}/></Frame>
# Use European Number Formatting

If you want to use English with European number formatting, use the **English (Netherlands)** setting. Follow the steps below.

<Note>
  **Note:** Domo European number formatting uses a dot (.) as a thousands separator, a comma (,) as a decimal separator, and other European numbering conventions.

  <Frame><img alt="european numbers.jpg" src="/images/kb/ka0Vq00000010Qz-00N5w00000Ri7BU-0EMVq000001Jqn7.jpg" style={{width: 300, height: 395}}/></Frame>
</Note>

1. Access the
   [user localization settings](#set_personal_interface_langauge_user_settings).
2. In the **Language** section dropdown, select **English (Netherlands)**. This changes the numbers in your interface to match Dutch number formatting conventions, including using commas as decimals.
     
   <Frame><img alt="england netherlands.jpg" src="/images/kb/ka0Vq00000010Qz-00N5w00000Ri7BU-0EMVq000000tvwv.jpg" style={{width: 232, height: 308}}/></Frame>
3. In the confirmation dialog that displays, select **OK**.

# Reset Interface Language to Company Settings

You can reset your personal interface language to the company default at any time from the [user localization settings](#set_personal_interface_langauge_user_settings). Beneath the **Language** dropdown, select **Set as language**.

<Frame><img alt="set as language.jpg" src="/images/kb/ka0Vq00000010Qz-00N5w00000Ri7BU-0EMVq000000txlp.jpg" style={{width: 401, height: 126}}/></Frame>
### Set the Company Default Interface Language

Admins can change the default interface language for the entire instance of Domo web.

##### Mobile App Language

The language for the Domo mobile application on iOS and Android matches the device language—to change the language of the application you must change your device language. If you use a language that isn't currently supported for Domo mobile, the application defaults to English.

##### Required Grant

You must have an Admin default security role or a custom role with the Manage All Company Settings grant enabled to change the default interface language.

# Change the Default Web Language

<Tip>
  **Tip:** We recommend notifying users before changing the company language settings.

</Tip>

Follow these steps to change the default interface language for Domo web.

1. Navigate to the Admin Settings.
2. Under **Company** **settings**, select **Company** **overview**.
3. On the **Company overview** page under **Formatting**, choose a language from the dropdown.
     
   <Frame><img alt="change language.jpg" src="/images/kb/ka0Vq00000010Qz-00N5w00000Ri7BU-0EMVq000000qt8T.jpg" style={{width: 599, height: 576}}/></Frame>
4. Select **Save** at the top of the **Company overview** page.
     
   <Frame><img alt="save company settings.jpg" src="/images/kb/ka0Vq00000010Qz-00N5w00000Ri7BU-0EMVq000000qt3d.jpg" style={{width: 601, height: 317}}/></Frame>
