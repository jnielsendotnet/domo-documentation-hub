### Intro

Domo requires a certain version of various systems to operate. Learn about each below.

| Domo Version | Requirements                                                                                                                                                                                                                                                                                                                                                                                                 |
| ------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Web          | You must have a web browser to use Domo on a desktop or laptop computer.<br/>Domo supports the two most recent non-beta versions of the following browsers:<br/>Google Chrome<br/>Mozilla Firefox<br/>Microsoft Chromium Edge (Old Edge version 44 is not supported)<br/>Safari (Mac)<br/>* Recommended<br/>The following items must also be enabled for your browser:<br/>JavaScript<br/>Popups<br/>Cookies |
| iOS          | For security purposes, Domo supports devices running on the latest iOS (iOS 14).<br/>After a new iOS version comes out, we support the new version and the previous version for 6–12 months.                                                                                                                                                                                                                 |
| Android      | Domo currently supports all devices running the latest Android or Android 10 and later.                                                                                                                                                                                                                                                                                                                      |
