## Intro

Mailjet brings marketers and developers together on one platform to accelerate their business growth with email. To learn more about the Mailjet API, visit their page ([https://dev.mailjet.com/](https://dev.mailjet.com/ "https://dev.mailjet.com/") ).

You connect to your Mailjet account in the Data Center. This topic discusses the fields and menus that are specific to the Mailjet connector user interface. General information for adding DataSets, setting update schedules, and editing DataSet information is discussed in [Adding a DataSet Using a Data Connector](https://domo-support.domo.com/s/article/360042926274?language=en_US "Adding a DataSet Using a Data Connector").

## Prerequisites

To connect to your Mailjet account and create a DataSet, you must have the following:

* A Mailjet API key
* A Mailjet API secret

You should be able to find these credentials in your Mailjet API Key Management page.

## Connecting to Your Mailjet Account

This section enumerates the options in the **Credentials** and **Details** panes on the Mailjet Connector page. The components of the other panes on this page, **Scheduling**, and **Name & Describe Your DataSet**, are universal across most connector types and are discussed in greater length in [Adding a DataSet Using a Data Connector](https://domo-support.domo.com/s/article/360042926274?language=en_US "Adding a DataSet Using a Data Connector").
