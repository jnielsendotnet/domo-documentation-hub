# Intro

The Asset Library contains all the assets in your organization's instance, including apps. You can manage apps that you have created and apps that have been shared with you. If you have an Admin default security role, you can also manage all apps that exist in the instance.

# Access Asset Library

You can access the Asset Library from the navigation header by selecting **More** > **Asset Library**.

<Frame><img alt="asset library.jpeg" src="/images/kb/ka05w0000012BE1-00N5w00000Ri7BU-0EM5w000006w7uP.jpg" style={{width: 327.99, height: 337.997}}/></Frame>
# Landing Page

The Asset Library landing page contains a list of apps in your instance. You can customize the way you view the list as well as filter, search, and sort the list. Learn more about these options below.

You can choose one of the following view options:

* **List View**
  **—** Displays apps vertically in rows. This is similar to DataSets.
* **Tile View**
  **—** Displays apps in smaller tiles so you can see more apps on the page.
* **Original View**
  **—** Displays apps in larger tiles.

<Frame><img alt="select view.jpg" src="/images/kb/ka05w0000012BE1-00N5w00000Ri7BU-0EM5w000006w7XU.jpg" style={{width: 697.99, height: 355}}/></Frame>
The following Quick Filters allow you to choose what part of the library you want to search, filter, and sort through:

* **My Apps**
  — Contains all the apps you have created.
* **Shared With Me**
  — Contains all the apps that have been shared with you
* **All Apps**
  — Contains all the apps created in your company's instance.
    
  <Note>
    **Note:**
    You must have an Admin default security role to access the
    **All Apps**
    Filter.

  </Note>

<Frame><img alt="Screenshot 2023-10-12 at 6.09.28 PM.png" src="/images/kb/ka05w0000012BE1-00N5w00000Ri7BU-0EM5w000006w7X5.jpg" style={{width: 183.999, height: 245.994}}/></Frame>
To find a specific app, enter one or more keywords in the search bar.

<Frame><img alt="search bar.jpg" src="/images/kb/ka05w0000012BE1-00N5w00000Ri7BU-0EM5w000006w7XK.jpg" style={{width: 686.996, height: 350.994}}/></Frame>
The **Sort By** menu provides the following options for sorting the list:

* **Name**
  — Sorts apps alphabetically: from **A to Z** or **Z to A**.
* **Created Date**
  — Sorts apps by the date they were created: from **Newest to Oldest** or **Oldest to Newest**.
* **Updated Date**
  — Sorts apps by the date they were last updated: from **Newest to Oldest** or **Oldest to Newest**.

# App Details

While the layout depends on your view of the landing page, each app in the list displays a title, a version number, the date it was last updated, the owner's avatar, the number of cards—instances of the app—and an option to create a new card.

Hover over the <InlineImage src="/images/kb/ka05w0000012BE1-00N5w00000Ri7BU-0EM5w000006w7XZ.jpg" /> **More** icon for an app to display the following options:

* **Download Assets —**
  Downloads the available files for the app.
* **Share —**
  Opens the **Share** modal, where you can choose to share the app with other users.
* **Create Card —**
  Creates a new instance of the app.

Select an app from the list/app tile to display the app Details. There are four tabs in the app Details, described below:

<Frame><img alt="Screenshot 2023-10-13 at 11.17.59 AM.png" src="/images/kb/ka05w0000012BE1-00N5w00000Ri7BU-0EM5w000006w7uT.jpg" style={{width: 907.983, height: 233.999}}/></Frame>
* The
  **Overview** tab displays the app name, design ID, version number, date last updated. This tab also allows you to create a new card (instance of the app), share the app, or download the available files for the app. (Select <InlineImage src="/images/kb/ka05w00000127Q7-00N5w00000Ri7BU-0EM5w000005vsLM.jpg" /> **App options** > **Download Assets**.)

  <Frame><img alt="overview options.jpg" src="/images/kb/ka05w0000012BE1-00N5w00000Ri7BU-0EM5w000006w7uK.jpg" style={{width: 382.997, height: 112.99}}/></Frame>
* The
  **Cards** tab displays the cards created with the app and displays the option to **Create** **Card**.
* The
  **Data** tab displays the data powering the app divided into **DataSets**, **Wired Collections**, and **Datastore Collections**. For each of these data sources, you can view the owner, a list of people with access to it, and the number of views.
    
  Navigate between the different groups of data (divided by card/app instance) by selecting <InlineImage src="/images/kb/ka05w0000012BE1-00N5w00000Ri7BU-0EM5w000006w8D6.jpg" /> <InlineImage src="/images/kb/ka05w0000012BE1-00N5w00000Ri7BU-0EM5w000006w8DV.jpg" /> **Expand/Collapse**.
* The
  **Versions** tab displays all of the versions of the app and allows you to release the version(s) to the Appstore.
