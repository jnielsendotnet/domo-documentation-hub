### Intro

Domo has many powerful tools for data analysis and visualization, like DataSet Views and Analyzer. App Studio brings more of these tools together in one place.

When you add a table to your App Studio app, you can explore your data in a spreadsheet-like interface with functionality formatting, aggregating, grouping, resizing columns, and adding a new calculation.

In App Studio View mode, you and your app users can interact with tables the way you do on dashboards. Learn how to [interact with tables](/s/article/360043429573#interact_with_table_charts).

<Warning>
  **Important:** This functionality is also available for existing tables in App Studio. Select a table element to open the configuration panel and select **Edit in Place**.

### Add a Table Element to App Studio

1. In App Studio edit mode, go to **Add actions and elements**. Under **Content**, drag a **Table** element to the canvas.
     
   <Frame><img alt="drag a table element to the canvas.jpg" src="/images/kb/ka0Vq0000008gy9-00N5w00000Ri7BU-0EMVq000003nwh3.jpg" style={{width: 898, height: 508}}/></Frame>
2. Choose the DataSet you want to explore.
     
   It displays as a Table Card on the app page.

<Frame><img alt="choose data.jpg" src="/images/kb/ka0Vq0000008gy9-00N5w00000Ri7BU-0EMVq000003w9cH.jpg" style={{width: 550, height: 431}}/></Frame>
3. Select the card to open the right configuration panel. Select **Edit in Place** to open the edit mode.
     
   <Frame><img alt="edit in place.jpg" src="/images/kb/ka0Vq0000008gy9-00N5w00000Ri7BU-0EMVq000003w9KX.jpg" style={{width: 900, height: 375}}/></Frame>
   You can now directly edit the table inside App Studio.
     
   <Frame><img alt="Screenshot 2024-11-20 at 3.45.50 PM.png" src="/images/kb/ka0Vq0000008gy9-00N5w00000Ri7BU-0EMVq000003wAA9.jpg" style={{width: 899, height: 309}}/></Frame>
### Edit a Table Element

When you choose to edit a table element in place, the table formatting bar displays at the top of the table, and the DataSet columns display in the **Data** column of the configuration panel.

<Frame><img alt="edit table.jpg" src="/images/kb/ka0Vq0000008gy9-00N5w00000Ri7BU-0EMVq000003w6L4.jpg" style={{width: 899, height: 309}}/></Frame>
#### Show/Hide Columns

The Table Card automatically shows the first columns in the DataSet. You can select **Show/Hide** for any individual column in the configuration panel.

<Frame><img alt="unhide.jpg" src="/images/kb/ka0Vq0000008gy9-00N5w00000Ri7BU-0EMVq000003wAi1.jpg" style={{width: 299, height: 471}}/></Frame>
#### Reorder Columns

Use the handles in the configuration panel to reorder columns on the Table Card.

<Frame><img alt="drag handles.jpg" src="/images/kb/ka0Vq0000008gy9-00N5w00000Ri7BU-0EMVq000003wAmr.jpg" style={{width: 300, height: 473}}/></Frame>
#### Rename Columns

Double-click on a column name in the configuration panel or in the Table Card to rename it.

#### Resize Columns

Drag the edges of the column headers to resize the columns in the Table Card. Your adjustments will be saved.

<Frame><img alt="resize columns.jpg" src="/images/kb/ka0Vq0000008gy9-00N5w00000Ri7BU-0EMVq000003wBfh.jpg" style={{width: 499, height: 250}}/></Frame>
#### Format Table

Use the formatting bar to format a column as currency or percentage, add or remove decimals, or align content inside the cells.

<Frame><img alt="format values.jpg" src="/images/kb/ka0Vq0000008gy9-00N5w00000Ri7BU-0EMVq000003wC77.jpg" style={{width: 500, height: 403}}/></Frame>
#### Add a Column

In each column, hover over the header and select **More** (three vertical dots icon) > **Add column** to add a new column to the right.

<Frame><img alt="Screenshot 2024-11-20 at 4.05.29 PM.png" src="/images/kb/ka0Vq0000008gy9-00N5w00000Ri7BU-0EMVq000003wCDZ.jpg" style={{width: 499, height: 315}}/></Frame>
You can then use the formatting bar to populate the column with the results of a formula and name the column directly on the card.

<Frame><img alt="Screenshot 2024-11-13 at 8.07.16 PM.png" src="/images/kb/ka0Vq0000008gy9-00N5w00000Ri7BU-0EMVq000003nxlB.jpg" style={{width: 549, height: 389}}/></Frame>
#### Group By

In the configuration panel, select **Group By** to group your columns and open up the **Aggregations** section of the configuration panel.

<Frame><img alt="group by.jpg" src="/images/kb/ka0Vq0000008gy9-00N5w00000Ri7BU-0EMVq000003w7aV.jpg" style={{width: 300, height: 409}}/></Frame>
For a group, select **More** (three vertical dots icon) to choose how to aggregate your data.

<Frame><img alt="more dots.jpg" src="/images/kb/ka0Vq0000008gy9-00N5w00000Ri7BU-0EMVq000003wCYX.jpg" style={{width: 300, height: 284}}/></Frame>
#### Show/Hide Total Row and Subtotal Rows

Go to the **Properties** tab of the configuration panel to choose to show/hide the total row and subtotal rows.

<Frame><img alt="Screenshot 2024-11-13 at 8.19.53 PM.png" src="/images/kb/ka0Vq0000008gy9-00N5w00000Ri7BU-0EMVq000003nxzh.jpg" style={{width: 300, height: 203}}/></Frame>
