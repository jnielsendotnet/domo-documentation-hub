## Intro

RoviTracker is an asset management software and hardware platform that collects critical metrics like fuel level, battery voltage, solar charge, engine run time, and GPS to provide insights into business operations. To learn more about the RoviTracker API, see [RoviTracker API Documentation](http://developers.rovitracker.com/ "RoviTracker API Documentation").

You can connect to your RoviTracker account in the Data Center. This topic discusses the fields and menus specific to the RoviTracker connector user interface. To add DataSets, set update schedules, and edit DataSet information, see [Adding a DataSet Using a Connector](https://domo-support.domo.com/s/article/360042926274?language=en_US "Adding a DataSet Using a Connector").

## Prerequisites

To connect to your RoviTracker account and create a DataSet, you need your RoviTracker account's Client ID and Client Secret.

## Registration Process

To Integrate with the RovioTracker API, you need to register your application and configure using an OAuth2 authorization process.

The registration process only happens once at the beginning:

1. Register your application with RoviTracker by providing a company name and logo.
2. You will receive a Client ID and Client Secret.

Note: To receive your Client ID and Client Secret, connect with Domo's support team.

## Connecting to Your RoviTracker Account

Authorization is done using the OAuth2 standard.

This section enumerates the options in the **Credentials** and **Details** panes on the RoviTracker Connector page. The components of the other panes on this page, **Scheduling**, and **Name & Describe Your DataSet**, are universal across most connector types and are discussed in great detail in [Adding a DataSet Using a Connector](https://domo-support.domo.com/s/article/360042926274?language=en_US "Adding a DataSet Using a Connector").

## Credentials Pane

This pane contains fields for entering credentials to connect to your RoviTracker account. The following table describes what is needed for each field:

| Field         | Description                                |
| ------------- | ------------------------------------------ |
| Client ID     | Your Rovi Tracker account's Client Id.     |
| Client Secret | Your Rovi Tracker account's Client Secret. |

Once you have entered valid RoviTracker credentials, you can use the same account at any time to create a new RoviTracker DataSet. You can manage connector accounts in the **Accounts** tab in the Data Center. For more information about this tab, see [Manage Connector Accounts](https://domo-support.domo.com/s/article/360042926054?language=en_US "Manage Connector Accounts").
