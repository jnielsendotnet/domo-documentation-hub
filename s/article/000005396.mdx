### Intro

The Task Center mobile app allows you to access Task Center from your mobile device, view your queues, and complete tasks.
Learn more about [Task Center](/s/article/000005172).

Task Center is available in the Apple App Store (Domo's Task Center) and Google Play Store (Domo Task Center).

### Getting Started

From the Task Center home screen, you can do the following:

* Select the **Your Tasks** tile to view a filtered list of all tasks assigned to you in all queues.
* Select the **Open Tasks** tile to view a filtered list of all open tasks you can complete.
* Select the **Completed Today** tile to view a list of tasks you completed today.
* Select and view each queue that you have access to.

### Task List

Selecting the **Your Tasks** or **Open Tasks** tile or any of your queues displays a list of tasks for completion.

Select a task to complete it.

<Frame><img alt="task list.jpg" src="/images/kb/ka0Vq0000002jKP-00N5w00000Ri7BU-0EMVq000000MvcL.jpg" style={{width: 300, height: 648}}/></Frame>
In a task, required fields are marked with a red asterisk (\* ).

<Frame><img alt="required field.jpg" src="/images/kb/ka0Vq0000002jKP-00N5w00000Ri7BU-0EMVq000000MvkP.jpg" style={{width: 300, height: 648}}/></Frame>
When you are ready to complete the task, select **Save and Submit**.

After submission, a **Task successfully completed** banner displays at the bottom of the screen.
