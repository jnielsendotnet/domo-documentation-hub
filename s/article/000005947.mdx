## Intro

**Box** is a cloud-based content management and file-sharing service for organizations. Use the Box connector to import files from a Box account into Domo.

You can retrieve the following file types:

- CSV
- JSON
- XLS
- XLSX

For more information about the Box API, see the **[Box Developer documentation](https://developer.box.com/)**.

You can connect to your Box FileWatch account in the Data Center. This topic discusses the fields and menus that are specific to the Box FileWatch connector user interface. To add DataSets, set update schedules, and edit DataSet information, see [Adding a DataSet Using a Connector](/s/article/360042926274).

## Prerequisites

To connect to your Box account and create a DataSet, you must have the email address and password associated with your Box account.

## Connecting to Your Box FileWatch Account

This section enumerates the options in the **Credentials** and **Details** panes on the Box FileWatch Connector page. The components of the other panes on this page, **Scheduling**, and **Name & Describe Your DataSet**, are universal across most connector types and are discussed in great detail in [Adding a DataSet Using a Connector](/s/article/360042926274).

## Credentials Pane

The Domo Box FileWatch connector uses OAuth to connect, so you don't have to enter credentials within Domo. Click **Connect** (or select **Add Account** if you have existing Box accounts in Domo) to open the Box OAuth screen, where you will be prompted to enter the email address and password associated with your Box account.

Once you have entered valid Box credentials, you can use the same account any time to create a new Box FileWatch DataSet. You can manage connector accounts in the **Accounts**tab in the Data Center. For more information about this tab, see [Managing User Accounts for Connectors](/s/article/360042926054).

Once you have entered valid Box FileWatch credentials, you can use the same account at any time to create a new Box FileWatch DataSet. You can manage connector accounts in the **Accounts** tab in the Data Center. For more information about this tab, see [Manage Connector Accounts](/s/article/360042926054).

## Details Pane

| Menu | Description |
| --- | --- |
| File Name | Enter all or part of the name of the Box file that you want to retrieve. |
| Match Type | Select whether the file to retrieve starts with or contains the name you entered in the **File Name** field. |
| File Type | Select the file type of the data file to retrieve. |
| Would you like to choose file from Folder ? | Select **Yes** to retrieve a file from a specific folder. If you select **No**, the connector retrieves all files that contain the name you entered in **File Name**. |
| Folder Name or ID | Enter a complete folder name or a Box folder ID to limit the search to a specific folder. This search is not case-sensitive and ignores subfolders. If you leave this field blank, the search includes all files in all folders that you can access. |
| Sheet Name | This field appears when you select the file type as **xls** or **xlsx.**  Enter the name of the sheet to retrieve from the spreadsheet. Ensure that the sheet name doesn't contain leading or trailing spaces. |
