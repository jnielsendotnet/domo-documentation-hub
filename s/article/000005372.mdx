### Intro

The Workflow Start Form Brick allows you to quickly start a workflow from a Domo Brick by gathering and passing customizable input values.

Learn more in the [Domo Bricks | Overview](/s/article/4423762260375).

---

### Access the Workflow Start Form Brick

1. In the navigation header, select **Appstore** to open the Domo Appstore.
2. Use the search bar to locate the Workflow Start Form Brick and open its Details page.
3. Select **Get**.
     
   <Frame><img alt="workflow start form brick.jpg" src="/images/kb/ka0Vq0000000KOX-00N5w00000Ri7BU-0EMVq000000Drkr.jpg" style={{width: 750, height: 460}}/></Frame>

4. Save the brick to an existing dashboard or create a new one and select **Save**.
     
   <Frame><img alt="Screenshot 2024-01-18 at 6.31.02 PM.png" src="/images/kb/ka0Vq0000000KOX-00N5w00000Ri7BU-0EMVq000000Drph.jpg" style={{width: 501, height: 373}}/></Frame>

5. Go to the dashboard to begin developing your brick. Learn how to
   [get started with Domo Bricks](https://developer.domo.com/portal/ec60c4980e1b5-getting-started-using-domo-bricks) <img src="/images/kb/ka0Vq00000001FR-00N5w00000Ri7BU-0EMVq0000000XDh.jpg" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/> in our developer portal.

### Use the Workflow Start Form Brick

<Warning>
  **Important:** Before you can successfully use the brick, you must have an existing workflow.
</Warning>

#### Input Controls

There are five available sample input controls in the Workflow Start Form Brick:

* Text
* Number
* Dropdown selector
* Single-select radio button
* Checkbox

Each of these can be configured and values passed through to the configured workflow.

<Frame><img alt="Screenshot 2024-01-18 at 6.44.57 PM.png" src="/images/kb/ka0Vq0000000KOX-00N5w00000Ri7BU-0EMVq000000Drxl.jpg" style={{width: 851, height: 291}}/></Frame>

#### Configuration

To configure your brick, hover over the card to display the <img src="/images/kb/ka05w00000127Q7-00N5w00000Ri7BU-0EM5w000005vsLM.jpg" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/> **Card options**.

Select **Card options** > **Edit Card** to display the brick settings. Here you can view the code editor for JavaScript, HTML, and CSS.

<Frame><img alt="Screenshot 2024-01-18 at 6.55.11 PM.png" src="/images/kb/ka0Vq0000000KOX-00N5w00000Ri7BU-0EMVq000000Ds0z.jpg" style={{width: 900, height: 423}}/></Frame>

Use the **Select Workflow** option to search for the workflow you want to connect to your brick.

<Frame><img alt="select workflow.jpg" src="/images/kb/ka0Vq0000000KOX-00N5w00000Ri7BU-0EMVq000000Ds2b.jpg" style={{width: 900, height: 423}}/></Frame>

Locate your workflow and select **Choose Workflow**.

<Frame><img alt="choose workflow.jpg" src="/images/kb/ka0Vq0000000KOX-00N5w00000Ri7BU-0EMVq000000Ds4D.jpg" style={{width: 550, height: 587}}/></Frame>

#### Edit Brick

In the brick editor, follow the instructions in the **JavaScript**, **HTML**, and **CSS** tabs to configure the parameters of your workflow.

<Frame><img alt="Screenshot 2024-01-18 at 7.07.30 PM.png" src="/images/kb/ka0Vq0000000KOX-00N5w00000Ri7BU-0EMVq000000Ds7R.jpg" style={{width: 801, height: 384}}/></Frame>

Select <img src="/images/kb/ka0Vq0000000KOX-00N5w00000Ri7BU-0EMVq000000Ds93.jpg" style={{width: 20, height: 20, display: 'inline', verticalAlign: 'start', margin: '0'}}/> **Run** to run the code in the app preview before saving and closing.

<Frame><img alt="run code.jpg" src="/images/kb/ka0Vq0000000KOX-00N5w00000Ri7BU-0EMVq000000DsCH.jpg" style={{width: 801, height: 384}}/></Frame>

If the code runs correctly, select **Save & Finish**.

<Frame><img alt="save and finish.jpg" src="/images/kb/ka0Vq0000000KOX-00N5w00000Ri7BU-0EMVq000000DsIj.jpg" style={{width: 802, height: 433}}/></Frame>
