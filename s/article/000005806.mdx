## Intro

Shopify is an e-commerce and point-of-sale software that helps merchants create online stores, market their products, and optimize the customer experience. The Domo's Shopify Multi Account connector integrates Shopify data, including data on customers, orders, marketing efforts, pages, and more.  
  
Using Domo and Shopify together enables users to create personalized dashboards with product, sales, inventory, and even customer data all in one place. Learn how to optimize your marketing and sales strategies for better results. Filter your view with easy drag-and-drop actions. With Domo and Shopify, you can easily create and share ecommerce reports across your entire organization in real time, on any device. To learn more about the Shopify Multi Accounts API, go to [https://docs.Shopify Multi Accounts.com/api](https://docs.shopify.com/api).

This topic discusses the fields and menus that are specific to the Shopify Multi Accounts connector user interface. For general information about adding DataSets, setting update schedules, and editing DataSet information, see [Adding a DataSet Using a Connector](/s/article/360042926274).

## Multi-Account Connector Vs Single Account Connector

Every Multi-Account connector has a corresponding single-account connector. For example, the Shopify Multi Accounts connector has a corresponding Shopify single Account connector.

You might prefer to use the Shopify Multi Accounts if the following are true:

1. You have multiple Shopify Accounts in Domo, OR you have a single Shopify Account in Domo and plan to add more Accounts later, AND
2. You would like to import the data from one or more of these Accounts into a single dataset.

You might prefer to use the single Account version of this connector if you have one or more Accounts in Domo, and you would like to keep the data from each Account in a separate DataSet.

### Credentials Pane

To connect to your Shopify Multi Accounts connector and create a DataSet, you need to select at least one Shopify Account in Domo from the drop-down list.

For more information on selecting and powering the accounts, see [Powering Up A Multi-Account Connector](/s/article/000005825) (this article explains the process of powering up multiple Connector Accounts and successfully creating a dataset).

To add a new Shopify Account, refer to the [Prerequisite](/s/article/360042927994)section of the Shopify single account connector.

After you have entered valid Shopify Multi Accounts credentials, you can use the same account in Domo any time to create a Shopify Multi Accounts DataSet. You can manage connector accounts in the **Accounts** tab in the Data Center. For more information about this tab, see [Manage Connector/Adapter Accounts](/s/article/360042926054).

### Details Pane

| **Menu** | **Description** |
| --- | --- |
| Report | Select the report which you want to run. The Execute GraphQL Query report is selected by default. |
| GraphQL Query | Enter the GraphQL query which you want to execute. For more information on Shopify GraphQL queries, refer to the Shopify API documentation. |

## DataSet Update History

Multi-Account Connectors display the history of running the Connector for each Account on the ‘History’ tab in the DataSet details. For more information on the update history for a Multi-Accounts Connector, and re-running a Multi-Accounts Connector dataset, see [https://domo-support.domo.com/s/article/000005825?language=en\_US](/s/article/000005825).
