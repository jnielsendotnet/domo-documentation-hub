---
title: "Edit a DataFusion"
---

import {InlineImage} from "/snippets/InlineImage.mdx";

You can edit a DataFusion in the Data Center. You can only edit a DataFusion if you are the owner OR have an Admin default security role OR a custom role with the Manage DataSet grant enabled. Learn more about [Managing Custom Roles](/s/article/360043438973 "Default Security Role Reference").

Follow these steps to edit a DataFusion:

1. From inside the Data Center, select
   <InlineImage src="/images/kb/ka05w00000123vE-00N5w00000Ri7BU-0EM5w000005vPVh.png" /> **DataSets** in the left navigation.
2. Find the DataFusion you want to edit.
3. Select the DataFusion to open the Details view and select
   <InlineImage src="/images/kb/ka0Vq00000013Ld-00N5w00000Ri7BU-0EMVq000001Q1MU.jpg" /> **down arrow** > **Edit DataFusion**.
     
   <Frame><img alt="edit datafusion.jpg" src="/images/kb/ka0Vq00000013Ld-00N5w00000Ri7BU-0EMVq000001PxkI.jpg" style={{width: 221, height: 125}}/></Frame>
4. Update the DataFusion.

<Tip>
  **Tip:** You can rename a DataFusion by selecting <InlineImage src="/images/kb/ka0Vq00000013Ld-00N5w00000Ri7BU-0EMVq000001Q2f7.jpg" /> **action menu** > **Edit Name & Description**. Change the title or description of the DataFusion and select **Save**.
