### Intro

This reference lists the available SQL expressions in Magic ETL. For a description and more information about a specific expression, see the Magic ETL UI.

Learn how to use these expressions in the [SQL tile](https://domo-support.domo.com/s/article/360044876874?language=en_US#sql_tile) while creating a Magic ETL DataFlow.
